　　“我漏考虑了一个人。”
<br/>　　罗昊用点射压制着压近的CiA行动队，说道：“除了我们和蛰龙接触过新型P病毒外，还有一个人也接触过。”
<br/>　　“詹姆斯！”
<br/>　　经过罗昊的提醒，周柏也是反应了过来。当时在那艘货轮上拦截下来西伦科特后，西伦科特先是被詹姆斯所杀，紧接着罗昊手中两支P病毒也是落在了西伦科特手中，如果不是蛰龙的出现，西伦科特那两支新型P病毒一定会落入美国军方手中。
<br/>　　“没错。”
<br/>　　罗昊点头，说道：“不过现在被他们识破也不算不能接受。本来我就没想着能用那瓶东西彻底忽悠住他们，如果就靠这么一瓶用葡萄汁加血浆勾兑出来的东西就能忽悠瘸他们，那他们的智商就真的太让人失望了。现在我只担心一个问题。”
<br/>　　“什么问题？”周柏问道。
<br/>　　“算了，暂时不说了。”罗昊摇头道：“但愿不会发生。”
<br/>　　战术中队这边，按照罗昊的吩咐，慕容飞带着佟千雪向山林深处撤离，梅庆伟带着C组负责保护，齐凡的A组进行掩护，B组则是负责断后。
<br/>　　“节约子弹，控制手雷的使用。”
<br/>　　罗昊从战术背心的弹匣袋中拿出两个用胶布捆绑在一起的P-MAg弹匣，塞入HK416的弹匣槽中，吩咐B组的队员控制射击。
<br/>　　这次行动，战术中队队员每个人携带了0.5个基数的弹药，因为有比洛多的机动部队作为支援，所以罗昊着重考虑如何能快速准确的切入进布克尔特的老巢，把佟千雪带出来；但是因为比洛多的突然倒戈，彻底打乱了罗昊的营救计划，罗昊不得不考虑自己这方的弹药消耗，以及撤离方案。
<br/>　　目前为止，罗昊还没想到恰当的撤离方案，如果比洛多调派联合政府军进行彻底封锁的话，那么对于战术中队而言就是雪上加霜了，想要从这个国家撤离出去就变得难上加难。
<br/>　　呼呼呼呼……
<br/>　　直升机螺旋桨转动时所产生的巨大轰鸣声从头顶上传来，两架“黑鹰”直升机出现在山林上空，两架直升机各抛下两根索降绳，搭乘在直升机山上的机动部队队员抓着索降绳滑降下来。
<br/>　　罗昊看着悬停在空中的那两架“黑鹰”直升机，瞳孔锁紧，他最担心也是最不愿意看到的情况还是发生了，比洛多彻底站到了他们的对立面，他派出了他手下精锐的机动部队来帮助美国人得到布克尔特手中的那支P病毒！
<br/>　　在几个小时前，他们跟这些滑降进山林的机动部队队员还是同一阵线的战友，可是在转眼之间，他们就成了敌人！
<br/>　　罗昊苦笑，他不会去责怪比洛多，大家都是为了各自的目的和利益才达成的合作，这样的联合是最牢固也是最脆弱的。当双方的利益保持一致时，这样的联合自然是牢不可破；可是一旦当利益发生分歧时，这种联合就变得跟一张厕纸一样，很轻易就会被撕碎，破裂。
<br/>　　罗昊从迷彩裤腿袋中拿出那部从布克尔特身上找到的手机，交给齐凡，说道：“小凡，我觉得这部手机中应该藏着布克尔特的秘密，很有可能从这部手机中能找到那支P病毒的线索，不然没办法解释为什么布克尔特会随身带着他。”
<br/>　　齐凡没有去接罗昊递过来的手机，对罗昊问道：“那你呢？”
<br/>　　“按照原计划，我会带B组的人拖住他们！给你们争取撤退的时间！”罗昊回答道。
<br/>　　“不行！”齐凡直接拒绝道：“要走一起走，要留一起留！”
<br/>　　“你不是个孩子了，不要这么幼稚！”
<br/>　　罗昊把手机塞到齐凡手中，说道：“你们留下也改变不了什么，只能减少我们的战斗力！拿着手机走，找到布克尔特藏起来的P病毒，我们还有机会！而且就算我们被困，我们暂时还是安全的，因为他们要从我这里得到关于P病毒的线索，所以他们只会对我用刑，但是不会杀了我！但是这个手机落到了比洛多手中，并且从中找到了有关于那支P病毒的线索，那么我们就失去了价值，比洛多只要随便给我们扣上一个罪名就能杀了我们，我们现在是黑身份人员，是得不到任何承认的！”
<br/>　　齐凡握了握手机，最后还是把手机放进了自己口袋中，带着A组的人转身离开！
<br/>　　“小凡！”
<br/>　　罗昊叫住了转身离开的齐凡，说道：“如果，我说如果我回不去了，跟小熙说声对不起。”
<br/>　　“不会有这个如果！”齐凡抬手擦掉眼角溢出的眼泪，说道：“你是‘龙刺’，你不会死，我也不允许你死！”说完，带着A组迅速撤入山林深处，很快就不见了踪影。
<br/>　　尽管罗昊跟其他B组队员拼尽全力的阻拦，可是人数和火力上的劣势，并没有出现什么逆转的机会。兰博，只存在于电影中！
<br/>　　数颗催泪瓦斯弹的攻击下，B组队员很快就陷入了包围中，加上空中“黑鹰”直升机上加特林机枪的虎视眈眈，B组队员彻底失去了反抗的机会。
<br/>　　砰！
<br/>　　Royce冲上前去，对着罗昊的腹部就是一拳，把罗昊打得弓起了身体，接着举起挂在M4A1战术步枪，用枪托对着罗昊砸了下去。
<br/>　　咚！
<br/>　　罗昊的额头被砸开一道口子，眼前看到的人和物也是变得模糊起来，额头上留下的血水遮挡在眼前，把视野中的东西全都染了色。
<br/>　　Royce只是对着罗昊一顿拳打脚踢，但是造成全都是非致命伤，正如罗昊所预料的那样，在没有得到P病毒线索之前，他们的生命安全是可以得到保障的。
<br/>　　得到五角大楼那边的指示后，Royce让人卸掉了B组队员的武器，并且把他们全都捆绑起来，押上降落下来的直升机。
<br/>　　远处山林中，藏在暗处的齐凡看着被打倒在地，被抓上直升机的B组队员，紧握着拳头，心中暗下决心，如果罗昊他们出事，他一定会去找比洛多报仇，就算是丢掉军籍，脱掉军装也在所不惜！
<br/>　　他会让比洛多知道，什么叫做不择手段的复仇！
