　　第751章
<br/>　　从李行哉口中，很难听出什么家国天下之类的话。他举着酒盏，如一个浪荡公子，哈喇子恨不得流出来。
<br/>　　“你可是不知道，当初本王在长安城可是一等一的人物，青楼酒馆，以能请我吃顿饭为荣。多少头牌想见我一面而不能，嗳，当初还真是年轻呐。”
<br/>　　程大雷不是很愿意搭理他，没好气道：“不如你现在下去，看她会不会与你斟杯酒？”
<br/>　　“嗳，现在不成喽，一代新人换旧人，如今的长安，谁还认得我六王子。”
<br/>　　听李行哉语气，颇为感慨唏嘘，像在怀念当初一去不回的好时光。
<br/>　　程大雷不是很认真的同他有一句没一句的闲聊，目不转睛的看着楼下，注意力在那花魁玉雀身上。
<br/>　　她体态娇小，占一个媚字，此刻立在银盘上，银盘动，美人舞，金乐齐鸣。掌心托着一盏酒，身摇体动，时快时缓，不知何时将酒盏搁在眉心，仰头托着，身体旋转。
<br/>　　观舞者颇多，都在担心她将酒洒在脸上，闹出个笑话来。一舞终了，杯中酒一滴不落，她双手托着，鬓边微微冒汗。
<br/>　　掌声雷动，有人忍不住欢呼，今日一舞，不知多少人倾心。
<br/>　　就连程大雷也倒抽一口凉气，感慨道：“动人如此，不愧长安第一花魁。”
<br/>　　李行哉抿着酒杯，淡淡笑着：“天下美人无双，谁能成欢，且要看男儿本事。今日百里胜将这美人祭出，不知打得什么主意。”
<br/>　　楼下的百里胜果然开口，他轻轻拍掌，款款笑着。
<br/>　　“这美人便如美酒，英雄当饮之，来，我敬诸位一杯。”
<br/>　　诸人都被玉雀一舞勾魂，不用百里胜招呼，便端起酒杯一饮而尽。有人的目光仍旧停留在玉雀身上，不掩饰贪婪。
<br/>　　在长安城他们是臣，但在各自封地都是一片天。杀人都是看心情，更何况是玩几个女人。
<br/>　　好险是没有把在家里那一套带到长安城，但性子里的霸道也不是说改就能改的。
<br/>　　百里胜哈哈笑着，对众人的目光视若无睹。他开口道：“诸位都是英雄，偏生玉雀姑娘是个爱英雄的人。今日恰逢盛会，有酒有月有水有美人，诸位谁愿下场试一试功夫，我就做主，将玉雀姑娘送给他。”
<br/>　　程大雷已经不太听得清楚百里胜口中话，他的注意力在玉雀身上。更确切来说，是在玉雀身边一老妪身上。
<br/>　　客人很多，有人喝酒吃肉，偶尔逗弄身边斟酒的歌姬，谈笑声响成一片。免不得有许多酒水汤汁落在地上，那老妪攥着抹布擦拭甲板。
<br/>　　大概是扰了一个公子哥的兴致，他一脚踹在老妪腰身，大骂道：“什么人，也有资格在这里么？”
<br/>　　急忙有几个仆人过来，将那老妪拖走。这件事发生在角落里，几乎无人发现。大家依旧在谈笑，百里胜侃侃而谈，说些风花雪月的事情，谁若能拔得头筹，今日便能占了玉雀的身子。
<br/>　　枕上联欢对玉，好生受用。
<br/>　　大家都色眯眯的笑了起来，有诸多人已经跃跃欲试，看能否摘了这朵鲜花。
<br/>　　几乎他的声音刚刚落地，就见一道人影从上楼跃了下来。大家同时抬头看，只见明月白袍翩翩落地。
<br/>　　“啧，还说自己不关心，竟然第一个出头，我还真是为我妹子担忧喔。”李行哉扒着窗户往下看，没办法，他没有程大雷的轻身功夫，只能一瘸一拐的，同徐神机还有刘发财一起往楼下走。
<br/>　　程大雷双脚落地，目光冷如刀，冷冷转了一圈，周围的热闹压下去许多。
<br/>　　又有几人不知道程大雷。
<br/>　　最后，程大雷的目光落在那老妪身上，目光是难以掩饰的哀婉和冷酷。
<br/>　　“经年不见，何至如此。”
<br/>　　程大雷已经认出了这老妪，不是旁个，正是程大雷在长安城鬼混时，受他庇护的满庭芳花魁一枝桃。
<br/>　　当初也是满城缠头，大小公子争着见一面而不得，但几年不见，她却沦落到这个地步。年纪其实并不大，但身材佝偻，看着像老了十几岁。
<br/>　　一枝桃大概已将程大雷认出，麻木的眼神渐渐有了光彩，但慢慢将头挪到一边，并没有说什么。
<br/>　　百里胜可不知发生了什么，哈哈笑了一声：“果然是程当家，我猜出头的也该是程当家。来来来，谁想与程当家较量，自可下场比试，输赢都是你们的造化。”
<br/>　　程大雷目光落到那公子哥身上，勾了勾手指：“来来来，你与我出来。”
<br/>　　那公子哥乃是兖州王雷天音的侄子，名唤雷荡仇，差不多是与程大雷同样年纪。
<br/>　　“我当是谁，原来是个山贼，这地方还真是鱼龙混杂，什么人都混了进来。”
<br/>　　“来。”
<br/>　　程大雷模样虽然不如何，但他一声清喝，自然威风凛凛。雷荡仇身子一跌，险些从板凳上掉下来。
<br/>　　他恼羞成怒，腾地一下站起：“真当我怕了你这山贼不成。”
<br/>　　雷荡仇也并非泛泛之辈，自幼跟着高人学艺，又兼神力惊人，不如此，也不会有跋扈性格。
<br/>　　他晃了晃膀子，虎扑向程大雷撞来。
<br/>　　程大雷踹出去一脚，雷荡仇反应不慢，也同时撞出去一脚。两人踹在一起，雷荡仇力气果然不小。但程大雷紧跟着又踹出去一脚，耳边只听砰砰声响，二人招式都是一样简单粗暴。
<br/>　　忽然听得咔嚓一声，雷荡仇倒在地上，双手抱着小腿，表情狰狞，冷汗直流。
<br/>　　程大雷一脚踹断了他的小腿。
<br/>　　紧跟着，又是一脚踹在他腰眼上。本来，雷荡仇还能哼哼几声，现在是痛得哼都哼不出来。
<br/>　　程大雷拍了拍手掌，不屑道：“什么货色，也配到这艘船上吃饭。”
<br/>　　他抬起头，发现许多人都看着自己。程大雷神情不变，淡淡道：“我便是程大雷，又想与我过手的，自可出来，我看今日有几人敢自称英雄。”
<br/>　　雷荡仇还有两个伴当，是他的贴身护卫，此刻同时撞了出来。一个虎拳，一个鹤形，封住了程大雷的步伐。
<br/>　　程大雷甩开胳膊，拳头已经攥紧。
<br/>(本章完)
