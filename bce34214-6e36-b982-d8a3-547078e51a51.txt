
三井美奈合起笔记本，看着眼前的三井家族队伍，心中忽然洋溢出了马到成功这个词，差不多有将近十年左右的时间了，他们家族都在为这批消失了的宝藏而努力，如今经过多方的查探，他们似乎逐渐离这批宝藏更加近了一些，因为这个笔记本的后半段被人给撕掉了，他们大概只知道宝藏是在喜马拉雅山脉内，可究竟处于什么位置就不清楚了，这片山脉太广泛了，并且有很多地方都是人迹难以达到的，但是在上个月，三井家族在喜马拉雅山脉的登山队发现了疑似藏匿宝藏的地方，就在山脉往珠穆朗玛峰延伸之处。<br/>　　<br/>　　三井美奈刚要起身，三井良河忽然快步走了过来，手中拿着导航仪，说道：“有三个目标正在朝我们这边接近着，速度不算很快，大概十五分钟左右，如果对方不改变方向的话，就会和我们遭遇了”<br/>　　<br/>　　三井美奈顿时诧异的问道：“是人还是动物？”<br/>　　<br/>　　“暂时不太清楚，估计是人的可能性很大，如果是动物的话，在山地上速度要比这个快上很多”<br/>　　<br/>　　三井良河抬起手，在自己的脖子上抹了一下，说道：“要杀掉么？”<br/>　　<br/>　　三井美奈略一沉思，说道：“有没有可能也是登山的人？”<br/>　　<br/>　　“美奈小姐，这里虽然人迹罕至，但喜马拉雅山地区有很多探险和登山的人，不排除这个可能”<br/>　　<br/>　　说话的人叫小林昙一，是个知名的探险家，几年前被三井家族雇佣来进入到了喜马拉雅山。<br/>　　<br/>　　“先等等，看下再说，收起武器不要让人发现了，这个时候清理掉人不太明智，这里是山地地形，如果枪响了的话能传出去几十公里的距离，被边防发现那就麻烦了，再一个就算人死了，尸体也很难处理干净……”三井美奈稍微犹豫了片刻，就做出了不能下手的决定，因为她现在需要的是能够顺利抵达藏宝所在地，可要是中途出现什么变故导致计划没办法进行下去，那可就亏大了，真要是被中方发现他们这只队伍的话，你根本就没办法解释，官方可能仔细一查，就探明出他们的目的了。<br/>　　<br/>　　在小心谨慎这点上，三井美奈做的一直都是如履薄冰的，他们这票队伍进入藏区都是分批进入的，她和三井良河还有三井小叶是从东jing飞来的，至于其他的人也至少是从三四个地方汇集到藏区的，并且有的是以探险，登山，还有的则是商务洽谈的名义，就是不想太引人瞩目了，让官方发现什么端倪。<br/>　　<br/>　　这帮人刚把武器收起来放进登山包中，然后装作登山队伍临时休息的状态，没过多久，就看见远方的山坡上出现了三道人影。<br/>　　<br/>　　王惊蛰，孔良和陈三岁是一路跟着獒王的痕迹过来的，三人离着挺远也看见三井美奈的队伍，初时他们还以为是碰到了探险队，孔良说喜马拉雅山脉附近，每年都有几十上百只这种队伍，碰上了也没什么稀奇的。<br/>　　<br/>　　两方人的相遇，其实特别平淡，因为双方都努力的把自己装扮成了探险登山的旅行者，谁都没有露出任何的敌意。<br/>　　<br/>　　“hello……”三井美奈先是用英语打了个招呼，然后询问道：“嗨，你们是从哪里过来的？”<br/>　　<br/>　　“这小娘们长得挺带劲啊”王惊蛰低声跟陈三岁嘀咕了一句，然后笑呵呵的朝着她摇了下头示意自己听不懂英语。<br/>　　<br/>　　随即，三井美奈反应很快，转而就用一口字正腔圆的普通话又问了一遍，王惊蛰这才点头说道：“我们之前是从拉sa过来的，但是那边不允许上珠峰，我们就转而打算去其他雪山攀登，比如喜马拉雅山脉附近的，卡拉帕塔峰和努布策山……”<br/>　　<br/>　　王惊蛰说的这套词都是刚刚孔良告诉他的，在藏区他就跟个老向导一样。<br/>　　<br/>　　接着，三井美奈身后的小林昙一就上前说道，他们这群人是来自于uiaa，也就是世界登山联合会，本身隶属于蔚蓝探险家组织。<br/>　　<br/>　　这番话说的没有一点水分，不光小林昙是国际登山联合会成员，他们这个队伍里的人，至少有一半都是来自于那里的，这都是早些年就做好的部署，为的就是来应付寻找宝藏时万一要是被官方发现后的说辞。<br/>　　<br/>　　双方聊的很熟络，还有种惺惺相惜的味道，就跟真正的探险者相遇了一样，说着一些客套话，而三井美奈操的一口纯正的普通话，似乎也在悄然间拉近了彼此不少的距离。<br/>　　<br/>　　聊了一会后，王惊蛰和三井美奈就询问了下彼此前进的方向，然后他们很惊奇的发现，对方的路线居然跟自己都差不多，当下就决定暂时结伴而行，于是三井家族的人走在前面，王惊蛰他们三人落在了队伍的最后方。<br/>　　<br/>　　陈三岁的脸“唰”的一下就沉了，小声跟孔良说道：“孔哥，从现在开始打起十二分的精神，要是有点什么变故的话，你先保护好自己”<br/>　　<br/>　　孔良顿时诧异的问道：“怎么了，他们有危险？不是登山队么……”<br/>　　<br/>　　王惊蛰阴着脸说道：“什么探险队还要干杀人越货的事啊？这帮人里至少有六七个手上都有人命，其中走在中间，围着那女人旁边的两个，杀过的人估计一双手都数不过来了”<br/>　　<br/>　　其实，在和三井美奈这伙人相遇的时候，第一时间王惊蛰和陈三岁就同时发现了他们中有些人的血腥味太重，杀气太浓了，这都是曾经杀过人的征兆，他们瞬间就觉得这帮人可不是什么普通的登山探险队了。<br/>　　<br/>　　于此同时，小林昙一就皱眉跟三井美奈说道：“美奈小姐，他们的方向跟我们暂时一致，有没有可能也是为了……”<br/>　　<br/>　　三井美奈顿时摇头说道：“不太可能，如果他们也是为了那座消失的宝藏而去，那就不只是三个人了，而且你没看到他们的补给和装备么，完全不像是深入腹地的感觉，可能只是单纯的探险吧……”<br/>　　<br/>　　“那要是他一直跟着我们，直到我们找到了那座宝藏呢？”<br/>　　<br/>　　三井美奈寻思了下，皱眉说道：“如果中途分开就算了，但如果一直跟随，那就找个地方，灭口了吧！”
