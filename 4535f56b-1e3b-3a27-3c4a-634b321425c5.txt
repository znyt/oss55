　　此时，慕清霄来到石屋中的最后一个书架前。
<br/>　　书架上摆放的不是竹简，却是最普通的卷轴，这种卷轴往往都是被用来记载一些历史事件。
<br/>　　拿起其中一册卷轴，果然，脑海中并未在响起系统的提示声。
<br/>　　慕清霄有些好奇的将卷轴打开，阅览起其中的信息。
<br/>　　蜀山仙剑志，相当于蜀山派自撰的史书，以干支纪年记载了蜀山仙剑派历史上各个年份的重大事件，也算是历史悠久了。
<br/>　　对于蜀山的往事，慕清霄并不感兴趣，将卷轴放回，拿起第二卷查看起来，降妖谱……
<br/>　　当慕清霄拿起第三册卷轴打开一看，神色一怔，星眸紧紧盯着上面《天书手卷》四个字体。
<br/>　　四个字体不禁让他想到了，青云志位面的五卷天书。
<br/>　　蜀山的天书手卷是祖师爷太清真人所留，记载有遭遇修行化境后进行恢复的方法。
<br/>　　难道两者间有什么联系不成？
<br/>　　稍微留意了下，将天书手卷记载脑海中，第四卷只是一部记录各类草药的册子，第五卷上记载的是江湖各地兵器之书。
<br/>　　“伏羲剑、女娲血玉、神农鼎……神农鼎！”
<br/>　　拿着手中的第六册卷轴，慕清霄瞳孔一缩。
<br/>　　其上，记载的赫然是三种传说中的神器，三皇神器，最重要的是有记载说这三种神器乃是蜀山的镇派之宝。
<br/>　　卷轴上的信息瞬间让慕清霄产生一丝兴趣，难道传说中的三神器真的存在？
<br/>　　慢慢的摊开卷轴，一柄宽刃剑就出现在了眼中，传说伏羲剑是三皇之一伏羲所留，持此可动天地，撼六界。
<br/>　　记载终归只是记载，微微摇头，慕清霄继续看下去，接下来描述的果然就是女娲血玉。
<br/>　　血玉是女娲所留，持此可驱动五灵之力，据传女娲造人时，是以吸收天地之灵气，以血注入泥土中而得，据册子上的记载，女娲血玉似乎就是一块红色的宝石。
<br/>　　继续往下翻，当反倒最后一页的时候，慕清霄瞳孔猛的一缩，映入眼帘的是一座青色丹鼎。
<br/>　　丹鼎上神纹交错，蟠龙刻凤，神农鼎！
<br/>　　神农鼎古称造世鼎，上古时代神农氏为苍生遍尝百草，也为后世奠定医学基础。
<br/>　　神农昔日炼制百药之古鼎，正因积聚千年来无数灵药之气，据说能够炼出天界诸神亦无法轻得之旷世神药。
<br/>　　望着册子图片上的神农鼎，天璇宫中摆放的丹鼎在慕清霄脑海中一闪即逝，两座丹鼎是何其相似，难道……
<br/>　　压下杂乱的情绪，慕清霄深吸了口气，将卷轴放回原位。
<br/>　　若是卷轴中记载的不错，那么天璇宫中的丹鼎可能真是传说中的神农鼎也说不定。
<br/>　　视线落在书架的最后一册卷轴上，慕清霄轻轻拿起，却敏锐的感觉到，此卷轴与其他卷轴略有不同。
<br/>　　当慕清霄打开卷轴的一瞬间，视线一黑，画面豁然转变，凝了凝神，此时他整个人似乎处于一片虚构的空间中。
<br/>　　旋即，一行金色的字体凭空而现：六界书！
<br/>　　待金色字体消散，慕清霄视线环顾四周，在视线的尽头有着无数的星体，熟悉的画面，似乎就是宇宙。
<br/>　　在宇宙的一角，一颗星球衍化而出，最初星球中没有任何生灵，随后衍化出山川河流，然后是狂风骤雨，雷鸣闪电……
<br/>　　数十万年后，名为神州大地的陆地上，凭空出现了三位伟大的神祇，分别为就是三皇：伏羲、神农、女娲。
<br/>　　伏羲耗费自身灵力在神界种下神树，以神树果实为实体创造出神族。
<br/>　　神农以草木为本，灌注自身气力，创造出心智未开的兽族。
<br/>　　女娲以土、水混合，附以自身血液和灵力，依凭自己的模样创造人族。
<br/>　　不知何时，三皇离去，分别留下一件至宝，分别灌输自身灵气于三种容器内，正是伏羲剑、神农鼎、女娲玉，合称‘三神器’。
<br/>　　世间随着时间的推移渐渐衍化，各族鼎盛到神魔大战开启，各族衰落，划分六界……
<br/>　　画面骤然崩碎，慕清霄脸色惨白无比，视线变得模糊，捂着眉心，脸上浮现一抹痛苦之色。
<br/>　　不多时，视线又回到石室中，望着手中的卷轴，声音虚弱道：“六界书，谁能构造如此恐怖的精神空间，难道是三皇不成？”
<br/>　　查看六界书中的内容，好似自己亲身经历过此世界的诞生到繁荣在到衰落，直到划分六界才结束。
<br/>　　此时，慕清霄觉得身心疲惫，怔怔的站在原地，不断整理着脑海中的庞大信息。
<br/>　　来到真武殿，对他来说收获确实出乎意料，也因为六界书的缘故，令他的思维一时间有些混乱了起来。
<br/>　　片刻后将卷轴放回，慕清霄拖着疲惫的身躯，转身离开真武殿。
<br/>　　……
<br/>　　……
<br/>　　回到天璇宫，慕清霄此刻也没有了去药园的心思，整个人都虚弱无比，现在只想好好的休息一下。
<br/>　　抛弃各种复杂的信息，慕清霄即刻来到天璇宫后的凉亭中，盘坐下靠在玉柱上，微微闭上双眼，旋即沉沉的睡去。
<br/>　　仙鹤振翅落下，杨柳随风飘荡……
<br/>　　只是，在他沉睡的一瞬间，体内浩瀚磅礴的阴阳灵气便翻滚起来，周围的灵气也渐渐汇聚，如同潮汐灌入他的体内。
<br/>　　随之，他的气息开始攀升，与此同时，异象悄然无息的降临。
<br/>　　天枢宫中的清微眼中山谷一抹异色，从蒲团上长生而起，走出天枢宫抬头望了一眼天空，转眼就消失不见。
<br/>　　转眼间，清微就出现在天璇宫的大门口，旋即四道身影紧随而来。
<br/>　　和阳长老感受着灵气的流动，惊骇失色。
<br/>　　“师兄，这小子突破的迹象可真是有些恐怖啊，若非蜀山有仙品聚灵阵法，山脉上的灵气非得被他全吸走不可。”
<br/>　　“他只是大乘期修士，但现在吸纳的灵气却是渡劫期修士也无法比拟的，当真是千年一遇的妖孽之才啊。”
<br/>　　站在一旁，清微没有说话，但眼中却闪烁着异色。
<br/>　　他当然知道慕清霄是天才，否则也不会在如此年纪就有如此成就，但仅仅突破小境界，此异象未免也太骇人了点。
<br/>　　……
<br/>　　……
