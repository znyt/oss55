        “漫雪，你到底是怎么了？”诺筱颖顿时一头雾水。
<br/>
<br/>        苏漫雪突然冷哼道：“没怎么，总而言之，你以后不要再跟我打电话。更加不要提起你认识我！”
<br/>
<br/>        “你是不是出什么事了？你跟我说说，我想办法帮你！”诺筱颖担忧地问。
<br/>
<br/>        苏漫雪顿时不耐烦起来：“我一切安好，你别想歪了！另外，我不会再回出租屋了，也不会再去那破公司上班。就这样，别再给我打电话了，烦，挂了！”
<br/>
<br/>        “嘟——”
<br/>
<br/>        被苏漫雪莫名其妙地说了一通后，诺筱颖这才恍然大悟地回想起今天下班之前所发的那些短信内容的事情。
<br/>
<br/>        诺筱颖懊恼地拍了拍自己的额头，她真的是工作忙过头了，不记事就算了，居然还拿自己的热脸去贴她苏漫雪的冷屁股。
<br/>
<br/>        诺筱颖苦笑，这世界上还有比她更加倒霉的人吗。
<br/>
<br/>        一夜之间，被人掠走了清白，还被自己最好的朋友说了绝交。
<br/>
<br/>        望着狭小破旧的出租屋，诺筱颖抽了抽酸涩的鼻头，没有谁会陪谁一辈子，不是吗？
<br/>
<br/>        另一边，苏漫雪到了傅家的依山别苑，看到别苑大厅的墙壁上挂着的那些金灿灿的功绩勋章，顿时能猜到傅大少爷的另一个隐蔽的身份是什么了。
<br/>
<br/>        “大少爷是军人？”苏漫雪不禁忧心忡忡地看着一旁的刘管家问道。
<br/>
<br/>        刘管家微笑着点点头又摇了摇头，他觉得能给大少爷当管家很光荣。
<br/>
<br/>        但苏漫雪看着刘管家，却愁眉苦脸起来。
<br/>
<br/>        如果她将要嫁的人，是一名军人，那岂不是相当于她将要“守活寡”？
<br/>
<br/>        军人常年不是在部队，就是在外跟那些歹徒交战。
<br/>
<br/>        命都不是自己的，再有钱顶个屁用？
<br/>
<br/>        早知如此，她就不冒充诺筱颖了！
<br/>
<br/>        苏漫雪有点后悔，眸光黯然失色。
<br/>
<br/>        刘管家一眼便看穿了苏漫雪的心思，又补充道：“大少奶奶放心，大少爷今年会退役，转战商界，接手老太爷的傅氏集团！”
<br/>
<br/>        “真的？”苏漫雪顿时眼前一亮。
<br/>
<br/>        刘管家干笑着微微点了点头，心底却在纳闷，大少爷怎么就看上了这么一个如此“现实”的女人？！
<br/>
<br/>        “大少奶奶，您是真心想要跟着我家大少爷的吗？倘若现在您后悔，还来得及。”刘管家微笑着提醒。
<br/>
<br/>        苏漫雪刻不容缓，斩钉截铁地回答：“我这辈子，跟定他了！”
<br/>
<br/>        “好！既然大少奶奶已经下定了决心，要嫁给我家大少爷。我家大少爷还说了，大少奶奶有任何要求尽管提，我都会尽量为您办妥。”
<br/>
<br/>        苏漫雪听着刘管家那一口一个大少奶奶，差点的笑出声。
<br/>
<br/>        这不正是她梦寐以求的豪门生活吗，只是这一切……原本是属于诺筱颖的……
<br/>
<br/>        想到这里，苏漫雪的眸光沉了下来。
<br/>
<br/>        晚饭过后，诺筱颖收拾了一下厨房，然后提起包包出了门。
<br/>
<br/>        白天没空去逛超市，家里的冰箱空了，诺筱颖打算去附近的水果超市里买一些新鲜的水果来填填冰箱。
<br/>
<br/>        如今，苏漫雪不在身边，以后的每一天，就她一个人进进出出这间出租屋了。
<br/>
<br/>        诺筱颖心里虽然感觉有点儿孤寂，但人生就是这样啊，没有不散的演戏，她一个人，也可以把生活过得有滋有味。
<br/>
<br/>        水果超市里，诺筱颖挑了火龙果、香蕉和苹果，提着袋子去前台打秤买单。
<br/>
<br/>        就在她掏出钱包，准备付款的时候，一只修长的大手，早已拿着一张百元大钞递给了收银员。
<br/>
<br/>        “找您五十，请收好，欢迎下次光临。”女收银员笑盈盈地说，将找好的余钱，递给了这个付了款买了单的人。
<br/>
<br/>        诺筱颖怔愣地顺着这只手，抬眸看了过去。
<br/>
<br/>        “美女，你男朋友好帅啊！”女收银员面带微笑地看着诺筱颖，又看了看诺筱颖身旁的男人，惊艳地夸赞。
<br/>
<br/>        这个男人，长得确实很帅！
<br/>
<br/>        一张犹如宫廷画师勾勒出来的俊脸，棱角分明，五官精致到无可挑剔。
<br/>
<br/>        男人穿着军装、军靴，宽肩窄腰，双腿修长，透着十足的禁欲感。
<br/>
<br/>        他的身高，足足高了诺筱颖一头半。
<br/>
<br/>        “你……”
<br/>
<br/>        “跟我来！”
<br/>
<br/>        不等诺筱颖把话说完，男人便将手伸了过来，这大手抓住了诺筱颖的小手，防不胜防地牵着她离开了水果超市。
<br/>
<br/>        “你是谁啊？快放开我！”
<br/>
<br/>        诺筱颖一手提着一袋水果，另一只手想挣脱掉男人的手，但却被男人拉着往前走，没法停下自己脚下的步伐。
<br/>
<br/>        一日不见，如隔三秋，甚是想念。
<br/>
<br/>        诺筱颖没法体会到男人的这种心情，她甚至有些抱怨，这男人是不是疯了？！
<br/>
<br/>        她压根就不认识他啊！
<br/>
<br/>        男人停下脚步，一个转身，诺筱颖一头撞入了他的怀中。
<br/>
<br/>        她刚一抬头，他便低头吻了下来。
<br/>
<br/>        男人霸道又强势。
<br/>
<br/>        他的心跳猛烈，呼吸急促，让诺筱颖的脑海里一片空白。
<br/>
<br/>        这熟悉的味道……
<br/>
<br/>        和昨晚的那个男人好像！
