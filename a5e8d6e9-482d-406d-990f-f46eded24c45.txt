　　随着叶尘的修为飞速提升，行事也开始发生逆转。
<br/>　　“你、你、你到底是什么妖孽？”
<br/>　　叶家家主终于开始慌了，以他的境界，实在无法理解眼前所发生的一切，简直太过于诡异了！
<br/>　　叶尘冷冷一笑，
<br/>　　“我说了，我的境界，可不是你能理解的！！”
<br/>　　轰隆！
<br/>　　叶尘抬手一掌拍出，一道硕大的巨掌虚影，朝叶家家主狠狠砸落过去。
<br/>　　砰！
<br/>　　叶家家主慌忙奋力抵挡，却被直接打飞了出去，体内气血翻滚不止，满脸的骇然之色。
<br/>　　对方修为的提升速度，实在是太快了，这才一小会的功夫，竟然就已经超过他！
<br/>　　至于叶家其余众人，看到这一幕，早已经吓得纷纷四散而逃。
<br/>　　“小子！这是你逼我的！”
<br/>　　叶家家主被叶尘一掌击退之后，顿时大吼了一声，长袖猛然一挥，
<br/>　　轰！轰！轰！
<br/>　　自叶家家主的储物法宝之中，猛然窜出三道庞然大物！
<br/>　　一虎，一豹，一狼，而且个个都高达十余米，煞气腾腾的朝叶尘飞扑了过来！
<br/>　　“这就是傀儡之术吗？”
<br/>　　造化大陆因为只有造化规则的存在，所以修者的战斗力普遍较弱，所以在傀儡术上的造诣，达到了炉火纯青的地步。
<br/>　　而叶家家主放出的这三具傀儡，每一个的实力都不在他之下，这也是他的最强的底牌！
<br/>　　“小子！我还就不信了，三具破神境的傀儡，难道还杀不死你？”
<br/>　　轰轰轰！
<br/>　　三具破神境的凶兽傀儡，在叶家家主的操控之下，顿时开始对叶尘发起疯狂的进攻。
<br/>　　叶尘却不慌不忙，身体突然腾空而起，躲过三头傀儡的攻击之后，猛然一掌拍落，
<br/>　　“规则之力，净化！！”
<br/>　　轰！
<br/>　　自叶尘的掌心之中，猛然发出一道璀璨的光芒，将下方的那三头傀儡笼罩其中。
<br/>　　滋滋滋~~
<br/>　　前一刻，还十分凶悍的三头傀儡凶兽，在那道璀璨光芒的笼罩之下，顿时开始急速的缩小，不一会的功夫，竟然就便成三具没有任何生机的木头玩具。
<br/>　　“什么！！”
<br/>　　远处原本还自信满满的叶家家主，看到这一幕之后，顿时彻底懵了，
<br/>　　“魔鬼！你简直就是魔鬼！”
<br/>　　叶家家主大吼了一声之后，眼看叶尘朝他这边望来，只感觉一股凉气从脚底板，直接冲到了头顶，浑身不由得一个哆嗦，转头拔腿就跑。
<br/>　　他是怎么也没有想到，自己的最强底牌，居然这么快就被叶尘彻底摧毁。
<br/>　　他是真的怕了！
<br/>　　然而，
<br/>　　叶家家主才刚刚奔出数十里，耳边忽然传来一道淡漠的声音，
<br/>　　“若不是因为你，我也不可能这么快便领悟到造化法则，今日我便不取你性命，就让你换一个形态继续苟活下去吧！”
<br/>　　声音响起之上，似乎还很遥远，等到声音落下，便已经尽在耳边，
<br/>　　“造化规则，改造！！”
<br/>　　叶尘追上叶家家主，直接再次一掌落下，
<br/>　　轰！
<br/>　　自他的掌心之中，再次发出一束璀璨的光芒，而且在这束光芒之中，还携带着一道道神秘的符文，隐隐透着一股，难以抗拒的规则之力！
<br/>　　“啊！怎么会这样？”
<br/>　　叶家家主忽然发出一道惊恐的呼叫声，因为他猛然发现，在那道白光的笼罩之下，他的身上居然开始出现毛发，而且身形开始缩小，四肢居然变成了猫腿，整个人不由自主的便躬着身体，匍匐在了地上。
<br/>　　叶家家主居然变成了一只猫的样子，而且还是那种看上去很萌的折耳猫。
<br/>　　“啊啊啊啊啊！”
<br/>　　看到自己变成了一只猫，叶家家主险些都要崩溃了，顿时发出一道不甘的吼叫，
<br/>　　“你这个恶魔！你到底对我做了什么？为什么？我为什么会变成这个模样？”
<br/>　　其实这个结果，叶尘也颇为感到意外，
<br/>　　他刚刚看到那些傀儡，对造化规则又有了一些新的理解，那便是运用造化之力，进行改造的能力。
<br/>　　修真者可以用灵木打造傀儡，将其制造成动物的样子，而人类本身，也是造化的产物，造化之力是不是也同样可以改造人类呢？
<br/>　　叶尘抱着这个念头，便将此法用在了叶家家主的身上，没想到居然真的可行，叶家家主居然真的被他改造成了一只猫。
<br/>　　而且更神奇的是，叶家家主虽然被他改造成了一条猫，但是他体内修炼的造化之力，居然也并没有消失！
<br/>　　也就是说，眼前这只猫，是一条具有破神境界实力的猫！
<br/>　　“这就是造化规则的能力吗？果然够强悍！”
<br/>　　叶尘又惊又喜，同时也感觉自己，对造化规则的领悟，还是太过肤浅了。
<br/>　　而另外一边，被叶尘变成猫的叶家家主，依旧在愤怒的咆哮着，
<br/>　　“小畜生！我不管你用了什么邪术，但是你如此羞辱于我，还不如直接杀了我！我要和你同归于尽！！”
<br/>　　已经愤怒到极点的叶家家主，扬起自己的猫爪，就朝叶尘扑了过来。
<br/>　　尽管他如今变成了一只猫，可还是实打实的破神境强者，这一爪挥出，连空间都能被他瞬间撕碎，气势依旧强大的惊人！
<br/>　　可惜，这种程度的攻击，对于已经掌握了造化法则的叶尘来说，简直不值一提。
<br/>　　“规则之力！禁锢！”
<br/>　　叶尘随手一指，
<br/>　　嗡~
<br/>　　一道白光自叶尘的指尖发出，携带着一股难以抗拒的规则之力和一道神秘的符文，瞬间便打入叶家家主的猫体之内。
<br/>　　“啊！！”
<br/>　　叶家家主顿时发出一道惨叫，在他的脖子里，顿时多出了一个项圈，死死禁锢住了他的脖子，全身也动弹不得了。
<br/>　　到了这个时候，叶家家主也终于意识到了对方的强大，情绪渐渐安静下来，一双猫眼之中，透着浓浓的惊恐之色，
<br/>　　“你绝对不是之前的那个废物叶枫！到底是什么人？不！人怎么会具有这样的能力？难道你是神？！！”
<br/>　　叶尘淡淡一笑，
<br/>　　“你可以这么认为！”
<br/>　　在造化大陆这个位面，只有造化规则这么一条规则，叶尘掌握了造化规则，就等于掌握了开发这个世界的钥匙，说是这个世界的神灵，也丝毫一点不为过。
