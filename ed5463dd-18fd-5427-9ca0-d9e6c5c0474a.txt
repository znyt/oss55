　　“不好！”
<br/>　　那妖兽心中暗道一声，这种声音化为神识攻击的手段正好是他亦擅长的，，他深深的知道这种攻击有多么可怕。他不得不打起十二分精神来对待。
<br/>　　此刻，他已经反应过来，来不及多想，必须赶紧应对才是。
<br/>　　妖兽张开大口，对着那正在弹琴的玉玲珑便是一声大吼：
<br/>　　“昂——！”
<br/>　　一声大吼，虚空之中出现一个巨大的虚幻狮子头，罡音震耳欲聋，周围一切全部被这狮子吼震得倒塌，树木破碎，大山崩溃，玉玲珑的天龙八音直接被攻破。
<br/>　　天龙八音对狮子吼，天龙八音败！
<br/>　　不是天龙八音不厉害，元始所传的东西，能不厉害吗？真正的原因是玉玲珑的修为还没有狮子妖兽强大。
<br/>　　功法再好，没有极强的修为驾驭，也是白搭。
<br/>　　四人都被这强大的狮子吼震的吐血倒飞，不怪他们修为不强，他们已经是太乙金仙顶峰了，只是对手却是远超他们的大罗金仙罢了。
<br/>　　此刻，他们同时被震伤，而妖兽却没有，此时妖兽趁着他们还没有完全反应过来之前，迅速来到四人面前，一个巨大的手掌对着四人轰了下去。
<br/>　　“不好！”
<br/>　　看着那遮天蔽日的巨大手掌，四人心中暗叫一声，匆忙之际，来不及抵挡，四人只好随意挥动手中的灵宝抵挡。
<br/>　　但，终究是匆促至极，完全没有准备，四人结结实实地挨了这巨大的一掌。
<br/>　　“噗嗤——”
<br/>　　四人吐血倒飞，直到飞出数里之远，这才撞倒一座大山之上，倒跌了下来。
<br/>　　“噗嗤——”
<br/>　　再次吐血，四人倒地不起。
<br/>　　“哈哈哈！去死吧！”
<br/>　　此时，狮子妖兽飞了过来，看着倒地的几人，抬手就是一掌，想要将他们全部解决。
<br/>　　师兄妹四人看着朝自己击来的一掌，心中已经任命般的闭上了双眼，他们心中极为遗憾，只叹平时没有好好修炼。
<br/>　　“死！”
<br/>　　就在此时，虚空之中，一道淡淡的声音传来过来。
<br/>　　紧接着，一个遮天般的滔天巨掌向着这狮子拍了下来。
<br/>　　“啊？什么？这怎么可能？！”
<br/>　　狮子妖兽心中感到不妙，抬头向上看去，只见一个这天蔽日的手掌向自己轰了下来，自己在这手掌之下，竟然完全升不起反抗的心思，这怎么可能？
<br/>　　“啪！”
<br/>　　手掌毫无阻拦的拍了下来，就这样，妖兽完全没有反抗，被虚空传来的一个巨大手印一掌拍死，尸骨无存。
<br/>　　师兄妹四人在此时却是愣住了，他们瞪大了双眼，看着虚空之中走出来的男子，只感觉他是那么伟岸，那么无边。
<br/>　　“怎么？被吓傻了？”
<br/>　　一道淡淡的声音传到四人耳边，令四人反应了过来，那声音还是那么的熟悉。
<br/>　　“啊？徒儿见过老师！多谢老师救命。”
<br/>　　四人连忙狼狈起身，对着虚空中的男子躬身道。
<br/>　　显然，这立于虚空之中男子是他们的老师元始天尊。
<br/>　　“嗯。”
<br/>　　元始淡淡的应了一声，看了一眼四人的模样，四人此次受伤颇重，许多地方都有重创，特别是神魂被那狮子吼震得受损，众所周知，元神受损，非常难治。
<br/>　　此次能够及时救到自己的四位弟子，还是因为他回昆仑的路上心血来潮，特意为他们算了一卦，这才算出他们有生命之危，之后顺便赶来一救。
<br/>　　若不是他心血来潮算了一卦，恐怕他们怕是以后再也看不见元始了。
<br/>　　元始抬手打下四道绿光，分别飞入四人体中，顿时四人全身绿光绽放。不过片刻时间，四人便全部好了起来，伤势恢复如初，完全看不出四人受过重创的迹象。
<br/>　　“多谢老师！”
<br/>　　四人再次躬身谢道。
<br/>　　“嗯，经过万年多时间的游历，尔等想必也了解到了这洪荒中的残酷之处了，此次回去想必也定能刻苦修炼了。如今，你们虽然修为不错，但比尔等修为高深之辈，大有人在。
<br/>　　此前，尔等虽然修炼依然够努力，但，你们并没有明确的修炼目标，这，不好。如今，想必你们都已明白了，今后亦会更加刻苦修炼。”
<br/>　　“记住，不证道，都是蝼蚁。”
<br/>　　“谨记老师教诲！”
<br/>　　四人脸色坚定，明显是对这句话有了深刻的理解，经过万年多时间的游历，他们亦知晓洪荒的可怕之处，弱肉强食，优胜略汰！故而，他们决定今后遵从老师嘱咐，努力修炼。
<br/>　　“老师，这是什么灵根？”
<br/>　　此刻，玉玲珑取出一根青色的藤条，对着元始问道。
<br/>　　这是他们之前在一个山谷之中发现的灵根，因为刚出世，就弄出了极大的动静，天空被染成了青色，玄光绽放，极为不凡。也正是因为这件东西，他们才会惹上了大罗金仙修为的强大妖兽。
<br/>　　“咦！”
<br/>　　元始瞥了一眼这青藤，只见这青藤不过三米之长，藤条通体青色，青如晓天，叶子宽大浓绿，青意盎然，生机勃勃，充满浓郁的生命力。
<br/>　　“这是不老长青藤，为极品先天灵根，蕴含浓郁的生命力，一生长青。三千年一开花，三千年一结果，再三千年，方能成熟。吃了不老果，一生不老。
<br/>　　而且，它还会凝聚生命之露，这东西极为珍贵，无论疗伤，还是增加寿命，都是极品中的极品。若是机缘足够，说不定还能领悟生命法则。
<br/>　　它的功效堪比生命树。不比先天十大灵根差，是个好东西，你们好好栽种。”
<br/>　　元始对着四人说道，这东西对于其他人来说，可能是一件逆天的好东西，但对于他来说，并不算什么。
<br/>　　长生不老？不好意思，他已经是了。生命法则？不好意思，他也已经领悟了。倒是可以让它与其他灵根种在一起，这样对其他灵根有莫大的好处。
<br/>　　“这……”
<br/>　　四人面面相觑，想不到，这还真是一件好东西啊。
<br/>　　“好了，吾如今住在昆仑三清圣地，你们的游历也结束了，也该闭关了。尔等是要回永恒岛，还是到跟随为师到昆仑？”
<br/>　　“老师，我们自是跟随在老师身边伺候老师。”
<br/>　　四人再次躬身道。
<br/>　　“好。”
<br/>　　元始亦不废话，一挥手，顿时将四人收进他的体内空间之中。之后，撕开虚空，向着昆仑山穿梭而去。
<br/>　　回到昆仑，安排好四位记名弟子后，元始便宣布闭关。此次，他不仅要恢复之前受损的元神，还要研究冥书生死簿，可谓是任务多多啊。
<br/>　　不再多想，很快，元始便沉浸在修复元神的修炼之中。修炼之中，他感受到阴阳元神的无上妙处，再结合这些年在星空之中的游历，以及自己修炼的星辰入神窍，他感受到了突破《神元九转》第七转星空的契机。
<br/>　　时间，便在元始修炼之中匆匆流逝。
