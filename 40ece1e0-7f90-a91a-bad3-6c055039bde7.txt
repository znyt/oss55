　　“除了他还能有谁。”
<br/>　　美华道：“可是这怎么可能，你说你来这个世界才十多天，十年之前，他怎么可能知道你要来？”
<br/>　　对这问题，叶少阳也是一头雾水。
<br/>　　毛小方沉吟半晌，突然一拍脑门，说道：“我明白了！”
<br/>　　叶少阳和美华一起转头看他。
<br/>　　“那个徐福，是最近才去通知美华姑娘的，也许就在昨天，甚至就在刚才，知道你来到这镇子上，这才去找她的！”
<br/>　　叶少阳皱眉道：“什么啊，我们说的是十年前。”
<br/>　　“对啊，少阳子，你可不要忘了，徐福有山海印，原则上可以穿越到任何时代。他没准在跟踪你，知道你要对付五通神，怕你对付不过，于是用山海印穿越到十年前，跟美华姑娘说了这事……那个时候的美华姑娘，就记住了这件事，然后这种记忆维持了十年……毕竟时间是顺延过来的。”
<br/>　　他这么一解释，叶少阳和美华都明白了。美华失声道：“怎么可能！”
<br/>　　“很有可能。”毛小方说道，“这种穿越时空的事情，本来就不是我们能够想得通的。我能想到的，就是这样。”
<br/>　　叶少阳按照他的理论，把事情想了一遍，说道：“如果你说的这个成立，那就说明，改变一个时空的某种东西，对别的世界也会造成连贯的影响——不然的话，为何徐福去通知了十年前的美华，今天的美华，也会跟着做出类似的反应。”
<br/>　　毛小方没太听得懂他这句话的意思，让他说清楚一点。
<br/>　　“算了，我也根本就说不清楚，而且这个也不是我能关心的，不管了。”叶少阳转头看着美华，说道，“现在你相信我的身份了吧？”
<br/>　　“你……”美华望着他，说道，“可是现在的我并不认识你，我跟你相识，应该在百年之后……我只想问一句，我是真心实意给你当鬼仆的吗？”
<br/>　　“这个……”叶少阳耸了耸肩，“是被我打服的，我本来想灭了你，但是你献出了魂晶，我只能收下。不过……在认主之后，我们的关系非常好。”
<br/>　　“有多好？”
<br/>　　叶少阳想了一下，道：“如果你遇到危险，我可以出生入死去救你，你对我也一样。”
<br/>　　美华定定地望着他，突然叹了口气，苦笑道：“好不可思议。”
<br/>　　“不过现在是一百年之前，你还不是我的鬼仆，所以，你也不用担心我要求你做什么。”叶少阳说道，“能见到你，我真的很开心了。”
<br/>　　美华目光闪动，似乎在犹豫，突然叹了口气，道：“我相信我百年之后的选择……既然那个时候的我把你当主人，就算是提前了，也是一样。”
<br/>　　说完，美华躬身对叶少阳施了一礼，叫道：“主人。”
<br/>　　叶少阳摆手道：“别这么叫，你一向都是叫我老大的。”
<br/>　　“老大？”
<br/>　　“对。我是你们的大哥，我们是一起混的，从人间打到鬼域，生死相依！”
<br/>　　美华有些动容，想了一下说道：“徐福让我在这里等你十年，也没有说什么事，你们是不是遇到麻烦了？”
<br/>　　叶少阳刚要开口，毛小方拉了他一下，说道：“隔墙有耳，小心点，我们回去说。”
<br/>　　大伙于是一起回到客栈。叶少阳把云儿安顿好，跟毛小方和美华一起来到隔壁房间，先贴了几张血精符在门窗上，隔绝了内外的空间，确定没人偷听，这才把自己要屠神的事跟美华说了。
<br/>　　美华听完之后，震惊地说不出话。
<br/>　　“你……居然想要屠神。你胆子也太大了。”
<br/>　　“我一向都是这样，这个你百年之后就知道了，而且有很多大事还是你陪我一起做的。”说这个的时候，叶少阳总感觉有点怪怪的。
<br/>　　美华喟然说道：“我知道徐福祖师让我找你做什么了……原来他早就安排好了，屠神我虽不敢，不过你既然是我主人，你要做什么，我也就听你的了。”
<br/>　　叶少阳道：“本来我是不想跟你相认的，但是既然都相认了，你确实得帮我，我在这边本来就缺少帮手。不过我不知道你刚才说的什么意思？”
<br/>　　美华道：“徐福祖师十年前让我搬到这里来修行，我不明白是什么意思，现在想来才明白，是提前让我过来，与五通神相邻，熟悉他的情况……我的洞府离蛤蟆山有几十里，跟五通神没什么来往，不过对他也算是了解的。”
<br/>　　叶少阳点点头，几十里的距离，对邪物来说并不算远，很可能只是相邻的两块势力范围。互相之间不可能不认识，于是问道：“你跟五通神关系怎么样？”
<br/>　　“他是神，我是鬼，没什么来往，不过我知道他的底细，他真身是一条三足金蟾，洞府就在庙宇后面的一口井里，修为在我之上，五通神，据说一共有五尊神灵，他是其中之一，另外四尊都不在此地，不过偶尔也曾前来与他会面，老……”
<br/>　　叶少阳看她表情，知道她是想叫自己老大，但有点叫不出口，摆手说道，“你要是不习惯，就直接叫我少阳没关系。”
<br/>　　“还是叫老大吧。老大，你要屠神，不得不提防那四尊神，要速战速决，免得他去搬救兵，五尊神灵一起来的话，我们必败无疑。”
<br/>　　叶少阳点点头，心想就一个蛤蟆精自己都对付不了，再来四个，自己还不分分钟就被拍死了？好在五通神现在也没把自己放在眼里，应该不会这就去搬救兵。
<br/>　　“我在洞府里也养了一些精怪，你对付三足金蟾的时候，我可以助你一臂之力，只是不能硬来。”
<br/>　　“这个我知道，想办法智取。”叶少阳笑道，“有你帮助，我现在有信心多了。”
<br/>　　毛小方道：“少阳子，我觉得我们应该把道渊找来，一起对付，他嫉恶如仇，性格比你还刚烈，一定会来帮忙的。”
<br/>　　叶少阳点点头，道：“就是路远，不好通知。”
<br/>　　这时代没手机没电话，通信是个大麻烦。
<br/>　　“我去雇一匹快马，这就过去，不到一天就能到，见到他马上接他一起过来。”
