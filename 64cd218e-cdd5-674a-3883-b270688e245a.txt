        &lt;h3&gt;第三卷第一百三十五章  低头碰上你的唇&lt;/h3&gt;
<br/>
<br/>        凌辰在贵州呆了一段时间，各经销商都回去了，他还得留下来和厂里进行沟通，原本在元旦之前就要走好的工作，因为中途出现了变故，计划基本都推后了。
<br/>
<br/>        凌辰这一次算是有惊无险，但是他不知道的是他总代没有丢掉的原因是因为董亚子姊妹俩。
<br/>
<br/>        有人问厂家要了总代，作为厂家来说，检验总代的标准就是帮着厂家打开市场，控制好市场规则。
<br/>
<br/>        还有就是要检验总代对厂家的忠诚度。
<br/>
<br/>        他们很难抉择，何少华的作为可圈可点，也没有太多的问题，而且有人给他们打了招呼，但是，甸城的情况也不容乐观，假冒伪劣产品据说都已经泛滥了，可何少华从来都没有反应过。他们曾经暗暗让人去看了一下，有些经销商都有，这就不得不让他们引起警觉了。
<br/>
<br/>        何少华的情况不一般，如果说他不知道或者说他没有能力进行整顿，那是不可能的。
<br/>
<br/>        当初也是看上了何少华的身份。
<br/>
<br/>        但是何少华狂妄自大，目中无人。
<br/>
<br/>        他们是厂家没有错，但是他们也是生意人，在考核代理商里面还是要考核代理商有没有能力。
<br/>
<br/>        凌辰无论是对厂家还是下面的经销商都很好，能将各方面关系处理得很好。
<br/>
<br/>        厂家需要的就是这样的人，总代就是厂家和各级经销商之间的润滑剂。
<br/>
<br/>        他们决定在好好商量一下，进行考察一下俩个人的人品。
<br/>
<br/>        第一次喝酒，有董总亲自来考察，酒品见人品，在酒桌上就有这样的一句话，其实说白了也算是在饭桌上能看到一个人的人品。
<br/>
<br/>        吃饭的时候何少华除了在他面前有所表示之外，对其他人都是一副爱理不理的样子。
<br/>
<br/>        而凌辰会将大家照顾得很好，喝酒的时候大家都想把对方喝倒，最后都装醉趴在桌子上，只有凌辰是实实在在的，最后将自己喝倒了。（当然董总要是知道凌辰是因为接了小雅的电话受到了打击再加上也喝酒的原因，怕是会大跌眼镜。）
<br/>
<br/>        第二次就是陪董亚子，这个是董亚子提出来，董巧奕长得比董亚子好看，这个考验两个男人的心性，面对美色会不会动心。
<br/>
<br/>        凌辰不为所动，连游戏都不参加，何少华不但极力的讨好一个不认识的美女，后来还直接以极其恶劣的态度驱赶董巧奕。
<br/>
<br/>        在他们返程的路上，董巧奕就将情况汇报给董总他们了，而且将何少华臭骂了一通。
<br/>
<br/>        最后说了一句：“如果总代交给这样的人做，肯定市场是一片乌烟瘴气。”
<br/>
<br/>        大家经过商议，决定还是让凌辰当总代，当然凌辰的代价就是打击假冒伪劣产品。
<br/>
<br/>        凌辰和董亚子具体负责市场规划和严打假冒伪劣产品。
<br/>
<br/>        董亚子才进入公司，对经销商不是非常的熟悉，一切都得和凌辰沟通。
<br/>
<br/>        为了更直观的告诉董亚子，全国各地经销商的布局情况，凌辰和刘诚之做了一份地图，标注各地区经销商的分布情况。
<br/>
<br/>        俩个人正在讨论的时候，董总过来了，董总看着他们的地图，问：“明年整顿市场你们准备从哪里着手？”
<br/>
<br/>        董总看着凌辰，凌辰盯着地图，这几年他对各地区的经销商还是很了解的，他们每年都会分给经销商一会货，但是有些销售得很好，有些人会囤一些货，有些人会在网上购买。
<br/>
<br/>        假冒伪劣产品也有，但是他控制不了，因为源头并不是他能查到的，他没有这个权限。
<br/>
<br/>        如今厂家将这个任务交给他，为了回馈厂家的这份心，他不干也得干。
<br/>
<br/>        他决定从消费大的地方也就是说消费能力强的地方着手。
<br/>
<br/>        他指了几个地方，但是董总皱着眉头，并不吭声，凌辰咬咬牙，指了另外一个地方。
<br/>
<br/>        “地处偏远，比较凶险。”董总点点头然后转身离去。
<br/>
<br/>        董亚子看了一眼凌辰：“凌辰哥哥，我们怕是要同患难了。”
<br/>
<br/>        凌辰：“董总，我们本来利益就是一直的。”
<br/>
<br/>        董亚子：“你能不能不要整天冷着一张脸，一副公事公办的样子？”
<br/>
<br/>        凌辰：“我们难道不是在谈工作吗？”
<br/>
<br/>        董亚子气结，他们是在谈工作没错啊，也不用那么划清界限的吧？
<br/>
<br/>        董亚子：“凌辰哥哥，你有没有女朋友？”
<br/>
<br/>        凌辰：“现在是工作时间，不谈私人问题。”
<br/>
<br/>        董亚子：“现在才发现你就是一个冷木头。不过你这冷木头我也能劈开烧了。”
<br/>
<br/>        凌辰抬头看了一眼，发现董亚子的样子很认真，但是他什么都不说，收好地图转身走了。
<br/>
<br/>        董亚子跺了跺脚。
<br/>
<br/>        刘诚之正准备和凌辰一起走，董亚子叫住了刘诚之。
<br/>
<br/>        “刘诚之。”
<br/>
<br/>        “董总。”
<br/>
<br/>        “听说你记忆力特别好，你记不记得你们凌总有多少个女朋友？都是什么时候开始谈的？”董亚子问刘诚之。
<br/>
<br/>        刘诚之一个激灵：“你当我们凌总是什么呢？我们凌总现在单身，只有一个女朋友。”
<br/>
<br/>        董亚子：“只有一个女朋友？”
<br/>
<br/>        “是啊，都谈了好多年了，基本每天都唧唧歪歪的。“刘诚之回答。
<br/>
<br/>        “他们关系确定了没有？”
<br/>
<br/>        “反正没有见双方父母，那个小雅姐家里的条件太好了，俩个人估计是怕家里人不同意，一直都没有说明。唉！”刘诚之叹了口气，不知道董亚子问这些是什么意思。
<br/>
<br/>        “小雅？”
<br/>
<br/>        董亚子陷入了沉思。
<br/>
<br/>        刘诚之看着董亚子不说话，挥了挥手一溜烟的跑去找凌辰去了。
<br/>
<br/>        刘诚之很机灵，记忆力也超好，公司的什么事情他都记得清清楚楚的，凌辰当他是个小弟弟一样带在身边，无论是自己的私事还是公事都不瞒着他，刘诚之的耳朵里早都灌满了宁雅的名字。
<br/>
<br/>        “嘿，莫不是董亚子看上凌总了？那可不行，宁雅姐姐那么漂亮，凌总那么帅的，不能让董亚子占了便宜。”刘诚之不跑了。
<br/>
<br/>        “董总，小雅姐姐是南方宁家的，人又漂亮又能干，和凌总是天造地设的一对，虽然这样说有点让人难为情，但是确实是事实。”刘诚之慢悠悠走到董亚子的身边。
<br/>
<br/>        “刘诚之？”董亚子看着刘诚之，一步一步的逼近刘诚之，刘诚之不知道董亚子要干嘛，一直退后，退到无路可退了，靠在墙上。
<br/>
<br/>        “退啊，退啊，你在退试试。”刘诚之低头看着抬头看着自己的董亚子，她的眼睛亮晶晶的，凶狠的看着自己。
<br/>
<br/>        “你想干嘛？”刘诚之心一慌，女人真的是老虎，惹不得。
<br/>
<br/>        “我不漂亮是吧？我不能干是吧？”董亚子双手一伸，拽住了刘诚之的两只耳朵。
<br/>
<br/>        刘诚之疼得一低头，恰好撞上董亚子的唇。
<br/>
<br/>        凌辰在回酒店的路上接到了陈子昂的电话。
<br/>
<br/>        “凌总，年前还来北京吗？”
<br/>
<br/>        “不一定。”
<br/>
<br/>        “那给我一个账号，最近周转开了。”
<br/>
<br/>        “你先用着吧，我最近不急。”
<br/>
<br/>        “凌总，不来看看宁雅？”
<br/>
<br/>        凌辰沉默了一会：“年前太忙，估计没有时间，以后再说吧。”
<br/>
<br/>        “好的，打扰了，如果凌总来北京了联系我。”
<br/>
<br/>        “好，再见！”
<br/>
<br/>        还有必要去找宁雅吗？追了这么久，一直都是他在努力，现在还有必要吗？
<br/>
<br/>        没有必要了！
<br/>
<br/>        这个刘诚之怎么还不来？
