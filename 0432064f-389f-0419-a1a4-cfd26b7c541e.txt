　　“立刻把你的枪放下！”
<br/>　　西装男手中握着格洛克手枪，神色紧张的对着罗昊大声吼道：“我命令你，立刻把你手里抢放下！”
<br/>　　罗昊按下弹匣扣，把手枪中的弹匣退出，拉动套筒，把枪膛中那颗已经上膛了子弹也退出。罗昊完成这几个动作用了不到一秒钟时间，西装男还没来得及反应，罗昊手里那把格洛克手枪就成了一把没有子弹的玩具。
<br/>　　甩手，卸掉全部子弹的格洛克手枪旋转着朝着西装男砸去，跟他那张微胖的脸来了一个亲密接触，西装男被砸倒在地，他的额角被砸开一个口子，鲜血立刻流了出来，沾着地上尘土的鲜血挂在他脸上，令他整个人看上去有些狰狞。
<br/>　　罗昊一步跨出，以左脚作为轴心，右脚甩出后旋踢，踢在一名第九处探员身上，在他黑色西装上留下一个灰扑扑的鞋印。
<br/>　　挨了罗昊一记后旋踢的第九处探员倒飞出去，撞在一辆轿车的副驾驶车门上，驾驶车门瞬间凹陷下去，玻璃窗也是被震碎开来。
<br/>　　被车门挡住的第九处探员吐出一口血，身体滑落到地上，气息萎靡。这一脚至少踢断了他三根肋骨，只要他稍微有动作，断骨处就传来撕心裂肺的疼痛，暂时失去了行动能力。
<br/>　　这些第九处探员胸前的徽章中藏着微型摄像头，所以发生在医院停车场里的一幕，完整的传到了K国安全局第九处指挥中心，第九处处长科尔看着巨型液晶屏幕中的画面，眉头紧皱，抓起桌子上的对讲机：“第一行动小队，立刻赶往现场把嫌疑人带回来！记住，必要时可以开枪射击，但是不要杀死他们。中国军方的现役军人如果死在我们国家的话，会被认为是我们主动挑起战争，等到那个时候，中方舰队、战斗机还有特种部队会进入K国，到那时候我们就死定了，巡航导弹会把我们炸上天，我们连逃跑的机会都不会有。”
<br/>　　“明白！”
<br/>　　K国安全局行动处停车场内，穿着防弹战术背心，身上挂着M4自动步枪的第一行动小队队长Guram拉开身边防弹吉普车门，用手在车门上用力拍了两下，大声说道：“我们出发！记住行动要求，留下活口！”
<br/>　　第一行动小队车队从地下停车库缓缓驶出，然后加速朝着医院的方向驶去。而在第一行动小队往医院驶去同时，一支从中方大使馆出发的车队同样也是火速赶往医院。
<br/>　　医院停车场，K国安全局第九处十几个探员此时已经全都倒在了地上，他们的手枪也全都被肢解开来，仍在他们身边。
<br/>　　罗昊把一根弹簧拉开仍在地上的时候，注意到了自己脚边那名第九处探员胸前徽章上的小孔。蹲下身，翻动徽章，罗昊发现了藏在徽章背后的微型摄像头。
<br/>　　“妈的！”
<br/>　　罗昊一把扯下徽章，仍在地上，把徽章背后的微型摄像机踩烂，对赵天宇说道：“他们身上有微型摄像机，估计很快就会有支援赶来这里。”
<br/>　　“就算没有这些微型摄像机，这里的情况也不可能瞒住。”赵天宇指了指停车场周围的医院保安，说道：“现在就看谁的速度更快一点了。”
<br/>　　赵天宇用手机拨通了云岳办公室的电话，等到电话接通后，对云岳说道：“暗影，我们遇到麻烦了。”
<br/>　　“什么情况？”云岳从办公桌背后站起身来，问道。
<br/>　　“K国方面认为我们涉及K国国家安全问题，打算把我们以间谍的身份扣留起来，但是我们都知道，这不过是一个借口，这个借口低级到连小学生都不会相信。”赵天宇说道：“大使馆已经了解情况，并且向医院赶来，但是K国方面的人应该也正在赶往医院。我跟傻小子被带走倒是无所谓，就算是被抓了我们也有办法出来，可是小熙那丫头因为子午的死晕倒了，到现在还没醒，如果她被带走，情况就不好说了。”
<br/>　　“谁敢动我女儿，我扒了谁的皮！”
<br/>　　云岳全身散发出一股子寒意，令得他办公室里的温度都是低了好几度，此时云岳又成了那个掌控着“炼狱”的鬼王！
<br/>　　“我会让‘炼狱’那边派人进入K国，如果事情往最极端的方向发展，他们能给你们提供帮助。”云岳深吸一口气，保持着冷静说道：“我会把事情报告给野狼。”
<br/>　　云岳结束了跟赵天宇之间的通话，打开办公桌最底下的抽屉，从里面取出一部加密卫星电话，拨通了一个号码。
<br/>　　“hello？”美杜莎充满着魅惑的声音从电话中传来。
<br/>　　“美杜莎？”云岳一愣，问道。
<br/>　　“Boss！？”
<br/>　　美杜莎放下搁在桌子上的双脚，坐直身体，接到云岳的电话令她感到很意外。云岳虽然离开了“炼狱”，不再是“鬼王”，但是“炼狱”中的人都清楚，以云岳的影响力，他在“暗月”还是有着话语权，甚至比起现任“鬼王”常青的影响力更甚！
<br/>　　“中东这边现在是你在管理吗？”云岳问道。
<br/>　　“是的，Boss！”美杜莎回答道。
<br/>　　“查一下，‘炼狱’中有多少学员是来自K国的？”云岳说道：“查清楚以后把他们全部找出来，然后给我扔到中东最混乱的地方，不要给他们一颗子弹、一滴水、一块面包！”
<br/>　　“哇哦~”
<br/>　　美杜莎夸张的叫了一声，用肩膀夹着电话，双手在电脑键盘上敲打着，对云岳问道：“他们到底怎么招惹到你了，让你要用这么残忍的方法玩死他们？”
<br/>　　“他们没有招惹到我，但是K国那些混蛋招惹到我女儿了！”云岳冷声说道：“把人给我扔出去之后，你亲自带一队人进入K国等候我的下一步命令。”
<br/>　　“明白！”
<br/>　　得到云岳命令之后，美杜莎脸上的笑容瞬间消失无踪，变得严肃起来。美杜莎站起身，把桌子上手枪揣进腿部快枪套中，看着电脑屏幕上那五个来自K国的学员，森然一笑。
