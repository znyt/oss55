　　青俞皇宫，金銮大殿。
<br/>　　青俞皇帝高坐于龙椅之上，下方左侧端坐着屈国师、青雅公主，以及一众青俞皇室的强者。
<br/>　　而下方右侧，则是只有苏莫一人。
<br/>　　众人的目光，均是不离苏莫的周身，所有人都是心中惊奇，这个年轻人到底是什么身份！
<br/>　　青雅公主美眸中异彩连连，不停的打量着苏莫，心中很是好奇苏莫到底是什么修为？为何实力如此逆天！
<br/>　　“这位小友，本皇名为青谷，不知小友名讳？”龙椅之上，青俞皇帝面带微笑，向苏莫微微抱了抱拳。
<br/>　　他没有询问苏莫来自哪个势力，而是准备先弄清苏莫的名讳，到时候就好查了。
<br/>　　“名字只是个代号而已！”
<br/>　　苏莫却是没有回答对方，他的目光直视屈国师，道：“我想问屈国师，在青俞帝国做了多久的国师，以前师承何处？”
<br/>　　苏莫没有强行逼问对方，因为对方很可能真的不是黄泉魔宗之人，况且，若是对方真是黄泉魔宗之人，有心隐瞒的话，逼问之下对方也肯定不承认。
<br/>　　所以，他随意询问一番对方的情况，看看其中有没有什么蹊跷之处。
<br/>　　“回小友，老夫在青俞帝国做国师已经四十余年，至于师承，老夫早年加入过几个宗门，分别是天岩岛的铁华门，北洲的金光门，金环岛万剑宗！”屈覃国师徐徐说道，没有任何隐瞒，将自己加入过的势力都告诉了苏莫。
<br/>　　苏莫闻言，目光望向青俞皇帝，以及青俞皇室的其他人，其他人均是向他点了点头。
<br/>　　显然，众人均表示屈覃长老所言为实。
<br/>　　苏莫面露沉吟之色，对方说的头头是道，不似作假。
<br/>　　而且，这种事情，稍一打听，便可知真伪，对方心中应该也比较清楚。
<br/>　　难道真的不是？
<br/>　　一切只是巧合！
<br/>　　不过，苏莫总感觉这里有些不对头，但具体哪里不对，他也说不上来。
<br/>　　“屈国师可知，在苍穹世界，有哪些势力的人修炼尸功？”苏莫目光灼灼的望着屈覃。
<br/>　　“修炼尸功？”屈覃闻言，略一沉吟，道：“据老夫所知，整个苍穹世界，除了黄泉魔宗的尸罗殿之外，修炼尸功的人，极其的稀少！”
<br/>　　“你的尸功在哪里所得？”苏莫又问道。
<br/>　　“就在北海，一处上古遗迹之中，距青俞岛月三日的路程！”屈覃道。
<br/>　　“你这四十余年，一直在青俞帝国？”
<br/>　　“……”
<br/>　　“……”
<br/>　　苏莫不时的问出一个又一个问题，都是询问关于屈覃的情况。
<br/>　　屈覃也是知无不言，一一回答苏莫的询问，将自身的情况完全告知了苏莫。
<br/>　　一众青俞皇室之人，虽然不清楚苏莫为何要了解屈覃的情况，但也没有多问。
<br/>　　询问了一刻钟后，苏莫基本上没有发现有什么蹊跷之处。
<br/>　　“诸位告辞了！”
<br/>　　苏莫叹了口气，长身而起，准备离开青俞皇宫。
<br/>　　既然问不出什么情况，他也不想多留。
<br/>　　“这……小友，宫中已经备下宴席，不如暂留片刻，让青某略尽地主之谊！”青俞皇帝见苏莫要走，顿时出言挽留。
<br/>　　“是啊！这位小友，我青俞皇室有最好的极品佳酿，定然不会让小友失望！”
<br/>　　“小友想必是远道而来，不如在宫中住上两日，略作休息！”
<br/>　　“我皇室一定将小友奉为上宾！”
<br/>　　其他一些皇室的强者，纷纷出言挽留，他们的想法和青俞皇帝相同。
<br/>　　此人杀了汪晟，若是一走了之，那麻烦就大了。
<br/>　　苏莫目光扫视众人，他心中跟明镜似的，自然清楚这些人为何要挽留他。
<br/>　　不过，他不可能留在皇宫之中，他并不完全相信屈覃，心中已经有了计划。
<br/>　　“诸位，我还有要事，以后若是有机会，再来做客！”苏莫摇了摇头，随即转身向大殿外走去。
<br/>　　“公子等等！”
<br/>　　就在此时，一声娇呼声响起，只见青雅公主快步的走到了苏莫身边。
<br/>　　“何事？”苏莫转头望向青雅，疑惑的问道。
<br/>　　“公子，汪晟是北海冰宫的得意弟子，死在了我青俞皇宫，北海冰宫不会放过我们。”
<br/>　　青雅俏脸上布满了担忧，她轻咬红唇，继续道：“公子实力不凡，想必也是出自某个超级势力，还请公子想办法，帮我们渡过此劫！”
<br/>　　青雅公主此言一出，青俞皇帝青谷，以及一众皇室的强者，顿时脸色一变。
<br/>　　他们本来都是想稳住苏莫，就算稳不住，但只要苏莫青俞岛的地域，到时候北海冰宫之人来了，他们只需将责任完全推到苏莫身上就行了。
<br/>　　人是苏莫杀的，和他们没有关系。
<br/>　　但现在青雅倒好，直接将此事的严重性说了出来，如此的话，这个无名青年，还不是有多远逃多远，怎么可能会留下来承受北海冰宫的怒火。
<br/>　　青谷等人心中大急，但他们也不能说什么，更不敢出手阻拦苏莫。
<br/>　　“这个丫头！”青谷心中既愤怒又无奈，现在事已至此，他也无可奈何。
<br/>　　“冤有头债有主，人是我杀的，北海冰宫应该不会对你们怎么样吧！”苏莫淡淡的说道，他和北海冰宫没有过多接触，也就见过两次北海冰宫的宫主北海女皇，以及那位玄冰女皇。
<br/>　　但想必，北海冰宫还不至于，将汪晟的死怪罪在青俞皇室的头上吧！
<br/>　　不过，这位青雅公主的表现，倒是让苏莫刮目相看，看来此女虽然有些刁蛮，但还算很明事理，知道为自己的宗族考虑。
<br/>　　而且，此女也没和其父以及皇室的其他人一样，想先稳住他，到时候来承担北海冰宫的怒火。
<br/>　　而是直接说明了情况，寻求他的帮助。
<br/>　　这一点，倒显得此女光明磊落。
<br/>　　“你不了解北海冰宫，作为超级势力，北海的霸主，北海冰宫的威严不容侵犯，虽然我们皇室没有出手，但人是死在我皇宫门前，北海冰宫不会放过我们！”
<br/>　　青雅公主一改之前傲气跋扈的姿态，变得颇为沉着冷静，俏脸上布满了深深的担忧。
<br/>　　言罢，青雅似乎想到了什么，俏脸微红，微低着脑袋，低声道：“公……公子，你若能帮助我青俞皇室渡过此劫，青雅愿意以身相许！”
<br/>　　青雅的声音低如蚊蝇，之前举行的比武招亲，本就是为她挑选驸马。
<br/>　　她，对于苏莫很是中意，虽然苏莫好像对她没有兴趣，但她也会尽力争取一番。
<br/>　　这不仅是为了她自己，也是为了青俞皇室的安全。
<br/>　　苏莫闻言无奈的摇了摇头，他可没有心思再娶亲了。
<br/>　　“以身相许就不必了，你青俞皇室，不会有事！放心吧！”
<br/>　　淡淡的留下一句话，随即苏莫身形一闪，身形掠出了大殿，消失无踪。
<br/>(本章完)
