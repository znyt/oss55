　　　　法相一出，谁与争锋？
<br/>　　人族的战斗之法简单直接，凶残至极，这些都是人族在外拼杀而来的经验武技，人族武者修为一到金丹境，便会被要求外出游历洪荒，斩杀各路妖族，多年的拼杀，早就让他们的战斗经验变得丰富无比。
<br/>　　而妖族众妖王，虽然他们有先天神通，更能变出妖兽本体，但这些对上人族的天地法相与人族的凶残，还是有些不够。
<br/>　　人族高手不畏死，不惧死，心中有着对妖族的仇恨，战士起来，个个犹如魔神一般，众多妖族妖王早已被打破了胆，这完全是一场屠杀的盛宴，如之前妖族吞食人族那般，只不过现在角色对换了，是妖族高手被人族高手屠杀。
<br/>　　死了，妖族大罗金仙的妖王，竟然被人族武者活生生地给屠杀了。
<br/>　　“嘶！怎么可能，我不是看错了吧，妖族竟被吊打了？”
<br/>　　“还真是，妖族妖王竟然被人族武者吊打了，这怎么可能？”
<br/>　　“人族真是恐怖，我们谁都不了解。”
<br/>　　“你们发现没有，人族的修炼的武道就是当年惊动洪荒所创之法，此法太过霸道，当初不觉得怎样，如今看来，此法就好像是专为人族所创之法。”
<br/>　　“不错，我也发现了，这武道与人族十分相配。”
<br/>　　“这下子帝俊与太一要好看了，嘿嘿！”
<br/>　　“……”
<br/>　　关注此战的众多修士们通通倒吸一口气，惊骇的同时也在暗自冷笑，妖族此次遭受如此奇耻大辱，妖帝帝俊与东皇太一不发疯才怪。
<br/>　　高空之上，看着不断被人族高手屠杀的妖族妖王，帝俊与东皇太一眼中闪过一丝惊讶，但更多的却是寒芒，在他们看来，人族是妖族的血食，天生就是给妖族作食物的，可是如今食物却要屠杀主人，这如何不让他们羞恼。
<br/>　　灭族，一定要将他们灭族，才能洗刷我妖族如此大辱，方解我心头之恨。
<br/>　　“找死！”
<br/>　　“轰！”
<br/>　　东皇太一大怒，抬手便是凝聚出遮天一掌，威势无双，准圣中期巅峰的修为暴露无遗，翻手之间，毁灭星辰，准圣之下者，谁能抵挡？这一掌向着人族武者拍去，就好似拍死一只蚂蚁一般。
<br/>　　“滚！”
<br/>　　正在所有人都认为此次人族再劫难逃之际，突然一声暴喝从人祖殿中传出，响彻九天，轰动苍穹，将东皇太一凝聚出的手中轰散开来。
<br/>　　“谁？给本皇滚出来！”
<br/>　　东皇太一大喝一声，眼睛死死盯着人祖殿。
<br/>　　“哈哈哈！太一小儿，想要灭我人族，休想！”
<br/>　　一阵爽朗的大笑声传出，紧接着，九道人影从人祖殿中走了出来，他们看上去普普通通，好似并没有什么特别的，但却让帝俊与太一脸色更加凝重了，因为他们能够从这些人中感受到深厚的气血，堪比准圣高手了。
<br/>　　“来着何人？”
<br/>　　东皇太一眼中寒芒更甚了，他看向几人眼中杀机毕露。
<br/>　　“人祖，燧皇，燧人氏！”
<br/>　　“人祖，巢皇，有巢氏！”
<br/>　　“人祖，衣皇，缁衣氏！”
<br/>　　人祖三族全身气势陡然大涨，气血如虹，贯穿天穹，与帝俊太一形成对抗之势。
<br/>　　“人族，太皇，太昊氏！”
<br/>　　“人族，少皇，少昊氏！”
<br/>　　“人族，雨皇，雨伯！”
<br/>　　“人族，风皇，风后！”
<br/>　　“人族，剑皇，赤松子！”
<br/>　　“人族，盘皇，盘王！”
<br/>　　剩下六人，一口气爆出自己的名号，全身气势恢宏，意志通天，形成大势，对抗帝俊与太一。
<br/>　　“嘶！”
<br/>　　这一刻，天地失声，天地间各路大能全都倒吸一口气，纷纷惊骇不已，他们简直不敢信心自己的眼睛。他们看到了什么，那是什么？
<br/>　　准圣强者啊！那可是准圣强者啊，九位，九位准圣啊！虽然他们都是准圣初期，后面六个更是刚刚晋升的准圣，但却也不能让人忽略啊。
<br/>　　恐怖如斯，真正的恐怖如斯！
<br/>　　准圣是什么？准圣可是天道圣人之下最强者啊，那可不是大白菜啊！
<br/>　　人族，作为一个仅仅发展万年的后天种族，就出现了九位准圣，这是多么令人难以置信的消息。
<br/>　　这怎能不让人震惊！
<br/>　　这一刻，无数人盯着人族，想要发现人族中到底隐藏着什么，竟然能有如此多准圣？这已经不是奇迹可以说明的了。
<br/>　　可惜，让他们失望了，他们并没有发现什么。
<br/>　　“人族，真是令人嫉妒啊！”
<br/>　　“人族，我已无力吐槽了，今日的震惊已经够多了，就算出现再大的奇迹，我也能够接受，只因为我已经震惊到麻木了。”
<br/>　　“人族，已成气候，将会成为除了巫、妖二族后的第三大族。”
<br/>　　“……”
<br/>　　此刻，就连圣人都瞠目结舌了，更不多说其他人了。
<br/>　　其实，人族并没有如他们猜测那般，修炼仅仅万年。他们并不知道，人族之中，武祖玄天领悟领时间法则，创造出一时空界，在时空界中，时间是外面的百倍。洪荒过去一年，时空界中已经过去百年了。所以，他们实际上是修炼了近百万年的时间。
<br/>　　虽然修为约到后面，待在时空界中的效果就越低，法相境以上的武者，甚至没有修炼效果，但对于炼体境到洞天境这段境界的武者，的确起到了很大的作用，能够为他们节约许多时间。
<br/>　　但就算如此，也足够令人震惊的了。试问，洪荒之中，哪一个准圣高手不是数千上亿万年修炼而成的，人族能够在仅仅百万年时间内，就出现九位准圣，这是多么不可思议的一件事。
<br/>　　高空之上，帝俊太一，眼中闪过惊骇之色，他们的确是被震惊了，他们实在是想不到，人族竟然如此恐怖，这一刻，他们深深的感受到了人族背后的巨大潜力，感受到了人族未来的巨大威胁。不，现在他们就是一个巨大威胁了。他们真不知道，若是再给人族一段时间的发展，天地间还有什么种族能够制约人族，妖族不行，巫族也不行。
<br/>　　这一刻，帝俊与东皇太一心中闪过无与伦比的强烈杀机，这比他们对巫族的杀机还要强烈。因为，巫族最多也只能与他们持平，但人族的表现，却是已经超越了妖族与巫族。
<br/>　　深吸一口气，帝俊眼中那无与伦比的杀机宛若实质，让人看之心寒无比。他刚想要提剑杀上去，不料，别人却已经找上他了。
<br/>　　“久仰妖帝大名，今日后辈人族斗胆前来挑战，还不忘吝惜赐教！”
