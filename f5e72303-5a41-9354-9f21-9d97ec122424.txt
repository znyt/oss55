　　他们这组的比赛除了一个皇族势力强一点之外，其他战队都是属于国内的三四线梯队，所以叶帝他们还是很轻松的。
<br/>　　回到家的时候蒋念他们都在比赛室内训练，对于今天叶帝他们的这场比赛几人倒是没有关注，谁要是告诉他们白队打不过EG战队的话，蒋念他们绝对会以为这是愚人节。
<br/>　　世界排名前3的队伍，遇到一个实力差距悬殊的队伍，在分配出这个阵容的时候，结果就早就已经定好了。
<br/>　　蒋念他们之所以这么认真的训练，完全是因为他们下一周对战的目标是雷霆战队，雷霆战队虽然整体实力不如他们，但是实力还是有的。
<br/>　　几人可不想阴沟里翻船，之前在bo3的比赛中，也输给过对方一局，所以他们自然是不想再有那样的事情发生，尤其是在bo5的对决中，第一场比赛如果输了的话，那对于队员的心态影响是非常的大的。
<br/>　　当然了只要他们赢下和雷霆战队的对决，那么接下来的比赛就要轻松多了，B组另外两支战队的比赛是王朝战队和AG，无论两支战队哪一支战队晋级，对于他们的影响都不会很大。
<br/>　　回到家的时候已经是差不多快五点钟了，中午几人都没敢吃的太饱，所以这个时候都差不多有点饿了。
<br/>　　在比赛之前大家吃饭其实都是吃的特别少的那种，大家经常在长时间的对决中看到队员们吃巧克力就是因为他们需要补充体力。
<br/>　　至于为什么吃的少，完全是因为如果吃的太多了，对于队员的发挥什么的都有很大的影响，而且吃的多还容易犯困，如果在场上一个注意力不集中，那可能就引发出非常大的后果的。
<br/>　　叶颖自然是知道他们的这个状况，所以这边吩咐厨房的阿姨赶着点做饭的，在看到叶帝他们的比赛结束后，他们这边就已经开始了，当叶帝他们回到基地的时候，饭菜都已经好的差不多了。
<br/>　　“蒋念，你们几个别玩了，赶紧过来吃饭。”叶颖对着比赛室内的蒋念一行人喊道。
<br/>　　“马上，这一局马上就要结束了，你们等一会在吃！”蒋念大声的喊道，闻到餐厅的饭菜味，这个时候他也是有些食指大动，生怕叶帝几人将菜抢光，连忙开口喊道。
<br/>　　大概过了5分钟左右的时间，蒋念一行人才兴奋的从训练室内走了出来。
<br/>　　“和谁打的训练赛，这么开心？”看着几人的表情，叶帝疑惑的问道。
<br/>　　“魔术师他们，最后被我们给虐了，当然开心了！”蒋念笑着说道。
<br/>　　刚刚他们是在和ST战队的队员打训练赛，最终他们以一波巧妙的运营成功破开了对方的大门，所以几人自然是十分的开心。
<br/>　　赢ST这样的顶尖强队和赢普攻的队伍，那感觉是完全不同的。
<br/>　　对战ST，他们心中也是没有多少胜算的，但是对战那些普攻的队伍，他们心中却是抱着必胜的信念的，两种心态，收获胜利之后自然是两种不同的心情。
<br/>　　“那确实值得开心一下，ST现在的实力比之春季赛强了很多啊，尤其是他们那个新打野小花生加盟后，中野联动就算是比之我也夜雪也差不了多少。”叶帝点头说道。
<br/>　　“没错，也不知道他们是从哪找来的这个打野，实力真的是太强了，应该是最近崛起的，如果是之前，那么就算对方上不了十王者，那么也应该在大师组的前列的，我们不能听都没听说过。”秦霄跟着开口说道。
<br/>　　“加上现在还有emp、god这些当初的十王者战队，以及新涌现出来的小石兄弟的山口战队，其实这届联赛还是有些难度的，大家千万不要大意，这些人的实力很强，一个大意，咱们可能就要阴沟里翻船了。”李景禹开口嘱托道。
<br/>　　“没错，对战这些战队都要拿出12分的实力来，官方将咱们三支战队排到了世界前3，还有有很多战队不服气的，到时候说不定他们就要过来找回场子呢。”叶帝点头应道。
<br/>　　吃过晚餐后，众人在客厅中休息了大概半个小时左右，便是来到训练室里继续训练，而小意和清浅两人这一段时间也都是在双排上着分，并没有参与到战队中的事情来。
<br/>　　知道这是叶帝还有秦霄他们的最后一次世界赛，两人虽然年纪小，但是还是很懂事的，他们自然是不会去剥夺这属于他们的最后的时间的。
<br/>　　而且叶帝之前也说过了，进入世界赛的时候，会给他们一些磨炼的机会的，让他们有一些大赛的经验，这样的话明年再次进入到世界赛的时候，他们也不会紧张。
<br/>　　而两人这一段时间的双排已经是打到了世界区大师组86名和45名的位置，一路上以百分之八十五的胜率上来，倒是惊到了不少的数据分析师。
<br/>　　甚至国外的许多专家还高呼，千万不要让这两人在一个战队里，要不然说不得又要出现当初的荣耀传奇，当初那个碾压世界无敌手的荣耀传奇。
<br/>　　而就在他们聊天的时候，叶帝的手机却是突然响了起来，看着来电显示，叶帝却是有些好奇。
<br/>　　“怎么了北北。”接通电话后，叶帝好奇的问道，北北是OG的领队，一个非常可爱的女孩子。
<br/>　　现在OG战队已经是荣耀旗下了，所以双方自然是存着对方的电话号码，不过北北给自己打电话的次数却是屈指可数。
<br/>　　“哥，我们战队之前参加了一个线下赛，不过下一次遇到的对手是EG战队，大家的心里都没有什么底气，想让你帮着打打气。”北北开口说道。
<br/>　　所谓的线下赛其实就是官方组织的其他赛事，类似当初地球上的德玛西亚杯这种，只不过这一次的线下赛荣耀蓝白两队没有参加，而也正是如此，才给了这些战队夺冠的心思。
<br/>　　那些没有跻身季后赛的队伍，一个个都将目光放到了那里，希望拿下这个冠军。
<br/>　　无论赛事大小，只要能够拿下冠军，那就是一个荣誉，对于他们之后的转会什么的，都是非常关键的。
