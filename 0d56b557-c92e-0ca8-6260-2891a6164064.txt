您如果以为我刚才所说的只是在吓唬你，那可是打错特错了。”<br/>　　<br/>　　    陆伊川呵呵笑了起来，看着傅华说：“傅主任啊，你别在我面前耍什么花腔了。我陆伊川虽然不能说吃的盐你吃的米多，但起码也是你痴长几岁的。商场的这些花样我玩的你溜。你不是要找什么霍处长吗？你现在找啊？”<br/>　　<br/>　　    傅华笑了，看着陆伊川的眼睛说：“怎么，陆董以为我说找霍处长是在骗你？”<br/>　　<br/>　　    陆伊川笑了起来，说：“不是我以为，而是你根本是在骗我。你以为事先打听出了香港警务处的一个处长的名字，可以拿来唬人的啊？你如果说是真的认识这个霍处长，你现在当着我的面给你说的这个霍处长打电话试试。”<br/>　　<br/>　　    傅华笑了，说：“陆董啊，看来今天你还非要跟我叫这个版了。”<br/>　　<br/>　　    陆伊川点了一下头，说：“对，我非跟你叫这个板不可。你打啊。”<br/>　　<br/>　　    傅华转头看了一眼邵依玲，说：“邵副市长，您可是听到了，这个可是陆董同意我们按照程序去走的，不是说我没给陆董留余地。”<br/>　　<br/>　　    邵依玲看了一下傅华，她是有些担心傅华并不真的跟这个霍处长有联系，而只是在跟陆伊川演戏，笑了一下说：“傅主任，陆董这么说是跟你看玩笑的……”<br/>　　<br/>　　    “别，”陆伊川打断了邵依玲的话，说，“邵副市长，今天您这个好人还真是做不成的，我没开跟傅主任玩笑，我说的是真的。你叫他马跟霍处长联系。如果他能跟霍处长联系，我陆伊川算是输了。”<br/>　　<br/>　　    傅华看了一眼陆伊川，笑了笑说：“陆董啊，这你可要把后果想清楚了，我今天如果打了这个电话，您的出路可只剩下了两条了，一条是主动跟我们回海川市去解决问题，另一条是从这里回去赶紧逃离香港，否则你只能等着坐牢了。”<br/>　　<br/>　　    “我不用想了，”陆伊川说，“如果你真的打通了这个电话，我陆伊川二话不说跟你回海川。”<br/>　　<br/>　　    傅华笑了，说：“这可是你说的。那我可打电话了？”<br/>　　<br/>　　    “是我说的，”陆伊川叫道，“你打啊。我不信你真的跟霍处长有联系。”<br/>　　<br/>　　    傅华拿出电话来拨了号码，在按拨出键之前，他看了陆伊川一眼，笑着说：“陆董啊，您现在后悔还来得及。”<br/>　　<br/>　　    “我绝对不会后悔的，后悔是你孙子。”陆伊川冷笑了一声，说，“可如果你无法联系这个霍处长，那你们也别在香港这里跟我耗了，赶紧给我滚回海川去。”<br/>　　<br/>　　    “行，那我们一言为定了。”傅华说完，拨通了霍处长的电话，为了能够让陆伊川听清楚，在拨通的时候，他还按了免提键。<br/>　　<br/>　　    几声手机铃声响过之后，电话被接通了，一个年男人的声音传了过来：“诶，您好，您是哪位？”<br/>　　<br/>　　    听到声音，陆伊川的神色微微变了一下，因为对方说的是字正腔圆的粤语普通话，这表示跟傅华通话的这个男人有八成以是香港人，那傅华刚才说的话有几分可能是真的了。<br/>　　<br/>　　    傅华笑了笑说：“您是霍处长吧，我朋友说可以打这个电话找您的。”<br/>　　<br/>　　    “哦，”霍处长笑了笑说，“您是傅先生吧？”<br/>　　<br/>　　    傅华笑了笑说：“是，是我。”<br/>　　<br/>　　    霍处长笑了笑说：“万博副总队长跟我说过您的事情了，说是您想要从香港这边引渡一个人回大陆受审。”<br/>　　<br/>　　    傅华笑了笑说：“对对，是这样子的，有一位香港的商人在我们那里涉及到一宗刑事案子，给我们造成了很大的经济损失，现在他躲在香港，不肯去大陆解决问题。所以我想请您协助一下我们。”<br/>　　<br/>　　    “好说，好说。”霍处长笑了笑说：“现在大陆和香港是一家人了，只要您提供出足够的证据，说明这位商人涉案了，那我们香港警方是会积极协助的。”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
