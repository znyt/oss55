　　第279章
<br/>　　地牢内。
<br/>　　云中龙五花大绑坐在椅子上，火盆烙铁，小皮鞭辣椒水已经准备齐全。
<br/>　　灯火昏暗处露出程大雷一张脸，他垂首将热茶的浮沫轻轻吹去。
<br/>　　“聊聊吧，关于你究竟如何变成现在的，我们是奉行人道，如果你不愿意了解人道主义，那么你就会体会惨无人道。”
<br/>　　“狗贼，有什么刑具尽管给我上！老子要是皱皱眉头，就是你养的。”
<br/>　　“文明用语，文明用语。”程大雷摆摆手：“我就是和你聊点关于莫鸣米的事，问你几个问题，你好好回答的话，我们就会优待俘虏。”
<br/>　　“你打听莫老大做什么，难道想和莫老大做对。哈哈哈，你算哪根葱哪根蒜，和莫老大做对，你们就不怕死么！”
<br/>　　“喔，忘了自我介绍，某家姓程名大雷，大家认识一下。”
<br/>　　“什么，你是谁？”云中龙一惊，嘴巴张得老大。
<br/>　　“程大雷啊，怎么了，你是要吃人么？”
<br/>　　“难道就是那在长安城刺杀杨龙停，威震草原的蛤蟆大王程大雷。”
<br/>　　“呃……哈哈，我这么大名气么，哈哈。”
<br/>　　“不对。”云中龙摇摇头：“据说那蛤蟆大王风姿潇洒，是一等一的俊品人物，可你看上去为何平平无奇。”
<br/>　　“这个……大概是每个人对风姿潇洒的理解不同吧。”
<br/>　　“真的是您老人家……”云中龙眼眶一红：“我们盼星星盼月亮，千盼万盼终于将您老人家盼来了。”
<br/>　　“……”程大雷。
<br/>　　“您是不是想对姓莫的动手，嘿，我早就看姓莫的不顺眼了，您想问什么，关于莫鸣米的我都知道。”
<br/>　　云中龙的表现和刚才截然相反，竹筒倒豆子般把知道的都说了。程大雷重点询问了附近的势力分布，像云中龙这样的路霸有多少，每个人有多少手下，其中又有那些是厉害的。莫鸣米知无不言，言无不尽，甚至把每个人有多少小老婆的事情都和程大雷说了。
<br/>　　“程当家，有个人不得不防，他叫窦竹仝，是莫鸣米的表妹夫，仗着和莫鸣米有亲戚关系，在这一带作威作福，您可得好好收拾他。”
<br/>　　窦竹仝……程大雷将这个名字记在心里，之后离开了地牢。
<br/>　　从云中龙这里知道很多情报，但也不能完全相信他，程大雷还要派出人去核实，看窦竹仝的话究竟有多少可信度。
<br/>　　目前要做的还是将山寨建设起来，只不过山寨只有一百人，纵然有建筑精通的技能加成，工程进度也快不了。
<br/>　　而在打探情报的过程中，程大雷已经得知，莫鸣米下了命令，以地盘为诱惑悬赏自己的人头。
<br/>　　十八家校尉，除了云中龙外都是蠢蠢欲动，蛤蟆岛顿时有种山雨欲来风满楼的气氛。
<br/>　　……
<br/>　　黄牛村，是大河之畔的一座渔村，这里的村民都以打鱼为生。
<br/>　　黄牛村渡口，有间豆腐店，豆腐店老板娘是个名叫吕春花的寡妇，小店的客人每天都是络绎不绝。只不过，虽然客人很多，但也没有村里那个闲汉恶霸敢打吕春花主意。这倒不是说老板娘面目丑陋，事实上，吕春花虽然年逾三十，依旧风情万种，丰腴的身子比新鲜豆腐还要水嫩。
<br/>　　真相是，大家都知道吕春花背后有人，她背后杵着那个男人可以令所有人望而却步。
<br/>　　白日，豆腐店却关着门，后院堂屋隐隐有了动静。
<br/>　　一具白嫩的身子从床上下来，身上只披着肚兜，鸳鸯戏水鲜红翠绿。
<br/>　　她坐在窗前，看着窗外的绿枝的石榴树发呆。
<br/>　　“你这死人……许久不来一趟，来了就要走，从来不肯在我这里过夜。”
<br/>　　“忙，忙嘛。”窦竹仝一边穿衣服一边说。
<br/>　　“三五日来一趟，把我这里当作快活场，你可知道村里的人怎么说我。”
<br/>　　“说就由他们说，闲话还能挡了吃饭睡觉。”窦竹仝穿上靴子，从床上坐起。
<br/>　　吕春华过来把他系扣子，嘴里哀怨道：“闲话是没传到你耳朵里，你是不知道人言可畏的可怕。”
<br/>　　“好啦，等我忙完这件事，就把你接过去，到时候看谁还敢说你。”
<br/>　　“嗳，你这话说了多少次，可有那一次算数，我知道你家里那头母老虎可怕，也不难为你。”
<br/>　　窦竹仝看着面前的吕春花，安禄山之爪攀上了她的身子，道：“好一个可人儿，知道你心里疼我，等我忙完这阵儿，一定把你接回家去。”
<br/>　　“好啦，你要走就走，别折腾得我不上不下。”吕春花满脸羞红：“你回去的路上慢点，我给你做了件袍子，等你下次过来就可以做好了。”
<br/>　　窦竹仝在吕春花腰上你捏了一把，之后推开门，骑着高头大马扬长离去。
<br/>　　窦竹仝的山寨就在附近，云中龙把着陆路，而他是水路的贼，当然贼现在穿上官服，也是在朝廷有档案的校尉。
<br/>　　山寨有五六百弟兄，凡是水上过的货都要给它交过路费，山寨吃得还不错。只不过家有妒妇，总是说些：如果不是我表哥你如何如何……
<br/>　　这次莫鸣米的命令已经传过来了，窦竹仝想要抓住这个露脸的机会，振一振夫纲。
<br/>　　这几日都在筹备这件事，等到这一日，窦竹仝把山寨弟兄点起，一共五百人，都是通水性的好汉。点齐十艘大船，在黄昏出发，逆流而上向蛤蟆岛出发。
<br/>　　在所有人中，窦竹仝是第一个向蛤蟆岛发动进攻的。
<br/>　　这几日，程大雷的精神绷得像弓一样，到这一日该来的终于来了。
<br/>　　窦竹仝本来想发动夜袭，来一次攻其不备。可程大雷早就提防着，当窦竹仝攻到的时候，黑暗的岛屿突然亮起篝火。
<br/>　　程大雷站在最高处，口中喝道：“是窦竹仝窦将军么，我等兄弟初来乍到，窦将军准备这样来看我么？”
<br/>　　窦竹仝站在船头，不想被程大雷察觉，他冲着岸边喊道：
<br/>　　“姓程的，我知道你是小河里的大鱼，但这里是大海，容不了你翻浪。”
<br/>　　“废什么话，这里才是河。”程大雷大喝道：“姓窦的，你看看这是谁！”
<br/>　　手下推着一人过来，在火光映衬下，窦竹仝看得清清楚楚，正是黄牛村的吕春花。
