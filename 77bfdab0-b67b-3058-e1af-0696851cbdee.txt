　　此时海盗船上，
<br/>　　几名凶神恶煞的海盗，已经将庄菲菲的衣裙撕的稀巴烂，把她死死的按在了地上，让她丝毫都动弹不得，
<br/>　　“放开我！你们这群混蛋！你们就不怕叶尘回来杀了你们吗？”
<br/>　　庄菲菲一边拼命的挣扎，一边大声呼喊起来。
<br/>　　那黑人首领满脸狰狞之色，
<br/>　　“哼！那小子虽然承诺不杀我们，可是一定会把我们交给华夏海军。”
<br/>　　“落入华夏军方手中，我们肯定必死无疑，既然如此，那老子在临死之前，不如好好再享受一把！”
<br/>　　“兄弟们，把她给我按好，老子玩完之后，你们人人有份！”
<br/>　　原来，
<br/>　　黑人首领料想，即便叶尘放过了他们，一旦落入华夏海军的手里，以他们平日里的罪孽，枪毙十次都绰绰有余了。

<br/>　　“咔嚓~”
<br/>　　黑人首领一把抓住庄菲菲的内衣，直接便撕扯了下来，那雪白如凝脂的皮肤，以及最隐蔽的部位，顿时便彻底展露在众人面前。
<br/>　　“啊！！”
<br/>　　庄菲菲顿时嘶吼了一声，满脸的惶恐之色，继而忍不住大吼了起来，
<br/>　　“叶尘，救我！快来救我！”
<br/>　　黑人首领顿时桀桀一笑，
<br/>　　“小美人，我劝你还是省点力气吧！我们这艘海盗船，可是经过特殊改装，用的是核动力，速度比一般的巡洋舰还要快将近一倍！那小子就算再厉害，难道还真能追上来救你不成？”
<br/>　　黑人首领说着，已经脱下了自己的裤子，便要朝庄菲菲的身上凑去……
<br/>　　然而就在这时，
<br/>　　忽然一道冰冷的声音，陡然在船上响起，
<br/>　　“你真的以为，你们能逃出我的手掌心？”
<br/>　　那黑人首领听到这话，顿时浑身一颤，瞬间就萎了下去，哆里哆嗦的回过头，
<br/>　　那道他此生都不愿意再看到的身影，不知道何时，已经如同鬼魅般出现在他的面前。
<br/>　　“不！这怎么可能！”
<br/>　　先不说海盗船和华夏军舰相隔近十里，他这艘船经过改造之后，速度已经接近40节，比核动力潜艇还要快！
<br/>　　他怎么也没有想到，居然这么快就被对方追了上来。
<br/>　　眼看着叶尘缓缓走了过来，
<br/>　　黑人首领顿时大吼了一声，闪电般拔出腰间的手枪，对准地上的庄菲菲，
<br/>　　“你、你不要过来！否则我就一枪崩了她！”
<br/>　　叶尘冷冷一笑，嘴角露出一抹嘲弄之色，
<br/>　　“在我的面前，你以为你还有出手的机会吗？”
<br/>　　说到这里，叶尘双眸微微一凝，
<br/>　　黑人首领顿时感觉自己的手臂，居然完全不受控制了，转而居然将枪口顶进自己的嘴巴之中，
<br/>　　“呜呜~~”
<br/>　　黑人首领顿时又惊又骇，偏偏嘴巴又被自己用枪堵住，根本无法说话，只能拼命的发出呜呜的声音，急的一身冷汗。
<br/>　　然而，根本无济于事！
<br/>　　“砰！”
<br/>　　一道枪鸣声响起，子弹顿时穿透了黑人首领的脑袋，整个头颅都几乎炸裂了开来。

<br/>　　可惜，
<br/>　　叶尘才懒得管他们说什么，轻叹一声道：
<br/>　　“我给过你们机会，是你们自寻死路，这可怪不得我！”
<br/>　　说完这话，叶尘直接一道神念发出，
<br/>　　噗噗噗！
<br/>　　数十米海盗，顿时全都软绵绵的倒了下去，个个两眼瞪得老大，至死都不知道他们是怎么死的。

<br/>　　庄菲菲两眼呆滞，过来好一会，方才反应了过来，顿时扑在叶尘怀里大哭起来。
<br/>　　……
<br/>　　等到李军洋把人质全部安顿了下来，庄菲菲依旧死死趴在叶尘的怀里，死活不愿意离开。
<br/>　　说实话，叶尘和庄菲菲并无太深的交情，不过当初若不是因为她，叶尘也不可能找到曦月的残魂。

<br/>　　因为叶尘着急回去，李军洋亲自安排了飞机，庄菲菲和他的经纪人红姐，也一同随叶尘返回华夏，至于其他人质则安排乘坐游轮回国。

<br/>　　“走吧！我陪你走一趟！”
