　　第383章
<br/>　　希曼的工作间，程大雷用镊子从药水中夹起一张青色的纸。
<br/>　　“这种纸是什么做出来的？”
<br/>　　“是采了大河中的一种水草造志，又浸泡过特制的药水。”希曼道：“如此可以很长时间不褪色。”
<br/>　　“别人能够仿制出来么？”程大雷问。
<br/>　　“大当家你来看。”
<br/>　　希曼夹起一张风干的纸，冲着太阳一照，本来青色的纸显出五彩斑斓的颜色，绚烂如彩虹。程大雷眼睛一亮，纸张显出这种特性，他以前莫说见过，听都不曾听过。
<br/>　　“成本高么？”程大雷。
<br/>　　“是用几种植物的汁液配的。原材料本地就有，可以就地取材。而颜料的配方是我们独有的。”
<br/>　　程大雷点点头，把事情交给希曼果然是一个正确的决定。这还不到三天，他们就拿出了可行的方案。当然，关于纸张的制造是山寨本有的技术，但颜料是希曼和托尔拿出来的。
<br/>　　凯恩将程大雷引到另一边，这是已经制好的几版样钞，但最终样式还是要程大雷来定。
<br/>　　一共七版，包括大小面额，图案却是大同小异，有飞天蛤蟆，有吞月蛤蟆，有水中蛤蟆……这图案是程大雷出的思路，冒险家西泽所画，不写意，只写实，惟妙惟肖。
<br/>　　程大雷瞬间陷入蛤蟆的海洋中，他定了一套吞月蛤蟆的图案，道：“就这套了，反正都差不太多，拿下去印出来吧。”
<br/>　　很快，琴川关的钱莊就正式开业。开始向外兑换纸钞，任何人可以把金银铜钱存在银行，兑换成纸钞，也能随时取出。
<br/>　　包括山寨兄弟的晌银，山寨内商品交易都用纸钞代替。
<br/>　　结果……
<br/>　　与和珅预料得一模一样，几乎没什么用。
<br/>　　平民还是习惯把银子拿在自己手中，何况他们手中也没多少积蓄，在他们的认知中，很难理解轻飘飘一张纸可以代替真金白银。
<br/>　　程大雷自然可以用强命以纸钞代替金银，但现在琴川关正是立牌坊的时候，牛不喝水强摁头，最后只能适得其反。
<br/>　　因为这次挫败，程大雷心情有些失落，整日唉声叹气，现在的他和刘悲差不多，大概也需要诸神的拯救了。
<br/>　　和珅还特意过来安慰程大雷，说些什么：事情不急于一时，慢慢总会想到办法的。大家现在的日子目前过来也算是不错么。实在不可以，先铸些铜钱也是可以的。
<br/>　　这话没能给程大雷带来安慰，反而狠狠地刺激了他。看着满屋子青色纸钞，程大雷就无可奈何的叹气，当真是擦屁股都嫌硬呐。
<br/>　　和珅无法理解程大雷，目前的琴川关的确称得上人强马壮，已经是西北一股不容小视的势力。但，也仅仅是如此罢了。想要在这个乱世中安身立命，凭目前这些是远远不够的，所以程大雷才希望将一级城池开出来。
<br/>　　然而万事万物都有其内在规律，某些超前的东西拿到这个时代，却也未必会适合。经过这件事，程大雷算是受了一次教训。
<br/>　　……
<br/>　　光阴荏苒，便又到这一年的秋天，屈指算来，程大雷这已经是程大雷到西北的第三个年头。
<br/>　　如今的程大雷，手里有三万精兵，八百里琴川地，有五六万流民安居乐业。商队往来不休，一切井井有条，程大雷也算是做了甩手掌柜。
<br/>　　只不过，程大雷还没升到一级城池，最主要原因是人口不足。
<br/>　　这一日，秋高气爽，程大雷正百无聊赖的在屋檐下晒太阳。心里想的，还是怎么把纸钞发行下去。这件事不解决，始终是程大雷的心结，让他无法安心。
<br/>　　“大当家。”和珅今天特意过来：“有伙人来投奔你。”
<br/>　　“你交给刘悲，看他做什么安排，这样的小事，不用来烦我吧。”程大雷端起茶碗抿了口茶，如今山寨已经走上正轨，这些事本不需要他处理。
<br/>　　“如果是一般人，自然不会来烦大当家，只不过这伙人说和大当家认识。”
<br/>　　“是我认识的，那会是谁呢？”程大雷心头有些好奇，反正今天也正好无事，他站起身道：“走，出去看看。”
<br/>　　还未走进，就听得山寨广场上乱哄哄一片。
<br/>　　“你们是不知道我们和老大是什么关系！”
<br/>　　“再敢对我们无礼，小心我们翻脸。”
<br/>　　“如果不是看老大的面子，你以为凭着你们几个，就能制得住我们嘛。”
<br/>　　程大雷循着声音靠近，待看清这些人时，猛地吓了一跳。
<br/>　　人还真不少，约有上百位，都是衣衫褴褛，像那路边乞讨的乞丐。关键是这些人每个还穷横穷横的，张牙舞爪，狰狞凶恶。
<br/>　　见到程大雷出现，上百人同时起身，同时高呼。
<br/>　　“大哥！”
<br/>　　和珅也吓了一跳，这些『乞丐』还真的和程大雷认识呐。
<br/>　　飞天蛤蟆白元飞、阿喜、阿悲……的确都是当年程大雷在长安城时结交下来的弟兄。他们来找程大雷，程大雷不奇怪，可他们如此落魄，程大雷真的有些不敢相信了。
<br/>　　“你们……怎么会变成这个样子？”程大雷失声问。
<br/>　　“看老大这个样子，又是拽了起来。”
<br/>　　“老大不愧是老大，到哪里都是吃独份喝独份儿。”
<br/>　　“我早说过，来投奔老大总是没错的。”
<br/>　　“好啦，好啦。”程大雷摆摆手：“拍马屁的话就不要说了，你们是怎么过来的，我记得当初留给你们的银子可不少。”
<br/>　　程大雷目光扫了一遍，虽然中间隔着三年，可每个人的相貌，还都是那样熟悉。他表情忽然一怔，呐呐道：“少了一些人，破落户杜达，秃子阿九……这些人怎么不来。”
<br/>　　当年程大雷在长安城，也有一干弟兄，后来他要到西北来，有些人拖家带口，也不方便跟着程大雷，于是这些人就留在长安城。
<br/>　　今日再重逢，见到很多过去的老伙计，但也缺了许多人。如果要来该是一块儿来，怎么只来了一百多位。
<br/>　　程大雷一句话问出口，乱哄哄的人群安静下来，大家目光闪烁，都像是在躲避着什么。
<br/>　　“老大，长安城出事了。”阿喜。
<br/>
