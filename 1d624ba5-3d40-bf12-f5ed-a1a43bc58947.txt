　　“起阵”
<br/>　　突然，纣王一声大喝。元始并没有阻止他们的行动，他心中亦想看看，这纣王有何手段可以对付他。
<br/>　　朝歌城东方，敖无天突然大喝一声，道：“我主黄泉！”
<br/>　　东方之地上，突然从虚空之中，隐现一条巨大的黄色浑浊长河，河面有无数怨魂沉浮，污浊不堪的气息弥漫到空间之中，让人闻之难受。黄泉之水既是圣水，同样是毒水，对于有用之人，即是圣水，对于一般之人来说，却是绝毒之水。
<br/>　　吕布脚踏大地，手中战戟一挥，插入中央大地一角，光华绽放，神秘的明亮纹路自中央开始向四周蔓延开来，刹那间便蔓延整片朝歌城，他大喝一声：“我主大地！”
<br/>　　诸葛亮手中羽扇一挥，南方虚空显化出一座大桥，桥上黄沙滚滚，桥下黄河滔滔，他轻笑道：“我主奈何！”
<br/>　　岳飞立于西方大地，手中长枪一指，一片血色花海开放，布满整片虚空，那是彼岸之花。
<br/>　　“我主彼岸！”
<br/>　　白发女子脸色冷酷，双手一翻，两柄弯刃握于手中，双刃一挥，一片死亡之海出现，她冷冷道：“我主死亡！”
<br/>　　“朕主天地，轮回起！”
<br/>　　纣王立于一祭坛之上，单手指天，沉声道。
<br/>　　瞬间，六道通天彻地的轮回通道拔地而起，笼罩四方，虚空上方，一镜、一塔、一鼎、一钟，定住天地四极，封天锁地。
<br/>　　此四件法宝分别是能够上探九霄，下通九幽，还能反射攻击的昊天镜；能够镇压妖魔的锁妖塔；能够炼化万物，熔炼药物的神农鼎；能够镇压诸天，定住时空，开启天界之门的东皇钟！
<br/>　　四件法宝元始都很熟悉，虽然这四件法宝是另一界的，受世界等级限制并没有洪荒的强大，但也不可小觑，毕竟每一件法宝都享有赫赫威名。
<br/>　　“六道轮回？时空大道？”
<br/>　　元始微微惊讶，自后土化轮回以来，洪荒之中，还没有谁能够执掌六道轮回，虽然元始分身三生大帝也悟出了六道轮回，但那其实好并不算真正的轮回大道，而此刻纣王借助大阵摆出的六道轮回，颇有几分真正轮回的样子。不过，此阵并不完全，元始何等眼光，一眼便看出，第六道还没彻底完善，导致此阵还有破绽。
<br/>　　一想到纣王是外界来人，元始心中也有些了然了，洪荒六道难悟，并不是其他世界轮回也难悟，不同世界法则构成不同，越完善的世界，强大法则越是难以领悟。也许纣王所在世界，轮回却是容易领悟的呢。
<br/>　　而且，除此之外，在此阵之中，元始还感受到了时空的气息，如此想来，纣王对时空一道也有领悟，这才是元始真正惊讶的地方，时空大道不论在哪个世界都是真正顶级的大道，纣王能够领悟此道，一点也不简单。
<br/>　　“没想到纣王竟有如此运道，领悟轮回与时空大道，真不简单。”
<br/>　　元始对着纣王夸赞一句，但随即便转变语气道：“不过，你以为凭借此残缺的时空衍生界，便想与本座对抗，不是本作看不起你，事实是这的确还不够看。”
<br/>　　“朕可不敢再小看天尊，方才小看了天尊，朕便已吃了大亏，如今可不想重蹈覆辙。”
<br/>　　纣王冷冷一笑，说着，他取出一块拳头大小灰白色的圆珠，将之扔起，那圆珠绽放出茫茫辉光，之后迅速变大，如同一个灰色大气球，最后越来越大，直至完全与朝歌城一般大小，融入到大阵之中，元始立刻便感受到其中的变化。
<br/>　　此刻的衍生界已经彻底被弥补，成为一方独立的时空，外界之力不能进入，而且整个朝歌城的空间都变得扭曲无比，朝歌已经与洪荒脱离了。
<br/>　　“怎么回事，朝歌城呢？怎么不见了？”外界，众多看守城都的将士们见朝歌眨眼间便消失不见，纷纷大惊。
<br/>　　“报——禀告武王，朝歌消失不见了。”有士兵来报。
<br/>　　“什么？不见了，你逗我呢？”武王此刻是一脸懵逼，整个朝歌如此之大，你却说不见了，这怎么可能。
<br/>　　不过，当他出来一看，的确是朝歌不见了，在之前朝歌城的地方，此刻是白茫茫一片，什么也看不见。
<br/>　　“难道是天尊出手将朝歌弄走了，但他又为什么要弄走呢？”武王不解道，不过，朝歌不见了，他的目的也达到了，只不过少了一座城，多少让他有些郁闷。
<br/>　　“空间规则世界？”
<br/>　　衍生界内，元始这是真正惊讶了，难怪能够领悟时空大道，有了这空间规则化成的世界，便相当于掌握了一部分空间规则，只需将之炼化，便能掌握其中的规则，领悟空间法则是最简单的。
<br/>　　“不错，天尊好眼力！”纣王道。
<br/>　　“有了这空间规则世界的确是对本座有些威胁，但这还不够。”元始依旧淡然无比，这纣王的底牌倒是有许多，若是对付一般圣人，也早已足够，但他并不知道元始是一位混元圆满的强者。
<br/>　　“够不够，只有战过才知道！”纣王冷哼一声，不再多语，站在祭坛之上，单手指天，操控六道大阵向着元始压将下来。
<br/>　　“也罢，既然纣王要战，那本座便奉陪一二。”元始轻笑一声，毫不理会压下的大阵，他一眼看去，那大阵便停了下来，不再落下。
<br/>　　“天尊果然厉害，那便试试东皇钟的攻击！”纣王也不意外，冷笑一声，操控上空的四件法宝向他打来。
<br/>　　“当——”
<br/>　　东皇钟鸣，吸收无尽星辰之力，变成一座巨大的天门，带着浓浓的威压，镇压诸天的气势，从天而落，将周边一切尽皆压碎，紧接着，锁妖塔变大，亦随之罩了下来。昊天镜反射出一道极昼之光，以迅雷不及掩耳之势射来，神农鼎倒悬于天，封锁天空。
<br/>　　元始目露不屑，一掌拍出，平平淡淡的一掌，看似毫无威力，但那一掌却将巨大的天界之门崩飞，电光火石之间，元始不停，又是一指点出，击在锁妖塔上。
<br/>　　“咔擦——”
<br/>　　一声脆响，那锁妖塔竟然出现裂缝，紧接着，哗啦一声，锁妖塔经受不住元始一指之威，竟直接爆裂开来，碎片如雨纷飞。
<br/>　　刹那间，那极昼之光射来，元始不急不缓，眼中已同样一道白光射出，与之相触，消散于无形。
<br/>　　这些只不过在眨眼都不到的时间之内完成，谁也不曾想到，会是如此一个结果。
<br/>　　“可恶，再来，空间错位！”
<br/>　　纣王控制大阵借助空间规则世界，直接在衍生界之中施展空间神通，轰隆隆间，那平时无形的空间，此刻竟然开始出现一层层的空间层，那一层又一层的空间不断错位开，向着元始延伸而来，若是一人站在那里，立刻会被这空间坍塌碾为无数段，最终变成血雾爆开。
<br/>　　元始目露精光，露出敢兴趣的神色，见那空间层不断向他压来，他脚步一抬，便踏上那空间层，之后随之那空间的不断错位，一步步地转换着空间层，待错位过去，元始便又重新立于原地，好似并没有动过半步。
<br/>　　“怎么可能？”
<br/>　　纣王惊叫道，脸上露出惊骇之色。空间错位的时间是非常迅速的，一息之间便会错位多少个亿万次，但元始却能在上面如履平地，这是何等可怕的眼力以及速度，这已经不是速度能够解释的了，他知道，元始同样精通空间法则或者是时间法则。
<br/>　　纣王不服，不知从何处取出一杆枪，那枪看似极为普通，感受不到丝毫的法则波动，但元始知道，这枪不简单。纣王一枪向前轰出，速度却极为缓慢。
<br/>　　“一枪——生世界！”
