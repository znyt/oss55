　　一场大朝会最后还是不欢而散了，李世民没有任何言语的直接回了含凉殿，李承乾尴尬的看着面下一群人，尤其是看上去似乎只剩一口气的魏征。
<br/>　　“殿下，老臣，老臣能做的只有这些了，只希望您不要忘记当初的誓言，一定要……”话说到一半，本就带着病的老魏双眼一翻，就此昏了过去。
<br/>　　“来人，快传太医！”李承乾眼看着老魏生死不知，哪里还有心思去管他刚刚说的是什么，一面招呼内侍去传太医，一面从御阶之上快步冲下来，将伏在地上的老魏托了起来，口中急叫道：“魏师，魏师醒醒！魏师……”
<br/>　　然而，此时的魏征脸上已经开始泛起一层青灰，眼见着人已经不行了。
<br/>　　事已至此，李承乾还能说什么呢，这老魏虽然平日时没少跟他对着干，这也不行那也不行，住个房子都不能过于奢华，马车又不能过于奢华，可是凭良心说，这老头儿所说的一切还真都是为了他李承乾好。
<br/>　　人啊，不可能没有私心，只不过有的人私心重一些，有的人私心轻一些。
<br/>　　老魏这个在李承乾看来，可以算得上大公无私，但临死时的这一举动属实让人有抓狂，但却不失一个父亲的职责，一个丈夫的职责。
<br/>　　这老头儿，用最后的一点力量，第一次也是最后一次为自己的子嗣留了一条后路。
<br/>　　从龙之功他魏征是享受不到第二次了，但是他的儿子还在，这老头子临死的时候估计一定希望自己能明白他的苦心，善待他的家人吧。
<br/>　　李承乾看着老魏的尸体被人抬着出去，心中如是想着。
<br/>　　……
<br/>　　含凉殿，大朝会上发生的事情早已经传了过来，长孙皇后急急忙忙赶来的时候，却李世民正在自己的书房之中拿着一本《道德经》看的津津有味。
<br/>　　“陛下，您，您没事吧？”长孙皇后有些担心的来到老李身边，不大却定的问道。
<br/>　　被逼着下台的滋味的确不怎么好受，老李应该不会受了什么刺激，一时想不开吧？
<br/>　　“朕能有什么事！”李二放下手里的书，拍了拍肩膀上皇孙皇后的手，有些感慨的说道：“朕早就知道会有这么一天，只是没想到会来的这么早而已。”
<br/>　　“那，乾儿他……”长孙皇后还是有些不放心，看了看李二，又看了看门外，发现没有李承乾的影子，这才稍稍有些放心。
<br/>　　“雏鸟大了，总是要单飞的，承乾能够把握朝局，能够让那些老鬼们服他，朕很满意！”
<br/>　　不管是瘦驴拉硬屎也好，倒驴不倒架也罢，总之李二此时表现的很是平静，丝毫没有被篡位之后应该有的那种愤怒。
<br/>　　长孙皇后正是因为如此，才有些担心，总是怕自己身边这两个最亲近的人会上演一出刀剑相向。
<br/>　　可是，让长孙皇后十分意外的是，就在她不知道如何劝说李二的时候，门外却传来了李承乾的声音：“父皇，孩儿承乾前来请罪！”
<br/>　　坏了！这是长孙皇后的第一个念头，一个现任皇帝，一个被众人捧出来的新皇帝，这两人一见面，那还不得天雷勾地火？
<br/>　　但是长孙皇后的担心明显有些多余了，只见老头子听到李承乾的声音之后，竟然没有任何发怒的表现，反而是一脸平静的说道：“进来吧！”
<br/>　　“喏！”李承乾的声音从外面传进来，接着人便缩头缩脑的走进了书房，还是一如继往的猥琐，还是一如继往的欠揍。
<br/>　　“朝堂上的事情处理完了？”李二看了一眼儿子，不动声色的问道，就好像刚刚什么事情都没有发生。
<br/>　　李承乾犹豫了一下，低声说道：“父皇，魏师……去了！”
<br/>　　“什么？”腾的一下，李二猛的站了起来：“你说什么？再说一次，魏征怎么了？”
<br/>　　虽然魏征刚刚在朝堂上带头谏言让他禅位，可是对于老魏身死的消息李二还是觉得无法接受，那可是他亲自封的‘人镜’，没想到竟然说没就没了。
<br/>　　“父皇节哀！”李承乾没有回答老头子的问题，但话里的意思已经确定了老魏死亡的消息。
<br/>　　“噗通”一声，李二重重的重回到了椅子上面，脸上泛起一抹悲痛之色：“没想到，真的没想到，朕的，朕的良臣竟然……”
<br/>　　“陛下，陛下节哀！”长孙皇后带头，房间中所有人全部单膝跪倒，齐声劝道。
<br/>　　要知道，除了老李渊死的时候，李二还从来没有如此痛苦过。
<br/>　　而现在先是在朝堂上被群臣建议禅位，接着又听说老臣病亡，双重打击之下，就算是铁人也受不了。
<br/>　　“唉！罢了，罢了！魏征啊魏征，你这是在用自己的老命在向朕最后谏言一次么！苍天啊！”
<br/>　　一声带着无限悲愤的怒吼从老李的口中传出，几可使闻着落泪。
<br/>　　一代帝王，为了一个想要推翻他统治的人，竟然有如此深厚的感情，不得不说，李世民的确是一个异类，又或者说他能够名动千古并不是没有原因。
<br/>　　……
<br/>　　但就算是李二的人格魅力再强，他的心志再坚定，他也还是一个人。
<br/>　　是人就有七情六欲，是人就有生老病死，所以在魏征带来的双重打击下，老李不出意外的病了，病的很重，神智不清，天天说胡话。
<br/>　　在这样的情况下，李承乾不得不再次监国。
<br/>　　只是与以前不同的是，这一次再也没有了任何的限制，再也不用向老头子去请示什么。
<br/>　　而事实上，他现在与皇帝也没有什么区别，只是没有经历过登基大典，老头子也还没有告诉他：“皇上你来干吧！”
<br/>　　满朝文武对于李承乾监国也并不意外，他们也全都在等消息，一是李二病故，二是李二身体恢复，然后禅位。
<br/>　　至于说老李继续当皇帝……，这个也行，法不责众嘛，只要老李好意思继续当的话，他们也无所谓。
<br/>　　反正该表态的时候他们都站出来了，李承乾将来就算是当了皇帝，也不可能有什么不良后果，最多就是大家继续该干啥还干啥！
