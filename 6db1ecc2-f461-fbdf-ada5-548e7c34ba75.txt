　　这冰漓长老，突然离开，应该便是接到了北邙学院高层的命令。
<br/>　　楚风眠如今展现出实力来，将这司空家的老祖司空景胜都给击败，学院高层应该已经知道。
<br/>　　所以才会如此匆忙的叫做冰漓长老。
<br/>　　只是这学院知道了，楚风眠也毫不担心，他现在的实力，已经是足以自保。
<br/>　　等到楚风眠度过了生死大劫，他的实力甚至足以可以超脱生死的地步，倒时候就算是北邙学院的高层来对付他，楚风眠都足以轻松化解。
<br/>　　“走，我们去武王城。”
<br/>　　楚风眠看了一眼旁边的寒月漓，直接开口道。
<br/>　　七日之后，便是武道大会开启的日子，虽然说楚风眠刚刚便是经历了连续的战斗。
<br/>　　不过楚风眠拥有神脉，这些灵力的消耗，对楚风眠来说便是没有任何一点的意义。
<br/>　　他必须要任何时间休整，便是已经将状态，达到完美。
<br/>　　“走。”
<br/>　　寒月漓也是点了点头。
<br/>　　两人的身形，顿时化为一道遁光离开。
<br/>　　武王城一路，路途遥远，提前前去，也免得赶不上。
<br/>　　这归墟天，每三年才会开启一次，楚风眠不想要错过这一次机会，错过这一次，楚风眠就要在等上三年。
<br/>　　楚风眠可没有三年时间可以来浪费。
<br/>　　两道遁光，直接离开北邙圣域，直接化为一道遁光，遁入到了虚空之中。
<br/>　　这北邙圣域，如今虽然极为隐秘，不过现在楚风眠已经是有着身份牌，知晓周围种种路线，就算是在这混乱的虚空之中，也一下子找到的通路。
<br/>　　一下子回到了北域之中。
<br/>　　这武王城，便是在北域的最北端。
<br/>　　武王城，不属于任何一方势力，但是却是由北域之中的无数宗门，合力维护的，其中北邙学院在武王城也有着一些权力。
<br/>　　当然像是地魔门，也依然占据这武王城权力的一角。
<br/>　　像是司空家，沧澜世家，风家这样的武道家族，还有一些二流势力，宗门都在其中拥有这一部分的势力，一起经营这武王城。
<br/>　　正是因为这个特殊性，武王城便是成为了每三年归墟天开启争夺名额的所在地，因此更是发展出了武斗大会。
<br/>　　每三年的武斗大会，便是北域年轻一代弟子排位，洗牌的一刻，每年都足以冒出来许许多多惊人的天才。
<br/>　　这已经不止是在争夺这归墟天的名额了，更是代表着武者的荣耀，北域天才，那一个不是天之骄子，人人都想要当第一，技压群雄。
<br/>　　所以这武斗大会，才会吸引那么多的强者都会前来。
<br/>　　“这武斗大会，其中遇到的所有弟子，都是可能随机的，所以就算是碰到同为北邙学院的弟子，也不能手下留情。”
<br/>　　寒月漓在一旁开口解释起来。
<br/>　　“武斗大会，便是跟生死台一样，在其中搏斗，都是生死搏斗，下台便是算输，只要是下台，便是不能再出手杀人。”
<br/>　　“但是在擂台上，无论什么手段，都可以动用，所以就算是碰到任何人，都不能手下留情。”
<br/>　　“恩。”
<br/>　　楚风眠点了点头，他也清楚，这种武斗大会上，每年每届出现伤亡，几乎是不可避免的一件事。
<br/>　　只有真正的生死之战，才能决定一尊武者的真正实力，所以这武斗大会每一场，都是生死之战，可以轰杀对方。
<br/>　　在上面，一个心软，只会给自己带来灭顶之灾。
<br/>　　不过楚风眠也不会在乎，以他的性格，本身便不会心慈手软，只要是敌人，在楚风眠的心中便是只有一个结果，便是轰杀。
<br/>　　武斗大会，说是为了归墟天的名额，但是实际上也是一场试炼，是各大宗门弟子之间的一场试炼。
<br/>　　一尊武者的境界在高，没有经历过无数生死之战，也不会成为真正的强者。
<br/>　　这武斗大会，正是给所有人一个机会，让北域的实力，从这一次次的残酷养蛊中提升。
<br/>　　三年一个周期，如今这三年，便是又到了。
<br/>　　楚风眠沉思之际，就在前方，突然出现了一大片山脉，这山脉中，布满巨树，一眼望去，楚风眠更是感觉到，其中妖气滔天。
<br/>　　“这里是荒尸山脉。”
<br/>　　楚风眠观察了一下周围地形，开口道。
<br/>　　这荒尸山脉曾经北域的一处战场，陨落了无数强者，无数妖兽吞噬了那些强者的血脉，在这里繁衍生息，已经是化为了一处妖兽聚集之地。
<br/>　　这其中，楚风眠感觉得到，有着不少强大的妖神，甚至是一些强大妖神的实力，都要接近超脱生死的地步，显然是妖神中的老祖宗。
<br/>　　“这荒尸山脉，危险重重，其中却也没有多少宝藏，是一处极为贫瘠的地方，进入其中也没有什么意义，我们还是快要绕路而行吧。”
<br/>　　寒月漓开口道。
<br/>　　荒尸山脉中，除了有着无穷无尽的妖兽之外，便是没有其他任何东西。
<br/>　　进入其中，也是在自找麻烦。
<br/>　　“现在想要绕开，似乎也有些晚了。”
<br/>　　楚风眠看着前方自言自语一声。
<br/>　　寒月漓顺着楚风眠的目光看过去，就在前方，突然便是出现了黑压压的一片。
<br/>　　如同乌云一般，直接冲着楚风眠这边飞了过来。
<br/>　　不过这种速度，可绝不是乌云，在其中，还蕴含有着恐怖至极的妖力。
<br/>　　“兽群！”
<br/>　　寒月漓看到这一片黑压压如同乌云般的飞来，都是脸色微微一变。
<br/>　　这兽群，绝对是北域之中遇到的最麻烦的事情之一。
<br/>　　一般兽群其中的妖兽，少说千只，万只，庞大的兽群甚至可能有着十万，百万只妖兽之多。
<br/>　　这种数量，让一尊超脱生死的强者来，都有可能陨落在其中。
<br/>　　眼前这兽群，无数妖兽，铺天盖地，足足是化为了一片乌云，其中的妖兽，至少有着十万之多。
<br/>　　“黑铁鸦？”
<br/>　　楚风眠展开灵识，仔细的打量了一眼，便是看到这妖兽，浑身漆黑，羽毛则是发出了一种金属般的光泽。
<br/>　　这黑铁鸦，是一种御风境的妖兽，成群成对，在妖兽族群中，不算强大，但是对武者来说，却是极为棘手。
<br/>　　因为黑铁鸦的羽毛，足足是有着精钢一般的坚硬，斩杀一只对任何一名圣者来说，都很简单。
<br/>　　可是黑铁鸦一出现，可就是成群数万，数十万一同出现，遮天蔽日，加上这黑铁鸦本身就很难斩杀。
