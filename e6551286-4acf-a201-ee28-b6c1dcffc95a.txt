　　将各自需要针对的英雄ban掉后，双方轮流选人，很快阵容便是确定了下来。
<br/>　　白队这边最终选择了：上单武器、打野雪人、中路蛇女、下路则是女警和娜美这对组合。
<br/>　　而对面则是选择了：上单大树、打野狮子狗、中路机械先驱、下路则是大嘴和布隆这两个英雄。
<br/>　　“他们这又是想跟咱们拼后期的节奏，这一局不能拖。”看着对方的阵容，叶帝第一时间部署战略，看到大嘴这个英雄的第一时间，他就已经知道了对方的想法了。
<br/>　　“找机会偷大龙，咱们这个阵容大龙很快的，蛇女加雪人，在加上一个武器，20分钟的时间完全可以很快的solo掉大龙的。”夜雪狂歌跟着开口说道。
<br/>　　而皇族这边此刻看到叶帝他们这样的阵容，他们也都在那里分析着，因为不是在赛场上，而且为了给几支战队加强难度，他们这边的教练也在场，而且还是现场指挥。
<br/>　　现在官方已经给出明确的答复了，明年就会加入教练战，所以皇族的教练也可以说是正在提前适应接下来的节奏，毕竟教练如果上场的话，那对于他们来说肯定是一个非常好的消息。
<br/>　　如果教练表现的好，那么他们完全可以像那些明星选手一样，成为一个明星教练，拥有着无数的粉丝。
<br/>　　“小龙、大龙做好视野，他们蛇女和雪人这个组合打龙的速度是非常快的，已经好久没看到雪人这个英雄打野了，这应该算的上是对面的一个小大招了。”皇族的教练开口说着。
<br/>　　“教练你感觉那个新英雄有机会在世界赛上出场没？”听着教练的话，楚白衣这边心思一动，开口问道。
<br/>　　“你说的是铸星龙王那个英雄？这个还真不了解，毕竟是在世界赛前才刚刚更新出来的英雄，大家还都不怎么了解，还是要等等才能知晓，不过我估计如果这个英雄真的要出场，那也应该是大魔王用。”教练开口说道。
<br/>　　每到世界赛前都会更新一个版本，而且还会出现一个新的英雄，这已经是这个世界上不成文的规定了。
<br/>　　火神这边也一直想给大家一种新奇的感觉，弄一个新英雄来提起大家的好奇心，不过大多数情况下，在不熟悉新英雄一般人是都不敢使用的。
<br/>　　而在他们谈论着这个英雄的时候，荣耀基地内，此刻叶帝等人在进入游戏后的闲暇时间，他们也都是在讨论着这个英雄。
<br/>　　“哥，那个铸星龙王是打中单的，这个英雄你感觉怎么样？”香蕉有些好奇的开口问道。
<br/>　　出于对叶帝的信任，每当出现新英雄的时候，众人都是会感觉叶帝肯定是了解的。
<br/>　　毕竟在之前的无数次出现新英雄的时候，叶帝都已经利用他的先知性征服了大家，如果一次那可能大家对他的信任还不这么大，但是每次都是这样的话，那就由不得他们不信任了。
<br/>　　所以在出现新英雄的时候，香蕉他们都是会问问叶帝的意见，之前的纳尔也是如此。
<br/>　　“适合打中单，等下次在打ST的时候我会用他，估计现在其他几个战队也都在练习那个英雄呢，小组赛之后，这个英雄应该会流行起来。”叶帝开口说道。
<br/>　　听着叶帝的话，众人的眼睛一下子亮了起来，既然叶帝这么说，那就代表着这个英雄是绝对可行的。
<br/>　　那么在接下来面对ST的时候，他们就相当于又放了一个大招，胜率的话肯定会提高的，只要不被ST追评，那么他们就会当之无愧的以小组第一的身份出线了。
<br/>　　之后的突围赛，排名在AB小组的第一位可是拥有着有限顺出位的，也就是说他们不用打比赛就会直接晋级到8强的名额，由剩下的10支队伍来争取这最后的6个名额。
<br/>　　少了几场比赛，那就说明了他们会拥有更多的休息时间，在连续大战的情况下，休息可是至关重要的，所以这也是为什么大家如此迫切的需要这个冠军的原因。
<br/>　　1分40秒，游戏正是开始，这一局白队这边是在紫色方，而皇族则是在蓝色方。
<br/>　　香蕉一级直接蛤蟆先手，用q技能吃了一口蛤蟆，又释放了一次惩戒之后，他便是快速的向着自己的蓝buff走去，将蛤蟆的经验让给了自家的下路组合。
<br/>　　这也算的上是他们的一个套路了，虽然现在的版本让下路单吃野怪的经验并不会让他们提前到达二级，不过毕竟会在经验上取得领先，这样的话在接下来的抢二中，他们的下路肯定会占据优势的。
<br/>　　而且这一点其实对方并不知道，也就是说他们还可以打一个出其不意，如果换成一个强势的下路组合，就算是逼闪或者是杀人，那也是很可能发生的事情的。
<br/>　　最重要的是雪人这个英雄本身打野其实是并不伤的，q技能不仅有着高额的对野怪的伤害，而且还附带着回血，所以说这个英雄打野的速度还是非常的快的。
<br/>　　而在这一局的游戏当中，为了创造机会20分钟过去打龙，夜雪狂歌是打算玩主q的这个套路的，毕竟q技能的等级越高，他对野怪造成的伤害也就会越高。
<br/>　　果然当萝卜和茄子两人到达二级的时候，司马贼他们并没有想到，直接吃了一个大亏，被破交出了闪现，这个版本吃野怪并不是击杀野怪的人独享经验的，依旧是两人共享，所以萝卜他们两个才会同时到达二级。
<br/>　　“我去，他们怎么这么快就到2了？”飞鸽一脸懵逼。
<br/>　　“雪人将经验让给下路吃了。”听着飞鸽的疑问，早在下路发生战斗的时候就将注意力放到这里的教练，第一时间给出了解释。
<br/>　　虽然没有见过这个套路，但是皇族的教练还是保持了自己超高的职业水平，第一时间给出了准确的分析。
<br/>　　“这个真的是太脏了，幸亏他们的下路前期并不强势，要不然的话这一波可能就要出现人头了。”司马贼有些心有余悸的说道，想想刚才发生的那波战斗，他还是有些后怕。
<br/>　　而这边在看到自己两人逼闪对方ad，茄子也是第一时间在屏幕上打字说出了大嘴下一发闪现转好的时间。
