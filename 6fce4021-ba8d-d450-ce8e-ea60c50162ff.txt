        麻烦？！
<br/>
<br/>        他说她是在给他惹麻烦。
<br/>
<br/>        从什么时候起，她成了他眼中的麻烦呢？
<br/>
<br/>        童筱颖顿时心塞得一句话也说不上来。
<br/>
<br/>        面对傅夜沉一个又一个的质问，她选择了沉默。
<br/>
<br/>        而她的沉默，于他来说，并不是一件好事。
<br/>
<br/>        傅夜沉拉开了跑车上副驾驶座的车门，童筱颖却愣在原地，转开了话题：“我的身份证和护照以及签证重新补办过来，大概需要多久？”
<br/>
<br/>        “半个月。”傅夜沉很没好气地说。
<br/>
<br/>        童筱颖不敢抬眸看傅夜沉的俊脸，只是将目光移到一旁，讷讷地接着说道：“那你能帮我在这镇上的宾馆里开半个月的房吗？我等到证件了，就回去。”
<br/>
<br/>        “有家给你住，你不住，你偏偏要住宾馆？好！很好！你想怎么着就怎么着吧！”傅夜沉恼火地关上副驾驶座的车门，转过身去踱步往镇中心走去。
<br/>
<br/>        他给她在镇上找了一家最好最安全的宾馆，给她开了房，给她塞了银行卡，然后二话不说地离开。
<br/>
<br/>        她没有主动挽留他，就连他自己也没有要留下来的意思。
<br/>
<br/>        其实，这一刻，童筱颖的脑海里是一片空白，直到傅夜沉再次弃她而去，她都未从这片空白里走出来。
<br/>
<br/>        一个人在房间里愣了一会儿，她才起身去洗澡，水淋在双膝的伤口上，“嘶嘶”作痛，她只能硬着头皮把澡快速洗完，然后裹着一条白色浴巾，蹲在洗漱间里洗衣服。
<br/>
<br/>        除了那张银行卡，她就只剩下这一身衣服了，今天洗了待会拿到阳台上去晾干，明天她好穿出去给自己买新的回来。
<br/>
<br/>        而这个时候，门铃不逢其时的响起，童筱颖甩了甩手上的水珠，起身往门口走去。
<br/>
<br/>        透过猫眼，她看到门外来访的人是傅夜沉时，心里又惊又喜，但隐隐地又有些发痛。
<br/>
<br/>        门铃再次响起。
<br/>
<br/>        童筱颖犹豫了一下，才将房门拉开一道缝隙，探出头去，弱弱地问道：“大哥，你找我？”
<br/>
<br/>        她探出头来的时候，也露出了半个香肩和半条雪白的胳膊，整个头发都是湿漉漉的像黑色的蛇尾一样缠绕在她的肩膀和手臂上。
<br/>
<br/>        傅夜沉随即将手里提的袋子朝她递了过去，淡漠地开口说道：“这里面是你的衣服，还有擦伤药。”
<br/>
<br/>        “谢谢。”童筱颖刚伸出手去，想要去接过那袋子，谁知室内和走道上的灯突然间灭了。
<br/>
<br/>        下一秒，一道闪电从窗外掠过，紧接着“轰隆”一声惊雷，几乎是一瞬间，童筱颖从门后窜了出来，本能地扑入了傅夜沉的怀中，紧紧地搂着他的腰身。
<br/>
<br/>        傅夜沉被她这突如其来的投怀送抱给怔住，原本还有一肚子的火气，结果在这一刻，闻着她身上那郁金香的沐浴乳香味，以及感受着她软软的身体，什么火全都消失殆尽了。
<br/>
<br/>        “我去帮你问问，什么时候来电。”他低沉道，欲要挪开她的双臂只身离开。
<br/>
<br/>        她闻言却反而将他抱得更紧，语气里像是在撒娇又像是在抱怨：“不要！我不许你走！”
<br/>
<br/>        “我没有要走。”傅夜沉的额际忽然冒出豆大的汗珠，咬牙忍着她此刻因为紧紧地抱着他，而撩拨起了一身的热欲。
<br/>
<br/>        “大哥……她是谁？是不是你的新欢？你是不是不要我呢？”她将他抱得越来越紧，身子完完全全地软贴在他身上，浑然不觉她的柔软对他所造成的影响。
<br/>
<br/>        “哪个她？”软玉温香在怀要他不心动都难，而她这软糯的声音里透着十足的醋意，更是让他爱不释手。
<br/>
<br/>        “今晚舞会上，和你跳舞的那个女人，她像公主一样高贵，和你很般配。”童筱颖喃喃细语，含糊不清说着。
<br/>
<br/>        傅夜沉这才恍然大悟她指的是朱斯蒂娜：“她确实是公主，法国皇室的公主之一，也是我们此次拓展化妆品项目的推荐人之一，而且她……”
<br/>
<br/>        他的话未说完，室内和走道上突然亮如白昼，总算是来电了。
<br/>
<br/>        但因为怀里这小女人的脑袋在他胸口上磨蹭着，惹得他抽气不断，更是将他推向水深火热的煎熬之中。
<br/>
<br/>        “你喜欢上她了，是不是？”她一边控诉，一边捶打着他厚实的胸膛，而本欲瞪视的眼却因充溢着泪水反而露出了一股柔媚的娇俏，“所以，大哥不要我了，是不是？”
<br/>
<br/>        傅夜沉好笑地看着童筱颖那一脸的醋意，意外的发现自己并不排斥她醋劲大发的撒泼，反而为她吃醋的俏模样感到窃喜不已。
<br/>
<br/>        “怎么会？”他失笑，话语里透着一股连他自己都没发觉的温柔，“我喜欢的人是你，爱的人也是你，要的人只有你。”
