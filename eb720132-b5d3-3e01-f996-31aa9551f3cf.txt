　　荒古禁地，北斗七大生命禁区之一，亦是最神秘的禁区之一。
<br/>　　此刻，混沌道君的身影径直来到深渊之上，接着毫不犹豫的向着下方一步踏出，却是消失于虚空之中。
<br/>　　刚一下落，便有一道恐怖的气息扑面而来，在他身体之中肆意侵略，可惜，这种气息虽然可怕，但还是不能损伤道君分毫。他知道，这是荒古禁地的荒之力，越到中心越可怕，对比，他毫不理会。
<br/>　　一路向下，越往下荒之力便越是浓厚，普通人一息不到便会被侵蚀，之后直接衰老，化为飞灰，就算是大圣准帝级别的修士，都不敢轻易踏足此地。荒之力中蕴含着浓郁的岁月之力，本身便是十分霸道的力量，无声无息将人寿命夺走，异常恐怖！
<br/>　　一道巨大的石门出现在道君眼前，幽静而深邃，充满了岁月的沧桑。这里宛若连接着一片浩瀚的仙域，女帝便是一直在这仙路上等待着什么！
<br/>　　道君心情升起一种奇异的感觉，这个地方的宁静让人感觉有些可怕，此处的荒之力近乎可以掠夺一个大帝强者身上的精力了，真不知道女帝当初是如何选择这样一个地方的。
<br/>　　“隆隆！”
<br/>　　这时，随着一声轰隆声响起，仙雾涌起，像是开启一方仙域，来到一个奇异的世界，这门就仿佛是一道天门！
<br/>　　有一方模糊的仙界在前方，不过，道君却是知道，这是女帝以无上法力推演创造出来的。
<br/>　　踏入门内，随处可见各种神药灵材，就好似是垃圾一般被扔在地上。女帝真是大气，不过想想也是，这里就是她的地盘，这天地间还有谁敢在她没有同意的情况下将此处的神药取走？
<br/>　　这时，一位身穿青色衣衫、面容枯老的老道来到门前，对着混沌道君微微行了一礼，接着便转身像是在带领道君去见某一个一样。
<br/>　　道君若有所思，这恐怕就是荒奴了。听说荒古禁地有好几位荒奴，都是当年侵占荒古禁地的圣地的绝世高手，被荒之力抹去神志，只留下一些身体本能，如同傀儡一般，但也正是这样，使得他们获得了一种另类的永生。
<br/>　　倒是对于青衣老道的行为，道君毫不意外。若是他不想让人知道他的行踪，这天地间还没有人能够知道他。但此刻他是来见一见这位无双女帝的，并不是来偷东西的，所以他并没有刻意影藏自己的气息，女帝能够发现他也很正常，若是没有发现，那才奇怪！
<br/>　　一路随着青衣老道向前走去，他又看到了好几位荒奴，正是当年天璇圣地的圣子圣女。继续向前走去，他看到了一位金毛怪物！
<br/>　　“吼……”
<br/>　　这怪物全身上下长满了诡异的金毛，他面目有些狰狞，像是在忍受着极大的痛苦。在他的身上，有着几道硕大牢固的锁链，这锁链所用的材料是道金，也正是这几道锁链将他锁住，才让他不能行凶。
<br/>　　“轰！”
<br/>　　这时，一道恐怖的气息从他身上升起，顿时金光爆闪，波浪声不断，雷闪电鸣，金色的神光遮蔽了整片天空，无上圣威爆发，浩瀚，无垠，宽阔，神圣不可侵犯！
<br/>　　这便是大成的荒古圣体！混沌道君看向那金毛已经渐渐消退的怪物，若有所思。
<br/>　　传说大成圣体可叫板大帝，道君感受一下，微微摇了摇头。大成圣体终究只是大成而已，说是可以叫板大帝，但实际上也只是能够与一般大帝强者对抗而已，终极是没有成道，不能打败或者镇压大帝，若是遇到变态一点的大帝，大成圣体也不算什么。
<br/>　　“你便是大成圣体？”道君开口淡淡问道，虽是在问，可语气之中却是无比肯定。
<br/>　　“我便是！”
<br/>　　那人看向混沌道君，神色之间无比凝重，他竟然看不透此人的修为，要知道，他可是大成圣体，堪比大帝强者，竟然看不透眼前此人。而且这人给他的感觉是深不可测，甚至比女帝给他的感觉还要危险。只要他有一点不好的念头，他便会感觉到有无穷危机笼罩心头，十死无生。
<br/>　　“你叫什么名字？”
<br/>　　“凌天。”尽管不知道这人要干什么，但他还是回答了他的问题。
<br/>　　“凌天？”道君看了大成圣体一晚，说到，“有没有兴趣跟本道君混？”
<br/>　　“……”大成圣体。
<br/>　　“本道君可以帮你除去身体之中的不详，亦可以将你恢复到全盛的状态，你可以去做你想做的事，你想想，你的仇人还在逍遥法外，你就不想报仇吗？只要投奔本道君就可以如何？”
<br/>　　大成圣体听了这话沉默了，他想起自己昔日的仇人此刻正躲在禁区之内，他也不想就此死去，他想起就将他变成这副模样的地府，他还有很多事没有做完，他不甘心就这样死去。
<br/>　　这时，他想起了眼前这个男子的话，心中思索着其中的利益，也许这是一个机会。至于男子所说的是真是假，他根本没有怀疑，这样一个强者，根本不屑于骗他。
<br/>　　“你好好想想，本道君不急。”
<br/>　　混沌道君说了一句，便继续向前走去，他此行的目的还没有达到，至于大成圣体，只是一个意外而已。他身边不缺人，多他一个不多，少他一个也不少。
<br/>　　再次走出数百米，混沌道君一晚便看到前方雾霭中有两道通天巨碑，岿然耸立，上面皆流露着苍茫亘古的气息，有着镇压乾坤的威势！
<br/>　　一道石碑上刻有三个大字：成仙路！
<br/>　　这种字十分古老，没有几人能够认识，有人说是仙文，也有人说是神话时代的神文。
<br/>　　另一道巨碑之上，同样刻有大字，但并不是三个，而是四个：六道轮回！
<br/>　　六道轮回，这自然不是真正的六道轮回之所，而是一道记载着传承的道碑。道君一眼便看出，这上面记载的是一种名为六道轮回拳的拳术！
<br/>　　六道轮回拳，传说乃是圣体的专用拳术，盖世无双，霸道绝伦，乃是一种无上杀伐圣术！
<br/>　　道君知道，这拳术的来历可不简单。乱古时代有一名仙王，名六道轮回仙王，开创了一门无上天功，名曰《六道轮回天功》，后来不知为何作为至尊殿堂的传承。若说两者之间没有关系，道君可不会相信。
<br/>　　道君看向道碑，眼中神光闪烁，有莫名的光彩在流动。
<br/>　　一个呼吸。
<br/>　　两个呼吸。
<br/>　　三个呼吸。
<br/>　　三个呼吸后，道君恢复常态，面色淡然，他已经领悟了这种拳术。
<br/>　　他不停，继续向前走去，一步踏出，他来到最深处。
<br/>　　雾霭沉沉，他看到一道修长的身影，那身影绝世而孤傲，秀发披散，眸子清澈如水，轻雾如纱，朦朦胧胧，仿佛有一股神秘的力量存在，让人看不清晰。
<br/>　　道君自是一眼便看得清楚，那是一位女子，躯体修长，傲世独立，风姿绝世，风华绝代！
<br/>　　在她的身上，还束有凰血赤金、仙泪绿金、永恒蓝金、龙纹黑金等道金组成的铁链，发出冲霄的光芒！
<br/>　　道君知道，这便是他此行的目标，荒主，亦或是狠人大帝！
