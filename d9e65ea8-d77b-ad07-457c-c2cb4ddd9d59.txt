　　“沈家主，我来助你！”
<br/>　　擂台上的龙傲天，忽然大喝一声，也加入了战团之中。
<br/>　　无论是龙傲天，还是沈青山，都具备神境后期的实力，两人联手之下，便是人仙境界的强者，也有一战之力！
<br/>　　方才还占据优势的唐清雅，被两人联手围攻之下，顿时有些手忙脚乱，根本无暇再出口反驳沈青山的话。
<br/>　　而且沈青山则趁机，又说了一些不堪的话语，把唐清雅气得肺都要炸了。
<br/>　　而网络直播间的观众，在沈青山的话语引导之下，顿时更加偏向沈青山。
<br/>　　而就在三人打的难分难解之时，数里外的一座山峰之上，几道气息相当恐怖的身影伫立其上，正遥遥注视着天台上发生是一切。
<br/>　　“狂仙门只来这么一个女娃吗？还真是让人失望啊！”
<br/>　　“是呀，原本以为用祝小白作为诱饵，至少可以将祝明飞引出来，只要杀了祝明飞，狂仙门便再无力抗衡我们蓬莱仙界，没想到却只钓出这么一条小鱼！”
<br/>　　“无妨！先将此女擒住，然后再徐徐图之，终究能钓出大鱼出来！”
<br/>　　“看来今晚已经不需要我等出面了，我们这几年安插在华夏武道界的势力，再加上这个沈山，应该足以控制住局面！”
<br/>　　“不错！虽然未能引出大鱼，但至少把狂仙门的名声搞臭了，如今他们已经成了众矢之的，就算他们今后重新出现，华夏也已经不是他们的天下了，哈哈哈！”
<br/>　　……
<br/>　　原来这几道强悍的气息，赫然是蓬莱仙界的强者！
<br/>　　而这整场武道大会，很显然都是蓬莱仙界设下的圈套，一个针对狂仙门的圈套！
<br/>　　就在几人议论纷纷之时，
<br/>　　“神！凰！变！”
<br/>　　天台之上，腹背受敌，已经开始节节败退的唐清雅，忽然一道娇喝，
<br/>　　轰！
<br/>　　自唐清雅的娇躯之上，陡然浮现出一道金色的凤凰虚影！
<br/>　　啨！！
<br/>　　随着一道凤鸣声，响彻天地之间，唐清雅的气息，竟陡然暴涨了一倍有余，一举压过了沈青山和龙傲天两人，所有人顿时全都为之一惊。
<br/>　　就连那几位蓬莱强者，也纷纷脸色微变，惊叹道：
<br/>　　“这虚影……是神鸟凤凰？”
<br/>　　“此女修炼的是什么功法？当真不凡的很呐！”
<br/>　　“务必将她擒下，拷问出此功法的来历！”
<br/>　　……
<br/>　　而天台之上，
<br/>　　众人看到这一幕，也纷纷惊叹了起来，尤其是青龙派那名地仙级的老者，一双老眼中顿时露出一抹炙热之色，显然对唐清雅施展出来的这门神秘功法，也相当的眼热。
<br/>　　轰轰轰！
<br/>　　唐清雅施展出神凰变之后，实力陡然翻身了一倍不止，战场上的形势也瞬间逆转。
<br/>　　“死！”
<br/>　　唐清雅大喝一声，猛然一掌拍出，狠狠砸落在龙傲天的身上，后者顿时狂喷了一口鲜血，直接倒飞了出去，重重栽落在擂台之下。
<br/>　　而沈青山见状，当即迅速暴退，同时不忘冲着远处高喊，
<br/>　　“龙长老，救命！！”
<br/>　　轰！
<br/>　　青龙派的那名地仙老者，在龙傲天被打飞的同时，就已经从座位上站了起来，
<br/>　　“大胆妖女！还不束手就擒！！”
<br/>　　这地仙老者虚手一拍，
<br/>　　轰隆隆！
<br/>　　天空之上，陡然出现了一道长达数百米的巨掌虚影，继而那巨掌虚影拍落下来，竟然又化成了一个巨大的牢笼，瞬间便将下方的唐清雅笼罩其中。
<br/>　　“嗯？”
<br/>　　唐清雅美眸一瞪，当即全力打出几掌，然而却犹如石沉大海，根本无法打破对方的真元牢笼。
<br/>　　“好强！”
<br/>　　唐清雅毕竟只有神境修为，即便施展出神凰变，最多也就能对抗人仙级别的高手罢了，面对地仙级别的强者，两者根本不在一个等级上，简直犹如婴儿之于成人！
<br/>　　“嘶~~”
<br/>　　众人看到青龙派强者，一招便将之前威风凛凛的唐清雅制服，顿时纷纷倒吸了一口凉气，原本还想站出来说两句的燕真人，看到那龙长老竟然有如此实力，也不由得微微一呆，旋即又重新坐回了座位之上。
<br/>　　在武道界，永远都是以强者为尊，谁的实力更强，谁才具有话语权！
<br/>　　唐清雅全力攻击了几次，都无法撼动龙长老的牢笼，似乎也意识到，自己的修为和对方相差太远，索性也不再浪费精力，冷冷道：
<br/>　　“成王败寇，你杀了我吧！”
<br/>　　龙长老微微一笑，一脸正气，道：
<br/>　　“女娃娃，我们青龙派乃是名门正派，可不像你们狂仙门这样的邪派，动不动就要灭人满门！”
<br/>　　“老夫看你资质不错，只要你能痛改前非，说出狂仙门弟子的下落，然后改投到我青龙派，老夫可以保你不死，如何？”
<br/>　　哗！
<br/>　　龙长老此话一出，众人顿时一片哗然，在场的一些强者们，其实已经猜到了龙长老的意图，暗道一声老奸巨猾。
<br/>　　而网络直播间里的观众，又哪里知道武林中的弯弯道道，众人见龙长老慈眉善目，而且语气温和，顿时一片称赞之声，甚至很多人纷纷劝说唐清雅，赶紧把握机会，改邪归正。
<br/>　　唐清雅冷冷一笑，
<br/>　　“真是唱的比说的还好听！别人不清楚，你当我也不知道吗？青龙派、巫山派、五行宗还有太一门，你们根本就不是什么华夏隐世宗门，而是蓬莱仙界混入华夏的奸细！”
<br/>　　“你们联手灭了七大家族，反而将脏水泼到狂仙门的头上，真是卑鄙至极！！”
<br/>　　唐清雅此话一出，顿时满座皆惊，尤其是她刚刚提到的那几大宗门的门人，更是个个脸色大变。
<br/>　　“一派胡言！”
<br/>　　龙长老当即一声暴喝，瞬间没有了之前的温和之态，眼眸中透着杀意，寒声道：
<br/>　　“没想到你为了给狂仙门洗白，竟然如此颠倒是非，当真以为老夫不会杀你吗？”
<br/>　　唐清雅冷笑一声，
<br/>　　“你要杀便杀！但是你要记着，等到我们狂仙门叶门主归来之时，便是尔等这些魑魅魍魉的末日！！”
<br/>　　听到唐清雅这话，那些年轻后辈一脸茫然，不知道唐清雅话中的意思，但是那些年纪稍长之人，则顿时个个脸色大变，似乎想到了什么恐怖的事情。
