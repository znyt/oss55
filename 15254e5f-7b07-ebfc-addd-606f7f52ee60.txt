　　结果……两只手拍在一起，瓜瓜立刻觉得自己好像是打在了一团棉花上，手上的妖气消失无踪，却被对方的手吸住，不管他怎么用力都挣脱不出来。
<br/>　　对方也没给他任何机会，另一只手从后面抓住了他的脖子，一股霸道的气息瞬间灌入身体，立刻失去了知觉。
<br/>　　这下子完蛋了……
<br/>　　瓜瓜脑海中闪过这个念头，不过对方并没用对他下手，而是提着脖子，窜了出去，在山野之间展开了极快的飞行。
<br/>　　为了防止尸族偷袭，就算是在青丘山后山的峡谷里，也驻扎着一批妖兵，负责巡逻的远远看到有人凌空飞来，作法阻挡，结果连对方的身边都没能靠近，就被震飞了出去……
<br/>　　那人提着瓜瓜的脖子，一路疾飞，速度之快，在瓜瓜看来都要赶上高铁了，最主要的是这家伙似乎一点都不累，一直飞行。
<br/>　　瓜瓜心中震撼，慢慢地适应了眼前的局面，虽然身体被控制，不过话还是可以说的，冲对方嚷道：“你是谁呀你，你要带我去哪？”
<br/>　　对方根本就不理他，只顾飞行。
<br/>　　“你要是想杀我，就来个痛快的嘛！你这样算是什么意思！我又不是花姑娘，你抓我干什么！”瓜瓜盛怒之下口不择言。
<br/>　　对方还是不理他，提着他的脖子，飞行了许久，最后上了一座山，总算是停下来，将瓜瓜扔下来。
<br/>　　瓜瓜转头看去，自己在一座山的山顶上，身后不远就是一个洞府，自己已经恢复了行动，但是瓜瓜没做任何行动，没跟他打，也没逃走。
<br/>　　之前见面时候那一个回合的交锋，已经让瓜瓜看出了对方的实力，自己绝对不是对手，打是打不过，跑也是跑不掉的。
<br/>　　那个蒙面人放下他之后，自己过去打开了洞府，这才回过头来看他，并且摘掉了面纱。
<br/>　　“是你！”瓜瓜叫了起来，“阿呆！”
<br/>　　这个把自己从青丘山上绑到这里来的不是别人，正是尸王后卿！瓜瓜曾经跟他打过交道，那时候他还叫阿呆，还被芮冷玉照顾着。
<br/>　　“阿呆不是你叫的。”后卿冷冷地说道。
<br/>　　瓜瓜耸了耸肩，转头看着四周，说道：“这是什么地方，是你老巢吗？你带我来这干什么。”
<br/>　　后卿看他表情自如，微微皱眉，说道：“你不怕我杀你？”
<br/>　　“你要杀我，之前就动手了，还用带我来这里干什么？不过我确实很奇怪你绑我来这里干什么。”
<br/>　　“带你见一个人。”
<br/>　　“谁？”瓜瓜脱口说出这个字，猛然想到什么，浑身一颤，表情也凝重起来，急忙说道：“我嫂子呢？”
<br/>　　后卿什么也没说，走进了洞府中。
<br/>　　瓜瓜愣了一下，紧忙跟了过去。
<br/>　　已经到了这里，不可能回去了。
<br/>　　从洞府中穿过，一路走到了尽头。瓜瓜看到在洞府最深处、那一块被夜明珠之类的物件照亮的水晶平台。
<br/>　　最出乎他意料的是，在这块空间里，居然摆着人间现代风格的沙发、茶几，还有一些人间常见的家具。茶几上摆着一摞书，一个女子很娴静地坐在沙发上，手里捧着一本书在看。
<br/>　　“嫂子！”瓜瓜激动地叫起来，眼前这不是别人，正是自己苦苦寻找很久、但是不能一见的芮冷玉。
<br/>　　不过在水晶平台的四周，有水幕阻挡，瓜瓜不笨，知道这是某种强大的结界，自己没有办法跨越过去，也没有必要去试。
<br/>　　在这里见到芮冷玉，他很激动。
<br/>　　芮冷玉浑身一颤，慢慢转过头来，看到瓜瓜，她也很激动。
<br/>　　之前，跟叶少阳在一起的时候，瓜瓜就总在他们身边晃悠，充当一个高瓦数的电灯泡，为此叶少阳踹过他很多次，但是现在回想起来，那一幕真的很温馨。瓜瓜就像是她跟叶少阳的孩子，一家三口……以前没觉得怎么样，此刻再见到瓜瓜，却是在这样一个特殊的环境下……再想起从前愉快相处的日子，都成了一种可望而不可及的奢望。
<br/>　　芮冷玉几乎潸然泪下。尤其是瓜瓜的一声“嫂子”，更是让她感到痛苦和绝望。
<br/>　　“嫂子，你在这怎么样，你是被关起来了吗！”瓜瓜怒不可遏，看了一眼站在一旁的后卿，吼道：“快把我嫂子放了！”
<br/>　　芮冷玉道：“不怪他，他是在保护我，若不是他，我早就死在外面了。”
<br/>　　瓜瓜无话可说，但是内心却是浮起了一股强烈的嫉妒，这种嫉妒，是替叶少阳产生的。不过人家的确是救了芮冷玉，瓜瓜也不能说什么。
<br/>　　芮冷玉道：“是我让他找你来的，我先问你，少阳现在什么地方？”
<br/>　　“老大被徐福用山海印带去别的时空了，不过他还活着，用特殊方法给我们带来了一封信，他现在在民国，正在想办法回来。”
<br/>　　说完，瓜瓜想了一下，又加上一句，“老大一定会回来的！”
<br/>　　芮冷玉长出了一口气，表情明显放松了很多，喃喃道：“我就知道，他一定会没事的。”
<br/>　　“嫂子，你别怕，老大一定会来救你的！”
<br/>　　芮冷玉像是被他这句话拉回到了现实，转头凝视着他，说道：“小九为少阳而死，不知道现在怎么样了？”
<br/>　　“这个……我相信，小九是不会死的，就算死了，老大也一定有能力把她复活！”
<br/>　　他没看到，后卿的嘴角浮起了一丝笑容。
<br/>　　芮冷玉道：“我也相信。我问你一件事，从前我没有问过你们，你要如实回答。”
<br/>　　瓜瓜点点头，道：“对嫂子，我当然有什么说什么，你问吧。”
<br/>　　“小九，是不是很喜欢少阳？你知道我说的是哪种喜欢。”
<br/>　　瓜瓜一下子懵了，平时他在叶少阳身边的时候，也观察过，这件事……他不可能不知道，现在被芮冷玉这么问，他实在不敢说假话，勉强只好点了点头。
<br/>　　芮冷玉早知道这件事，也没有什么反应，接着又问了一句更切中要害的话：“少阳，是不是也很喜欢小九。”
<br/>　　瓜瓜浑身一震，本能地嗫嚅道：“我……我不知道。”
