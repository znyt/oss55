　　远洋货轮甲板上的动静自然是引起了船舱中其他武装毒贩和西伦科特的注意，西伦科特从椅子上站起身来，把桌子上装有两个液氮低温瓶的背包背在身上，西伦科特知道联合行动特种部队来了。
<br/>　　西伦科特料想到联合行动特种部队一定会找到自己，但是却没有想到会这么快，这一刻，西伦科特几乎可以断定，在这艘远洋货轮上有卧底，这个卧底未必是联合行动司令部派遣过来，但却肯定和联合行动司令部有联系。
<br/>　　不过，此时西伦科特也顾不上去找出这个藏在远洋货轮上的卧底，他现在最需要考虑的是，如何保住自己的性命。
<br/>　　远洋货轮上，武装毒贩用AK自动步枪对着盘旋在空中的直升机进行射击，子弹打在直升机机身上，溅起点点火星。
<br/>　　咻咻咻咻咻！
<br/>　　轰轰轰轰轰！
<br/>　　又是一轮火箭弹朝着远洋货轮甲板上的武装毒贩倾斜而来，火光冲天而起，远洋货轮甲板上的武装毒贩死的死，伤的伤。
<br/>　　两架“黑鹰”运输直升机上，Alpha和Bravo小队队员准备进行索降。
<br/>　　龙隐部队所搭乘的“黑鹰”直升机中，罗昊拿起一张照片，照片上是一个穿着AOR1迷彩作战服的黑人，罗昊把照片在机舱中那些龙隐部队队员看了眼，大声说道：“这就是DEA在这艘远洋货轮上的卧底，代号‘JR’，一会儿动手的时候，记得注意，不要伤及这个DEA的卧底。”
<br/>　　机舱中的龙隐部队队员看了眼照片上的DEA卧底，将这个代号“JR”的卧底的相貌记下后，在战术手套外又戴上一层厚布手套，罗昊站起身将盘在舱门口的索降绳一脚踢下，对着机舱中的龙隐部队队员大声吼道：“抓紧时间，进行索降！快！快！快！这次绝对不能放走西伦科特！”
<br/>　　机舱中的龙隐部队队员纷纷起身，走到舱门口，抓住索降绳依次滑下直升机。
<br/>　　最后滑下直升机的罗昊松开索降绳，对着直升机驾驶舱竖起一个大拇指，盘旋在货轮甲板上空的“黑鹰”直升机立刻调转方向离开了甲板上空，飞到了货轮周围的海面上待命，至于那两架“长弓阿帕奇”依旧盘旋在远洋货轮的甲板上空，为登上远洋货轮的特种部队提供火力掩护，货轮上那些武装毒贩的AK自动步枪并不能对这两架“长弓阿帕奇”造成威胁。
<br/>　　罗昊右手握着AR战术步枪，竖起左手对着前方船舱的方向挥了挥手，周柏、李然、孙涛、周立等人紧随其后，跟着罗昊朝着船舱的方向推进。
<br/>　　“黑鹰”直升机上，李毅峰和慕容飞端着狙击步枪，对货轮上的武装毒贩进行远程狙杀，他们的主要目标是货轮上那些对直升机有直接威胁的火箭筒手。
<br/>　　一个接一个手持火箭筒的武装毒贩遭到狙杀，一个个还来不及发射的RPG-7火箭筒掉落在地上。
<br/>　　罗昊脚下踩着战术步伐，手持AR自动步枪以半自动模式对着货轮上的武装毒贩进行射击，尽管罗昊是在用半自动射击，可是高射速却使得半自动射击有了全自动射击的强大火力。
<br/>　　这些武装毒贩在两支代表着当今世界最高水准的特种部队面前，根本没有任何抵挡之力，沉闷的点射声中，货轮上的武装毒贩不断的中枪倒地，滚烫的弹壳从抛壳窗中飞出，掉落在甲板上，发出清脆的声响，轻快的跳动着。
<br/>　　冲出船舱的武装毒贩又被逼退回船舱中，继续对着船舱外的特种部队进行射击。
<br/>　　罗昊取出一颗手雷，打开保险，大喊一声“手雷”，从船舱破碎的玻璃窗扔进船舱中。
<br/>　　轰——！
<br/>　　船舱中生疼起一团火焰，灼热的气浪从窗户中翻涌而出，原本船舱中激烈的枪声，随着这颗手雷的爆炸声消失了。
<br/>　　一名武装毒贩挂在碎了窗玻璃的铁窗框，留在窗框上尖锐碎玻璃刺破了他的胸膛，鲜血顺着船舱舱壁流淌下来，除了胸口那个被碎玻璃扎破的口子，他的口、鼻、眼、耳中也是有着鲜血流淌出来，只是因为刚才那颗手雷爆炸产生的冲击波造成的。
<br/>　　躲在船舱中的武装毒贩全都被手雷炸死，他们身上除了被高温灼伤造成的伤痕外，并没有明显的伤痕。进攻型手雷不会像防御型手雷那样对人造成明显的伤痕，但是在巷战和城市大楼中作战时，进攻型手雷所造成的杀伤力远比破片式防御型手雷恐怖，对藏在建筑物内的敌人更致命。
<br/>　　罗昊端着AR战术步枪站在船舱门口往里面打量了一番，确认里面并没有西伦科特后，示意继续往船舱内部搜索，寻找西伦科特。
<br/>　　“货轮已经被重重包围了，除非西伦科特会飞天遁地，不然这次我看他往哪里逃。”周柏握着AR战术步枪，眼睛中的点点寒光犹如冰冷刺骨的冰碴子一样，仿佛随时都能将人掩埋起来，冻僵。
<br/>　　“注意！”罗昊侧头看了眼身边的其他几个人，提醒道：“西伦科特手中握有P病毒，行动过程中必须要保证P病毒的安全，绝对不能让他将P病毒释放到海洋中。”
<br/>　　“明白！”
<br/>　　众人沉声应答。
<br/>　　船舱中，西伦科特听着逐渐逼近的脚步声，并没有太多的紧张。西伦科特抓起桌子上那把银色的沙漠之鹰，带着装有液氮低温瓶的背包，走出船舱，跟走廊中的联合行动特种部队正面相对。
<br/>　　西伦科特用沙漠之鹰指了指背包，眼神冷漠的看着眼前的特种部队，说道：“你们找到我的时间，比我想象中的早，是你透露了我的行踪吧？”西伦科特看向跟Alpha和Bravo小队站在一起的DEA卧底JR。
<br/>　　“我本来的任务只是摧毁掉这批被运往北美洲的高纯度cocaine，这只是一个巧合。”JR露出雪白的牙齿，看着西伦科特说道：“这次的收获有些出乎我的意料。”
<br/>　　“那我们来赌一把，我们谁会先死。”西伦科特把枪口顶在背包中的液氮低温瓶上，说道。
