　　高空中，长孙赤烈凌空站着的地方，已是一片火海。
<br/>　　百条拥有绝世9990道实力的火焰游龙，一瞬间爆发，其威力，果然惊人。
<br/>　　“长孙赤烈。”远处，第一云与花若怜惊呼一声。
<br/>　　萧逸看了眼火海，摇了摇头。
<br/>　　炎龙御火术，确实厉害。
<br/>　　可他的九阳轮回诀也不弱。
<br/>　　虽说他的九阳轮回诀还未大成，但他的控火兽武魂已然直追冰鸾剑的层次。
<br/>　　故控火兽的‘控火专精’这一天赋，已被发挥到极致。
<br/>　　自然的，萧逸如今的控火能力，强悍得可怕。
<br/>　　论控火，萧逸本就没怕过谁。
<br/>　　加上如今强悍得可怕的控火能力，单论控火本事，萧逸早已达到强得发指的地步。
<br/>　　“散。”
<br/>　　萧逸摆摆手，低喝了一声。
<br/>　　远处，火海顷刻消散，露出了里头被火焰吞噬的长孙赤烈。
<br/>　　此刻的长孙赤烈，被烧得狼狈至极。
<br/>　　那硕大的光头脑袋，更是被烧得一片黑。
<br/>　　“噗。”长孙赤烈，嘴中吐出一口腥血。
<br/>　　当然，只是轻伤。
<br/>　　萧逸及时散去了火海，故长孙赤烈并不至于重伤。
<br/>　　“你败了。”萧逸淡漠地说了一声。
<br/>　　“哼。”长孙赤烈冷哼一声，不语，但那双原本狂傲的眼眸，再也不敢直视萧逸分毫。
<br/>　　嗖…
<br/>　　忽然，一道疾影，直朝萧逸而来。
<br/>　　萧逸早有准备，一道紫炎凝聚包裹与手臂，轻轻一挡。
<br/>　　啪…一声重响，疾影被挡下。
<br/>　　出手的，正是第一云。
<br/>　　此刻，第一云正一拳在萧逸手臂之上。
<br/>　　萧逸皱了皱眉，“长孙赤烈已败，你三人实力在伯仲之间，修为也皆在绝世9991道，还需再战？”
<br/>　　这三人，修为一致。
<br/>　　虽分别来自三殿，各人所修功法、手段亦不同。
<br/>　　但以萧逸现今的眼光，绝不会看错，这三人的实力，在伯仲之间。
<br/>　　长孙赤烈已败，再战下去，并无意义。
<br/>　　他能败长孙赤烈，同样能败第一云。
<br/>　　第一云摇了摇头，桀骜的面容上，写满了自信。
<br/>　　“易霄，你也是猎妖师，你该知道猎妖师没有那么简单。”
<br/>　　“如果你真是一个合格的猎妖师，你应当知晓我的意思。”
<br/>　　萧逸点了点头，“那随你吧。”
<br/>　　轰…第一云，瞬间收拳，随后又一拳破空而来。
<br/>　　这一拳，快到了极点，又狂猛到了极点。
<br/>　　萧逸堪堪躲过。
<br/>　　拳头的破空声在萧逸耳边擦过，空气震动破碎。
<br/>　　萧逸肩膀之上，一道火焰凝聚，拦住了破空之力。
<br/>　　第一云一招不成，拳风狂风暴雨般袭来。
<br/>　　双手如舞，脚下凌空飞动。
<br/>　　手脚间的配合，或攻或退或袭，竟是掌控得淋漓尽致。
<br/>　　萧逸身上凝聚的火焰，则已渐渐密布全身，以增幅自身的爆发力。
<br/>　　第一云的实力，着实不俗。
<br/>　　而且，他也明白刚才第一云的话语意思。
<br/>　　猎妖殿，不同于其它七殿。
<br/>　　其它七殿，或炎殿武者的精通控火，或药尊殿武者的精通炼药，或风刹殿武者的御风之能，又或者修罗殿的体修强悍…
<br/>　　而猎妖殿，以‘猎妖’二字名之，殿内一切，几乎都是为了对付妖兽、斩杀而存在。
<br/>　　包括功法、武技、手段等等，皆是为了斩杀妖兽。
<br/>　　而妖兽，本身就肉体力量强大，又狡诈过人。
<br/>　　加之偌大个大陆，妖兽无数，种类也不一。
<br/>　　妖兽的手段和攻击，或狂猛、或急速、或诡异、或难缠…
<br/>　　而猎妖殿，存在了无数年，早便有了无数办法应付这些种类不一的妖兽。
<br/>　　这便有了猎妖殿内的功法、武技、手段。
<br/>　　或者更简单来说，第一云，几乎是个全能的武者。
<br/>　　他手上拥有着极多的手段，无论应付怎样的对手，都能游刃有余。
<br/>　　无论他要使出怎样的攻击，都能将其完美契合。
<br/>　　这个常年游走于险地，其击杀妖兽效率更甚于萧逸的妖孽，绝非泛泛之辈。
<br/>　　数分钟后。
<br/>　　高空中，二人的战斗，已然化作了一道道幻影。
<br/>　　二人，同为一身劲装。
<br/>　　不过，萧逸是如若一道火焰幻影。
<br/>　　而第一云，则是招招凌厉，身影如风如雷，刚劲过人。
<br/>　　又是数分钟后，萧逸摇了摇头。
<br/>　　第一云此人，萧逸也有所听闻。
<br/>　　如果说，长孙赤烈是狂傲过人的话。
<br/>　　那么第一云，便是桀骜不驯。
<br/>　　此人以第一为姓，其作风，性格，亦是样样必争第一。
<br/>　　他和长孙赤烈、花若怜三人，虽是齐名，实力也在伯仲之间。
<br/>　　但若真说到实力极限爆发下，第一云的实力，绝对是三人之最。
<br/>　　“好了，就陪你打到这了。”萧逸淡漠地说了一声。
<br/>　　“破。”
<br/>　　一声轻喝，漫天紫炎，瞬间暴走。
<br/>　　第一云的身影，瞬间在火焰爆发下被轰飞千米。
<br/>　　待他站稳身影后，已是一口腥血喷出。
<br/>　　“你败了。”萧逸说了一声。
<br/>　　他特地与第一云鏖战一段时间，也是想看看第一云掌控了多少猎妖殿的功法武技和手段。
<br/>　　就刚才一战来看，第一云的水准，怕是不亚于总殿内的寻常主殿主。
<br/>　　当然，比不上灸炎、唐音那种。
<br/>　　但和一般的主殿主相比，差之不远。
<br/>　　如此年纪，便能比得过那些动辄修炼数百年的老一辈强者，而且还是猎妖殿主殿主，这等成就，确实斐然。
<br/>　　“你是炼药师，想来，便不必与你比了吧。”萧逸看向花若怜，淡漠道。
<br/>　　花若怜，有药公子之名，乃是药尊殿公认的年轻一辈第一炼药师。
<br/>　　甚至于，传闻此人同样破了药尊殿最快成就绝世炼药师的记录。
<br/>　　更有传言，药尊殿总殿主曾断言，此人十年之内，必定踏入传奇炼药师行列，成为药尊殿有史以来最年轻的传奇炼药师。
<br/>　　此人的炼药本事，确实厉害。
<br/>　　不过，炼药师的战力，一般不怎么强。
<br/>　　八殿武者，自然是强于外头的寻常武者的，但论及八殿之中，无疑是药尊殿的武者战力最低。
<br/>　　“谁说不必比？”花若怜冷笑一声。
<br/>　　“你当我站在这里如此久，何故？”
<br/>　　“花神大阵，起。”花若怜大喝一声。
<br/>　　嗖…嗖…嗖…
<br/>　　空气中，一堆堆丹药，凭空而现。
<br/>　　轰…下一秒，一股滔天阵法之力，撼动长空。
<br/>　　“哦？以丹药为阵基之力，以药道融合阵法一道？有些意思。”萧逸眼前一亮。
