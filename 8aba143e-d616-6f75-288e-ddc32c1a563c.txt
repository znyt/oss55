　　其实说老实话，李承乾并不能确这眼下这个情况张亮会不会造反，可是历史书上说这小子不老实，那么他就不会放心让老头子带着他出海。
<br/>　　天知道这货会不会玩儿什么花样，万一把老头子绑架了可怎么办，到时候天高皇帝远，自己拉不长他捶不扁他，麻烦不是一般的小。
<br/>　　不得不说，人这种生物还真是不能留下任何污点，原本张亮其实并没有造反的打算，可是在另一段历史里面，他就是个反骨仔，于是这一段历史上不可避免的悲剧了。
<br/>　　……
<br/>　　十余日后，在一个风和日丽的早上，老头子的车驾离开了大明宫，转过半个长安城，来到渭河之上的唯一码头。
<br/>　　整个长安能动的，不能动的所有五品以上官员全部随行于老头子的车队后面，李承乾自然也不例外，不过他是坐着马车的，而与他同行的则是秦琼。
<br/>　　“秦叔叔，这次父皇远行安全方面就交给你了！小侄在此拜谢！”看着越来越近的马头，李承乾望着一脸踌躇的秦琼，转身对其深施一礼。
<br/>　　“陛下不可如此，折杀某家了！”秦琼被李承乾突如其来的动作弄的手忙脚乱，连忙上前将其扶住，口中承诺道：“秦某可以用项上人头担保，不管遇到什么事情，也不会让人伤到太上皇一根汗毛。”
<br/>　　“有秦叔叔的保证，承乾就放心了，不过叔叔路上也需要注意身体，平日里多食一些青菜，船尽量不要离岸太远，一来可以少遇到一些风浪，二来也可以就地补充物资。”李承乾已经不知道是第几次这样嘱咐了，可他还是有些不放心。
<br/>　　这些老头子一起出海，玩的嗨了天知道他们能干出什么事情，由其是程老魔头还在南海，等这两伙儿人遇到一起，那特地么可就热闹了。
<br/>　　想到程咬金，李承乾突然觉得很有必要把这个老货给调回长安，一来可以多个自己人，二来也可以避免他与老李汇合之后搞出什么不可收拾的场面。
<br/>　　一路想着，嘱咐着，时间不大已经到了河边码头，看到那庞大的钢铁巨兽之后，李承乾的心才算多少放下了一些。
<br/>　　其实他这种感觉很多朋友应该都体会过，比如年迈的父亲母亲出去旅行，而我们因为工作又不能跟着的时候，往往都会有这样那样的焦虑。
<br/>　　一会儿担心老头儿老太太路上会不会安全；一会儿担心路上会不会水土不服，过一会儿还要担心一些外地的食物老两口能不能吃的惯……，总之当儿女的一般来说都会有这样的纠结。
<br/>　　李二和长孙皇后还有杨妃、阴妃等虽然有着庞大的随行队伍，但说起来还是让李承乾有些不放心，于是便借着礼官唱礼的时候凑到了老头子的身边：“父皇，儿臣把孙思邈叫来了，您看要不要带上他一起？”
<br/>　　“带他来干什么？”老李狐疑的看了李承乾一眼。
<br/>　　“儿臣这不是不放心么，山高路远的，多个医生总是好的。”李承乾一边说，一边看了一看自己老娘。
<br/>　　“是啊陛下，不如就让孙神医跟着吧，妾身也觉得那几个太医有些不大靠谱。”长孙皇后对孙思邈印象不错，闻言也劝了一句。
<br/>　　“那长安这边怎么办？他是医学院的院正，若是他离开了，这边岂不是要乱了？”李二犹豫了一下，转而向李承乾问道。
<br/>　　“医学院这边已经走上正轨了，只要做好未来的发展规划就好，孙思邈在不在其实一个样，而且这次父皇您带他出去走走也好，正好可以出去开开眼界，总好过天天在缩在研究院里固步自封不是。”李承乾笑着给老头子递了个台阶。
<br/>　　他早就已经摸清了老头子好面子的属性，所以一切事情只要说的冠冕堂皇一些便可以蒙混过关，在孙思邈的问题上同样如此。
<br/>　　果然，老头子听了李承乾的安排之后点了点头，不再坚持自己刚刚的观点：“照你这样说好像也有些道理，那便带上他吧，反正路上也不差一个人的位置。”
<br/>　　“是，儿臣一会儿就去安排！”李承乾见老头子答应了，脸上露出一丝如释重负的笑容，只等着一会儿礼官唱完了礼，便安排人去找后面车上的孙思邈。
<br/>　　……
<br/>　　其实孙思邈是不想参与这次远行的，他在医学院里还有一大推的事情没有处理完，但是这老头儿终于还是没有扛住李承乾的那条三寸不烂之舌，被他从医学院里忽悠了出来，而且用的理由与他跟李二说的差不多——读万卷书，不如行万里路。
<br/>　　要知道，这个世界可是大的很，只有走出去了，才会知道大唐一偶之地是多么的渺小，船行大海之上，望着看不到头的广阔天际，那种独孤与沧桑的感觉足以改变一个人。
<br/>　　孙思邈也是一个喜欢研究新鲜事物的人，对于李承乾口中说的会唱歌的鱼很感兴趣，还有就是那种比船还要大的鱼，也就是传说中的鲲！
<br/>　　所以送李二出行的队伍中，孙思邈自然也在其中。
<br/>　　李承乾派人回去之后很快就找到了这个老道士，并且迅速通过其他通道安排他上了战船。
<br/>　　当然，也幸亏李承乾这次长了个心眼安排孙思邈跟着老头子，事则的话估计还真有可能出大事，不过这些都是后话而已。
<br/>　　……
<br/>　　唱礼已毕，祭天、祭地、祭河神，最后老头子又亲自念了一份告天文书，点火烧了之后这出行的礼仪便也算是结束了。
<br/>　　长孙无忌显然并不想跟着李二出海，可是李二的命令再怎么说也是皇命，他又不能不听，所以只能亦步亦趋的跟在老李的身后，一脸的郁闷。
<br/>　　而尉迟恭和秦琼两人则显的有些兴奋，上了战舰之后这里摸摸，那里看看一副好奇宝宝的样子。
<br/>　　要知道虽然他们几年前登上过一次铁甲舰，但那时候的铁甲舰毕竟还没有安装武器系统，与现在简直就是天地之差。
<br/>　　……
<br/>　　“呜……”巨大的气笛声中，战舰缓缓移动，码头上的众臣齐齐躬身拜了下去。
<br/>　　主政十七年之后，伟大的帝国主义头子李二陛下，终于要开始他新的征途，向着茫茫未知的大海而行。
