　　“我们五灵宗的圣子，都来到了九域之中，应该说整个圣龙秘境的圣子，都已经来了九域了。”
<br/>　　炎撼世听到楚风眠的话开口道。
<br/>　　就在楚风眠跟洛汐动身前往九域后的几天，圣地之中的各个势力也按奈不住了，都前往了九域。
<br/>　　五灵宗的一群圣子，也全部都来到了九域，他们来到九域打探了一下情况，便是都来了归墟天。
<br/>　　只是后来这一群五灵宗的弟子在一处宝藏之中寻觅的一刻，突然碰到了一群魔族，以至于大量的五灵宗弟子都被魔族抓住。
<br/>　　唯有炎撼世一个人逃了出来，但是还被三尊阿修罗追杀，也就是好运碰到了楚风眠，才逃过一劫。
<br/>　　那些阿修罗要活捉的原因，却是因为炎撼世，似乎是触动了宝藏之中的一道禁制。
<br/>　　只有他，才可以开启那一道宝藏，那一群魔族，纵然是占据了宝藏，也无法得到其中的宝物。
<br/>　　“楚风眠，只要你帮我救出我们五灵宗的诸位圣子，我可以把宝藏之中的宝物，全部给你。”
<br/>　　炎撼世说完，突然对着楚风眠开口道。
<br/>　　楚风眠的实力，他已经是看了出来，强横到了一种不可思议的地步。
<br/>　　虽然那一群魔族难以对付，但是楚风眠一旦是出手，未必没有这个机会。
<br/>　　“哦？”
<br/>　　听到炎撼世的话，楚风眠眼前一亮。
<br/>　　他现在的境界，已经是无限接近于帝尊境，在吞噬了厚土之精，离火之精，玄水之精后他的实力再进一步。
<br/>　　现在楚风眠已经是足以清晰的感觉这周围的天地之力，这就是楚风眠距离帝尊境，只差一步之遥的信息。
<br/>　　现在楚风眠只差一个契机，或许就是一个宝藏，便是就足以踏入帝尊境。
<br/>　　而且五灵宗的圣子，现在都被魔族困住，如果楚风眠出手相救，也足以让他们欠下人情来，未来纵然是要跟六绝宗，圣院宗翻脸，如果五灵宗跟楚风眠站在一起的话，也算是一大助力。
<br/>　　并且楚风眠的心中，也有着打算要将这一群五灵宗圣子，全部纳入到剑道门的想法。
<br/>　　这些五灵宗弟子，无一不是天才，如果能加入剑道门，足以让剑道门的实力提升无数。
<br/>　　“那一群魔族的实力如何有没有九劫古帝？”
<br/>　　楚风眠虽然心中已经是下了决定，但还是打算仔细的问一问。
<br/>　　那一群五灵宗圣子，可不是那么好对付的，那一群魔族，居然是足以将这些五灵宗弟子全部困住，显然也不好对付。
<br/>　　楚风眠也要以防万一才行。
<br/>　　“一共是五名八劫古帝，现在那阿修罗皇死在你的手里，应该还有夜叉皇，鬼魅皇，独魔皇，阎魔皇，四名八劫古帝。”
<br/>　　“其他的，还有阿修罗，鬼魅，独魔，阎魔，夜叉等大约三十几名七劫古帝级别的魔族。”
<br/>　　炎撼世开口道。
<br/>　　“之前我们联手，已经是打伤了一批魔族。”
<br/>　　阿修罗，夜叉，鬼魅，独魔，阎魔！
<br/>　　这些可都是魔族之中赫赫有名的族群了，都是魔族之中高贵的血脉。
<br/>　　这一次居然有着如此之多的魔族，都来到了九域之中，不过四名八劫古帝巅峰级别的魔皇，楚风眠倒是不在乎。
<br/>　　除非是碰到九劫古帝，不然楚风眠一身的实力，几乎是所向无敌，至于三十几名七劫古帝，楚风眠更是连看都不会多看一眼。
<br/>　　这些魔族的血脉尊贵，血脉之中蕴含的力量也无比强大，杀了他们，利用吞天秘术吞噬殆尽的话，对神树也有着极大的好处。
<br/>　　“走，我跟你去救人，不过你必须要答应我两个要求，第一那宝藏之中的东西，全部归我，第二，日后我圣龙宗跟六绝宗，圣院宗敌对的时候，五灵宗必须要出手相助。”
<br/>　　楚风眠看着炎撼世开口道。
<br/>　　“好，我答应你。”
<br/>　　炎撼世想了一下，还是点了点头。
<br/>　　这宝藏现在落入到了魔族的手中，就算是他想要得到，也没有这个机会，楚风眠出手的话，就算是给楚风眠一个顺水人情。
<br/>　　至于第二个要求，在见识了楚风眠的实力之后，五灵宗的人，本就是在拉拢楚风眠，日后撕破脸，站在圣龙宗这一边也没什么。
<br/>　　“那好，带路。”
<br/>　　楚风眠看了一眼炎撼世，点了点头道。
<br/>　　炎撼世化为了一道遁光，一路飞去，楚风眠则是跟随在后面。
<br/>　　一路飞行，就在赤红大陆的一处山谷前，突然间炎撼世停了下来。

<br/>　　炎撼世转过头来，对着楚风眠说道。
<br/>　　“我们要不要定一个计划。”
<br/>　　“不用什么计划，直接杀进去就行了！”
<br/>　　楚风眠目光冰冷无比，一步踏出，刚刚踏入到山谷的一刻，周围的景象突然天旋地转起来，随即便是化为了一片草原。
<br/>　　另成一方世界的小世界，在这归墟天之内，并不少见。

<br/>　　炎撼世指着灵力波动的方向，焦急的喊道。

<br/>　　不过现在这一群五灵宗圣子的处境可不妙。
<br/>　　而在这大阵之外，大量的魔族，正远远不断的轰击这大阵，每一次轰击，都足以让这大阵，开始变的摇摇欲坠。
<br/>　　显然是这一群五灵宗圣子，已经是支撑了很久的时间了，在这围攻之下，终究有着油灯枯尽的一刻，已经是快要支撑不住了。
