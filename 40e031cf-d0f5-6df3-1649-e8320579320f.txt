　　林峰冲出烟雾，就看到古月稳稳的落在地上，抬头看了眼林峰，然后杀了几个丧尸然后就飞快跑了。
<br/>　　“缩骨功？这种异能都有？不过说实话，这异能不怎么强。”
<br/>　　林峰摇了摇头，异能者到目前为止他也就遇到那么几个，真的正如系统所说，熊猫血这种万中无一的比例，极其珍贵。

<br/>　　林峰把烤肉全部消灭。

<br/>　　今天的几个空投都拿了，外加一个千年僵尸，也算是收获丰厚。

<br/>　　轰隆隆。
<br/>　　意外的林峰听到了一阵阵爆响。
<br/>　　就好像有人在爆破大楼一样的声音。

<br/>　　居然是一个大概四五米高的巨大藏獒。

<br/>　　此时此刻。

<br/>　　“居然是个八阶的存在？”
<br/>　　林峰大感意外，这是他现阶段见到的最强的恐怖存在了。
<br/>　　“究竟那群人在猎杀这种级别的存在。”
<br/>　　林峰立即摸了过去这一看顿时看到了熟面孔，就是之前从他手底下跑掉的那个棒球棒。
<br/>　　“这么看来，这是什么狼窝的人了。好强。”
<br/>　　林峰扫了一眼，这狼窝还真是够厉害。
<br/>　　一个七阶，一个六阶，七个五阶，剩下下最弱的都是三阶的。
<br/>　　可以说简直就是精锐了。
<br/>　　“七阶、六阶，这两个应该就是那个老外说的什么疫苗研究所的研究员了，他们在大爆炸之前就发现了不对劲的地方，发现了丧尸的秘密，算是赢在了起跑线上，所以等级才会这么高。不过都是些黑铁战士而不是异能者。”
<br/>　　林峰眯着眼道。
<br/>　　这是两个年轻人，三十上下。
<br/>　　其中一个戴着眼镜，看上去就是个知识渊博的家伙，但是他下手可一点都不轻，拿着个砍刀疯狂的劈砍。
<br/>　　“这个家伙要不行了！”
<br/>　　“杀了他！”
<br/>　　“今天只要成功干掉这个家伙，每个人五十颗晶石！”
<br/>　　众人立即是咬紧牙关施展了全部的力量。
<br/>　　这藏獒的确凶猛，作为八阶的，更是无敌。
<br/>　　此时此刻，地上都已经有十几个人的尸体了，大部分都要么是脑袋没了，要么是肚烂肠流，非常的血腥。
<br/>　　不过这个藏獒也身负重伤，浑身不知道差了多少弩箭，一些刀口更是不断喷着鲜血。
<br/>　　“八阶怪物的晶石，没理由不抢一枪啊。”
<br/>　　林峰之前在千年僵尸墓里他玩的就是一招鹬蚌相争。
<br/>　　眼下很显然又有机会玩一次鹬蚌相争了。
<br/>　　想到此，林峰就在旁边潜伏下来，准备随时出手。
<br/>　　而场中谁都没有注意到林峰的到来，藏獒也好，人类也罢，双方都是玩命的干。
<br/>　　不过狼窝这边准备的太充分了，他们似乎还在这弩箭上涂抹了毒药之类的东西。
<br/>　　这藏獒哪怕是八阶，但也架不住四十人狂殴。
<br/>　　再又杀了七八个人后最终惨叫一声倒在了地上剧烈的喘气着。
<br/>　　“八阶的怪物！你是我的了！”
<br/>　　狼窝的狼老大眼中露出一抹兴奋，来到那藏獒面前一刀劈开了对方的脑袋彻底的让这条藏獒走上了末路。
<br/>　　“恭喜老大！贺喜老大！”
<br/>　　“恭喜老大！贺喜老大！”
<br/>　　“恭喜老大！贺喜老大！”
<br/>　　其余人都立即狂呼起来。
<br/>　　“哈哈哈哈，今天大家表现的都非常好，我史一表在这里发誓，只要有我史一表吃肉的地方，就有兄弟们喝汤的地方！”
<br/>　　“多谢老大！”
<br/>　　一群人都是立即表示了自己的忠臣，能够跟这样强大的老大干是，也是他们的荣幸。
<br/>　　“哈哈。我倒要看看这八阶的晶石有多大。”
<br/>　　史一表非常兴奋的挖开了藏獒的脑子。
<br/>　　刷刷刷刷。
<br/>　　也就在此时。
<br/>　　几十枚飞镖洋洋洒洒的破空而来。
<br/>　　“啊啊啊啊。”
<br/>　　瞬间十几个小弟全部猝不及防之下被爆头秒杀。
<br/>　　这些都是三阶上下的。
<br/>　　他们刚才都处于懈怠状态，根本没有任何的警惕，一时之间全部倒了下来。
<br/>　　而几个四阶以上的全部凭借着超快的反应闪躲了。

<br/>　　“有人偷袭！”
<br/>　　“妈的，是谁？”
<br/>　　一群人是又气又怒。
<br/>　　史一表楞了一下也是勃然大怒，妈的，杀八阶怪物没有死多少人，这眼下大功告成了，反而倒是直接死了这么多。

<br/>　　又是几十枚飞镖飞去，这一次主要是朝史一表飞去。
<br/>　　“妈的！”
<br/>　　史一表根本来不及去挖晶石立即被逼退。
<br/>　　而林峰也趁这个功夫立即跑了过来。

<br/>　　一个人抬起手中的一把仿六四手枪就是要开枪。

<br/>　　那人立即原地闪躲。

<br/>　　“僵尸？”

<br/>　　狼窝等人都是明显的目瞪口呆，这僵尸他们在电影里看过，亲眼还是第一次所见。

<br/>　　林峰立即下命令。

<br/>　　拿着仿六四式手枪的家伙再次开枪。
<br/>　　不过却被千年僵尸给挡住了，对方不过身子一顿就冲了出去。

<br/>　　那个人彻底的呆住了。

<br/>　　史一表面色难看。
