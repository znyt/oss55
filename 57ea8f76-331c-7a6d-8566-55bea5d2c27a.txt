　　第37章
<br/>　　这一夜的青牛山人人自危，谁都感觉头顶像是悬着一柄刀，不知何时会落下来。
<br/>　　“大哥，后山我又加了两道岗，今晚让弟兄们轮流值夜。”高飞豹从外面进来。
<br/>　　“今天让兄弟们都机灵点，一有风吹草动都给我抄家伙。”说这话的时候，高飞虎下意识摸了摸后脑勺。自从今天见到朱家寨被屠的惨状，他就有了这习惯。
<br/>　　回到山寨后，他立刻加强了警戒，可即便如此，仍然有些不放心，生怕半夜三更在自己睡着的时候，会有柄刀砍掉自己的脑袋。
<br/>　　“军师，你觉得怎样，还有没有其他疏忽的地方？”高飞虎问于虬然。
<br/>　　“大当家这么安排，应该不会有差池，也无非是嘱咐大家小心为上，只要能撑过这三天，等他们从山下经过，应该就没事了。”
<br/>　　高飞虎自然知道于虬然口中的『他们』是谁，他颓然坐在椅子上，道：“本来还想劫了这批寿礼，弟兄们踏踏实实熬过这个冬天，谁能想到，陆哼竟然这么狠，血屠朱家寨，亏得是向朱家寨动手，要是冲我们飞虎寨动手，我现在也脑袋搬家了。”
<br/>　　说到这里，他又摸了摸后脑，忽然想起什么，问道：“程大雷那边什么消息？”
<br/>　　“呦，大哥你快别提他了，那小子直接被吓尿了，听说回山就病倒了，现在还和个娘们似的在床上做月子呢。”高飞豹直接不屑道。
<br/>　　“唉，毕竟年纪太小，那见过这种惨状，也是陆哼太狠，别说他，今天我也被吓着了。”高飞虎说着，又摸了摸后脑，感觉那里飕飕的冒凉气，今夜，注定要失眠了。
<br/>　　……
<br/>　　蛤蟆寨。
<br/>　　李行哉回来后，也将朱家寨发生的事情同李婉儿说了。李婉儿听罢之后，沉默许久方才慢悠悠道：“连孩子都杀，虽然是山贼，可陆哼做事也太绝了些。”
<br/>　　李行哉叹口气：“从前听人说打仗，一场战役十几万人就死了，今天见到才知道，杀人还真不是一件好玩的事情。”
<br/>　　“哥哥，咱们找机会走吧，现在的这里不太安全。”李婉儿。
<br/>　　“嗯，走的时候我们悄悄离开，也别惊动他们。”李行哉道。
<br/>　　“呦，聊天呢，还没睡呢？”
<br/>　　正这个时候，屋门打开，程大雷走了进了。李行哉二人一激灵，也不知道刚才的谈话他听见没有。
<br/>　　“大当家，听说你病了，怎么还过来了？”李婉儿看到程大雷脸色还是有些发白。
<br/>　　“染上点风寒，也不太严重。”程大雷在炕上坐下，冲李行哉道：“还是想找你聊聊。”
<br/>　　“今天聊什么？”李行哉。
<br/>　　“聊聊帝国以前的事吧，随便聊聊就行。”
<br/>　　经过这十几天的交流，程大雷对帝国十三州一百零八城的位置已经有模糊的印象，这几天找李行哉了解的，是帝国的历史。
<br/>　　李婉儿挑亮了灯芯，灯下李行哉对帝国上下三千年的历史侃侃而谈，程大雷认真记在心中。
<br/>　　刚来到这个世界的时候，程大雷可以说对这个世界两眼一抹黑，也不知如今的历史可以和前世那一段时间相对应。如今，听李行哉聊了这么多，总算是对这世界有了自己模糊的印象。
<br/>　　这个世界和程大雷的前世一样，也有盘古开天，女娲造人，然后三皇治世，五帝定伦，之后夏商周春秋五霸天下大乱，一直到秦一统，楚汉争霸，汉立国四百年，东汉末年，天下大乱，十常侍之乱，董卓乱天下……
<br/>　　也就在这个时候，历史出现偏差。是有曹操这个名字的，但他在刺董卓时就被抓住杀死了，没在历史上掀起多大水花。
<br/>　　所以也就没有了后来的三国争霸，而是……十九国分立，逐鹿天下。
<br/>　　程大雷熟悉的那些名士或名将的名字，也是有的有，有的没有。比如没了武侯诸葛，没了刘关张，没了曹魏五虎将……没有赵云和貂蝉。
<br/>　　然后历史就是一团乱麻，十九国纷争不断，打仗，打仗，不停的打仗，与之相随的就是死人，死人，不停的死人……直到大武立国，直到如今。
<br/>　　程大雷根据时间推断，这差不多能对应前世的隋唐时期，但其实也对不上，历史一团乱麻，地理位置上也是千差百错。
<br/>　　“……最后戎族虽然被赶出去了，可这些年还是缕缕犯边，摩擦不断。”李行哉口若悬河，滔滔不绝，说着说着，便聊到三十年前戎族入侵的那场战役。
<br/>　　“帝国一年多少军费投到边军，可还是屡战屡败，败得一塌糊涂。哥哥，你说这么多年，我们和戎族打，怎么就打不赢呢。”李婉儿感慨道。
<br/>　　“还不是边军将领都是一帮酒囊饭袋，只知道中饱私囊，敷衍军事！”说到这里，李行哉显得极其气愤。
<br/>　　程大雷投过来目光：“你这么生气干什么？”
<br/>　　“哈，不过是有感而发而已，这些年和戎族的作战，可以说是十战十败，而且还虚报军功，一场仗死了几百人，反而要上报帝国，说杀敌五十……这样报来报去，胜仗越来越多，边军的战斗力却越来越低。”李行哉说着看了程大雷一眼：“大当家，依你之见，帝国和戎族的作战，帝国怎么才能打赢？”
<br/>　　程大雷打了个喷嚏：“我怎么知道，我只是个山贼啊。”
<br/>　　李行哉道：“曾有人对我说过，天下的军队可以分为三等，最低等为了生存而战，每个士兵都只顾及自己性命，一遇敌军便做鸟兽散；第二等是为了利益而战，打胜仗时士气如虹，一旦遇到败仗，便人心惶惶。第一等当是为了荣耀而战，以战死沙场为荣，那才是真正的精锐之师呐。”
<br/>　　“这话是对你说的？”程大雷忽然道。
<br/>　　“呃……”李行哉抬起头，看到程大雷一双大眼炯炯有神的望着自己：“一个老人而已。”
<br/>　　“那么你确定，他不是在忽悠你？”程大雷认真问。
<br/>　　“……”李行哉再次无语，他如何告诉程大雷，对自己说这番话的人，是当今帝国元帅。
