　　　　用膳结束后，慕清霄几人心满意足的回到西苑，赶了一天的路，众人都有些许疲惫。
<br/>　　曾书书嬉笑，道：“今晚就在这里休息吧，我问过老板，在过半个时辰，酒楼门口有烟花节目，免费的夜宵，如果有兴趣的话大家一起去？”
<br/>　　慕清霄微微摇头，道：“烟花节目就算了，我们明日还要赶路前往空桑山，说不定什么时候就会碰到敌人，还是早点休息，养精蓄锐。”
<br/>　　“慕师弟说的有道理，不能因小失大，还是早些休息吧。”张小凡应了一声，曾书书摸了摸鼻子有些尴尬，陆雪琪瞪了慕清霄一眼，‘砰’一声将房门关上。
<br/>　　林惊羽道：“小凡，今晚好好休息，我先走了。”
<br/>　　两个大男人眉来眼去，谁能受得了，慕清霄嘴角抽了抽，若非他知道张小凡喜欢女人，恐怕会将两人当成基友。
<br/>　　等林惊羽和曾书书离开，张小凡将怀中的‘六合镜’拿了出来，道：“慕师弟，你帮助我获得七脉会武冠军，这是当初的承诺。”
<br/>　　慕清霄并没有接过六合镜，想到张小凡身上已经没有噬血珠，倒不如将此宝物给他当保命法宝，反正自己也用不上。
<br/>　　若是张小凡死了，引起蝴蝶效应可就不妙了。
<br/>　　“张师兄，六合镜你就留着吧，到时候遇上危险也够保命，此宝物对我来说，帮助不大。”
<br/>　　“这...这怎么能行，当初...”
<br/>　　“行了行了，一个大男人怎么唧唧歪歪跟娘们似得，你若是不要扔了便是。”慕清霄摆了摆手，转身离去。
<br/>　　...
<br/>　　...
<br/>　　此时已是深夜，仰望苍穹，繁星满天，皎洁的圆月挂在天边，夜风习习，隐约带着一丝芬芳。
<br/>　　慕清霄叹息一声，地球上的妻子们，是不是也能看到相同的美景呢？
<br/>　　抛开脑海中的杂念，慕清霄沉吟起来，呢喃道：“伤心花，那少女的身份就一目了然了，只是没想到这么快就遇到了，也不知道剧情轨迹会不会变化。”
<br/>　　小径曲折幽深，通往前方不知名处，小道两旁青草灌木，各色花朵变得开放，微风拂面，带来丝丝凉意。
<br/>　　忽然，慕清霄脚步一顿，抬头望去，只见一名水绿衣衫少女，站在月光下，肌肤如雪，清丽无双，宛若仙女一般。
<br/>　　那少女察觉到有人来此，将手中鲜花收起，转过臻首，一双明亮的大眼睛盯着慕清霄，道：“喂，你叫什么名字，这么晚来此有何事？”
<br/>　　“深夜赏花，姑娘真是好雅兴，我叫慕清霄，至于为何来到此处，好像与姑娘无关吧。”
<br/>　　少女闻言有些薄怒，道：“...你这人怎么如此没礼貌，先前用目光猥琐我，现在又出现在此处，你不会是跟踪我来的吧，传说中的跟踪狂？”
<br/>　　慕清霄嘴角抽了抽，这也太特么自恋了，老子只是看伤心花，怎么就用目光猥琐你了，我确实是打算去窃玉偷香来着，可对象也不是你啊！
<br/>　　只是碰巧路过这里，谁特么知道你半夜三更不睡觉，反而跑来这里赏花？
<br/>　　“姑娘若是无事，在下就告辞了。”说罢，慕清霄就准备离开。
<br/>　　走了没几步，前方就走来一名轻纱蒙面的女子，正是刚才与少女坐在一起的金丹修士，只是身上的紫色长裙已经换上了黑色丝裙，在朦胧也夜晚，如同幽灵一般。
<br/>　　慕清霄只是惊鸿一瞥，轻纱蒙面女子娇躯一颤，脸色瞬间变得煞白，一丝丝冷汗自她的额头滑落。
<br/>　　从始至终，慕清霄只是瞥了她一眼，蒙面女子也未曾说过一句话，只是身躯僵硬的站在原地，眼中注视远去的神骏青年。
<br/>　　待慕清霄的背影消失在黑夜中，她紧绷的心弦才放松下来，狠狠的吸了几口新鲜的空气，露出一副惊魂未定的模样。
<br/>　　那是怎样的眼神，宛若君王，看自己就像是看待一只蝼蚁一样，使她连喘气都不敢，哪怕是面对鬼王，她都没有如此失态过，那名青年到底是谁，怎会如此恐怖？
<br/>　　许久，她才回过神来，发现自己已是浑身冷汗，无奈向花园深处走去，很快就看见那名绿衣少女，依然站在原地，手里把玩着一朵折下的鲜花。
<br/>　　绿衣少女抬起头来，并没有吃惊，微笑道：“幽姨，你回来了。”
<br/>　　蒙面女子看了她手中的鲜花一眼，面纱情动，微微点头。
<br/>　　“我刚刚已经仔细查探过一番，刚才那五人都是青云门的弟子。”她的声音回荡在花园深处，幽深飘荡，虽然轻柔，却带着一丝鬼气。
<br/>　　“虽然是青云门的弟子，但都是未曾见过的新面孔，看来都是年轻一辈，还有，若是遇到那名样貌神骏的青年，切记不要招惹，知道了吗？”
<br/>　　说到慕清霄的时候，幽姨眼中闪过一抹微不可查的惊惧！
<br/>　　绿衣少女眼中有些困惑，刚才的青年到底是什么人，居然能让幽姨谈之色变，虽然好奇，但还是乖乖的应了一声。
<br/>　　“嗯，我知道了，刚刚过去的那名青年，叫做慕清霄，真是奇怪的名字。”
<br/>　　蒙面女子看了她一眼，严肃道：“记住，千万不要去招惹他......碧瑶，许久没见你赏花了。”

<br/>　　离开花园后，慕清霄并没有多做停留，而是来到西苑最左侧的房门口，‘咚咚咚’轻轻敲响房门。
<br/>　　“谁？”
<br/>　　旋即，客房内传来一道清冷悦耳的声音。
<br/>　　“雪琪，是我。”
<br/>　　客房内沉默片刻，“吱”一声房门打开，陆雪琪早已换下白衫，湿漉漉的秀发散发着幽香，上面水渍还未散去，显然刚沐浴不久。
<br/>　　嗅了嗅空气中的幽香，慕清霄不管不顾，直接推门而入，陆雪琪压着房门，但力气哪有他大，根本拦不住。
<br/>　　见他脸色一如既往的平淡，陆雪琪差点没气死过去，真把这里当自己房间了，这流氓先前就色眯眯的，半夜三更来自己房间，肯定没安好心。
<br/>　　慕清霄夜闯她的客房，心里不免有些心虚，臻首向外面看了看，连忙将房门关上，不知何时，一双大手已经放在她的腰间。
