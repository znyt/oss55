        &lt;h3&gt;第三卷第一百八十四章  张家来人了&lt;/h3&gt;
<br/>
<br/>        因为俩个孩子生病了，新年陈子昂哪里都没有去，给父亲母亲打了电话，告诉了情况，也给财叔和春姨放假了，家里就是他们母子三人。
<br/>
<br/>        每年家里都会贴对联的，之前是财叔贴，今年财叔走得早，就是他们母子三个人贴，当陈子昂准备贴最后的一条对联的时候，家里来了三个人。
<br/>
<br/>        陈子昂看了一眼张函和他身后的弥青云，心里想着:“来得真早。”
<br/>
<br/>        她没有见过张函的爸爸，但是只有他们三个人，估计也错不了。
<br/>
<br/>        “子昂。”张函看着手中拿着对联的陈子昂和团团，嘟嘟手中拿着胶水。
<br/>
<br/>        “爸爸。”俩个孩子喊张函，可是看到张函身后的俩个人时，便警惕的看着他们。
<br/>
<br/>        “张总，来的可真早。”陈子昂冷冷是说。
<br/>
<br/>        “团团，来，给爸爸来贴。”张函伸手拿过团团手中的对联，张沛尧也拿过陈子昂手中的对联，父子俩个人将对联贴好。
<br/>
<br/>        全程一片充满。
<br/>
<br/>        “谢谢张总，请吧。”陈子昂打开门，请张家三人进家门。
<br/>
<br/>        弥青云从见到孩子的那一刻就一直盯着孩子看，她发现她竟然见过孩子。
<br/>
<br/>        弥青云进了房间之后，想要去拉团团和嘟嘟的手，俩个孩子躲开了。
<br/>
<br/>        他们也认出来了，这个老奶奶是那次丢了东西的老奶奶，团团心里想:原来她就是爸爸的妈妈，早知道她是爸爸的妈妈，他就不去给她捡东西了。
<br/>
<br/>        弥青云有一瞬间的尴尬，她看了一眼陈子昂，这孩子比几年前成熟了很多，原来那种青涩已经不见了，取而代之的是沉稳和一种强势。
<br/>
<br/>        “陈虞、落妍洗手去房间，家里来客人了，妈妈陪客人说说话。”陈子昂手中拿着茶杯，团团接过妈妈手中的茶杯放在茶几上、看了一眼张函转身拉着嘟嘟离开了。
<br/>
<br/>        “子昂，这个是我爸爸，这个是我妈妈。”张函介绍着。
<br/>
<br/>        “叔叔好，阿姨好。”
<br/>
<br/>        “子昂，对不起。”弥青云的声音有点发颤。
<br/>
<br/>        “已经过去了，不用说对不起。大家来我家有什么事情就直说吧。”陈子昂不喜欢拖拖拉拉，遮遮掩掩。
<br/>
<br/>        张沛尧看着陈子昂，他是第一次见陈子昂，陈子昂看着他们心情平和，似乎能料到他们会来，陈子昂的平和已经超过了她本身的美丽，让张沛尧刮目相看，他的儿子错过的不仅仅是一个漂亮的妻子。
<br/>
<br/>        “子昂，我们前几天才知道你为我们张家生了俩个孩子，对于我们之前所做过的事情我们深表抱歉，我们今天来呢不是想和你争抢孩子的抚养权，我们只是来看看我们的孩子。”张沛尧说。
<br/>
<br/>        陈子昂沉默不语，她知道张沛尧的话没有说完。
<br/>
<br/>        “当然，我们心里还是希望孩子能和我们在一起，但是，张函的身份不允许，陆家有可能已经知道你有孩子了，为了孩子的安全，我们来和你商量一下，后期该怎么办。”
<br/>
<br/>        “孩子是我的，和你们没有任何关系，只要你们不来找我们就好了。”陈子昂说。
<br/>
<br/>        “子昂，就算是我们不来找你们，你觉得你们会安全吗？”弥青云问。
<br/>
<br/>        “为什么我碰见你们就会要不安全了呢？”陈子昂觉得无语，这么多年他们不是安安全全的过来了吗？
<br/>
<br/>        为什么她碰见张函就会有很多的矛盾要去想要去解决？她以为他们不会再有任何纠葛了，这一刻她有些憎恨自己的优柔寡断和感情用事了。
<br/>
<br/>        留下张函和她的照片，让俩个孩子知道了他们的父亲，为了寻找谷强加快耗材进度导致公司出现危机，为了想报答江俞轩这几年在公司的付出等江俞轩来命名耗材名称，导致耗材计划推迟。这一切都是怪她自己太纠结私人情感了。
<br/>
<br/>        “子昂，我们是来解决问题的。”张沛尧看着陈子昂“我们来分析一下现在你和孩子的处境，我们知道了孩子是张函的，同样别人也会知道孩子是张函的，你现在是一个公众人物，张函也是一个公众人物，还有一点，你的二嫂告诉你阿姨和蔡少芬，这俩个孩子是江俞轩的，如果这件事情被有心人利用的话，你们的处境将会很困难，影响不是一般的大，你的公司现在正在准备上市，江家已经是上市公司。你们的每一步路都不能有差错。”
<br/>
<br/>        “这些不用你们担心，我过完年就将孩子送走。”陈子昂听说冯睿告诉了弥青云和蔡少芬江俞轩的一句玩笑话，心里有些气，冯睿真是秉性不改，一直用她做踏脚石。
<br/>
<br/>        “还有，我的俩个孩子一个姓陈，一个姓落。所以以后的事情就不劳几位操心，如果你们真的为了我们母子三人好的话。”
<br/>
<br/>        “子昂，你准备将孩子送到哪里？”张函听说要将孩子送走，心里有些难过，他以为他认回孩子就会和孩子能光明正大的在一起，没有想到会是分离。
<br/>
<br/>        “恕难奉告！”陈子昂是再也不会告诉他们孩子会去哪里的。
<br/>
<br/>        “子昂，我可以陪着孩子。”弥青云开口，她希冀的看着陈子昂。
<br/>
<br/>        “不必，谢谢！”陈子昂冷冷的拒绝了，弥青云会那么好心？当初的羞辱历历在目。
<br/>
<br/>        “子昂，请相信我，毕竟我是孩子的奶奶，我也马上退休了，找个病假的理由，我出去陪着孩子吧？”弥青云现在是满心满脑子的悔不当初啊，看着俩个可爱的孙子，恨不得立马抱在怀里。
<br/>
<br/>        想想陆玉，想想陆家，为了苟延残喘他们竟然拿孩子的幸福来交换他们的安宁，实在是汗颜。这么十几年看着自己的孩子如此痛苦，弥青云的心里在滴血。
<br/>
<br/>        “不，谢谢阿姨。既然不能让人知道孩子是张函的，你们就更不能和我们来往。”陈子昂看着弥青云也算是真心的话坚定的拒绝了。
<br/>
<br/>        张函的父亲分析的很对，他们不能因为孩子引发一系列的风波，而且如果曝光了这件事情尤其是对孩子的成长不好，唾沫淹死人啊。
<br/>
<br/>        现在唯一的办法只能是将俩个孩子送出去，雪藏起来，等孩子大一些了再说，或者她带着孩子出去，但是想到还没有找到哥哥，想到现在的公司，陈子昂的心在滴血，目前她还是公司的领头人，她需要为了公司的所有员工负责。
<br/>
<br/>        “子昂，感谢你的理解，我们张家欠你的只有以后找机会回报了。”张沛尧站起来“我们可以和孩子说几句话吗？”
<br/>
<br/>        “团团，嘟嘟！”陈子昂喊着俩个孩子，既然张家来了，她没有必要和他们结仇，原本孩子留着张家的血，说说话又何妨？
