　　NGL边境检查站，数名工作人员在不停的忙碌。
<br/>　　“最近怎么有那么多人进入ngl？”其中一个大胡子中年人一边操控仪器，一边和旁边的助手闲谈。
<br/>　　“里面可别出什么事故。”年轻女助手一脸担忧：“这几天进去的人，都不太一样。”
<br/>　　“你也看出来了？”中年人笑了笑。
<br/>　　“也不能说看出来，就是不太敢站在那些人身边。看着挺普通，却让人害怕。”女助手皱着眉头。
<br/>　　“当然让人害怕。”大胡子笑了笑，脸上带着神往：“仅仅几个人，甚至单枪匹马，就敢进原始森林一样的ngl，这些人很可能是猎人！猎人协会的正式猎人！实力强大啊。”
<br/>　　“猎人都来啦？ngl境内难道真的出事啦？”女助理一脸惊讶。
<br/>　　“出不出事和我们没关心，我们只是被雇佣来，负责检查过境人员的。反正又不在ngl生活，只要记录过境人员，做好监测就好。其他的和我们没关系。”
<br/>　　“不会有人强闯吧？”女助理不安道。
<br/>　　“强闯？不会！别看咱们这里简陋，我告诉你，这边境上可藏着不少军队，火力强着呢！谁敢强闯！”大胡子脸上带着得意。
<br/>　　“火力？ngl不是不允许存在机械造物的吗？”女助理大吃一惊。
<br/>　　“只要不在ngl境内，放在境外不就行了。不用怕，没人敢强闯的……”
<br/>　　轰轰！哒哒哒……
<br/>　　剧烈的爆炸声，密集的开枪声，陡然想起。
<br/>　　两个人脸色猛然一变，检查站内的一群人，呼啦啦连忙走到窗口向外张望。
<br/>　　一群白衣士兵，举着机枪、火箭筒，不断的往天上射击。
<br/>　　爆炸声不断响起。天空顷刻就布满硝烟。
<br/>　　一个遮天蔽日的黑影缓缓从空中掠过，庞大的体型，黝黑坚硬的鳞甲，狰狞的尖角。
<br/>　　“这……这是什么怪物！”
<br/>　　不只是检查站内的人恐惧，就变手持枪械的凶悍士兵，也满脸惊惧。他们哪里见过这种怪物！
<br/>　　以火箭弹的射程，根本就达不到对方，子弹更没戏！只能眼睁睁看着这个庞然大物悠然过境。
<br/>　　“快看，那怪物头上有人！”女助理眼睛大睁，捂着嘴满脸的不可置信。
<br/>　　这……竟然有人能降服这种怪物！
<br/>　　苏重盘膝坐在龙头之上。
<br/>　　身下这头身长百米的魔龙，自然不可能是真的。是和线分身类似的念兽。念线的特殊性，让苏重的能力变化万千。能制造和本体一模一样的分身，就能制造栩栩如生的魔龙！
<br/>　　他自己当然能飞，但那里比得上让魔龙带着飞来的悠闲。以他如今的实力，除了即将诞生的嵌合蚁王，其他蚂蚁很难引起他的兴趣。他有大把的时间赶路。
<br/>　　……
<br/>　　ngl境内，某处山崖上，一个隐秘洞窟坐落在山崖顶端。
<br/>　　站在洞**，能够清晰看到远处日渐庞大的嵌合蚁巢穴。
<br/>　　下方森林内，布满白色烟雾，伸手不见五指。
<br/>　　诺布和莫老五两人坐在山洞边沿，悠然的看着下方的森林。
<br/>　　嘟嘟嘟……
<br/>　　“喂，会长。”诺布接通手机。
<br/>　　“在放一些进来吧，我需要继续热身。”尼特罗平稳的声音中，带着丝丝兴奋。
<br/>　　“嚯，老爷子玩的很开心那。”莫老五咧嘴嘿笑。
<br/>　　诺布抚了抚眼睛，自信道：“除了数量多，这些蚂蚁个体实力并不怎么样。”
<br/>　　他的能力特殊，叫做四次元公寓。是一种空间类能力，拥有一处独立空间。
<br/>　　莫老五的能力，则是下方的那些浓雾。
<br/>　　借助浓雾掩盖，诺布能悄无声息的把蚂蚁拉进空间内，然后让尼特罗来动手收拾。这些天来，他们已经用这种方法消灭了数百的蚂蚁。
<br/>　　“嗯？那些蚂蚁发现不对了。”莫老五通过浓雾感知着森林中蚂蚁的动向：“已经在有组织的地毯式搜索，蚂蚁相互之间保持着很小的距离。”
<br/>　　“没关系，以会长的实力，一次性多拉进去几只，他反而会高兴。”诺布浑不在意。
<br/>　　“咦？有人进入蚂蚁包围圈！”莫老五心中一惊。
<br/>　　“是哪个混帐猎人，竟敢闯进这里！”诺布脸色难看。
<br/>　　他们不怕嵌合蚁，是因为个人实力强大。但一般的猎人，对付蚂蚁却不那么容易。如果被围攻，后果不堪设想！
<br/>　　“真是乱来，我去把他救出来。”诺布身前陡然出现一个黑色圆盘，一脚迈进圆盘，身影迅速陷落消失。
<br/>　　苏重漫步在密林之间。ngl的森林实在是太茂密了，到处都是参天大树。根本就看不到现成的道路，只能他自己开辟。
<br/>　　走着走着，身边逐渐起了丝丝雾气。
<br/>　　苏重第一时间察觉其中的不同寻常。这雾气中有着淡淡的念力波动。
<br/>　　他瞬间就感觉到异样。感知迅速放出，方圆一公里之内，所有影像立刻进入脑海。
<br/>　　笼罩着整个森林，足有十公里的地域内，全是这种白色浓雾。那种念的波动，越发清晰。
<br/>　　这是念能力？
<br/>　　在浓雾之中，还有大量奇形怪状的嵌合蚁，正在成地毯式搜索前进。
<br/>　　看他们的方向，正好朝着苏重而来。
<br/>　　“这里有人！”一个长着鹿脑袋的嵌合蚁，从浓雾中猛然窜出，正好看到迎面走来的苏重，立即发动独特的心电感应，开始召唤其他蚂蚁。
<br/>　　苏重不懂蚂蚁的语言，但魔龙七段淬炼出的强大身体，让他清晰的波捉到了这股波动。
<br/>　　知道对方实在呼唤同伴，苏重不以为意。这些天，他沿着蛛丝马迹，一路朝嵌合蚁巢穴前进，却没遇到任何一只嵌合蚁。没想到这次，一下就碰到一大群。
<br/>　　“怪不得见不着嵌合蚁，原来都被拖在了这里。”
<br/>　　“找到你了，狡猾的人类！”鹿头嵌合蚁竟然开口说话。
<br/>　　“果然是吸收了人类基因，具备了人类语言的能力。”
<br/>　　嵌合蚁能够吸收食物中的优秀基因。也就是所谓的摄食交、配，既然能说话，必然是曾经捕食人类，并吸收了人类的基因。
<br/>　　“有人类的智力，有更坚硬强大的身躯，加上更容易觉醒念能力的体质，这就是战争兵器！”苏重恍然，有些了解了比杨德等人的想法。
<br/>　　嵌合蚁试验，是对黑暗大陆物种的研究试验，是针对尼特罗的阴谋，同时也是在积蓄兵力！
<br/>　　用嵌合蚁的能力吸收人类、野兽的各种基因，产生易于觉醒念能力的优势兵种。牺牲数百万人，供养嵌合蚁繁殖壮大。最后只要有那么几个操控系的念能力者，就能很快掌握一只庞大的念力部队！
<br/>　　“真是打得好算盘啊。可惜，敢惹我，我偏不让你们如意！我让你竹篮打水一场空！”
<br/>　　把所有蚂蚁全灭了？嗯，这可是个不错的主意。苏重咧嘴狰狞一笑，露出一口细密尖牙！
