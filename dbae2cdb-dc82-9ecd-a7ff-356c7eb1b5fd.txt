　　“咕噜...!”
<br/>　　陡然，死寂的岩浆世界中，一道黑色闪电掠过，使得压抑的环境中，响起一阵沉闷的声音。
<br/>　　“真是够深的，此处距离地面，最起码有数千米了吧。”
<br/>　　双翅一阵，周围岩浆一阵翻滚，慕清霄抬起凤首，打量了一眼四周，周围全都是一模一样的景色，全是赤红色一片。
<br/>　　低头一看，依旧是无边无际的岩浆领域。
<br/>　　类似的情景，慕清霄曾经在风云位面，凌云窟地底经历过，但凌云窟中的岩浆池，相比此处，简直是小巫见大巫。
<br/>　　哪怕是现在，他所在的位置，都已经超过了凌云窟中岩浆池的深度。
<br/>　　这里到底有多深，他并不知道，但想要继续深入，就必须拥有强横的实力。
<br/>　　否则，下面炽热的高温，以及粘稠岩浆所带来的压迫力，就不是普通修炼者能承受的。
<br/>　　按照慕清霄的推测，想要继续朝下，最起码也需要斗宗级别的实力。
<br/>　　如果斗气不雄浑的斗皇，想要往下，恐怕就祸福难料了。
<br/>　　当然，慕清霄完全是个例外，哪怕不变化成黑凤，他体内生生不息的阴阳灵气，不管他在其中待多久，都没有任何问题。
<br/>　　岩浆世界中，死一样的寂静，四周只有粘稠的岩浆，深浅不知，若是胡乱闯入其中，谁也不知道会通往何处。
<br/>　　若是他猜的不错，这整个岩浆世界，应该是类似于古界，魂界一样的小世界，否则不至于如此庞大。
<br/>　　而且，最底层还有一座陀舍古帝洞府，想想也就释然了。
<br/>　　别人或许在其中会迷失，但慕清霄不同，他能根据温度的变化，异火的气息，在其中穿梭。
<br/>　　想要找到目标，轻而易举。
<br/>　　不过，陨落心炎似乎是察觉到了危机，一直在四处逃窜，而慕清霄则一直在与它拉近距离。
<br/>　　“终于不跑了吗？想要耗死我，你终究是太嫩了点。”
<br/>　　这时，慕清霄眼中闪过一丝戏谑，旋即化作一道黑色闪电，继续向下掠去。
<br/>　　茫茫无尽的岩浆世界中，周围呈现一片赤红，而在这片赤红的，飞速下潜的黑色身影，格外的显眼。
<br/>　　几分钟后，慕清霄双翼一振，停在了半空中。
<br/>　　映入他眼中的，是一条由精纯的火属性能量，凝聚而成的赤红色巨蛇。
<br/>　　巨蛇通体赤红，面露狰狞，在它的额头上，有着一缕无形的火焰不断闪烁，周围的温度，再度攀升。
<br/>　　“嘶嘶...!”
<br/>　　巨蛇口中发出一阵嘶鸣声，声音中充满了愤怒与恐惧。
<br/>　　它已经诞生灵智，完全能清楚，对面的生物是何等恐怖，仅仅是威压，便使得它不敢靠近。
<br/>　　但是后者穷追不舍，实在是让它愤怒不已，俗话说兔子急了也咬人...
<br/>　　于是，伴随着它的嘶鸣声，一团略微偏白的无形火焰，突兀而现，随着火焰的出现，周围的岩浆，在瞬间沸腾起来。
<br/>　　岩浆世界中，气泡翻滚不休，紧接着爆裂开来，释放着一股高温与火毒。
<br/>　　“高温火毒，这种东西对我可没用，乖乖束手就擒吧。”
<br/>　　慕清霄凤眼中闪过一丝不屑，不顾周围的温度与火毒，双翼一振，以迅雷不及掩耳之势向陨落心炎掠去。
<br/>　　岩浆翻滚间，利爪已经朝着它巨大的蛇身爪了过去。
<br/>　　利爪接触蛇身的瞬间，就像是抓住了橡皮筋一样，极其的坚韧，陨落心炎也没想到，他的速度会如此迅捷。
<br/>　　反应过来，嘶鸣声再次响起，巨大的蛇身不断扭动，想要以此将慕清霄给缠住。
<br/>　　“找死！”
<br/>　　冷笑一声，无尽的神火自体内狂涌而出，周围的岩浆都有退避的趋势。
<br/>　　携带着神火，利爪猛的用力，伴随着‘噗’一声闷响，巨大的蛇身瞬间被利爪给刺穿。
<br/>　　黑色火焰蔓延而上，陨落心炎凄厉的嘶鸣起来，声音中满是惊惧，面对慕清霄，它没有任何的应对办法。
<br/>　　高温不管用，火毒也不管用，简直就是它的克星，不过，它似乎不甘心就此放弃。
<br/>　　只见巨大的蛇身上冒出无形火焰，想要抵挡，可就在无形火焰接触到神火的瞬间，立刻就被吞噬殆尽。
<br/>　　伴随着神火包围网形成，巨大的蛇身已经被吞噬的差不多，而在中心位置，却亮起微弱的荧光。
<br/>　　周围依旧是赤红色的岩浆世界，但随着巨蛇的身体消失，神火中心区域却是变得透明起来。
<br/>　　在那荧光之中，盘着一条巴掌大小，有些偏向乳白色的火焰小蛇。
<br/>　　小蛇眼瞳中充斥着无形火焰，眼珠转动间，看上去颇为灵动，很明显，这是一条拥有灵智的小蛇。
<br/>　　黑炎闪烁间，慕清霄单翼一卷，神火就包裹着火焰小蛇漂浮到了面前。
<br/>　　小蛇身躯一颤，抬起头颅望着眼前的生物，眼中尽是恐惧。
<br/>　　感受到其中传来的温度与能量，慕清霄嘴角牵起一丝笑意：“这就是陨落心炎的本源吗？”
<br/>　　说着，嘴巴一张，就将陨落心炎吞入腹中。
<br/>　　随着陨落心炎入体，神火收敛，慕清霄就化作人形，盘坐在岩浆世界中，开始炼化陨落心炎。
<br/>　　...
<br/>　　...
<br/>　　随着时间的推移，慕清霄如同老僧入定一般，静静的盘坐其中。
<br/>　　两个时辰后，一股无形的波动自他体内，向四周扩散而出，在岩浆底部掀起一阵暗流。
<br/>　　剑眉一颤，眼眸中闪过一丝精茫，慕清霄伸出右手，神火猛的冒出。
<br/>　　“身体素质全方位的提升，九霄琉璃焱的威力也再度变强了一些，化神九重初期，距离练虚期就差最后一步了。”
<br/>　　想到此处，无尽的神火自体内涌出，黑凤再现，慕清霄伸展了下羽翼，惊喜道：“果然又大了一圈，都快近三丈了，力量也增强了数倍。”
<br/>　　羽翼一挥，左侧的岩浆猛的翻滚起来，一阵阵岩浆气泡向上涌去。
<br/>　　“好强的力量，化神九重初期，化身黑凤的我，哪怕是对上高阶斗宗，也完全不惧。”
<br/>　　只是，让慕清霄没想到的是，陨落心炎居然只让他提升了一小层境界。
<br/>　　但想到，他突破所需要的能量，本就比普通修炼者要多的多。
<br/>　　化神九重初期到巅峰，估计就需要一股庞大的能量，想要突破练虚期，就需要更庞大的能量了，他的身体，简直就是无底洞。
