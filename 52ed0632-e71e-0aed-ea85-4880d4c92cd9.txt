　　“卢兄，你也知道在下来自小地方，对于高深的武道境界，了解的并不多，还请告知这武尊境之上，还有什么境界？”
<br/>　　房间中，苏莫与卢云仿佛老朋友一般，相谈甚欢，他开始询问关于一些武道方面的情况。
<br/>　　“武尊境之上，还有两个大境界，分别是武圣境和虚神境，武圣境的圣者在我们古灵星河，那是巅峰般的强者，至于虚神境的大能，我们古灵星河并不多，很少能见到！”
<br/>　　卢云面上充满唏嘘之色，继续道：“我们都只是普通人物，无论是武圣境的圣者还是虚神境的大能，对于我们来说，都是遥不可及的存在。”
<br/>　　苏莫闻言深深的吸了口气，武圣境、虚神境，他的路还很长远啊！
<br/>　　目前，他才是武皇境九重的境界，连武帝境都没有达到，距离武圣境与虚神境，可谓是天与地的差别。
<br/>　　“那虚神境之上呢？”苏莫又问道。
<br/>　　“不清楚，传闻达到虚神境之上，就能成就神位，长生不死，但具体如何，没有人知道！”卢云摇了摇头道。
<br/>　　“苏兄，你到底来自哪里？为何连这些基本的武道境界都不知？”卢云好奇的问道，现在他心中越来越疑惑了。
<br/>　　按理来说，苏莫就算出生再差，再乡巴佬，这些人人都知道的武道境界，应该也听说闻吧！
<br/>　　现如今，不管苏莫问什么，卢云都是知无不言，争取和苏莫搞好关系。
<br/>　　因为，说不定几日后，苏莫就是公会的炼器师了。
<br/>　　“呵呵，我的家乡比较偏僻，以前从未出来过，最近才开始出来历练。”苏莫笑着说道。
<br/>　　“原来如此！”卢云恍然的点了点头。
<br/>　　少倾，苏莫沉吟一番，再次问道：“卢兄，你可知道巫族？”
<br/>　　“巫族？”卢云闻言一怔，随即点了点头，道：“当然知道，巫族是我们古灵星河的大族，是名列太古万族榜上的种族。”
<br/>　　“太古万族榜？”苏莫闻言面带好奇之色。
<br/>　　“不错！”卢云点头，随即手掌一翻，手中出现了一块玉简，递向苏莫，道：“这里面便是太古万族榜，记录了整个洪荒宇宙最强大的一万个种族，此玉简便送于你了！”
<br/>　　苏莫没有拒绝，接过玉简，便凝神查看了起来。
<br/>　　玉简内，的确是记录了一万个种族的名称，并且还有先后的名次。
<br/>　　第一，混沌族！
<br/>　　第二，古神族！
<br/>　　第三，古魔族！
<br/>　　第四，冥族！
<br/>　　银月狼族！
<br/>　　圣猿族！
<br/>　　玄冰族！
<br/>　　……
<br/>　　苏莫粗略的查看了一番，这所谓的太古万族榜，很快他便看到了巫族的排名。
<br/>　　第三百八十五名，巫族！
<br/>　　苏莫不免心中一震，巫族居然如此之强，在太古万族榜中，都能排在第三百八十五名！
<br/>　　少倾之后，苏莫看完之后，收起了玉简，心中稍稍有些沉重了起来。
<br/>　　巫族越强，对他便越不利。
<br/>　　“怎么？苏兄难道和巫族之人有过节吗？”卢云好奇的问道，他想到苏莫之前说受到歹人袭击，精神力大损，不禁开始暗暗猜测了起来。
<br/>　　“这倒是没有！”
<br/>　　苏莫摇了摇头，解释道：“之前在城中，听到有人谈论巫族，说是非常强大，不免心中好奇而已！”
<br/>　　苏莫当然不会告诉对方，他和巫族有大仇，以免引起不必要的麻烦。
<br/>　　“这是当然，巫族作为太古万族榜上的强族，族中虚神境大能都不止一位，当然强大！”卢云说道。
<br/>　　苏莫心中暗叹，这巫族居然如此的不简单，虚神境的大能都不止一位！
<br/>　　他以后的路，怕是不好走啊！
<br/>　　“对了！卢兄，这太古万族榜之上，怎么不见人族呢？难道人族很弱，上不了榜？”苏莫想到太古万族榜上并无人族，不免疑惑了起来。
<br/>　　“当然不是！”
<br/>　　卢云微微一笑，道：“这是因为人族只是一个统称，很多种族都是从人族中分化而出！”
<br/>　　“统称？”苏莫依旧不解。
<br/>　　卢云见此，详细解释道：“就比如你刚才问的巫族，在太古年代也是人族，只不过后来形成了自己独特的体系，从人族中分化而出，才形成了如今的巫族。”
<br/>　　苏莫闻言恍然，这说的简单点，就像是地球上的亚洲人、欧洲人和非洲人。
<br/>　　虽然种族不同，但都是人类。
<br/>　　“那古灵星河之内，都有那些强大的种族？”苏莫又问道。
<br/>　　“古灵星河之内，最强大的种族，莫过于巨灵族、飞灵族、巫族、黑鳞族等十数个种族了！”
<br/>　　卢云说道，言罢他又补充道：“当然，我们普通的人族，也是丝毫不弱，只不过，普通的人族分为了一个个势力，实力较为分散罢了！”
<br/>　　苏莫点了点头，通过和卢云一番交谈，他对于这无尽星空，古灵星河，都有了大概的了解。
<br/>　　“卢兄，在下的事情，还需多多费心了！”
<br/>　　又聊了片刻之后，苏莫长身而起，婉转的下了逐客令。
<br/>　　“好说，这点小事，要不了一日时间便能搞定！”卢云同样站了起来，满脸笑意的说道。
<br/>　　苏莫让卢云购买的炼器材料，正是重渊斧的炼制材料，这让卢云心中疑惑不已。
<br/>　　这个时候，苏莫要这么多炼制重渊斧的材料做什么？
<br/>　　难道是考核之时没有把握，现在想临时抱佛脚？
<br/>　　卢云暗暗猜测，不过他也没有多问，反正他现在不能再得罪苏莫了，免得自己日后遭到报复。
<br/>　　在卢云离开之后，一直在一旁静听的秦不死，不由的叹了一口气。
<br/>　　“苏莫，看来外界比我们想象的要复杂的多！”
<br/>　　秦不死面带忧虑之色，继续道：“没想到巫族居然如此强大，以后我们怕是步步艰辛。”
<br/>　　“无妨！”
<br/>　　苏莫摇了摇头，道：“只要避过了这次危机，天大地大巫族想找到我们，也不容易！”
<br/>　　苏莫倒是没有太多的畏惧，敌人越强，他越是不会服输！
<br/>　　“哎！希望吧！”秦不死长叹了口气。
<br/>　　随后，两人也没有多聊，便各自修炼了起来。
<br/>　　苏莫盘膝而坐，一边等待卢云送来炼器材料，一边静静的凝练玄力、巩固修为，只待逃过此劫之后，便疯狂提升修为。
