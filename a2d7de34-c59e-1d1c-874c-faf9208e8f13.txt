        &lt;h3&gt;第三卷第五十七章  耗材研发成功&lt;/h3&gt;
<br/>
<br/>        陈子昂和江俞轩到处飞，忙碌的到处拓展业务，时间就像是手中的沙，一晃一年多过去了，陈子昂的甸城计划也因为一些事情在向后推迟着。
<br/>
<br/>        陈子昂出差回到北京的时候，并没有回家，而是和江俞轩直接回到了公司，临走之前，他们已经将工作重组，这一段时间人员已经开始各司其职了。
<br/>
<br/>        大家的行动力都不是盖的，皇甫双双已经从南方提溜了一个技术员回来了，样品都已经放在了工作台上。
<br/>
<br/>        苟江凌也已经将南方的一个小作坊已经收购，带着那个小老板和他的技术员也到了北京。
<br/>
<br/>        陈子昂不知道他们是怎么在这短短的一段时间里就完成了这件事情，当她回到厂房的时候，耗材项目组的人员已经全幅武装，开始了他们的研发和实验，各种型号成品的硒鼓堆成了一排排
<br/>
<br/>        原本南方有现成的模具，他们要做的是让现有的东西更加有卖点和质量更可靠，在这些产品上进行更深层次的研发和加精，他们也拿到了原装的硒鼓进行拆分和对比。
<br/>
<br/>        赵绾绾将一些做鼓芯的厂家也找到了，目前市场上有，但是并不是很多，只是国内的质量并不太好，赵绾绾联系了给各大品牌厂商做鼓芯的供应商，拿到了一些份额。
<br/>
<br/>        陈子昂说得很对，他们是在和时代奔跑，也是和时间在奔跑，只有赶在前面他们才有机会，既然要做就不能拖拖拉拉，要做就做好。
<br/>
<br/>        “哈哈，你们太厉害了啊，我这个组长还没有回来呢，你们就已经试验好了？”
<br/>
<br/>        “你的那个组长是虚的，我们人人是组长。”倪小美毫不客气的打击着江俞轩。
<br/>
<br/>        “成批量的生产还得多久？”陈子昂问。
<br/>
<br/>        “我们正在试验，这些个质量都有点瑕疵。怕用不了多久。”赵绾绾手中正在鼓捣着鼓芯，虽然她是个女孩子。
<br/>
<br/>        “先拿我们自己公司来试用。”
<br/>
<br/>        “我们公司能用几个？我正好要和你说这件事情，必须得找一个大量用耗材的单位来进行投放测试，我们自己是可以测试的，但是还是要收集客户的想法和意见。”倪小美说。
<br/>
<br/>        陈子昂和江俞轩相互看了一眼，有大量需求的只能是大单位。
<br/>
<br/>        “俞轩，我去找一下某仁医院，看看他们愿不愿意换耗材？”陈子昂说。
<br/>
<br/>        某仁医院是她挖掘第一桶金的地方，到现在还一直供着货。
<br/>
<br/>        “行，我也去找一找另外的单位。”一个单位肯定是不行的，必须要找几家单位，而且还是能靠得住的单位。
<br/>
<br/>        “行，我们分头行动。”
<br/>
<br/>        陈子昂去了某仁医院，找到了老总；
<br/>
<br/>        “赵总，您好。”陈子昂突然袭击，她对这里已经很熟悉了，赵总也已经见怪不怪了。
<br/>
<br/>        “哟，小陈来了啊。”赵总热情的招呼陈子昂。
<br/>
<br/>        这个陈子昂是落家的亲戚，落家是这里最大的股东，他们也无非就是给人家打工的。
<br/>
<br/>        “赵总，我找您有点事情。”陈子昂也是开门见山的说。
<br/>
<br/>        “什么事情，还要你亲自来找我？”
<br/>
<br/>        “是这样的，我们公司最近会研发一批新的硒鼓，准备投放到市场上，我想找一个可靠的单位来对我们的硒鼓做个评测。”
<br/>
<br/>        “那你们的这个新的硒鼓会不会对打印机产生影响？”
<br/>
<br/>        “一般的情况下是不会的。”
<br/>
<br/>        “那对打印质量呢？”
<br/>
<br/>        “打印质量也应该和原装硒鼓能够进行pk。”
<br/>
<br/>        “那这个硒鼓多少钱？”
<br/>
<br/>        “第一批我们免费给您们用，因为您们是我们的优质客户，享有优先享受优惠的权利。”
<br/>
<br/>        “你们研发也是需要陈本的吧，这样，你们前期按照你们的陈本价给我们，用得好了，我们在讨论价格的问题。”
<br/>
<br/>        “赵总，那太感谢您了。”陈子昂都没有想到会这么顺利，她大喜。
<br/>
<br/>        “不用谢，你这个小姑娘很能干嘛，现在我们用的耗材都是原装品牌，比较起来说，都是很贵的，一般人是用不起的。如果你能做到大部分人能用得起，那就很了不得了。”
<br/>
<br/>        “是啊，我们的打印机和耗材都是国外的品牌呢，只是这个需要一个过程的。”陈子昂懂赵总的意思，她心里何尝不是这样想的。
<br/>
<br/>        “慢慢来吧，总得一个过程的。”
<br/>
<br/>        陈子昂和赵总聊了一会就告辞回到了公司，她都没有想到会这么顺利，她要回去给大家分享这个喜悦。
<br/>
<br/>        原本她怕赵总对产品质量会进行质疑，不会接受的，所以她说前期免费让他们单位使用，没有想到赵总让他们先按照成本价给他们，这样他们的陈本就回来的，不赔钱就好啊。
<br/>
<br/>        回到公司，江俞轩悠哉悠哉的坐在那里喝着茶，而且也已经给她泡好了茶。
<br/>
<br/>        她一看到江俞轩那样，心里就知道江俞轩也搞定了一个单位。
<br/>
<br/>        “俞轩，你这么快？”
<br/>
<br/>        “那是当然，我是谁啊？最帅最棒的江俞轩。”
<br/>
<br/>        “切，王八卖瓜--自卖自夸。”
<br/>
<br/>        “你以为我和你一样，还要屁颠屁颠的跑过去啊，我，只要一个电话就搞定了。”
<br/>
<br/>        “那你告诉我，你找的是哪个单位？”
<br/>
<br/>        “事事都要向你汇报？我这手头上也有好几个单位吧？”
<br/>
<br/>        “哟，还保密呢，反正我以后也会知道的。”
<br/>
<br/>        “那不就得了？你还问？”
<br/>
<br/>        “他就坐哪里打了几个电话。”苟江凌看不过去江俞轩那嘚瑟的样子。
<br/>
<br/>        “就打了几个电话啊？”
<br/>
<br/>        “喂，老大，你老婆要搞研发，需要单位进行投试。你说怎么办？”赵绾绾绘声绘色的表演着江俞轩打电话的样子。
<br/>
<br/>        “你老婆？”陈子昂问
<br/>
<br/>        “啊，我老婆啊，我有老婆不行嘛？”江俞轩瞪了一眼赵绾绾。
<br/>
<br/>        “我有那么好的老婆不用，干嘛还要我亲自去跑一趟呢？”江俞轩更加嘚瑟了。
<br/>
<br/>        “你什么时候有老婆了？你这样的人竟然还有人要？”倪小美的关注点总是和人不一样。
<br/>
<br/>        “哎，陈子昂，你说，你这都是什么人啊？我好歹也是公司里的一枝花，他们竟然敢说我没有人要。过几天你要将我风风光光的嫁了啊。”江俞轩适时的转移了话题。
<br/>
<br/>        “俞轩，你真要结婚了？”倪小美问。
<br/>
<br/>        “当然啊，难道你们要我打一辈子光棍你们就开心了吗？”
<br/>
<br/>        “唉，单身狗又少了一只。”苟江凌唉声叹气，他什么时候才脱单啊。
<br/>
<br/>        “不怕，不怕，江凌，到时候我多给你喂喂狗粮，你很快就会脱单了。”江俞轩挤兑苟江凌。
<br/>
<br/>        “反正，单身狗不是我一个人，哈哈哈。”苟江凌看了一眼陈子昂，幸灾乐祸的笑着。
<br/>
<br/>        “你这是恨不得全世界都和你一起单着吧？”陈子昂看着苟江凌那幸灾乐祸的样子就想给他一脚。
<br/>
<br/>        “子昂啊，你美艳天下，没有人能配得上你，单着是应该的，我这一猥，琐，男，凭什么要让我单着啊？”苟江凌看了一眼陈子昂的尖跟高跟鞋，赶紧自黑了一下。
<br/>
<br/>        “你不但猥琐，你还渣，你的大学女同学可是等你很久了。”皇甫双双冷冰冰的声音传来。
<br/>
<br/>        苟江凌觉得脖子凉嗖嗖的。
<br/>
<br/>        “干活，干活！”便一溜烟的跑到自己的位置上去了。
<br/>
<br/>        耗材研制成功给了陈子昂很大的信心，而且有几个单位答应试用更是让大家欢欣鼓舞。
<br/>
<br/>        大家都沉浸在一片欢乐之中。
