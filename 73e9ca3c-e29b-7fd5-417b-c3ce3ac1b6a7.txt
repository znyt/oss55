<br/>　　不知不觉中，或许连洛冰玄都没有意识到，自己的声音，居然也变得温和起来。
<br/>　　叶尘乖乖转过身。
<br/>　　洛冰玄当即从妖兽皮之中，飞快的伸出玉臂，一把抓住自己的白裙法袍，心念微微一动，法袍便已经自动穿在了她的身上。

<br/>　　一袭白色长裙穿在身上之后，洛冰玄也渐渐恢复了之前的冰冷气质。
<br/>　　叶尘见状，不由得再次暗自嘀咕，
<br/>　　‘如果不是看在曦月的面子上，我才懒得救你这个冰块脸！’

<br/>　　她身为绝情居的圣女，自小养尊处优，何曾吃过这种俗物，自然十分不喜。
<br/>　　不料，
<br/>　　她这话才刚刚落下，
<br/>　　“咕噜~~”

<br/>　　洛冰玄那原本犹如冰霜一般的俏脸之上，顿时露出一抹淡淡的红晕，眼神中闪过一抹尴尬。

<br/>　　“不吃饭怎么有力气干活？待会还要族击杀冰甲猛犸象，我可不想带着一个累赘！而且，我做的烤肉，可是很好吃的哦！”

<br/>　　因为她发现，这些看上去让人毫无任何食欲的俗物，味道竟然还挺不错！
<br/>　　等到两人先后走出这处山洞，洛冰玄手中的烤肉串，居然已经被她消灭了七七八八。
<br/>　　叶尘望了一眼嘴角还泛着油光的洛冰玄，不由得微微一笑，忍不住调侃道：
<br/>　　“你现在的这个吃货样子，可比之前那张冰块脸可爱多了！”
<br/>　　洛冰玄听到这话，顿时柳眉倒竖，脸上登时浮现出一抹寒霜，重新恢复了之前的冰冷状态，正要开口。

<br/>　　“嗷！！”

<br/>　　周围忽然一阵地动山摇，犹如发生了大型地震一般。

<br/>　　叶尘和洛冰玄两人，不由得对视了一眼，不惊反喜，

<br/>　　“现在的当务之急，是那道冰甲猛犸象的妖丹，其他的事情，以后再说！”

<br/>　　不由得，洛冰玄又想起了之前，两人在山洞中的那一幕幕，芳心顿时一下子跳的更厉害了，而且各种奇奇怪怪、从未有过的念头，纷纷涌现在脑海之中，
