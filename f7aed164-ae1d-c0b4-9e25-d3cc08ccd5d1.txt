　　第452章
<br/>　　明月照山川，山川一片静默。
<br/>　　月光之下，三道人影，鬼鬼祟祟，蹑手蹑脚靠近杀虎口。
<br/>　　“大当家……”徐神机轻声唤道。
<br/>　　“搞什么，你想把人召来么。”程大雷提心吊胆道。
<br/>　　“我们快到了吧？”徐神机。
<br/>　　“应该……是快到了。”
<br/>　　与老兵组激战过后，程大雷三人便一路向北奔来，幸运的是，路上再没遇见伏击他们的敌人。在夜晚三更时分，他们终于赶到杀虎口外围，此刻正摸索着迫近杀虎口。
<br/>　　李行哉曾说过，程大雷既然敢来，必然已有救出林少羽的把握。老实说，他委实有些高估了程大雷。凉州和并州相距万里，程大雷对这边的情况两眼一抹黑。既然不知道确切情况，又如何想得好对策，不过是走一步看一步，心里觉得以自己本事，救出林少羽，应该并不是绝对做不到。
<br/>　　现在已靠近杀虎口，不管是帝国军还是起义军，程大雷马上就可以接触到他们，必须思考下一步行动计划了。
<br/>　　牵着黑牛，程大雷登上了一座山坡，在看到山坡下的情形时，程大雷三人同时呆立在那里。
<br/>　　杀虎口外，是一眼望不到头的营帐，士兵人人披甲，长矛如林。
<br/>　　程大雷干咽了口吐沫，道：“军师，你估计他们有多少人？”
<br/>　　徐神机手搭凉棚，向前面望了一眼：“至少有十几万吧。”
<br/>　　“我们事先得到的消息，少羽手中的残兵败将，应该只剩下两三千人。”程大雷倒抽一口凉气：“以十万对三千，少羽现在还有可能活着么。”
<br/>　　徐神机也觉得情况有些不妙，思索了片刻，发出一声感慨：“就算他还活着，想要从十万大家中将人救出来，谈何容易。大当家，你可有什么法子。”
<br/>　　“我能有什么法子，我也是血肉之躯，不是三头六臂，金刚不坏之身。”程大雷。
<br/>　　“咦，大当家，你不是仙人传艺么，难道就不能让仙人给你些神通，一道天雷，劈开一条生路。”徐神机。
<br/>　　“仙人忙，没时间搭理我。”程大雷白了他一眼，目光看着前面的大军：“既然已经到了，就没有望而却步的道理，无论有没有法子，都是要试一试的。”
<br/>　　程大雷低头沉思片刻，道：“军师，你和银眸守在这里，我自己进去，看有没有机会把人救出来，最好，少羽现在还活着。”
<br/>　　“你一个人进去？”徐神机一惊：“这太冒险了吧。”
<br/>　　“怕是带着你，更加冒险吧。”程大雷白了他一眼，道：“让银眸保护你，假若我一个人去，只想要脱身，应该是不难做到的。”
<br/>　　徐神机想了想，觉得程大雷的话也有道理。他孤身前往，要照顾的只有林少羽一个，但如果三人一起进去，还需要留心照顾自己。在千军万马中，他可没自信不被误伤。
<br/>　　“那大当家你自己小心。”
<br/>　　程大雷没打算骑牛，这一路过来，黑牛已经成了自己的标志。如果骑着黑牛进去，肯定第一时间被识破身份。
<br/>　　他将斧头背在身上，手摁住匹夫剑，临走之前看了银眸一眼，道：“照顾好军师。”
<br/>　　然后，程大雷裹紧大氅，提着剑下了山坡。山坡的密林中，就只剩下徐神机和银眸中。
<br/>　　本来徐神机还很镇定，程大雷这一走，他感觉周围的寒风都分外瘆人些。目光看了银眸一眼，想与她说说话，驱逐心中的恐惧。然而，银眸依旧冷冰冰一张脸，似乎根本不知道恐惧为何物，徐神机只好打消了这主意。
<br/>　　好好一个姑娘，竟然不会说话，唉……
<br/>　　徐神机心底发出一声感慨，目光看向杀虎口方向，只盼着程大雷可以马到功成，平安脱身。
<br/>　　程大雷鬼鬼祟祟向前走，尽量不惊动周围的人。现在他已经混入敌人大营中，周围到处都是持矛往杀虎口中冲的士兵。程大雷穿着蛤蟆寨的制式皮甲，因为黑夜的缘故，倒也没人注意到皮甲背后的***案。当然，他背后的大斧是很醒目的，不过各路诸侯，也免不了有些乌合之众，用铁棍的，狼牙棒的……而且，因为程大雷声名鹊起，免不得有人有样学样，军中兴起一股用斧之风，一柄柄斧头，一个赛一个大。所以程大雷混在大军中，竟没有被人发觉。
<br/>　　“冲！那个狗娘养的敢后退，老子一刀砍了他！”一个头目模样的人大喝。
<br/>　　听到这头目的话，略微松了口气。既然还在调兵，证明战斗还未结束，林少羽有很大可能还活着。
<br/>　　他也随着大军向杀虎口中涌，心中迫切想要知道林少羽如今的情况。
<br/>　　“喂，那个小子，你过来，对，就说你呢，给我过来。”头目指着程大雷道。
<br/>　　程大雷一头雾水，不知道这头目怎么在万军丛中发现了自己。难道是自己生得格外英俊的缘故，走到那里都是最亮眼的存在，根本不可能被忽略。
<br/>　　程大雷来到他面前，手一直摁着剑柄，情况只要有丝毫不对，他就先把这头目宰了。
<br/>　　“大人，我……”
<br/>　　“你这斧头很不错么？”头目拍着程大雷肩膀：“摘下来给我看看。”
<br/>　　“呃……大人，这斧头是我家传的。”
<br/>　　“家传的又怎样，我只是看看，难道我还能吞了你的不成。”头目咧开大嘴：“不给我看，莫非你是贼人程大雷派来的，不，我看你就是程大雷。”
<br/>　　程大雷心底激灵一下，由衷的感慨，这头目看人的眼光真准呐。现在不是暴露身份的时候，他无可奈何的将斧头摘下来，道：“我是怕太重了，大人拿不动。”
<br/>　　“哼，什么话，大人我天生神力，双臂有千钧之力，有什么拿不动的。”
<br/>　　说着，这头目接过斧头，忽然哎呦一声，斧头差点砸在脚上。
<br/>　　程大雷赶忙将斧头接住，关切的问道：“大人，你没事吧。”
<br/>　　“没事，没事，我能有什么事。”头目摆摆手，看了程大雷一眼，道：“小子，你叫什么名字？”
<br/>　　“启禀大人，小的叫牛三斤。”
<br/>(本章完)
