　　如叶少阳之前猜测的第一种可能，这山谷里有阴气之源！
<br/>　　这是对叶少阳二人来说，最不好的一种可能，阴气之源可以有很多种，有一些邪性的法器或凶器，或者某种阴巢鬼穴，但是能在这种空旷的山谷里造成如此浓郁的阴气，并长期维持，如果是法器或凶灵之器造成的，这得是多么恐怖的东西？
<br/>　　得出这个结果之后，两人心里都有点凉凉的，商量了一下，决定还是把这阴气的源头找出来，看看是什么鬼东西。
<br/>　　叶少阳拿出了阴阳盘，毛小方也拿出了自己门派的法器，两个人从山谷一头走向另一头，然而走了没多久，叶少阳就发现自己的阴阳盘失灵了：指针乱转，根本找不到具体的方位所在，毛小方的道钟更不好用，两人好歹勉强走完了山谷，坐在地上讨论。
<br/>　　“这办法不行，”叶少阳道，“阴气的源头太强大了，整个山谷阴气弥漫，我的阴阳盘没法用。”
<br/>　　“我的也不行。”毛小方摆弄了两下自己的道钟，有些沮丧。
<br/>　　叶少阳望着山谷深处，心中很有些不甘，虽然这阴气之源十分强大，但假如自己拥有十成法力，完全可以用茅山解灵术找到阴气源头的所在，只是现在……确实没什么好办法。
<br/>　　两人商量了一下，决定还是等到晚上，看一看百鬼夜行的形态，然后再根据这些鬼魂的走向和表现，必定会泄露出蛛丝马迹，然后可以根据这个，或许能找出阴气的源头在哪。
<br/>　　但是在山谷里呆着是不行的，倒不是两人害怕，而是万一阴气的源头跟某个邪物有关，在山谷里呆着就暴露了了自己，影响调查，于是两人还是回到之前呆过的山峰上，从上面俯瞰山谷。
<br/>　　这时候已经是下午五点多，日薄西山。两人山顶吃了干粮，然后坐在山顶上聊天。
<br/>　　叶少阳闲着也是闲着，跟他打听起有关法术界的一些事情。
<br/>　　毛小方是名门正派的天师，对于法术界的情况，比叶少阳之前接触过的那些法师知道的都多，跟叶少阳聊了起来。
<br/>　　“当今法术界，最强的几个人，江湖上给他们编成了一句话：一道二僧三头陀，四丐五仙六巫婆。一共是八个人。”
<br/>　　叶少阳听完之后，当场就懵了，问道：“什么情况，怎么搞的跟武侠小说似的？来给我解释解释！”
<br/>　　“喏，一道，说的是龙虎山的紫云真人，紫云真人早年行走天下，建立过很多功德，不过他之所以被摆在第一位，主要还是品行方面没任何问题，一生恪守正道，又是张天师后人，威望极高，坦白说，你们茅山因为代代单传，对于法术界的影响是不如龙虎山的。”
<br/>　　叶少阳点点头，这个是茅山的祖训，在培养弟子方面，与龙虎山截然相反，而且茅山掌教不是不强，只是在自己之前，几代掌教修的都是清静道，习惯性地低调，不到万不得已不会用出全力。青云子之前还总是被人低估，如果不是悬空一战，爆发出全力，与无极天师同归于尽，世人还以为他仅仅只是地仙的实力。
<br/>　　“二僧，说的是五台山方丈云空禅师，和九华山方丈痴空上人，这二位也是法力绝顶，三头陀，说的是西藏密宗的三位苦行僧，这三人长年在中原活动，法力高深，而且与中原法术不太一样，至于后面三人，乞丐，说的是一位民间散修陈七袋，这人早年是丐帮的，据说还是七袋长老，这人最强的不是法术，而是堪舆术，是一位地师，在奇门遁甲、阵法布置方面造诣极深。有好几次法术界伏击强大的邪物，都请陈七袋去主阵。
<br/>　　五仙，说的是东北马仙中最富盛名的陈大仙，会的也不是中原法术，而是东北一脉的通灵术，我也没见过，据说很神奇。
<br/>　　最后一个巫婆，说的是波月洞鬼婆，其实不是鬼，而是云南的一位巫婆，据说是古南诏国师的传人，一手巫术十分了得。
<br/>　　这一共八个人，是法术界现在最富声望的几位，功参造化，登峰造极，当然其他几大门派的掌教和一代弟子中，还是有很多宗师的，实力也都很强。”
<br/>　　叶少阳听他说完，内心是震撼的，这个时候的法术界，虽然四分五裂，还是有一点江湖意思的，至少不像百年后自己所处的现代社会，因为科技和主流信仰的变更，法术界只能在地下活动，影响力也要小的多，更没人有工夫排出这种花名的榜单。
<br/>　　“对了，无极天师呢，跟这几个人比怎么样？”
<br/>　　“无极天师在蓬莱岛，久不出世，不在此列，而且还有很多隐修宗派，不怎么在法术界活动，因此也没有拍进去。而且这只是一代弟子，还不算那些太上掌教级别的。”
<br/>　　叶少阳缓缓点头，所谓的一代弟子，就是当代的掌门中同一辈分的法师数量最多的，一般来说，以六大门派为主，一般上了年纪的宗师会把掌教之位让给弟子，自己去做太上掌教，基本上不管事，自己跑去清修，掌教数量多的一代人，就被称为一代弟子。
<br/>　　像叶少阳这样的，虽然当了掌教，不过那是青云子挂了的意外结果，别的门派还都是壮年人把控，所以他仍然是二代弟子。
<br/>　　毛小方又补充道：“不过这些都是老一辈的排行了，三位头陀里，有一个已经不在人世。现在是二代弟子的天下。前几年，时局还没有这么坏的时候，法术界举办了一场龙华会上，选出了一批二代弟子作为培养对象，一共七八个人，都是二代弟子中天赋最高的，经过几年的成长，已经有很多崭露头角了。”
<br/>　　叶少阳问了几个人的名字，大概有一半自己都听说过，跟道渊真人算是同辈，这几个人中，除了个别成了一代宗师，实际上单就个人实力来说，反而不如后来的道渊真人。
<br/>　　至于青云子那一代，在这个时代算是三代弟子。
<br/>(本章完)
