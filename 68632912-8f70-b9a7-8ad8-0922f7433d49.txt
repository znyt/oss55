　　下方木屋前，白晟回首冷冷瞥了叶永阳一眼，再次看向楚涵时，眼底已然带上了与白忧极其相似的阴沉感。
<br/>　　“每次，都是你在坏事。”白晟的声音明显带上了不甘和狠辣：“你这个人，变数太大又身怀重宝，杀又杀不得……”
<br/>　　楚涵眼神瞬间一闪，不禁嘴角勾起：“怎么杀不得？”
<br/>　　“哼！”白晟冷哼一声：“别想套我话。”
<br/>　　楚涵笑容更加灿烂，扬起的修罗战斧笔直的指向白晟：“你们神秘家族，到底还有什么秘密？”
<br/>　　白晟冷笑，环顾了一圈幽幽开口：“等这一代制裁者也死了，就是你们这些人的死期，楚涵不能杀，不代表你们几个不能杀。”
<br/>　　楚涵的双目眯起，这句威胁似乎另有所指，眼前这个白家家主的确忌惮制裁者，但却并非没有解决办法。
<br/>　　神秘家族真正想要得到的，似乎是楚涵本身？
<br/>　　想到这里，楚涵又一次联想到了回炉系统，这个自出现到今就一直神秘的玩意儿，到底与阴阳谷有什么关系？
<br/>　　白家秘术被楚涵一下子识破，又无法直接动手让楚涵交出他想要的东西，最关键的是只有极少数的老一辈神秘家族成员知道，楚涵现在还不能死。
<br/>　　杀不得，又得不到，这让白晟相当的不痛快。
<br/>　　局面也因此被楚涵拉回，让白晟好不容易铺垫得到的优势荡然无存。
<br/>　　望着楚涵越发刺眼的笑容，白晟一边划动轮椅远去，一边留下了一句话：“不要以为这次我无功而返，白允儿还在我手上，你要是有胆，就来阴阳谷。”
<br/>　　话落，白晟就如同来时的那样，一下子远离在众人视野，而后消失的无影无踪。
<br/>　　长久的寂静，在这片满是战斗后的废墟地带蔓延，整个狼牙基地仅存的几个人，都在很长的时间里没有说话，唯有呼呼的风声不息。
<br/>　　一片荒凉，一地血迹，满目尸首。
<br/>　　这是狼牙史上最严重的一次灾难，也是第一次被人打到家门口。
<br/>　　“去，看下步纱的情况。”楚涵忽然在这时候出声，打破安静。
<br/>　　尚九娣瞬间清醒过来，立即跑向木屋。
<br/>　　陆玟丞‘啪’的一声跌坐在地上，大口的喘着气，有些不可置信：“结，结束了吗？”
<br/>　　楚涵点着头，一步步走向顾凉晨的尸体，蹲下，抬起右手将他死瞪着的双眸捂住：“结束了，兄弟。”
<br/>　　楚涵的声音沙哑，每说出一个字，面色都苍白一分，话落之时猛然一大口鲜血呕出，淅淅沥沥的在旁边的地面浸·湿了土地。
<br/>　　再次抬起手，顾凉晨的双目已经闭合，面色安详。
<br/>　　陆玟丞连忙爬起来，想要扶住单手撑地，还在持续咳血的楚涵。
<br/>　　楚涵无力的挥了挥，声音一顿一停：“你先，处理伤势。”
<br/>　　陆玟丞这时候哪还有心思处理伤口，面色焦急无比：“老大你？”
<br/>　　这时候的叶永阳从远处走来，由于失去了听觉，说话的声音听上去有些怪异：“他强行从白家秘术里拉回心智，一直坚持到白家家主离开，这段期间表现的一点不受影响，骗过了白家家主，否则要是被看出有一点反常，再来一次秘术，他根本抵挡不了。”
<br/>　　叶永阳的解释让陆玟丞大惊，这才明白刚才有多惊险，楚涵并非天生对白家的秘术有抵抗效果，而是纯靠着无比坚定的信念在坚持。
<br/>　　信念的来源，正是眼前这些死去的狼牙战士们。
<br/>　　谁都动摇不了！
<br/>　　正面抵挡的后果，便是眼下的精神萎靡，大脑如同被轰炸般的剧痛。
<br/>　　楚涵伸手摸了摸自己的额头，苦笑一声，这大概是他进入末世纪元以来，第一次高烧，滚烫的额头用手摸上去，感觉都被烫了一下。
<br/>　　这时候的尚九娣背着步纱跑出木屋，面色焦急：“急救！我需要直升机！”
<br/>　　那名狼牙的情报部成员立即站出来：“城主请跟我来！”
<br/>　　他不是太了解这个从木屋里出来的陌生红衣女子做了什么，但他知道这是楚涵的救命恩人，也是整个狼牙的恩人。
<br/>　　楚涵跟尚九娣对视了一眼，朝着她点了点头。
<br/>　　尚九娣眼中带着担心，但还是头也不回的将步纱放在那名情报部成员背上，大步跑向直升机所在的方向：“我来驾驶，快！”
<br/>　　整个狼牙都被毁了，根本没有任何医疗团队，最快的办法只能到最近的安罗城。
<br/>　　楚涵和陆玟丞虽然也伤势严重，但对比步纱还不至于到濒死状态，眼下只能先将步纱送走。
<br/>　　“你俩，还能动么？”楚涵看着陆玟丞和叶永阳问道，顺带比划了几下手势。
<br/>　　“我可以，我来开车。”叶永阳虽然听不见，但很了解当下情况，立即开口。
<br/>　　陆玟丞伸出另一只没受伤的手，就想要去扶楚涵，狼牙基地只剩下最后一架直升机，他们三个同样需要治疗，只能选择开车，索性楚涵的座驾一直安置在仓库。
<br/>　　但这时的楚涵却又一次摆手，道：“你俩去，我还有事。”
<br/>　　“老大？长官！”陆玟丞急了，激动道：“我知道你担心白小姐，但是……但是你现在情况很不好，需要立即输液！况且你现在去阴阳谷，有什么用吗？”
<br/>　　楚涵没有理会陆玟丞，自顾自的站起来看了眼四周，继续吩咐：“这些死去的兄弟们不能就这么放着，让安罗城的人立即过来。”
<br/>　　“长官！”陆玟丞双眼发红，颤抖着抬起手敬着军礼：“请长官回城接受治疗！”
<br/>　　楚涵转身看向他，定定的站了会儿后，抬手回礼：“命令：中将陆玟丞立即与叶永阳返回安罗城，毕！”
<br/>　　陆玟丞没有放下手，神态倔强：“请长官回城！”
<br/>　　楚涵眯起眼，迈开步伐从陆玟丞身旁走过：“回头去刘玉定那领罚。”
<br/>　　“请长官回城！”陆玟丞大喊。
<br/>　　楚涵脚步不停，不理会陆玟丞大步向前走，哪怕高烧让他意识模糊脚步虚浮，背影也依然笔直。
<br/>　　“请！长官回城！！”
<br/>　　身后传来陆玟丞撕裂般的沙哑声音，倔强的坚持。
<br/>　　“楚涵，你需要休息。”旺财的声音在楚涵的脑海中响起。
<br/>　　“不。”楚涵坚定而答，眼底波涛汹涌。
