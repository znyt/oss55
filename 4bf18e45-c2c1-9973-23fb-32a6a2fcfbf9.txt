　　第116章
<br/>　　清早醒来，阳光很好。
<br/>　　不过程大雷精神状态不是很好，他昨夜没有睡好。一个人精神愈差，食欲便也愈糟糕。
<br/>　　“这是什么东西，给狗吃的么！”
<br/>　　“拿下去，这样的东西也能入口。”
<br/>　　……
<br/>　　牛本善昨夜也没有休息好，金龙镖局和漕帮两家势力走后，他翻来覆去睡不着，心里琢磨着程大雷的来历。早上醒来格外派了两个人注意丁字号院落的状况。
<br/>　　送饭的进去了三次，都被赶了出去。这一次，牛本善亲自进去，陪着笑道：“客官，是对小店的饭菜不满意么，定是本店的厨子不知道您的口味，您有什么可口的，我可以吩咐伙计去城里买来。”
<br/>　　“来之前家里就交待过，扬州是个小地方，诸样事情，定不能像家里一样称心如意。该受些委屈便受些委屈。但我却是没想到，扬州竟然连一份能入口的菜都拿不出来。”程大雷叹了口气：“罢了，也别难为他们，把我们带的辣椒拿出来吧。”
<br/>　　“辣椒……那是什么东西？”牛本善心里嘀咕着。
<br/>　　“早就给公子备好，这就让下人端上来。”
<br/>　　这时有人端着饭菜上来，牛本善偷眼瞧了瞧，见里面红的绿的霎是喜人。
<br/>　　“客官，有件事我要向您禀报一声，甲字号的客人想来拜会您，只是他怕冒昧来犯，惹您不开心。所以特意让我转达，请您参加晚上的辩宝大会。”
<br/>　　昨夜的事情当是已在渔阳楼传开，已经有人注意到程大雷这伙人，目前看来扬名的目的已经达到，虽然过程有些稀里糊涂。
<br/>　　关于甲字号的客人，徐神机已经昨天已经打听过。对方名叫耿星河，据说是京州来的商人，做的是丝绸和茶叶生意，在岳阳楼住了些日子，出手阔绰，很有面子。
<br/>　　入夜，甲字号院落里灯火通明，隐隐有乐声传到外面。
<br/>　　程大雷一伙到的时候，只见一个锦袍胖子大步迎来。
<br/>　　“欧公子，刚才我们还谈起您，没想到正说着您就来了。”
<br/>　　“昨夜就想来拜访耿兄，奈何俗事缠身，今日才得空叨扰。”
<br/>　　“快请，快请。”
<br/>　　耿星河将程大雷引到上座，一路诸多人都打量着程大雷，琢磨着程大雷的来历。究竟是怎样的背景，能同时招惹漕帮和金龙镖局两家势力却又安然无恙。
<br/>　　座上，耿星河和程大雷闲谈，耿星河这个人很和气，也很健谈，天南地北无所不知，程大雷显得有些相形见绌。他只好闭上嘴巴少说话，伪装成沉默寡言，气质高冷也是好的么。
<br/>　　中央有一舞女在表演舞蹈，这女人生得身材高挑，眼窝深陷，一双眼睛却是金色的。
<br/>　　混血儿！
<br/>　　程大雷眼前一亮，忍不住在对付脚踝腰眼打转。
<br/>　　“这胡姬是我从一个波斯商人手中高价买来，今年一十九岁，欧公子若是喜欢，我便送给公子。”
<br/>　　“咦！”
<br/>　　程大雷对耿星河顿生好感，此人当真是好大方。
<br/>　　瞥了身后立着的苏樱一眼，程大雷哈哈笑道：“君子不夺人所好，耿兄客气了。”
<br/>　　耿星河自然也注意到程大雷身后的苏樱和柳芷，二人虽然都是小童打扮，却逃不过耿星河的眼力。
<br/>　　“欧公子左拥右抱，享尽艳福，一般的庸脂俗粉自然看不在眼里。胡姬，你过来敬欧公子一杯酒。”
<br/>　　坐拥右抱个鬼，事到如今一个都没抱上的，都是体面人，看看人家小酒喝着，小妞抱着，小舞跳着，再看看自己，唉，人比人气死人呐。
<br/>　　那胡姬款移莲步，行到程大雷身边，双眼脉脉含情，腮边两个梨涡，身上肌肤如流动的牛脂。程大雷喝了她倒的一杯酒，酒中仿佛也有了她身上的胭脂香味。
<br/>　　美人计什么的，我一点抵抗力都没有，要坚持住啊，欧巴。
<br/>　　辩宝大会已经进行过一日，在酒席宴间，时不时有人拿出什么『宝物』，南洋的珍珠，北山的宝石，东海的沉木，极西之地的美人……乱七八糟，什么都有，有些大家闻所未闻，见所未见。有些拿出来后，便立刻有人出价买下，聚在一起嘀嘀咕咕商量交易。
<br/>　　“欧公子，怎么一直不用菜，是饭菜不入你的口味么？”耿星河问。
<br/>　　程大雷顿时精神一震，又是自己表演技术的时候到了。
<br/>　　正在这时候，忽听得院门出乱哄哄一声，打断了程大雷和耿星河之间的谈话。
<br/>　　“什么事？”耿星河眼神一历。
<br/>　　程大雷在旁边观察着，见耿星河平素和和气气，可只是眉头一拧，便有威严渗出，令人忍不住心中一寒。
<br/>　　有一耿星河的随从快步而来：“大……老爷，门外有个人抱着东西进来卖，被我们拦下了。”
<br/>　　知道这里是辩宝大会，便有人捧着些东西过来撞运气，都说编出一套故事，这种事情也常见。
<br/>　　“这点小事还过来问我，赶走便是，无端扰了我们的酒兴。”
<br/>　　“嗳……”程大雷拦住了：“耿兄不必动怒，让他进来，看看他卖得什么东西也好。”
<br/>　　“既然欧公子开口，就让他进来吧。”耿星河道。
<br/>　　从院外走进来一人，破衣烂衫，脸若菜色，不知多少天没吃过饭。在他身上背着一样长条状物事，被厚厚的破布缠着。
<br/>　　“你要卖什么东西？”
<br/>　　“家父去世前，留下一件兵器。”
<br/>　　“什么兵器？”
<br/>　　年轻人把背上的东西取下，解开上面破布，一柄旧矛出现在大家眼前。大家都递过去眼睛瞧，只见这矛已不知放了多久，上面都生满铁锈。
<br/>　　“欧公子，我在岳阳楼住了些日子，总是会遇到这样的人，言必称传家之宝，真是搞得我烦不胜烦喔。”
<br/>　　其他客人也发出笑声，看来这样的事情他们都没少遇见过。
<br/>　　程大雷却是离席走过去，这长矛的形状隐隐有些熟悉，他不禁问道：“这兵器有名字嘛？”
<br/>　　“有，家父生前曾是一镖师，从北地得到这件兵器，据家父说说，这兵器的名字叫做……”年轻人顿了顿：“丈八蛇矛。”
<br/>　　程大雷眼前一亮，咦，竟然恐怖如斯。
<br/>　　
<br/>(本章完)
