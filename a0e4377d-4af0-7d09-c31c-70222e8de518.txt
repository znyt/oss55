　　大伙面面相觑。
<br/>　　虽然说古墓本来就是停放棺材的地方，但是把棺材摆放在这种地方，大伙都是头一次见，心中十分好奇。
<br/>　　“不知道这棺材里有没有尸体。”毛小方嘀咕了一句，立刻遭到陈晓宇的反驳：“废话，棺材就是放尸体的，怎么可能没尸体。”
<br/>　　妙心说道：“关键是，这里面停放的是谁的尸体，这不可能是墓主的尸体。”
<br/>　　这么大一座古墓，墓主当然是停放在古墓的正中间、风水气场之上，不可能摆在这里一个狭小的不通风的地方。
<br/>　　叶少阳想到什么，在棺材边上点了两根蜡烛，让他们先观察着，自己走出墓室，往墓道深处走去，进了相邻的一间墓室，举着三色莲花灯照过去，看到了一模一样的石棺……又走了几个墓室，都一样，里面都停放着一模一样的石棺，叶少阳不敢擅自乱来，回到第一间墓室里，把自己的发现跟大伙说了。
<br/>　　听他这么一说，大伙更是感到震惊。
<br/>　　“也就是说，这条墓道里，所有的墓室里都放着一副石棺？”毛小方震惊地说道。
<br/>　　“可能是吧，前面没去，不过想来应该是差不多。”
<br/>　　大伙互相望着，心中都产生了同样的疑问：为什么会有这些石棺的存在，如果里面真的有尸体的话，那这些尸体又是怎么回事？
<br/>　　“想这么多干什么，把这石棺打开看看不就知道了。”陈晓宇提议道，朝石棺走了过去。
<br/>　　“不妥！”妙心立刻拦住他，说道：“这些石棺绝不是随便摆在这的，在弄清楚之前，最好不要随便动。”
<br/>　　“不动的话，又怎么知道是怎么回事啊，至少让我们看看这些尸体都是什么样子，才好去推测啊。”
<br/>　　陈晓宇说完，卢晓清和曹雨兴也跟着附和。
<br/>　　叶少阳明白，他们并不能确定看到尸体之后、就能推测出个所以然来，一切都是出于好奇，想要看看这石棺里面的尸体究竟是什么样子。
<br/>　　这也算是一种本能的对未知的好奇，叶少阳自己也是一样。
<br/>　　“妙心，就开棺看看吧，想找别的先所还不知道在什么地方，先开棺看看应该不碍事，就算是尸变，有我们几个在，应该也没什么事的。”
<br/>　　妙心自己其实也好奇，听梧桐这么说，也不好违背大伙的意思，只好说道：“只是开棺可以，如果有什么不对劲的东西，大家最好别碰。”
<br/>　　得到了妙心的许可，陈晓宇、卢晓清和曹雨兴三人十分高兴，找来蜡烛，点燃之后，捧在手中，对着石棺的下沿照了一圈，确定石棺的棺盖和棺材本身是可以分开的，于是让曹雨兴一个人抠住棺材的小头，两人站在大头前面，一人抠着一边，用力往上抬。
<br/>　　叶少阳本来以为用蛮力是抬不起这石棺的，结果没想到三人还真的把石棺的盖子抬了起来，虽然有些吃力。
<br/>　　“你们小心点，端平了！”
<br/>　　在妙心的嘱咐下，三人将卸下来的棺盖放在了一边地上，叶少阳立刻提起三色莲花灯，朝棺材里照过去——
<br/>　　一片明晃晃的液体。
<br/>　　凑近了看，是一汪水，很清，里面并没有尸体，而是……有一条鱼，在水里漂浮着。
<br/>　　鱼有成年人一只手左右的长度，身上鳞片是墨绿色的，在灯光下反射出冷冷的幽光。
<br/>　　“什么鬼东西！”
<br/>　　在场几人都震惊了。在他们的设想中，已经在期待在棺材里看到尸体……不管是什么样的尸体，哪怕是僵尸，也完全可以接受，结果却没有尸体，有水，还有鱼……这简直大大超出了大伙的预料，一时之间都盯着这条鱼看起来。
<br/>　　“这是鬼鱼。”叶少阳开口说道。
<br/>　　大伙都倒吸冷气。
<br/>　　“鬼鱼”这两个字，不光是法师听说过，就算是在民间，也有不少关于它的传说：在很多地方的水域里，塘里，或者水沟里，都有这种鬼鱼的存在，说来奇怪，这种鱼没有正式的名字，也没人知道属于什么品种，数量也极少，有时候撒网捕鱼的时候，偶尔会带上来这种鬼鱼，一般都不大，最多也就手指长短，抓到这种鱼，是绝对不能吃的，要赶紧放生。因为民间传说，这种鱼是死人变成的，因此只有在死过人的水域里才会看到。
<br/>　　这当然只是传说，在法术界的说法，鬼鱼，是心怀怨气的人死后所化，前提是尸体在水中腐烂。这种鬼鱼不算是正经的生灵，严格说算是邪灵，但是不伤人，也不吃东西，消耗的是死者的怨气，等到怨气消耗光了，鬼鱼也就融化消失了。
<br/>　　因此，鬼鱼一般都长不大。
<br/>　　在棺材里看到鬼鱼，大伙都觉得很吃惊。不由仔细打量起来。卢晓清把蜡烛凑了过去，刚要仔细看，这鬼鱼突然扇了一下尾巴，卷起了一些水花，正好飞溅到了卢晓清脸上。
<br/>　　这突如其来的变故，把卢晓清也吓了一跳，急忙后退，伸手把脸上的水抹掉。
<br/>　　其他几个人也都被这突然的变故吓了一跳，纷纷后退。
<br/>　　“没事吧卢师兄，这水会不会有问题？”陈晓宇对卢晓清还是很关心的。
<br/>　　卢晓清不断地擦着脸，一边往地上吐着唾沫，骂道：“没事是没事，就是太恶心了，这是特么的什么水啊，有鬼鱼的，一定躺过尸体……”
<br/>　　卢晓清越想越恶心，打开随身挟带的水壶，用手洗了洗脸，然后回到棺材边上，往里面看了一眼，顿时愣住了，失声问道：“鱼呢？”
<br/>　　叶少阳指了指棺材的水底，哪里有一个圆形的孔洞，之前那鬼鱼被卢晓清惊到之后，就顺着这个小孔游了下去，用蜡烛照下去，只见黑黑的一片，也不知道是通往什么地方。
<br/>　　这一情况的发现，比之前发现鬼鱼还要让人震惊。
<br/>　　“棺材里怎么会有洞呢，通到哪里去的？”毛小方的一句话，道出了大伙心中的疑问。
<br/>　　“最重要的是，为什么水不会流下去呢？”梧桐喃喃说道。
<br/>　　叶少阳道：“既然鱼可以游得下去，说明下面肯定也有水。”
<br/>(本章完)
