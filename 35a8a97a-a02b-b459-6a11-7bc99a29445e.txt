　　水火两族史上的第一次合作，便是由水族圣女领头，与炎良及一批火族人进入阳谷。
<br/>　　极长的通道对这批人来说自然不成问题，而走到制裁之地上方的时候，鱼柔女直接朝着前方踏出左脚，步伐与之前并未有任何区别，而同时她的脚下便自动幻化出一条水线，随着她步步朝前延伸。
<br/>　　炎良也想用同样的方式过去，无奈对比眼前这个有些特殊的水族圣女，他这个火族少主的控火能力要弱上很多，无法支撑他一路到达对面，于是他便和其他火族成员一样，踏着半空中肉·眼不可见的石砖大步朝前跃过。
<br/>　　这帮人前进的速度飞快，诡异的是他们明明是双方第一次见面，甚至在此之前的整个千年两家族都毫无联系，但却在第一次合作时便默契惊人，很多事几乎不用交代双方便有共识。
<br/>　　与不久前到来的肖七等人一样，水火两族从制裁之地上方度过后，便一路延着通道往外走，一直走出那座黑色宫殿的地下。
<br/>　　明亮的光线忽然出现，让眼前的视野忽然开阔，也让水火两族这批人心中的阴霾一扫而空。
<br/>　　连鱼柔女都一瞬间的松了口气，望着眼前的场景久久未能回神。
<br/>　　炎良等火族人则惊叹的看着自己走出来的身后宫殿，黑色的压抑感扑面而来，整个建筑风格都带上了阴郁，但更多的是赤·裸裸的压迫。
<br/>　　“这是什么？”
<br/>　　很多火族人发出感叹，望着眼前这座高耸入云的恢宏建筑无比震惊。
<br/>　　从来没人告诉过他们，阳谷里还有这个！
<br/>　　炎良也被震撼的不小，但比起火族的其他成员，他在震惊之余却想到了更多方面。
<br/>　　此时，鱼柔女就已经转过身正面向那座恢宏建筑，紧接着，她便做出了与不久前肖七一样的动作。
<br/>　　单膝下跪，朝着那恢宏建筑低头。
<br/>　　这一行为震惊了所有在场的火族人，他们都不可思议的看着鱼柔女，这一路走来这个女人的话很少，偶尔露出来的神情也带着难以言喻的冷漠和高傲，与他们有种极其遥远的距离，以及实力差距下的巨大鸿沟。
<br/>　　作为水族圣女她无疑是极其高傲的人，尤其炎良还知道更多有关初代的秘密。
<br/>　　但让他们没想到的是，这样一个女人，竟然也会低头？
<br/>　　不经意间，火族看向那座让他们压抑畏惧的黑色建筑，都带上了种种复杂的情绪。
<br/>　　好奇，猜测，疑惑……
<br/>　　甚至于很多人都想到了阴阳谷的历史，尤其是有关那个回炉一门的历史，谁都知道这里是墓园，八门墓园在制裁之地之下，可这座宫殿？
<br/>　　莫非是墓碑么！
<br/>　　鱼柔女却没有任何解释的意思，很快就站了起来，冷然转身跨出脚步：“走吧。”
<br/>　　火族人全体发愣又跟不上鱼柔女的节奏，对身后那黑色的巨大宫殿疑惑更浓，但却也无人在这时候问什么，全部快速的朝前奔走而去。
<br/>　　就在水火两族也进入了阳谷之时，这时候的阳谷入口又来了几人，对比之前的来人无论是白罗两家还是高肖两家，亦或是水火两族，这一批人都与之前的人有明显的差别。
<br/>　　他们只有十人不到，但却平均年龄在四十岁以上，尤其是好几人都白发苍苍，看起来年岁近百，是与之前截然不同的一批长者。
<br/>　　而然处于这些人最中间，明显是这群人领头的那位，却是出乎意料又似乎理所应当的坐在了轮椅上。
<br/>　　一身极简的白衣，空空荡荡的裤腿看上去有些惊悚，神态却淡然自若，散发着淡淡的沉静与压迫力量。
<br/>　　这便是白家的现任家主，被上一任制裁者斩断了双·腿的那位——
<br/>　　白晟！
<br/>　　白晟的身旁还站着几位白家长老，他们年级都在七八十岁的样子，虽然白发苍苍却一点不显老态，精神抖擞又神采奕奕。
<br/>　　再往后站着的，则是白家的几名极少会出动的精英战力成员，他们年级约莫三十上下，每一人都极具强大战力。
<br/>　　其中一名站在最边上的成员，手中还提着一个人。
<br/>　　仔细看去这是个女人，长发垂落在地，一身白衣上沾着星星点点的深红色，显然那是干透的血渍，只是在这极深的颜色里，还不断有鲜血溢出，时不时的一两滴落于石台之上。
<br/>　　这是白允儿！
<br/>　　白允儿还处于昏迷之中，距离上回在狼牙基地的一战早已过去多日，但她却被人一路像是沙袋一样提着来到这里，重伤未治新伤旧伤一起在颠簸中复发，消瘦的身躯看上去随时会断气，更别说那摇晃中时不时露出的苍白面容，早已不复之前的样子。
<br/>　　一名白家长老瞥了眼白允儿，冲着白晟迟疑道：“族长，我们是不是至少给返祖者治疗一下比较好？我怕她熬不到关键时刻。”
<br/>　　白晟神色不变，只是最近勾起笑容轻蔑：“她既然是返祖者，就没那么容易死，放心，这个女孩命硬的很。”
<br/>　　那名长老顿了顿，又一次开口道：“我们这次行动不少长老都很不满。”
<br/>　　白晟这回直接冷笑：“不满？不满什么？不满我想要这个女孩的命，还是不满返祖者竟然不是白忧？要不是我出关，他们恐怕还想将白允儿推上家主的位置吧！”
<br/>　　那名长老不再敢说话，略微一叹气便闭口不言。
<br/>　　白允儿到底是个返祖者，可惜了……
<br/>　　可无奈白晟太过于强大，哪怕被上任制裁者斩断了双·腿战力减半，也依旧强大无比，强大到一出关便随意碾压其余家族！
<br/>　　这样的存在哪怕再如何违背白家观念，在这个混乱末世的当下，也无人有能力反抗，尤其是个别对白允儿忠心异常的旁支，由于表现的太过激直接被白晟处罚，从旁支继承人到其支脉的好几名年轻苗子，都被一次性铲除。
<br/>　　年轻一代的全部死亡，便是断了这条旁支的命脉，没有任何一个白家旁支经受得起这种毁灭式的打击！
<br/>　　所以让步与放弃，是他们唯一的选择。
<br/>　　至于白允儿，白晟非常需要白允儿的命，来替他完成一件事！
