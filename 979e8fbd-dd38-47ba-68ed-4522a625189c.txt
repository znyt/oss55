　　第198章
<br/>　　大厅内的气氛干了干，仿佛有凉水浇在炙热的火炭上，温度瞬间降低。
<br/>　　刘阿吉察觉到大厅冷下来的气氛，微微顿顿，道：“王爷手下有这样的勇将，咱家只是好奇他这个人，想见见而已。”
<br/>　　“只是，只是……这程大雷可是个山贼。”
<br/>　　“听说啦。他本来盘踞在青牛山，是王爷将他招安到麾下。山贼又如何，能够打败戎族，就是帝国的功臣。明帝陛下说了，这次能击退戎族，王爷，方城主，程大雷功劳各居三分之一。王爷为帝国找到这样的人才，居功至伟啊。”刘阿吉顿了顿，冲在场其他人道：“当然，各位的功劳，陛下也不会忘记的。”
<br/>　　“哈哈……”杨龙停笑笑道：“为帝国选拔人才，本也是我该做的，其实，因为他是个山贼，要不要用他，我也考虑了很久，最后还是冒险启用此人。”
<br/>　　“也为帝国找到一个了不起的人才喔，没让良玉埋没山野。”
<br/>　　“应该的，应该的。”
<br/>　　刘阿吉笑笑，道：“那么，这个程大雷在何处，能不能让我先见一见，陛下心里其实也想见见。”
<br/>　　“这个……山野鄙俗之人，不通礼数，原先又是个山贼，怕冒犯了刘大人，等日后调教好了，再在刘公公面前垂听教诲。”
<br/>　　“那好吧。”刘阿吉显得很遗憾，道：“其实我这次来还有一件事，这次戎族乱国，在帝国造下不少杀业。朝廷决定开科选武举，如今具体日子还在定。等日子定下来，诸位手下有勇猛的武将，莫要自珍，可以送到京城参加武举，会一会天下英雄。”
<br/>　　“朝廷好像很重视这次武举？”
<br/>　　“嗯，陛下亲自定的。”
<br/>　　……
<br/>　　诸人散去后，杨龙停来到崔云间的房间，屏退手下人，杨龙停立刻问道：“二弟，京城那里什么情况？”
<br/>　　杨龙停的夫人，是崔云间的长姐，而崔家在京城是有很高地位的家族。在外人面前，双方论官职，而私下里，大家却是自己人，可以穿一条裤子的那种自己人。
<br/>　　崔云间将京城最近的动向与杨龙停说了，大概和刘阿吉说得差不多，当然，同样的话从二人口中分别说出来，底下的意义就完全不同了。
<br/>　　这次帝国和戎族的战斗，帝国输得那叫一个惨，整条战线被人扒掉裤子，摁在地上狠狠摩擦。最后还被戎族大军临城，逼得帝国陛下签下城下之盟。
<br/>　　在全面溃败的情况下，胜利就显得那样珍贵。憋屈得一直在宫里揪胡子的明帝，忽然发现帝国还有个幽州，幽州还有个青牛山，竟然击败戎族，全歼敌军十万有余。
<br/>　　事实上，帝国和戎族在谈判桌上吵架的底气，完全是靠这一战给的。
<br/>　　那么，对于这一战领兵的将领自然要赏，不仅要赏，还要在所有人面前大赏特赏，让整个帝国的人知道：我们也是有英雄的，我们也是有胜利的，我们并没有输得那么惨。
<br/>　　于是，杨龙停被竖立成了典型。而也因此，崔家可以在朝堂上，获得不少政治资源。
<br/>　　至于即将到来的武科举，也是被打得太惨了，想要从下面选拔出一些人才。这，的确是帝国近年来的一件大事。
<br/>　　崔云间和杨龙停谈了很久，其他小事还有很多，例如：某某御史竟然上本参我们崔家，不知道是谁在背后主使的。
<br/>　　某某过寿礼时，某某某竟然没有亲自去贺寿，莫非两家已经有了嫌隙。
<br/>　　等等之类的，崔云间说得嗓子都干了，他抿了口茶，想起什么似的问道：“那个程大雷……究竟什么情况？”
<br/>　　“程大雷那里……不太好办呐。”
<br/>　　“怎么……他不听你的，难道你还命令不了他。”
<br/>　　杨龙停心道：何止不听啊，若是见了面，他一斧头劈死我的心情也有啊。
<br/>　　他干笑两声：“也不是完全不听啦，只是山贼么，你也知道，不是那么喜欢被控制。”
<br/>　　“不，这个人一定要控制在你手中，这一点很重要。”
<br/>　　“嗯，我明白。”杨龙停轻轻点了点头。
<br/>　　……
<br/>　　青牛山，蛤蟆寨。
<br/>　　如今程大雷身边留下的，只剩下徐神机、秦蛮、林少羽、林冲、刘关张赵以及黄三元等千余人左右。
<br/>　　把戎族留下的物资用来建设山寨，地址还是原先的蛤蟆岭上，至于那山谷……如今里面有十万具尸体，程大雷想一想都怕做噩梦，更不必说住在里面。至于那山谷中埋藏的财报，也先留着吧，反正也不会长腿跑了。
<br/>　　大战过后，休养生息。
<br/>　　程大雷还没确定要不要把逃到南方的弟兄接回来，或者说，大家直接去南方发展。幽州这个地方，实在太穷了，商品经济极其的不发达，抢了路人浑身上下没有一个铜板，一打听，老娘卧床，孩子吃不饱饭，妻子整天骂他窝囊废，听得你一心酸，可能还给他几块铜板。
<br/>　　做个山贼，这样抢下去，怕只怕会越抢越穷。
<br/>　　南方好啊，江南美女多，当然，官兵也更厉害些。
<br/>　　“老程，老程……”
<br/>　　山下突然传来呼声，齐德强和李善言被刘关张押着上山。
<br/>　　“大当家，这两个人说是奉幽州王的命令来，想要招安我们。”
<br/>　　“老程，你肯定没认出我么，我是你的兄弟小强啊，还记得咱兄弟一见如故，不是亲兄弟胜似亲兄弟啊。”齐德强。
<br/>　　“杀了。”程大雷做了个割喉的手势
<br/>　　“老程，老程，你这是把我忘了呐，难道你忘了咱们秉烛达旦，促膝长谈了么？兄弟我这次来，是特意奉幽州王命令，授予你官职的。”
<br/>　　“先别杀，把他们手脚剁下来，丢在粪坑中。”
<br/>　　“老程，老程……你可不能不认兄弟啊，老程。”齐德强听到程大雷的话先是一喜，随之一惊，手伸在空中，突然喊了一声：“程寨主，你可愿为一城之主。”
<br/>　　程大雷一顿，摆摆手，让押着他们的手下先停下来。
<br/>　　“黑石，落叶，铁河，赤沙任选其一。”齐德强高声道。
