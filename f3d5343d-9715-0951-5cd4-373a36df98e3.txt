<br/>?    黑暗降临，一轮圆月高悬，在仙客来客栈之中，十多位大大小小的将军汇聚一堂，等待着陈明的安排。
<br/>    
<br/>        陈明吃完了自己的晚餐，拿起一块手帕擦了擦嘴，朝着楼下走去。
<br/>    
<br/>        陈明走到主座之上坐下，刚想要开口，却是王蒙忽然起身道：“世子殿下，王爷传命，敌军人多势众，而且占尽了地利，此事，已经超出了世子殿下可以控制的局面，请世子殿下撤军，后续安排将由王爷和大将军决断，这里是大将军和王爷的联名密信。”
<br/>    
<br/>        王蒙道完，将手中的密信递了上去。
<br/>    
<br/>        这都安排好了，居然让我临阵撤军？要是我这次撤军了，那这个好不容易得到能够改变自己炮灰命运的任务岂不是和自己没有什么干系了？怪不得是剧情挑战任务，若是没有任何难度，怎么可能改变自己一个炮灰的命格。
<br/>    
<br/>        陈明伸手接过了密信，也不打开，两指夹住密信，直接放在了烛火之上点燃，烛火点燃了密信，将烧成灰烬的密信丢在面前，陈明冷冽的眸光扫视了一圈，道出不容置疑的话语，“将在外，君命有所不受！”
<br/>    
<br/>        陈明看向在座的将军们，笑了笑，道：“我也不怕告诉你们，杀生殿堂有这十万大军，而我等，只有三万大军，而且对方，还占尽了地利，姜王陵更是一个易守难攻之地。不愿意听从我的命令的，可以出去。”
<br/>    
<br/>        将军们面面相觑，却是不发一言，只敢用眼神交流，过了一会儿，终于有一个白袍将军站了起来，拱手道：“对方有着十万大军，我等却是只有三万，而且对方还占尽了地利，这要去攻打，无异于送死，末将，恕不奉陪！”
<br/>    
<br/>        道完，这个白袍将军起身，转身离开。
<br/>    
<br/>        就在这个白袍将军走到客栈门口之时，陈明手中掐诀，背后的素王剑落在手中，凭空一掷，素王剑划过一抹素白色的剑光，只听噗嗤一声，已经是将那个白袍将军扎了个对穿。
<br/>    
<br/>        在座的将军们露出一抹惊诧的神色，身子微微一抖，不愧是摄政王之孙，和摄政王一般狠辣果决。
<br/>    
<br/>        陈明走到客栈门口，伸手提起满是鲜血的素王剑，提起白袍将军的白袍擦拭着素王剑上的鲜血，陈明回头笑着看向客栈之内的将军们，露出一抹人畜无害的笑容，“还有谁要走吗？要走的话就快点，省的我还要再擦一次剑。我飞虎军，可从来不少想要当将军的人，死了十几个，可没有什么影响。或许去进攻杀生殿堂只有死路一条，但是现在就想要临阵脱逃，我可以送你们早点去死。”
<br/>    
<br/>        话里的威胁之意已经图穷匕见，将军们哪里还敢安坐，一个将军连忙起身跪下道：“唯世子殿下马首是瞻，世子殿下之命，末将必定以死献忠！”
<br/>    
<br/>        陈明的意思已经很明确了，要么现在死，要么跟他去进攻杀生殿堂寻一条活路。别说什么一起上杀了陈明，就算是一起上，还不一定打得过陈明。而且陈明有着虎怅卫守护，现在上不是找死吗？而且就算能够逃走，摄政王和大将军估计就是一笑置之，还会真的惩罚陈明这个亲孙子亲儿子不成？而他们，却是要等待陈明的报复。
<br/>    
<br/>        一个将军表明了态度，其他将军也是有样学样，起身朝着陈明跪下道：“唯世子殿下马首是瞻，世子殿下之命，末将必定以死献忠！”
<br/>    
<br/>        陈明点点头，道：“很好！”
<br/>    
<br/>        陈明走回自己的主位，坐下，然后看向王蒙，“王蒙泄露军机，去领三十军杖，不可再犯！”
<br/>    
<br/>        王蒙见陈明已经控制住了局面，无奈苦笑，只得拱手道：“属下遵命！”
<br/>    
<br/>        道完，便出去领军杖了。
<br/>    
<br/>        陈明继续安排道：“三日之后，我要你们在葫芦口上集结两万大军，其他一万大军，带齐能够搜集到的所有的猛火油，分散包围姜王陵，等候我的命令。命杀生殿堂之中的暗卫，在我等准备好之后,不惜一切代价破坏掉杀生殿堂之中的地道。事成之后，立即发信号弹汇报消息，并从葫芦口一带突围。”
<br/>    
<br/>        陈明已经安排完了，却不见这些将军们有反应，当即冷冷问道：“明白了吗？”
<br/>    
<br/>        将军们齐齐起身，拱手道：“我等这就去办！”
<br/>    
<br/>        王蒙虽然领了军杖，不过还是将陈明执意进攻的消息禀报给了陈狱虎和老王爷，陈明虽然知道，但是也没有去管，如今大权在握，等到老爹他们知道这个消息，只怕这里已经打起来了。
<br/>    
<br/>        三天之后，夜晚降临。
<br/>    
<br/>        摄政王府。
<br/>    
<br/>        老王爷披着一身黑袍走进摄政王府，不过刚刚走进大厅，便将身上的黑袍一把扔在了地面之上，“又是什么事，这是第二次了，第二次了！陈狱虎，你必须给老子一个解释！”
<br/>    
<br/>        陈狱虎也是无奈啊，这小王八蛋不听话，他老子只能把火散在他身上了，“父王，明儿当着所有将军的面，将我们传命的密信给烧了，还说什么将在外，君命有所不受。”
<br/>    
<br/>        老王爷伸手摸了摸胡子，笑道：“将在外，君命有所不受，这小子，倒是有我年轻的时候的风范，毕竟是我们陈家的种，等等，你是说他准备抗命不遵！”
<br/>    
<br/>        陈狱虎无奈地点点头。
<br/>    
<br/>        老王爷气得吹胡子瞪眼，“这小王八蛋，还反了天了，快派人，无论如何，一定要将这小王八蛋完整的带回来！”
<br/>    
<br/>        陈狱虎道：“我已经将剩下的二十虎怅卫全部派出去了，并且已经下令调集姜王陵附近所有的飞虎军前去救援，只是恐怕我们消息还没到，明儿那里已经打起来了，我请父王过来，只是想要将这个消息禀报父王。”
<br/>    
<br/>        老王爷的右拳紧紧一握，咔咔作响，“难道我陈王，要绝后不成！这个孽障！”
<br/>    
<br/>        姜王陵之外，陈明骑在一只猛虎之上，只见从密林之中窜出一个黑影，在陈明之前恭敬跪下，拱手道：“夜鹰大人已经动手破坏地道了！”
<br/>    
<br/>        陈明点点头，拿出地图，让周围的几个将军过来，挥手在葫芦口的一条线上划了一下，道：“速速将这些伐木带连接起来，立刻去办！”
<br/>    
<br/>        将军们完全不懂陈明的意图，但是现在，箭在弦上，不得不发，只得拱手道：“末将遵命！”
