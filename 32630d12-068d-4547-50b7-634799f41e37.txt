　　第109章
<br/>　　这里是扬州，是天下三分明月夜，二分无赖在扬州的扬州。
<br/>　　扬州是个好地方，好山好水，既有名垂千古的好文章，也有腰身比水还软的好女人。
<br/>　　当然，也有别的……
<br/>　　码头停泊着大小船只无数，客商船夫往来如梭，单单一个扛麻袋的苦力，穿着气质也与幽州不同。
<br/>　　乱哄哄的码头上，有一伙人格外明显。码头上的人见了，纷纷避让，实在躲不过的，也只得硬着头皮打个招呼。
<br/>　　“路爷，您辛苦，最近一向可好。”
<br/>　　“小六，别废话，这个月的份钱你还没交，今天挣了多少钱，拿出来吧。”
<br/>　　“路爷，您手下开恩，这是我给老母看病用的，等下次，我一定凑齐给您送过去。”
<br/>　　“废什么话，你拿过来吧你。”
<br/>　　一伙人一拥而上将这汉子踢到在地，把钱袋从他怀里搜了出来。路子野用手掂了掂，喊了声晦气，挥挥手道：“滚滚滚，别在爷眼前碍眼。”
<br/>　　路子野是漕帮的人，漕帮不仅扬州一地有。在这个时代，水路运输有极重要的经济地位。码头上总会聚集着这样一拨人，靠水吃饭。大家都是用短刀、棍子抢地盘，拼下来的就是漕帮。
<br/>　　路子野算是这个码头的负责人，进港的货船，做经济的牙行，扛麻袋的苦力……总之，就算是在码头讨饭的乞丐，也得给漕帮贡献一份钱。
<br/>　　程大雷在甲板上往下望，此地的繁华景象也是令他大开眼界，心中迫不及待想要领略一下扬州的风土人情。徐神机、高飞豹、张肥，赵子龙、柳芷、苏樱、小白狼随他一起下船，其他人仍留在船上。
<br/>　　“下船后记得唤我公子，咱们是北国世家，别把山贼那一套带出了，扬州是个文质彬彬的所在。”下船之前，程大雷在做最后的吩咐，其他人他都不担心，担心的是张肥和高飞豹这两个憨货。
<br/>　　几人都已迫不及待，有种土老帽第一次进城的期待。
<br/>　　可是，几人还未下船便被拦住了。
<br/>　　“怎么都是第一次来扬州，连扬州城的规矩都没人懂么。”
<br/>　　七八个人堵在船头，路子野斜睨着脸，满口扬州官话。
<br/>　　徐神机几人面面相觑，倒不是因为被人拦住，而是这些人的话他们听不太懂。扬州土语和幽州口音大相径庭，众人虽然勉强能听懂几个字，却连不成一串话。
<br/>　　程大雷是听得懂的，而且还会说，不过此刻他没有开口。
<br/>　　路子野拧起眉头，满脸不耐烦道：“一个会说人话的都没有，一群土老帽。”
<br/>　　大概是能猜出个意思，一时间都有些自卑，目光闪烁想寻个主事的人，目光都望向苏樱。
<br/>　　苏樱也不会说扬州土语，表情有些尴尬。
<br/>　　此刻一个戴小帽的人钻出来，道：“诸位，诸位，我是此地的牙行，诸位可以叫我富贵。”
<br/>　　路子野：“富贵，是你小子啊，你和这群乡巴佬讲讲规矩，别让我们动手。”
<br/>　　富贵满脸堆笑，冲徐神机道：“诸位爷，这位是漕帮的路爷，这里是路爷的地盘。”
<br/>　　徐神机倒是知道漕帮，也明白漕帮在码头的势力，他拱拱手道：“原来是漕帮的兄弟，诸位辛苦辛苦……”
<br/>　　“谁是你兄弟。”路子野连眼皮都没抬。
<br/>　　“……”徐神机。
<br/>　　“诸位，你们到漕帮的地盘做生意，就要讲扬州的规矩。”
<br/>　　“这里什么规矩？”
<br/>　　“咱们是过十抽一。搬运货物的工人，必须是漕帮的人。你们能卖给谁不能卖给谁，也得由漕帮做主。”富贵道。
<br/>　　“过十抽一，意思是十成货有一成是漕帮的？”徐神机知道漕帮在码头的势力大，但没想到漕帮的势力大到这种地步。
<br/>　　“如果这规矩我不想守呢？”这时候程大雷分开众人走到船头。
<br/>　　程大雷说的是扬州官话，口音还十分干脆。路子野上下打量他几眼，见他穿着锦袍，脖子上缠着貂绒，腰带上一颗硕大的宝石。
<br/>　　只是看了一眼，路子野便差点捂嘴笑出声来。一群人叽叽喳喳，窃窃私语。
<br/>　　“这是哪儿跑来的蛮子，这幅打扮是不知道咱们扬州的天气呐。”
<br/>　　“估计在他们那里，这都是好打扮了。”
<br/>　　“呵呵，乡巴佬。”
<br/>　　程大雷满头黑线，他便知道扬州的审美观和幽州是不同的，得，现在被人当成穿貂挎蜜的东北大哥了。
<br/>　　路子野摆摆手，示意周围安静下来，他道：“这规矩是漕帮定的，为的也是大家生意兴隆，如果你不想守也成，那船上的人和货都不能下船，万一其中藏污纳垢，窝着山贼海盗什么的怎么办，我们漕帮也得保一方安定不是。”
<br/>　　咦，他怎么知道我们是山贼的。
<br/>　　“是这样么……”程大雷笑笑，招招手道：“来，你近前来，我给你讲讲规矩。”
<br/>　　“讲什么规矩？”
<br/>　　“我还以为什么事呢，不就是想要钱么，告诉你，爷手指头缝里露出个三瓜俩枣也够你一辈子吃喝。”
<br/>　　路子野打量程大雷一眼，这人虽然看上去没什么教养，但估计很有钱，是个北国来的暴发户。
<br/>　　他咧嘴笑笑，搓着双手靠近：“算你小子懂规矩，你有什么东西啊？”
<br/>　　“看仔细了。”程大雷手摁绷簧，锵啷一声宝剑出鞘，甩手一挥，从路子野脖子上划过。
<br/>　　一剑封喉。
<br/>　　血滴答答流出来，路子野的尸体晃了三晃，噗通一声倒在地上。
<br/>　　程大雷将剑甩了甩，血滴在空中划过，黑铁剑身不沾一丝血。若说李行哉给程大雷留下的这柄剑委实不错，杀人不沾血，有时候也不能小瞧这个世界的锻造技术。
<br/>　　满场俱惊，张肥和高飞豹睁大眼睛看着程大雷，脑中想的是：不是说要讲文明懂礼貌文质彬彬么，可程大雷怎么……
<br/>　　他就算在蛤蟆寨当山贼的时候，也从未这样过呐。
<br/>　　富贵吓得跌了一跤，结结巴巴道：“你，你……”
<br/>　　程大雷还剑入鞘，用绣帕捂嘴，口中不耐烦道：“告诉诸位一声，我欧巴欧公子到了，从今天开始，这地界儿爷立规矩。”
<br/>　　
<br/>(本章完)
