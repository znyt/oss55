　　“苏莫！”
<br/>　　黑衣青年走到苏莫身前，立刻冷喝一声。
<br/>　　沉浸在感悟之中的苏莫，闻言顿时睁开了眼眸，扫了一眼面前的黑衣青年，又瞥了一眼其身后的华服青年，他顿时心中明了。
<br/>　　麻烦来了！
<br/>　　苏莫没有说话，只是淡淡的望着对方，等待对方开口。
<br/>　　“你抢劫我兄弟等四人的储物戒，真是胆大包天，我给你一个机会，双倍返还，并且赔礼道歉！”黑衣青年冷冷的说道。
<br/>　　他本是想杀死苏莫的，但考虑到苏莫可能和火元圣师有关系，以免惹下麻烦，便退了一步。
<br/>　　“不好意思，没有！”苏莫摇了摇头道。
<br/>　　双倍返还？
<br/>　　真是痴人说梦！
<br/>　　别说他没有这么多财富，就算有的话，也不会交出来。
<br/>　　他之所以抢劫对方，是对方要敲诈他们，咎由自取。
<br/>　　“不知死活！”黑衣青年闻言，眸中顿时杀机爆闪，华服青年更是气的双拳紧握，捏的咯嘣作响。
<br/>　　“好，既然你不同意，那就接受我的挑战，敢不敢？”黑衣青年顿时暴喝一声，声震四野。
<br/>　　奥义圣山周围的神武学府弟子，全部被惊动了，所有人都停止了参悟奥义，向此地望了过来。
<br/>　　“这不是北苑的冷江吗？”
<br/>　　“还有何传岩！”
<br/>　　“那个小子是哪苑的弟子！”
<br/>　　“他是我们南苑弟子，好像叫苏莫，上次还得罪了太史轩！”
<br/>　　不少人低声议论了起来，都在观看事态的发展。
<br/>　　“没有兴趣！”面对黑衣青年冷江的挑战，苏莫淡淡的说道。
<br/>　　“是吗？”冷江闻言轻笑一声，下一刻，他陡然出手了。
<br/>　　轰！
<br/>　　铁拳如山，直击而出，轰向苏莫的脑袋。
<br/>　　这一拳力量之强，堪称无与伦比，咆哮的拳劲如同一条怒龙，充满毁灭般的力量。
<br/>　　此拳速度极快，瞬间便临近了苏莫的面门。
<br/>　　“找死！”苏莫见此顿时大怒，他没想到对方居然毫无征兆的出手了。
<br/>　　仓促之间，他急忙脑袋后仰，并且抬手封挡。
<br/>　　下一息。
<br/>　　轰！
<br/>　　冷江的拳头，狠狠的轰在了苏莫的手掌上，霎时之间，如同两座大山狠狠的撞击在了一起，发出惊天动地的爆响。
<br/>　　嗖！
<br/>　　苏莫直接被对方一拳轰飞了出去，身形划破空气，倒飞万丈，才缓缓的落在了地上。
<br/>　　“废物，敢不敢战？”冷江身形挺拔，面带奥义，冷冷的凝视着远处的苏莫。
<br/>　　他就是要打出苏莫的火气，逼苏莫应战。
<br/>　　不过，他心中也是暗暗震惊，本以为刚才一拳，能将苏莫重伤，没想到苏莫却是丝毫没有受伤。
<br/>　　虽然他已经从华服青年何传岩的口中知晓，苏莫的实力很强，但现在亲眼所见，还是非常的惊讶。
<br/>　　一个准帝境的武者，居然能接住他五成实力的一拳，这种越级战斗能力，的确恐怖，都堪比学府的那些圣子了。
<br/>　　苏莫稳住身形，望着冷江，脸色阴沉无比，心中杀机升腾。
<br/>　　“鬼圣前辈，此人是什么修为？”苏莫用意念向阎魔鬼圣询问。
<br/>　　他看不太清楚对方的修为，感觉对方像是武帝境三重，又像是武帝境四重。
<br/>　　“武帝境三重！”阎魔鬼圣说道。
<br/>　　苏莫心中了然，武帝境三重的话，倒是没有太过强大。
<br/>　　只不过，他现在也没有把握战胜武帝境三重的武者，他的实力虽然比武帝境二重的武者强一些，但很难战胜武帝境三重武者。
<br/>　　“好，我接受你的挑战！”望着满脸傲意的冷江，苏莫点了点头。
<br/>　　被人欺负到了头上，他自然不会退缩，即便对方实力超强，他也不惧，一战而已。
<br/>　　“好！明日一早，战界你我一决胜负！”冷江见苏莫答应了下来，顿时面上露出了笑意。
<br/>　　只要苏莫答应一战，那一切就迎刃而解了。
<br/>　　“一个月后吧！”苏莫淡淡的说道，现在与对方一战，他并无把握，所以只能将时间推后，争取再提升一些实力。
<br/>　　“拖延时间吗？”冷江闻言双眸微眯了下来，他并不傻，一个月的时间，会有很多的变数。
<br/>　　当然，他可不认为一个月后，苏莫的实力能超越他。
<br/>　　他担心的是，这段时间内，苏莫会不会找到什么帮手，或者是通过火元圣师的关系，向他施压。
<br/>　　不过，略一沉吟，他还是点了点头，道：“好，那就一个月后，你我一战！”
<br/>　　冷江也没有太过担忧，他也是有些背景，虽然远不如火元圣师，但也算有人撑腰。
<br/>　　“我们走！”随即，冷江招呼何传岩一声，便转身离去。
<br/>　　“苏莫，我说过，你会付出代价的！”华服青年何传岩瞥了苏莫一眼，眸中露出戏虐的笑意。
<br/>　　他仿佛已经看到了，苏莫被强势镇杀，魂飞魄散的场景。
<br/>　　随即，何传岩也转身离去了。
<br/>　　苏莫叹了口气，心中郁闷不已，他想安心修炼，总有人来找麻烦。
<br/>　　当然，这也是他自己做事不留情面，惹下的祸端。
<br/>　　少倾之后，苏莫也不再多留，通过出口返回学府。
<br/>　　而苏莫和冷江的约战，也被奥义圣山周围的弟子听到，很快便在学府中传开了。

<br/>　　苏莫回到自己的阁楼，便盘膝坐在蒲团上，低头沉思，怎么样最快速度的提升实力。

<br/>　　他可不会信对方的话，在这种强者为尊，杀伐无尽的世界，低调才是王道。
<br/>　　实力不够强大，高调只会死的更快！
