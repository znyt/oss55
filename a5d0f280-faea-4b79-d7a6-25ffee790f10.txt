        “这跟我有什么关系吗？”傅夜沉突然间冷脸了。
<br/>
<br/>        郑忠威顿时一脸懵然地凝视着傅夜沉。
<br/>
<br/>        难道不是他这大少爷对那落以柔有意思吗？
<br/>
<br/>        不然，他这当“忠臣”又岂会跑到“太上皇”那儿去吹耳边风！
<br/>
<br/>        这三年来，傅老太爷没少给他傅夜沉物色婚约对象。
<br/>
<br/>        除了女明星，傅老太爷看不来，其余的婚约对象上到官家小姐、军门将女、政界精英，下到商家千金、职场白领、就连平凡百姓家的女儿，只要是家庭健全、品学兼优、相貌端正、德才兼备，傅老太爷都会列入他的候选长孙媳的名单。
<br/>
<br/>        傅夜沉对于爷爷傅炎廷的念叨，向来是左耳进右耳出，就连爷爷傅炎廷每次特意安排的“相亲宴”，他都从来不出席，完全不给爷爷傅炎廷和女方那边的客人的面子。
<br/>
<br/>        结果，这件事情的第二天，郑忠威就莫名其妙地因身体不适，请了长假。
<br/>
<br/>        所以，这总裁特助的所有工作，全压在了落以柔的身上。
<br/>
<br/>        落以柔第一次进总裁办，找这位傅氏集团的帅总裁签字时，还闹了一个乌龙。
<br/>
<br/>        “大哥，你好厉害，居然应聘上了傅总的贴身男秘！噢，对了，傅总呢？我这儿有三个文件，需要他签字。”落以柔走到傅夜沉的办公桌前，怀里抱着一叠文件，微笑着问道。
<br/>
<br/>        她一直都不知道他叫什么，所以干脆一直唤他“大哥”，至少，他看起来确实是比她大。
<br/>
<br/>        傅夜沉从文件前抬起头来，无语地看了落以柔一眼，便向她伸出手来。
<br/>
<br/>        落以柔却把怀里的文件抱得紧紧的，一脸誓死不从的样子，严肃道：“师父说了，文件必须亲自交到傅总手中，不能让傅总以外的第三个人碰。虽然咱俩认识，但是也不能坏了师父的嘱托。”
<br/>
<br/>        她嘴里所说的师父，是她的上司郑忠威，因为她还有很多地方需要学习，索性直接拜了郑忠威为师。
<br/>
<br/>        “……”傅夜沉汗颜，面对落以柔的“傻里傻气”，简直无言以对。
<br/>
<br/>        “这里是什么地方？”傅夜沉忍不住冷冷地质问。
<br/>
<br/>        落以柔咧嘴笑了笑，露出了两颗可爱的小虎牙：“当然是总裁办啊！”
<br/>
<br/>        “男秘书会坐在总裁办工作？”傅夜沉没好气地反问。
<br/>
<br/>        落以柔愣了半天，还未反应过来，纳闷道：“其他公司里，都是老板和秘书同办公室的啊！方便工作交流。”
<br/>
<br/>        “……”傅夜沉不想再多说什么了，而是从老板椅上起身，靠着办公桌边缘，伸手直接扯走了落以柔怀里的文件。
<br/>
<br/>        落以柔缓过神来后，才后知后觉地发现自己的文件被夺走，连忙绕过办公桌，想要拿回自己的文件：“大哥，你别为难我啊！”
<br/>
<br/>        岂料她的话刚说完，手里的签字笔未拿稳，掉在了地上，刚好滚在了她的脚下，她一脚踩上去打了趔趄，整个人直接虎扑到了傅夜沉的身上。
<br/>
<br/>        落以柔紧接着下意识地抬起头来，谁知差点点就跟傅夜沉的鼻尖撞到了一块。
<br/>
<br/>        两人四目对上。
<br/>
<br/>        一个青涩娇羞。
<br/>
<br/>        一个情绪莫测。
<br/>
<br/>        她眼神清澈，身体娇软，散发着和筱颖身上那近乎一样的柠檬清香。
<br/>
<br/>        傅夜沉也未料到，自己竟然又一次在这个女孩身上看到了童筱颖的影子。
<br/>
<br/>        落以柔讷讷地看着傅夜沉的俊脸，双颊瞬间染上了两片红晕。
<br/>
<br/>        她不敢再与他对视太久，羞涩中，她移开了目光，有些不知所措地从他的身上起来。
<br/>
<br/>        他的帅气，让她心里直打小鼓。
<br/>
<br/>        而殊不知，她刚刚虎扑到傅夜沉身上的那一瞬间，被不知何时出现在门口的连华生拿手机给拍了下来。
<br/>
<br/>        “咳——咳——”就在落以柔不知道自己该说些什么的时候，连华生故意轻咳了几声，示意这儿除了他两，还有他这么一个大活人在这。
<br/>
<br/>        傅夜沉淡定地看向连华生，询问道：“你怎么来了？”
<br/>
<br/>        “我怎么就不能来呢？哈哈，你是嫌我打扰到你们的二人世界了吗？”连华生一边走过来，一边打趣地说道。
<br/>
<br/>        落以柔见状，趁机将傅夜沉夺走的文件一一拿了回来，抱在怀里，就像一个含苞欲放的青涩少女一般，向连华生微微欠身，红着脸害羞地跑开了。
<br/>
<br/>        傅夜沉再次无语，他只想说，那几个文件，他还未签字。
<br/>
<br/>        连华生目送落以柔的背影离开后，忍俊不禁地说道：“这个女孩好单纯啊！呆萌呆萌的，倒是挺适合你这座移动的高傅大冰山。”
<br/>
<br/>        “……”傅夜沉听着连华生这话，一脸不悦，剑眉微微蹙起。
