　　众人都一阵激动，立即跑向出口，罗杰和白忧都慢了半步，在看到有人安全走出后才跟上。
<br/>　　敞开的道路的确就是出口，也完全没有其他诡异，二十人一出来便是那座宽大平台，也就是阳谷的入口。
<br/>　　终于确认自己等人都无恙的出来之后，很多人都松了一口气，在阴谷内的遭遇实在惊心动魄，只是同时他们又不禁奇怪，刚才那个忽然出现的出口实在诡异，并不像是白忧找到的出路，反倒是像这个迷宫自行回到原样。
<br/>　　“刚才是怎么回事？”罗杰问出声。
<br/>　　白忧紧皱眉：“迷宫自己把出口露了出来。”
<br/>　　“莫非重点是时间？”罗杰忽然领悟过来。
<br/>　　“谁知道呢？”白忧幽幽的开口，望着剩下为数不多的人又是一阵烦躁，这大概是他们神秘家族损失最大的一次。
<br/>　　“还继不继续？”罗杰看着脚下的平台，神态复杂。
<br/>　　白忧回头直视着他：“当然。”
<br/>　　没有任何犹豫，在白忧强势的态度之下，这两家族的人哪怕损失惨重，也继续朝着阳谷进发。
<br/>　　此时在火族旧址内，一名火族人小跑着来到炎良面前。
<br/>　　“少主，那些人逃出去了。”他开口汇报，眼中带着不甘心。
<br/>　　炎良愣了愣，而后问道：“还剩多少人？”
<br/>　　“二十来个。”那人回道。
<br/>　　炎良一声冷笑：“只是封死道路而已，竟然就死了这么多人，这些四偏门真是越活越回去了。”
<br/>　　汇报的人抬起头，有些担忧的说道：“还有一件事，就是那些在阴谷内的两家族人，好像有人在乱窜之中发现了家族旧址。”
<br/>　　炎良一愣，显然没有料到这样的发展，他皱眉问道：“白家和罗家的旧址都有人？”
<br/>　　“应该是。”汇报的火族人不能肯定，解释道：“我们行动受限，能确定的是那些人的数量在减少，也没有人从出入口离开，唯一的可能就是去了旧址。”
<br/>　　炎良眼神深邃起来，下令道：“这些人中如果有人往出口走，别杀，把他们赶回去。”
<br/>　　那名火族人一惊，错愕不解：“少主？”
<br/>　　炎良没有过多解释，只是又加了一句：“确保他们待在旧址即可。”
<br/>　　远在阴谷另一尽头的水族旧址内，鱼柔女巨大的金色鱼尾在水中摇摆，她表情玩味的看着远方，身旁已经喋喋不休半个小时的鱼勇男在说什么，她似乎一句都没听进去，整个人都看上去漫不经心。
<br/>　　“我说，你到底有没有在听我说话？”鱼勇男终于忍不住，不再用尊称，直接朝着圣女大吼起来。
<br/>　　鱼柔女回过头，鱼尾上下起伏带起一阵水花，她笑道：“不就是两族有人去了旧址，阴谷禁制只剩一半？”
<br/>　　鱼勇男叹了叹气：“这么大的事，你怎么一点反应没有？难道你都不惊讶，不着急？”
<br/>　　“阴谷的出入口刚才重启了一下，我当然能感受到是禁制的变化，没什么可惊讶的。”鱼柔女淡笑着回答，紧接着又道：“至于着急就更没必要了，四偏门早晚都会进入旧址，制裁一门早晚会回归，回炉早晚会重现，我着急什么？”
<br/>　　听着这话鱼勇男更加焦虑：“我说妹妹啊！咱们水族的战力现在可是非常弱，如果七门回归，守护者重现，这阴谷禁制一旦解除，我们可敌不过任何一族的攻势，更别说一直对我们虎视眈眈的火族！”
<br/>　　“不是有火族圣物在手么？”鱼柔女又是一阵轻笑，回身看向鱼勇男的眼神意味不明：“大不了大家一起死啊。”
<br/>　　鱼勇男彻底噎住，当场无话可说，只觉得越发看不懂自己这个妹妹，不过传闻里都说水族圣女一旦上任都会性情大变，原因便是接受了前几任圣女的意志，从此便不再是单一的人格。
<br/>　　比起火族的爆发性强和家底丰厚，水族更加注重的便是意志的传承，以及超乎常人的敏锐感和灵性。
<br/>　　只是忽然间，鱼勇男又觉得一阵不对劲，忍不住脱口而出：“你知道火族圣物在哪？”
<br/>　　鱼柔女没有回答，只是淡笑着看着他。
<br/>　　鱼勇男连忙低头不敢再多问，眼前这个早已不是他熟悉的亲妹妹，而是继承了千年前几任圣女意志的当代圣女大人！
<br/>　　——————
<br/>　　在阴阳谷悄然改变的时候，此时的狼牙基地也爆发了一场血雨腥风。
<br/>　　防守队在步步逼出了几名暗杀者后，四周忽然涌·出了另一批暗杀者，来势凶猛一下子便让那批防守队成员死伤不少。
<br/>　　围在木屋外圈的人只能步步后退，范围圈也越来越小，越发难以施展身手。
<br/>　　尚九娣已经加入了战斗，时不时的开枪助阵，白允儿依旧死守着木屋门，注意着一切企图靠近的暗杀者。
<br/>　　在时间一点点的流逝中，白允儿的脚边也躺了几具尸体，此时忽的一阵刀光剑影，两名暗杀者猛然冲破防守队的防御圈，直逼木门而去！
<br/>　　没有人能当得了这忽然爆发的强烈攻势，甚至那一条直线上的防守队成员，都瞬间血花飞溅的倒地不起，血流成河的场面让顾凉晨心中警铃大响，这才发现他们之前对付的那些人不过是一些最低战力的暗杀者。
<br/>　　顾凉晨大惊之中立即回身，四周的人也一阵慌乱，当所有人都以为他们暗杀的对象是木屋中的楚涵，连白允儿也下意识的以身躯挡住木门之际。

<br/>　　顾凉晨猛然愣住，忽然察觉到有一丝不对劲。
