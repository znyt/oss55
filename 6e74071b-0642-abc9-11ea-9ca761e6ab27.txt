　　各种各样的舆论铺天盖地，最后就连皇城之内的禁军都开始讨论起这个问题。
<br/>　　原本还信心满满的长乐在这样的讨论中也开始心情忐忑，找到李承乾商量对策：“哥，你确定真的没有问题么？万一那些异族不来怎么办？”
<br/>　　“喜欢来就来，不来也无所谓，但如果不来的话，他们最好能考虑一下后果。”李承乾对长乐提出的问题毫不担心。
<br/>　　正所谓真理只在大炮射程之内，大唐的火炮可以打到什么位置，他李承乾的话在那里就是圣旨，敢不同意最好先考虑一下后果。
<br/>　　“哥，我只是不明白，为什么你一定要搞什么华夏杯，为什么要把那些异族都找来呢？大唐已经够大了，我们自己的队伍就有不下百支，应该足够打比赛了吧？”长乐对李承乾的做法有些不理解，很是纠结的问道。
<br/>　　在她看来，李承乾这样根本没有必要找那些异族番邦来大唐参加什么球赛，而且为这件事情付出那么多的代价似乎也有些不大划算。
<br/>　　“长乐，你说错了，现在的大唐虽然已经足够强大，但在世界上的影响力却并不大。”李承乾放下手中的笔，坐直了身体沉声说道。
<br/>　　刚刚他一直在处理各地送来的奏折，并没有将长乐的问题放在心上，可是随着问题的深入探讨，他觉得很有必要与长乐好好说道说道。
<br/>　　“为什么？”长乐好奇的问道：“我们只要过好自己的日子不就行了，在异族那里有没有影响力有什么关系？”
<br/>　　“呵呵……”李承乾看着长乐意味深长的笑了笑：“你是只知其一不知其二，虽然我们大唐现在已经足够强大，可是你知道大唐留给周边那些异族的印像是什么么？”
<br/>　　“是什么？”长乐问道。
<br/>　　“霸权、侵略、野蛮！”李承乾苦笑说道：“这样子是不对的，长期积压下来会导致异族对大唐的敌意越来越强，虽然他们现在不敢怎么样，但如果万一有一天大唐不如现在强大了，谁又能保证他们不起来反抗，共同推翻这个他们眼中的‘大魔王’呢。”
<br/>　　“这……”长乐显然没有考虑到李承乾所说的问题，现在听他一说也有些不知如何回答。
<br/>　　要知道，虽然每个人都希望自己一手打造出来的王国可以长盛不衰，但这显然是不现实的，毕竟人不能长生不老，国家自然也不可能长盛不衰。
<br/>　　这是客观事实，不会以个人意志为转移。就好像我们都想发财，想有一个好工作，有一个好老婆，但现实的情况却往往是相反的一样。
<br/>　　所以李承乾所担心的问题并不是不存在，而是很有可能会发生。
<br/>　　但就算是这样，那么改变大唐在异族眼中的形象就能解决这个问题么？带着深深的犹豫，长乐对李承乾提出了心中的疑惑。
<br/>　　“当然不能从根本上解决这个问题，只是这样可以减少一些异族对大唐的敌意罢了。”李承乾耸了耸肩膀，无可奈何的说道：“现在大唐已经无力再扩张了，改变一下对外政策也是不得已！”
<br/>　　“哦，那我明白了。”长乐在李承乾的解释下点了点头，不在坚持自己的观点。
<br/>　　“明白了就好。”李承乾轻轻敲了敲桌面，那样子几乎与老头子考虑问题时的样子差不多，片刻之后对长乐说道：“如果你这段时间忙不过来，可以找晓晓和雨馨她们两个帮忙，别让她们两个觉得被排斥就好。”
<br/>　　长乐听了李承乾的安排，掩口轻笑道：“哥，难怪母后说你是个情种，竟然连这种事情你都要惦记着！”
<br/>　　“臭丫头，连你哥都调侃起来了？胆子真是越来越大了。”李承乾被长乐笑的有些尴尬，恼羞成怒的瞪了她一眼。
<br/>　　“好啦，这件事情我早就已经安排下去了，一定不会让你的那两个小美人觉得被冷落。”长乐笑嘻嘻的看着李承乾，似乎一点也不在乎他是不是在发脾气。
<br/>　　对此，李承乾也有些无奈，只能装成很生气的样子，摆了摆手示意长乐快点离开。
<br/>　　……
<br/>　　而尽管民间为了华夏杯的事情吵个不休，但准备工作还是有条不紊的一点点展开了。
<br/>　　首先便是场地的选择，长安城北面是皇城，大明宫和太极宫都在北面，赛场自然不能设置在这里；而城的西面又长安火车站，每天都有列车进站装卸货物，再加上客运列车也会进不时的进站，赛场的布置自然也不能安放在这边。
<br/>　　所以考虑来考虑去，最后所有人一致认为，不如就把赛场设在城东和城南，在这两个方向分别各自修建两座马球场。
<br/>　　而随着四座马球场的施工，长安杯马球预选赛也如期开始。
<br/>　　一直支支由各大家族培养的球队纷纷露脸，细数之下竟不下百余支之多。
<br/>　　十六卫的球队、勋贵家里的球队、世家的球队、还有下面一些州府的球队，名字起的也是五花八门，比如像什么李家王牌队；程家必胜队；王家种子队；长孙夺冠队……。
<br/>　　细数之下就会发现，类似于十六卫这样，以每一卫的名字来命名的队伍竟然算是最正常的队伍。
<br/>　　对此，李承乾虽然觉得很尴尬，但是奈何这帮家伙认死理儿，不管他怎么说，就是不改名字，劝了几次之后索性他也不再劝了，爱咋咋地吧，反正丢脸的又不是他这个皇帝陛下。
<br/>　　而与此同时，随着各个马球队之间展开较量，那股子比赛的气氛甚至被那些老家伙们带到了朝堂之上。
<br/>　　每次上朝的时候，李承乾都能感受到大殿之上浓郁的火药味，彼此间互为对手，或者将来会成为对手的老家伙们，一个个全都吹胡子瞪眼，看着彼此的神眼中仿佛都带着敌意。
<br/>　　不过，这样的情况正是李承乾所需要的，老家伙们彼此对立，必然会分散精力，到时候就不会一心一意与他这个当皇帝的为难，他今后的日子也会过的舒服一些。
<br/>　　这是李承乾第一次在皇帝这个位置上尝试老头子所说的分化之法，现在看来似乎多少有那么一点点的作用。
