        “我都不急，爷爷你比我还急。”明司寒跨过门槛，弯腰伸手挪了一条圆木凳过来，在连星儿的身边坐下后，接着说道，“虽然明冷两家是豪门联姻，但怎么着，我跟星儿也得培养一下感情是不是？”
<br/>
<br/>        连星儿听后，不禁睐了明司寒一眼，嘴角瘪得像抹下弯的月牙，在心底嗔怪：哼，谁要跟你培养感情啊！
<br/>
<br/>        明护国却是一副过来人的样子：“我看你两感情挺好的。豪门联姻，不需要你们爱得死去活来。”
<br/>
<br/>        明司寒咧嘴一笑：“爷爷，我跟星儿都还没有谈过恋爱。那你们做长辈的是不是要给我们谈恋爱的机会？”
<br/>
<br/>        “那行，你两今天就开始谈恋爱。所有开销，爷爷全包了。”明护国拍着胸脯说。
<br/>
<br/>        连星儿突然觉得这明司寒不是来帮她的，而是来搅局了，还火上浇油。
<br/>
<br/>        “那这样吧！我和星儿先谈一段时间的恋爱，谈好了，觉得差不多了，我们就跟爷爷提订婚的时间。”明司寒接着提议道。
<br/>
<br/>        连星儿一脸狐疑地瞥着明司寒。
<br/>
<br/>        明护国倒是不糊涂：“呵呵，万一你们谈着谈着，各奔东西，两家这婚事岂不是黄了。”
<br/>
<br/>        “爷爷放心好了，一年的时间怎么样？明年星儿十九岁生日那天，就是我和她的订婚礼。”明司寒信誓旦旦地说。
<br/>
<br/>        连星儿听他这话，更加头疼不已。
<br/>
<br/>        但是，明司寒的提议，总比现在急着订婚要好。
<br/>
<br/>        明护国随之看向连星儿，温和地询问：“星儿，你可否有意见？”
<br/>
<br/>        连星儿见状，连忙摇头。
<br/>
<br/>        她还能有什么意见？
<br/>
<br/>        连星儿接着又补充了一句：“爷爷您还可以找我爸妈商量。”
<br/>
<br/>        “好，那就先这么定了。具体的情况，等我和傅家商讨了之后再告诉你们。”明护国顿时笑不拢嘴地说。
<br/>
<br/>        明司寒会心一笑，三五两下就把碗里的早餐吃完后，起身抓住连星儿的手腕，跟爷爷明护国告了辞，拉着连星儿转身便走。
<br/>
<br/>        回头见这丫头仍旧一脸不乐意，他又停下脚步，伸手捏住这丫头的脸颊，拉动着她的嘴角微微向上扬起。
<br/>
<br/>        “我说你这丫头开心点，解除婚约这事急不来，我们越是反抗，双方父母越是会阻扰。还不如随机应变。”明司寒咧嘴一笑。
<br/>
<br/>        连星儿抬起眼皮子，眼神里满是狐疑地看着明司寒：“你又想诓我？”
<br/>
<br/>        “我若是诓你，刚刚爷爷说要马上举行订婚礼，我就不会找借口推迟一年了。”明司寒云淡风轻地说。
<br/>
<br/>        连星儿觉得他这话说得有道理，于是抬起手来，再次指着他的鼻子，凶巴巴地说道：“这一年里，你赶紧想办法把婚约解除了啊！”
<br/>
<br/>        “你放心好了！解除婚约的这种事情，就包在我身上。你就不要去想这么费脑筋的事情了。毕竟，我看你很碍眼来着对吧？”明司寒会心一笑。
<br/>
<br/>        连星儿又想了想，觉得明司寒这话有理。
<br/>
<br/>        “那你去收拾一下行李，我们一起返校。”明司寒接着说道。
<br/>
<br/>        连星儿下意识地点了点头。
<br/>
<br/>        北城回临海城路途还很遥远，临时订的机票没订到，明司寒只能改订高铁票。
<br/>
<br/>        因为是两个人一起订票，所以明司寒拿到连星儿的身份证后，只看了一眼，便将她的身份证号码给记了下来。
<br/>
<br/>        跟明护国告别后，出门前，明护国还不忘对明司寒谆谆教诲：“臭小子，你要照顾好我孙媳妇。你要是敢欺负她，我就打断你的狗腿。”
<br/>
<br/>        “我知道了，爷爷。”明司寒一脸憋屈。
<br/>
<br/>        连星儿看着这对爷孙俩，捂着嘴忍俊不禁。
<br/>
<br/>        返校的途中，明司寒不仅给连星儿充当“搬运工”，还充当着“扫码支付钱包”。
<br/>
<br/>        连星儿只要负责吃喝玩乐即可，不仅如此，她还买了北城的特产，说是要带回学校去跟室友们一起分享。
<br/>
<br/>        进入高铁站后，在候车期间，连星儿想吃冰激凌，明司寒便去买了两只香草味的冰激凌回来，和她坐在等候区里边吃边聊天。
<br/>
<br/>        两人俨然就像一对正在热恋且颜值超高的小情侣。
<br/>
<br/>        明司寒看着吃得满嘴都是冰激凌的连星儿，忍不住轻笑了一声，然后拿起纸巾，一边贴心的替她擦拭，一边笑着说道：“瞧你这贪吃的模样，多像只小花猫。”
<br/>
<br/>        “你才小花猫了！”
<br/>
<br/>        “好好好，我是小公猫，你是小母猫。”明司寒嗤笑了一声。
<br/>
<br/>        连星儿看了明司寒一眼，用手指沾了点冰激凌，往明司寒的鼻头上一抹，笑嘻嘻地说道：“这才像只小公猫的样子。”
