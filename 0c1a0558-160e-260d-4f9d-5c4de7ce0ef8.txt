　　楚涵以一己之力在尸潮战中上演了贪吃蛇战术，在众人的惊艳连连中当场突破八阶，一件件壮举让战场上的无数双眼睛都聚集在他身上。
<br/>　　末世爆发以来，这是宝岛上头一个出现的怪胎级别人物！
<br/>　　人们都在相互询问，这是谁，谁认识他？又是从哪冒出来的？原先在哪个基地？
<br/>　　一个个问题环绕在众人心中，同时T市基地的危机也已经解除，因为丧尸集体围聚在楚涵那一块地方，基地大门都开了！
<br/>　　城墙和城门口陆续走出来大·片的基地居民，呆滞的望着满目苍夷的战场，谁能想到在过去的十几个小时之内，这里差点被尸潮淹没。
<br/>　　而在末世纪元中最威胁普通人生活的危险，就这么一点点的当着全基地居民的面解决。
<br/>　　新人类，真的好强！
<br/>　　尤其是眼下还在与丧尸群作斗争的那一位！
<br/>　　前来支援的新人类队伍，快速的将各种物资送进基地，生怕晚了就来不及。
<br/>　　毕竟那头大丧尸还没死，尸潮的统领没干掉之前，都不是真正意义上的胜利。只是在搬运过程中，无论是支援者还是基地居民，全部都时不时的朝楚涵看去。
<br/>　　贪吃蛇的战术还在继续，那头大丧尸已经杀死好几千个小弟了！
<br/>　　这是尸潮中最后一批丧尸，大部分都死于大丧尸之手，眼看偌大的战场上丧尸逐渐减少到寥寥无几，这场盛宴令无数人动容。
<br/>　　只是少数心里有数的人，则是随着战局的越来越靠后，心中恐慌越来越盛。
<br/>　　他们很清楚贪吃蛇战术的初始原因，就是这头十阶大丧尸过于强大，没人能战胜！
<br/>　　他们在等待支援的最强队伍到来，却没想到半路杀出来一个楚涵，直接用战术把最后一批丧尸搞死，眼看所剩无几了该怎么办？
<br/>　　大丧尸，谁来抵挡！
<br/>　　就在众人焦急万分的时候，忽的一辆大型皮卡冲入战区，开车的人似乎根本不懂什么叫刹车，一路横冲直撞甚至还撞翻了好几波运送物资。
<br/>　　这一幕引得很多人破口大骂，现在是战局的最关键时刻，竟然还有人想搞事情？
<br/>　　陈正奇第一个不能忍，直接横栏在皮卡面前，怒喊道：“什么人？没看到我们尸潮战还没结束吗？你当这里是什么地方，开车横冲直撞想做什么！”
<br/>　　皮卡稳稳停下，车上一连跳下来五个人，各个精干异常面带傲色。
<br/>　　尤其领头的那位，嘴里还叼着一根雪茄，看到陈正奇的发问直接冷笑：“尸潮战？我正好想问问，不是有人买我们五人出战一次吗？我们好不容易跑了这么长时间过来，哪来的什么尸潮？耍我是不是！”
<br/>　　这话一出，旁边的战役指挥双眼一亮，连忙上前道：“您一定就是万博吧？您可终于来了！尸潮战虽然可以说是结束，但那头十阶大丧尸还未解决！”
<br/>　　万博吐出一口烟圈，瞥了眼远处还在带着大丧尸跑圈的楚涵，不经意露出了一抹嘲讽：“还真是，但尸潮战这么快结束也太扫兴，而且你们怎么没提路途堵塞的事？害我们在路上浪费这么多时间！”
<br/>　　“堵塞？不可能啊！”周军词不解：“这又不是文明时代，哪来的人流量造成堵塞？”
<br/>　　万博冷哼：“我什么时候说人流了？我说的是丧尸，漫山遍野的丧尸档在前面，让我们怎么过来？等了好几个小时才通，你们这给的什么情报？”
<br/>　　一番话说的不少人傻眼，尸潮战尸潮战，当然会有丧尸了，当然很多了！
<br/>　　这人不会是故意的吧？
<br/>　　战役指挥半响都接不上话，张了张嘴一脸的无语。
<br/>　　万博缺不打算放过他，继续道：“我们五个人的身价你也知道，耽误这么长时间得损失多少单生意？可不是只有你们基地有危险，这世上的大丧尸多了！”
<br/>　　战役指挥皱眉：“那您的意思是？”
<br/>　　万博嘴角一勾：“给我双倍价钱，我就帮你把这头大丧尸解决掉，否则，我现在就带人走！”
<br/>　　此话一出，不少人都咬牙切齿，坐地起价也太过分了，而且理由如此荒诞！
<br/>　　战役指挥咬着牙，道：“可是我们基地现在并没有这么多的资金。”
<br/>　　万博无所谓的一挑眉：“那我可管不着，要么给钱，要么等死，你自己选一个。”
<br/>　　战役指挥身体颤抖，整个T市基地的安危，他不能不顾！
<br/>　　好不容易挺过尸潮，又飞来横祸……
<br/>　　在场众人都愤恨不已，却又毫无办法，除了眼前这五人，的确没人能够帮他们解决最大的问题。
<br/>　　难道真的要将基地都掏空，喂这个贪婪无厌的家伙？
<br/>　　就在战役指挥两难的时候，远处听到这里谈话的楚涵喊了一声：“让他走吧，这头丧尸我来解决。”
<br/>　　哪怕给得起钱楚涵也不肯把这头丧尸让出去啊！
<br/>　　积分啊！积分！
<br/>　　但他的这一番话，一下子让很多人都皱眉，现在想到的就是自不量力。
<br/>　　万博更是哈哈大笑：“我来了也有几分钟了，就看到你小子在转圈，难道你还指望把这丧尸绕晕还是绕累？动点脑子想想，这可是十阶的大丧尸！体能与平衡比你好！而且，你带着其绕圈的小聪明来拖延时间，不就是为了等我来吗？”
<br/>　　没有人相信楚涵可以战胜大丧尸，已经十阶的万博更加如此，于是话语之间狂傲无度，跟着他一起来其余四人更是毫不掩饰的嘲笑。
<br/>　　眼下的楚涵，一蹦一跳的简直像个跳梁小丑。
<br/>　　就这样还敢大言不惭的说能解决十阶丧尸？
<br/>　　笑话！
<br/>　　但就在一声声嘲笑刚响起的时候，那头的楚涵却忽然不跑了。
<br/>　　只见他双脚从一头八阶的丧尸脑袋上跳下，稳稳的背对它落于地面，八阶丧尸一路被楚涵当跳板跳了一路，这时候早就忍无可忍，发现楚涵竟然还敢背对它，当场发狂大吼，锋利的双爪狠狠朝前刺去。
<br/>　　极近，极快！
<br/>　　远处不少人都内心一颤，这根本躲不过去！
<br/>　　但就在这时候，楚涵忽然动了，原本修罗战斧的斧刃尖端轻轻点在地面，但却在那利爪过来的瞬间，在半空中划出一道肉·眼可见的轨迹，完美的像是艺术。
<br/>　　很慢，很清晰！
