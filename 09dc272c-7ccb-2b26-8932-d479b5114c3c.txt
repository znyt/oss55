　　黑岩城，城主府，后院，
<br/>　　“丫头，这就是破灵丹，你服下这颗丹药，便可以重塑经脉，不过这个过程会很痛苦，你确定要吃吗？”
<br/>　　叶尘将一颗赤色丹药，递到灵溪的跟前，一脸郑重道。
<br/>　　灵溪没有任何犹豫，重重点了点头，
<br/>　　“叶尘哥哥，我确定！”
<br/>　　说着，拿起破灵丹，直接丢入自己的嘴巴，然后一口吞了下去。
<br/>　　轰！
<br/>　　灵丹进入灵溪的腹中，顿时化作一股狂暴的能量，在她的身体之中肆虐开来。
<br/>　　“啊！！”
<br/>　　巨大的冲击力，顿时让小丫头痛苦的哀嚎了一声，险些晕厥过去。
<br/>　　叶尘见状，慌忙将手掌按在灵溪的后背之上，将自己的真元渡入对方体内，同时低喝一声，提醒道：
<br/>　　“撑住！试着将这股能量运转周天，我会护住你的心脉，你不会有事的！”
<br/>　　“嗯！”
<br/>　　灵溪已经无法开口说话，微微点了点头，当即开始按照叶尘所说，将体内那股狂暴的能量，按照真气运行的路线，开始缓缓运转起来。
<br/>　　滋滋滋~
<br/>　　灵溪顿时感觉自己的全身经脉，犹如被火烧的一般，甚至皮肤上，已经开始渗出血来！
<br/>　　“哼！”
<br/>　　强大的痛苦，让小丫头不由得发出一道闷哼，不过她并没有因此而放弃，依旧在坚持着。
<br/>　　一个小时之后，
<br/>　　灵溪终于将体内的那股能量，运转了一个周天。
<br/>　　轰！
<br/>　　全身经脉霍然贯通，
<br/>　　“啊！”
<br/>　　小丫头惨叫了一声，因为承受不住巨大的痛苦，直接晕厥了过去，倒在了叶尘怀里。
<br/>　　叶尘望着怀里，这个满头汗水的小萝莉，脸上也不由得露出一抹赞许之色。
<br/>　　破灵丹虽然可以为资质差的人重塑经脉，可是这种痛苦，一般人很少能一次坚持下来，大都需要服用好几次破灵丹，方才能看到效果。
<br/>　　可是这灵溪小丫头，小小年纪，居然一次就成功了，这其中虽然也有他的功劳，可是这份毅力，也算是十分难得了。
<br/>　　“叶尘哥哥……我、我还能坚持……”
<br/>　　小丫头人虽然晕倒了，可是还在低声喃喃着，犹如梦呓一般。
<br/>　　叶尘随手一翻，从乾坤镯中取出一枚养神丹，塞入灵溪的口中，然后趴在她耳边轻声道：
<br/>　　“丫头，你做的很好！吃了这颗丹药，好好睡一觉，等明天醒来，一切就都好了！”
<br/>　　“嗯！”
<br/>　　小丫头迷迷糊糊应了一声，这才昏昏沉睡。
<br/>　　然而，
<br/>　　就在叶尘要将灵溪抱入房中歇息的时候，一道惊怒的女子声音陡然响起，
<br/>　　“你对我妹妹做了什么！”
<br/>　　却是大美女灵韵，方才听到妹妹的惨叫声，便匆忙赶了过来，恰好看到方才那一幕。
<br/>　　站在灵韵的角度，叶尘将自己的妹妹抱在怀里，还给她服了一颗不知名的丹药，然后又趴在她的身上，这场景实在过于暧昧，她想当然的便以为，叶尘是要对灵溪图谋不轨！
<br/>　　倘若换成其他人，灵韵肯定想都不想，直接把这个意图侵犯自己妹妹的恶徒杀了，可是这人偏偏是叶尘，她心中虽然怒气冲天，却也不得不强忍着。
<br/>　　“叶、叶先生，您、您这是在做什么？”
<br/>　　灵韵冲到跟前，强压着心头的怒火，缓缓开口道。
<br/>　　叶尘哪里知道这女人此刻的想法，他现在的注意力全都在灵溪的身上，不由得眉头一皱，
<br/>　　“灵溪才刚刚睡下，你在这大呼小叫什么！”
<br/>　　说着，叶尘很自然的将灵溪抱起，直接便朝房间里走去。
<br/>　　灵韵顿时美眸一瞪，简直肺都要给气炸了，心中早已经把叶尘骂了一个狗血喷头，
<br/>　　‘枉我还以为你是一个正人君子，没想竟然如此无耻！禽兽！’
<br/>　　可是这些话，灵韵也只能在心中骂一骂，万万是不敢说出口的，万一激怒了这个煞星，灭了整个黑岩城，恐怕都不是什么难事！
<br/>　　可是眼看着，叶尘抱着自己的妹妹，已经走进房间，直奔床上而去，灵韵无论如何也坐不住了，慌忙也跟了过去。
<br/>　　“噗通！”
<br/>　　叶尘才刚刚将小丫头放到床上，正准备将她外衫脱掉，后面跟进了的灵韵见状，当即跪倒在了一旁。
<br/>　　叶尘顿时一愣，一时间也没有反应过来怎么一回事，不由得眉头紧皱，
<br/>　　“灵韵姑娘，你这又是做什么？”
<br/>　　灵韵先是沉默了片刻，贝齿咬着朱唇，似乎犹豫了片刻，最终缓缓抬起头，美眸直视着叶尘，轻声道：
<br/>　　“叶先生，我妹妹，她还是个孩子啊！求求你、放过她吧！如果您不嫌弃的话，我愿意……愿意替我妹妹来服侍您……”
<br/>　　灵韵飞快的说完这些话，然后又将身上的衣衫随手一拉。
<br/>　　哗啦！
<br/>　　修真者所穿的衣服，大多都是由特殊材料打造的法衣，并没有世俗界那么多的分类，全都是整体一套。
<br/>　　灵韵将法衣收起，身上的衣服，便顿时全部消失不见，将她那堪称完美的酮体，彻底展露了出来~~
<br/>　　这灵韵，不愧是名动整个巨门星域的大美人，无论是脸蛋，还是身材、皮肤、气质，都堪称极品！
<br/>　　尤其是此时，果体跪俯在地上，一副任君采劼的模样，更是有着一股难以形容的诱惑。
<br/>　　便是叶尘如今的心境，也不由得心头微微一热，不过更多的却是惊愕。
<br/>　　“这女人，搞什么？”
<br/>　　叶尘足足愣了好一会，终于想明白过来了，不由得一阵好笑，
<br/>　　“灵韵姑娘，你是不是误会什么了？我刚刚是在给你妹妹疗伤，可不是要对她做什么龌龊的事情！”
<br/>　　说完这话，叶尘便不再理会这个异想天开的女人，将灵溪放倒在床上，用被褥盖好之后，便站起身来，直接转身朝门外走去。
<br/>　　“这……”
<br/>　　看到这一幕，这下顿时轮到灵韵傻眼了，
<br/>　　“原来他真的只是为灵溪疗伤？并非是要……”
<br/>　　搞清楚了情况之后，灵韵顿时羞的不行，想起之前自己举动，真恨不得赶紧找个地缝钻进去才好。
<br/>　　迅速将法衣穿回身上，灵韵再回头望去，发现叶尘早已经消失不见。
<br/>　　不知为何，心中竟然有一种莫名的失落……
