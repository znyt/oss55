　　慕清霄并没有直接离开，几分钟后，落在沙漠中的一座黄土高坡上，静静的等待，为了蜕变，美杜莎肯定会跟过来。
<br/>　　果不其然，几秒后，慕清霄十多米外的空间就一阵扭曲，随后一道曼妙雍容的身姿就走了出来。
<br/>　　望着毅力在黄土高坡上的神骏黑凤，美杜莎声音清冷，道：“你愿意将青莲地心火借给本王吗？”
<br/>　　慕清霄凤首摇了摇，旋即浑身燃起九霄琉璃焱，见到诡异的火焰，美杜莎莫以为他要攻击，急忙爆退，美眸中尽是冷漠与愤怒。
<br/>　　然而，就在她爆退到百米开外的时候，黑色火焰消失，黑凤也消失不见，取而代之的是一名拥有完美面孔的神骏青年。
<br/>　　青年负手而立，一袭白衣翩翩，温润如玉，最显眼的是他的额头上，有一枚黑色纹路，看上去极其神秘。
<br/>　　看到青年的瞬间，美杜莎神情一愣，美眸中闪过一丝错愕，暗道：“世间怎会有如此神骏的男子。”
<br/>　　慕清霄见美杜莎已经爆退百米，无奈的摇摇头，难道自己就如此可怕吗？
<br/>　　“怎么，你不是想借用异火吗？”
<br/>　　“你当真愿意将青莲地心火借给我？”见慕清霄终于肯开口说话，愣了一秒，美杜莎也急忙回复。
<br/>　　微微摇头，慕清霄道：“青莲地心火是不可能给你的。”
<br/>　　“......你是在戏耍本王吗？”美杜莎瞳孔中尽是冷意，气息也逐渐攀升，一直都高高在上的她，何时被如此戏耍过。
<br/>　　“你误会我的意思了，我不借你青莲地心火，却有更好的火焰，帮助你蜕变。”
<br/>　　说罢，慕清霄伸出手掌，掌心窜起一缕黑色的火焰，接着说道：“你可以将它当作异火的一种，虽然没有温度散出，但威能却远不是区区异火能比，最重要的是，火焰任凭我控制。”
<br/>　　听闻慕清霄的话语，美杜莎冷笑一声，寒意四射，道：“你当本王是白痴吗？你们凤族的兽火虽强，但怎可能堪比异火！”
<br/>　　“......我何时说过自己是凤族？”慕清霄觉得有些好笑，难怪这女人刚才没有动手，原来是忌惮凤族。
<br/>　　“你不是凤族？可你的真身......”
<br/>　　慕清霄眼眸中的不屑之色毫不掩饰，道：“你说那些血脉稀薄的杂毛鸟？呵，给我提鞋的资格都没有。”
<br/>　　这下轮到美杜莎懵逼了，好狂妄的语气，居然将凤族说成是杂毛鸟，还不配给他提鞋！
<br/>　　刚才他说凤族血脉稀薄，刚才的黑凤真身，确实与自己当初见过的凤族有些不同，颜色也极其的妖异。
<br/>　　见她犹豫不决，慕清霄也不理会她，转身就要离开此处，自己夺走了她进化的机会，本想弥补她一下，既然她自己不信，那他就没办法了。
<br/>　　见他头也不回的离开，美杜莎气的胸口起伏不定，何时有人敢这样无礼的对待她。
<br/>　　“等等，本王相信你！”
<br/>　　美眸闪烁了几下，美杜莎最终选择了相信慕清霄，一来后者拥有一张人畜无害的面孔，而来他的话有一定的说服力。
<br/>　　至于其他原因，就只有她自己知道了。
<br/>　　慕清霄脚步一顿，转身望着美杜莎，眼中闪过一丝赞赏，任何事情都伴随着风险，选择相信他，自然也有一定的风险。
<br/>　　但美杜莎却毅然决然的选择相信，这女人心性倒是不错，否则，错失现在的机会，她想要进化就不知道要等到猴年马月了。
<br/>　　看了一眼她妖艳的容颜，说不动心的都是伪君子，慕清霄自然也喜欢，但正事要紧，道：“走吧，回到刚才的地方，我要借助岩浆中的狂暴能量突破，等我突破后在帮你进化。”
<br/>　　“可以，你与别的男人不一样，本王选择相信你。”
<br/>　　虽然慕清霄不征求她的意见，让她心中有些不悦，但比起接下来的事情，美杜莎选择忍耐，同时心中暗道：“等本王进化成功，你这小子还不是任本王宰割。”
<br/>　　几分钟后，回到刚才的地洞中，慕清霄将洞口封印，美杜莎则站在通道前，紧紧盯着眼中的岩浆。
<br/>　　盘坐在滚烫的岩浆中，慕清霄一阵舒爽，宛如泡在温泉中一样，双手结印，开始炼化整个青莲。
<br/>　　随着心法的运转，慕清霄神色也变得极其严肃，眼眸中也反射出青色的妖异火焰。
<br/>　　青莲地心火本源火种，在起初形成时，它的体积应该有着半座山峰大小，但经过大自然的千年磨练，体积不断压缩，当它体积压缩的只有巴掌大小的时候，方能够形成一点火灵。
<br/>　　而此时的它，才能真正的称之为异火。
<br/>　　将千年吸收的恐怖能量，压缩到手掌大小，其中蕴含的恐怖能量，可想而知。
<br/>　　即便是斗宗面对这种恐怖的力量，都只有一种后果，那就是陨落，好在慕清霄的本命神火直接压制住青莲地心火。
<br/>　　否则，慕清霄是凤凰，有在多的命，也不可能生生吞下如此恐怖的能量，更别说炼化。
<br/>　　已经炼化过骨灵冷火，对于炼化步骤，慕清霄都相当熟悉，若非是他，旁人直接吞下异火，估计会疼的死去活来，而他只是隐隐感受到一丝刺痛罢了。
<br/>　　看了一眼站在洞口的美杜莎，慕清霄渐渐闭上双眼，开始炼化步骤。
<br/>　　此时，四肢百骸中，青莲地心火已经莲花莲子已经化作青色火焰与能量，这些青色火焰与能量极其恐怖，在他的体内乱窜起来。
<br/>　　好在慕清霄体质经过多番改造，早已没有堵塞的经脉，如此一来，也减轻了不少痛苦，否则经脉哪里经得起冲撞。
<br/>　　随着青色火焰与能量在经脉中穿梭，身为凤凰的他，依旧是感受到一丝刺痛。
<br/>　　随着心法运转，体内的能量也顺着路线流淌起来，不断的洗刷经脉，强化筋骨，最后通过经脉，全部汇入心脏中。
<br/>　　心脏，相当于九霄琉璃焱的容器。
<br/>　　半个小时过去，慕清霄浑身一阵，气息也开始攀升，化神二重巅峰，化神三重初期，化神三重巅峰......化神四重初期！
<br/>　　将莲花莲子，青莲地心火全部炼化，慕清霄的实力不断攀升，直到化神四重初期方才停止。
