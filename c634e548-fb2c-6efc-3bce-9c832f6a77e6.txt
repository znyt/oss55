        “你……”仲正浩欲言又止。
<br/>
<br/>        明司寒忽然咧嘴一笑：“如果你是真心地看上贺思思了，你可以去追求她。”
<br/>
<br/>        “我……”仲正浩顿时语塞。
<br/>
<br/>        明司寒放下双手，抓住椅子板两边，连人带椅子一起挪回了书桌旁，打开笔记本电脑继续刚刚的游戏。
<br/>
<br/>        张子晋和邢乐游见状，也纷纷重新进入了游戏状态。
<br/>
<br/>        因为刚刚的挂机，他们这一局正处于下风。
<br/>
<br/>        但是，明司寒的回归，加上张子晋和邢乐游的配合，三个人带着游戏里的另外两个队友，成功逆袭。
<br/>
<br/>        明司寒赢了后，直接退出了游戏。
<br/>
<br/>        张子晋和邢乐游则打算继续玩。
<br/>
<br/>        明司寒起身，走到仲正浩的面前，抬起手来拍了拍他的肩膀，接着说道：“我和贺思思虽然自幼相识，但是我并不了解贺思思。所以，我帮不了你什么。”
<br/>
<br/>        仲正浩怔愣了一下，刚刚还紧握成拳的双手，此刻已经微微松开了。
<br/>
<br/>        “我出去一趟，你们继续玩吧！”明司寒随后跟张子晋和邢乐游打了声招呼，便只身离开了寝室。
<br/>
<br/>        另一边，傅家大庄园里，连星儿坐在水榭的石桌前，正在看祖爷爷了傅炎廷给哥哥冷晏琛挑选的“未来媳妇”的资料。
<br/>
<br/>        “你看看，你有没有喜欢的未来嫂嫂。”傅炎廷笑呵呵地问。
<br/>
<br/>        连星儿一份份资料看下来，发现，祖爷爷给哥哥冷晏琛挑选的“未来媳妇”，一个个都是精英。
<br/>
<br/>        其中，不是律师就是医生，还有警察、教师和女兵，以及海归爱国的白领和在蓝天里“翱翔”的女机长。
<br/>
<br/>        重要的一点是，还都长得很漂亮。
<br/>
<br/>        难得祖爷爷没挑选什么豪门贵族里的千金大小姐。
<br/>
<br/>        “祖爷爷，我挑了没用啊！得我哥哥喜欢才是。”连星儿一本正经地说道。
<br/>
<br/>        本来这一大清早的，她还想睡懒觉来着。
<br/>
<br/>        结果被祖爷爷拉到这儿，看了一个多小时的个人信息资料。
<br/>
<br/>        “你就先挑一个，然后你祖爷爷我好先去打声招呼。再安排你哥哥和她相亲。”傅炎廷摸着胡子，笑呵呵地说道。
<br/>
<br/>        连星儿很是头疼地问道：“祖爷爷，这是要给哥哥逼婚吗？”
<br/>
<br/>        “当然！你祖爷爷我这一生，快走到尽头了。想在进棺材之前，抱上小玄孙。”傅炎廷语重心长地说道。
<br/>
<br/>        所以说，每个老人在儿孙满堂这方面，其实都很贪心的。
<br/>
<br/>        抱上了孙子就会想着抱曾孙，抱上了曾孙就会想着抱玄孙。
<br/>
<br/>        其实，说白了，老人也希望自己能一直长寿。
<br/>
<br/>        连星儿看了一眼傅炎廷，不再反驳了，反而顺着傅炎廷的心意，从这数十份资料里随手抽了一个出来。
<br/>
<br/>        在给傅炎廷看之前，连星儿自个儿先看了一眼，是一个在蓝天里“翱翔”的女机长，而且还只比他哥哥小两岁。
<br/>
<br/>        “就这个吧！”连星儿满意地点了点头，双手呈给了傅炎廷。
<br/>
<br/>        虽然是她随手抽的，但运气似乎也不差。
<br/>
<br/>        傅炎廷看到连星儿选的资料，只是会心一笑地说道：“不愧是我家小乖孙最讨我欢心。”
<br/>
<br/>        “祖爷爷之前看中的也是她吗？”连星儿好奇地问道。
<br/>
<br/>        傅炎廷微笑着点了点头：“众多女孩子里，我觉得就她不错。”
<br/>
<br/>        “可是，祖爷爷，您急着抱小玄孙。但是，这位女机长还这么年轻，应该不会这么早就替哥哥怀孕生宝宝吧？”连星儿若有所思地说道。
<br/>
<br/>        资料里的这位女机长虽然只是副驾驶座的见习副机长，但是升职为正机长的机会是绝对的百分百。
<br/>
<br/>        傅炎廷意味深长地回答道：“这种事情，你这个小丫头就不要过问了。”
<br/>
<br/>        “噢！”连星儿顿时尴尬地笑了笑。
<br/>
<br/>        傅炎廷忽然想到了什么，上上下下将连星儿打量了一番后，问道：“如果你有空，替你哥哥先去观察观察一下你这个未来的嫂嫂。”
<br/>
<br/>        连星儿愣了一下，立马咧嘴一笑，做了一个OK的手势。
<br/>
<br/>        不就是观察未来的嫂嫂嘛！
<br/>
<br/>        完全没问题。
<br/>
<br/>        陪完祖爷爷后，连星儿又得去陪奶奶。
<br/>
<br/>        哄两位老人开心了一上午，吃过午饭后，连星儿打算返校，刚出傅家大庄园的大门，只见明司寒双手抱臂，帅气地倚着车头。
<br/>
<br/>        见到这丫头从门内出来，他嘴角微扬，放下双手，踱步向她迎了上去。
<br/>
<br/>        “你怎么来呢？”连星儿走下台阶，来到明司寒的跟前停住了脚步，她虽然很好奇，但是满脸的笑容如沐春风。
<br/>
<br/>        明司寒微笑着转身，一边拉开副驾驶座的车门，一边说道：“自然是对你有了心灵感应，知道你要返校了。就特意来接你。”
