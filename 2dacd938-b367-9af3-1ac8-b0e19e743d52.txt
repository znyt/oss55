<br/>    黑色的夜幕之下，大雨滂沱，一群身穿黑衣的人在大雨之中疾行，脚步落在地面上，溅起一地的泥水。
<br/>    
<br/>        无疆坐在一块青色的大石之上，背后的披风之上，雨水顺着披风汇聚成水帘，他的眉毛之上，雨水不住的滴下。
<br/>    
<br/>        雨水从他的眼中流过，如同流过一尊雕塑。
<br/>    
<br/>        无疆是一个看起来多愁善感的中年人，发丝微卷，平添几分沧桑，然而实际上，他从来不多愁伤感。
<br/>    
<br/>        从他的背后，一个身穿黑衣的人来到此处，朝着无疆拱手道：“大宗主，魔刀堂十二个王级军团，刀已经磨好了！”
<br/>    
<br/>        无疆并没有回头，也没有回答，还是看着歇歇雨幕。
<br/>    
<br/>        黑衣人问道：“大宗主，你在看什么？”
<br/>    
<br/>        无疆似乎是回过神来，道：“我在看一柄刀。”
<br/>    
<br/>        黑衣人道：“能够让大宗主如此认真去看的刀，想必一定是一把好刀。”
<br/>    
<br/>        无疆点头，“确实是一把好刀，在这把刀还没有开锋之前，我就确定这是一把好刀。”
<br/>    
<br/>        黑衣人问道：“敢问这柄好刀的名字！”
<br/>    
<br/>        无疆回味了一番，然后道：“应该是叫做李素衣吧。”
<br/>    
<br/>        无疆继续道：“不说这些了，既然你们已经准备好了，那就准备出发吧，这一次，你带着我的第七柄刀去，师弟，你还记得吗？曾经我们拜在魔刀堂麾下，师父想要用你的血去血祭这柄刀，我无意之中得知了这个消息，然后，我挥刀杀了我们的师父，背上了弑师的罪名，一时之间，魔道追杀，仙道不容，可我，终究还是活下来了，而且还成为了魔宗的大宗主，师弟，你可曾怪我杀了师父，还杀了师父的全家，包括你的最爱，师父的女儿。”
<br/>    
<br/>        黑衣人摇头，然后道：“师兄，我从未怪过你，只恨当年你被整个天下追杀的时候，我不能帮到你一丝一毫。”
<br/>    
<br/>        无疆笑了笑道：“如果不是那一次被整个天下追杀，我可能也没有与整个天下为敌的心思，我在整个天下的面前逃过一次，但是我不会逃第二次，这一次，我要整个天下都在我的刀下颤抖，这柄刀的名字，我不喜欢，以后，就叫它斩草吧，毕竟，它会因为这一次斩尽仙道所有有望大君的天才而天下闻名。”
<br/>    
<br/>        无疆解下背后的一柄狭长的刀，朝着背后丢去，黑衣人伸手接住斩草，然后拱手道：“师兄，这一次，我绝对会永远站在你的身后，帮你屠尽这天下仙道的天才，斩草除根！”
<br/>    
<br/>        无疆没有回头，只是淡淡道：“去吧，师弟，早点回来。”
<br/>    
<br/>        黑衣人转身，如同消失在雨幕之中，片刻之后，黑衣人带着斩草来到一眼望去黑压压的黑衣人前，他举刀，大喝道：“奉大宗主之命，斩尽仙道所有有望大君的王级！”
<br/>    
<br/>        黑衣人们齐齐抬头，那是一双双血红色的眸子，一时之间，群魔乱舞，“杀！杀！杀！”
<br/>    
<br/>        黑衣人手中的斩草出鞘，朝着身前一座传送阵挥刀，“出发！”
<br/>    
<br/>        无数的黑衣人朝着传送阵涌去，然后消失在传送阵之中。
<br/>    
<br/>        连日以来，陈明已经吞了不知道多少阴阳和合丹，他的身体就像是一个无底洞，因为道宫有着将近百座的缘故，这些阴阳和合丹就像是归入大海之中，瞬间消于无形，增强了陈明的心念。
<br/>    
<br/>        而陈明现在，施展玄关剑舞已经不是问题，只是还没有达到巅峰，玄关剑舞，最高可以控制万柄飞剑，而他现在，可以同时施展两千柄飞剑。
<br/>    
<br/>        陈明扫了一眼自己的危险指数，1750，距离大君已经不远了，但是还是有一段差距。
<br/>    
<br/>        这一段时间，自己的危险指数可谓是暴涨，而陈明，无比的希望这个暴涨的速度能够继续下去。
<br/>    
<br/>        还需要两个多月，他才能达到巅峰。
<br/>    
<br/>        这一天，陈明正在自己炼丹，忽然外面有着吵闹声，陈明将这一炉丹药炼制完毕，看向在旁边等待他炼制完毕的紫霞仙子，陈明问道：“怎么了，外面发生了什么事？”
<br/>    
<br/>        紫霞仙子急切道：“在外面，来了一条龙鲲，随着龙鲲而来的，还有着众多的凶兽！”
<br/>    
<br/>        陈明道：“怎么会这样，这里怎么会来这么多的龙鲲？”
<br/>    
<br/>        紫霞仙子道：“这里有着一具龙鲲的尸骨，可能是龙鲲的沉眠之地，我们打搅了龙鲲的沉眠，龙鲲震怒，所以带着如同沧海般多的凶兽来到了这里。”
<br/>    
<br/>        陈明问道：“现在情况怎么样了？”
<br/>    
<br/>        紫霞仙子道：“刚刚在外面请求炼制丹药的人王不少，他们正在为你离开这里争取时间，我是来保护你离开这里的！”
<br/>    
<br/>        “叮，你触发了剧情任务，龙鲲的亲切问候，请给龙鲲的亲切问候给予一个完美的答复。任务奖励：未知。”
<br/>    
<br/>        系统小姐姐这个时候就不要开玩笑了吧，还龙鲲的亲切问候！
<br/>    
<br/>        亲切你妹啊！
<br/>    
<br/>        陈明倒是没有直接说离开，而是道：“走，出去看看。”
<br/>    
<br/>        陈明来到头骨之外，现在外面已经乱成了一团，有不少人看向陈明，“那位就是炼丹宗师吗？”
<br/>    
<br/>        “看紫霞仙子的尊敬态度，应该就是了！”
<br/>    
<br/>        “宗师，快走，我等还能抵挡一二，只要宗师安全撤走，我等就放心了！”
<br/>    
<br/>        “不错，宗师为我等不辞昼夜的炼制阴阳和合丹，宗师一定不能出事，不然，我等良心不安！”
<br/>    
<br/>        陈明此刻看向场外，那铺天盖地的凶兽，如同潮水一般涌来，放眼望去，只怕不下数十万，而在凶兽之中，一只巨大的身影在其中游动，想必这就是那龙鲲了！
<br/>    
<br/>        众多的人王此刻正在尽出手段，在凶兽潮之中和这些凶兽搏杀，鲜血在水中蔓延，将周围彻底染成一片血色，红色的血水之中，只看得到隐约闪烁的夜明珠光芒，不知道是生是死。
<br/>    
<br/>        紫霞仙子看向陈明，问道：“怎么了，看到这些被你骗了的人王为了你的安全拼死搏杀，心中产生了愧疚？”
<br/>    
<br/>        陈明笑道：“怎么可能？欺骗的人之中，不是还有你吗？不过这一次，倒是刷新了我对正道的感觉，倒也不全是一群伪君子，还是有一部分人心中有着热血的。”
