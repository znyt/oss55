　　燕碧的身躯，从这擂台之中，被直接打飞了出去，跌落到了外面。
<br/>　　按照这擂台战的规则，这边说代表着，如今的燕碧，已经是败了，只要是离开擂台，便算是失败。
<br/>　　“好强！”
<br/>　　“天级灵器，居然都被这一道剑光震破！”
<br/>　　“北域百绝榜前十的人物，居然又败了一个，怎么可能？”
<br/>　　下方无数武者，看着燕碧就这样被打飞出去，一个个也都是猛然站了起来。
<br/>　　他们谁也想不到，这一届的武斗大会上，居然会出现如此之多的意外。
<br/>　　楚风眠，寒月漓，葬天，他们三个人，都是踩着百绝榜前十的人物，进入下一轮的。
<br/>　　像是这样突然冒出的天才，往届之中出现一个，都是极为罕见，没想到这一口气，居然是出现了三个人。
<br/>　　这燕碧，本来也是这一届几乎板上钉钉可以进入前十的人物的，可如今碰到寒月漓，便是就被如此轻而易举的给击败了。
<br/>　　就算是手持天级灵器，都被寒月漓击败。
<br/>　　“燕师姐！”
<br/>　　北邙圣域的无数种子弟子，一下子围了过去，将燕碧搀扶了起来。
<br/>　　燕碧在北邙学院之中，也是赫赫有名的巨头了，没想到如今这样，便是败了。
<br/>　　他的苍生鼎，都被打破，如今威力失去五成，更是损失了这次进入归墟天的资格，一下子付出了极大的代价。
<br/>　　“可恶！走！”
<br/>　　燕碧被搀扶起来，眼神中充满着怨恨，冷眼的瞥了一眼寒月漓，便是直接跟无数北邙圣域的弟子，直接离开了。
<br/>　　“你的冰莲剑术，已经是快要达到大成了吧。”
<br/>　　楚风眠看着旁边从擂台中已经落下的寒月漓，笑着开口道。
<br/>　　听着楚风眠突然开口，寒月漓的脸色都是微微露出几分惊讶。
<br/>　　她的剑术即将大成，这是她这几日一来最新的领悟。
<br/>　　如今刚刚展现出来，居然都被楚风眠直接看穿？这也让她似乎有些惊讶。
<br/>　　看着寒月漓的惊讶，楚风眠也没有解释什么。
<br/>　　对着冰莲剑术，楚风眠倒是真的熟悉的不能在熟悉了，寒月漓的剑术有所进展，楚风眠便是第一时间都已经是感觉得到。
<br/>　　“以你的实力，除非是碰到葬天，夜天君，冷面魔君，风无心四人，剩下的任何对手，应该都足以击败了。”
<br/>　　楚风眠扫视了众人一眼，开口道。
<br/>　　“这些人中，你最需要小心的人，便是葬天，其他人，你都有着一战之力，但是碰到葬天的话，可以选择弃权。”
<br/>　　说道这里的时候，楚风眠的脸色都是露出了几分凝重。
<br/>　　在场所有人中，不管是夜天君，还是冷面魔君，甚至是北域百绝榜的第一人风无心，给楚风眠的感觉，都没有那葬天给他感觉中的威胁要大。
<br/>　　楚风眠感觉得出。
<br/>　　寒月漓碰到这三人，未必没有一战之力。
<br/>　　但是一旦碰到葬天，可就只有一个下场，便是必败无疑。
<br/>　　“嗯。”
<br/>　　寒月漓点了点头，却是脸色丝毫不变。
<br/>　　她也同样是如今这一届的黑马之一，楚风眠的提醒，也最多让她小心一点，但是让她放弃，却是绝无可能。
<br/>　　楚风眠自然也深知这一点，他话中的意思，只是提醒。
<br/>　　葬天的实力虽然强大，但是寒月漓碰到，自保，也应该没有问题。
<br/>　　“轰！”
<br/>　　擂台之上，又是一阵灵力肆意爆发，一尊武者，被直接碾碎，令这一场擂台战告终。
<br/>　　这一轮擂台战，也终于结束。
<br/>　　除了这两战之外，这一轮，倒是没有太多的震惊。
<br/>　　下一轮，便是到了决定这一届武斗大会前十名次的时刻了。
<br/>　　只要是这一场在胜，便是就进入了这一届武斗大会的前十，可以得到归墟天的资格了。
<br/>　　能够来到这一轮的武者，已经是没有任何一位是弱者，其中最弱的一人，也要是北域百绝榜上十五位以上的人物。
<br/>　　楚风眠一眼望过去，夜天君，风无心，冷面魔君等人，一样全部晋级到了这一轮之中。
<br/>　　武斗大会，已经是开始进入到了白热化的地步，越来越激烈，无数强者都要碰撞了。
<br/>　　短暂的三个时辰的沉寂休息，在三个时辰后，终于这擂台，便是再一次的开启。
<br/>　　新的一轮战斗，几乎开始。
<br/>　　这开始的瞬间，楚风眠手中的身份牌，便是亮起。
<br/>　　这争夺前十的第一战，便是楚风眠率先出场了。
<br/>　　楚风眠一跃来到擂台之上。
<br/>　　而楚风眠这一轮的对手，不是别人，正是北域百绝榜的第一人，榜首，风无心。
<br/>　　“居然是他？”
<br/>　　楚风眠看着面前的对手，脸色也是微微变化一些。
<br/>　　这争夺前十的第一战，居然便是碰到了这北域百绝榜的榜首第一人，风无心。
<br/>　　“楚风眠，你能够走到这一步，倒是的确没有让我失望，我早就期待着跟你一战，如今倒是得偿所愿了。”
<br/>　　风无心看着楚风眠，突然开口道。
<br/>　　“我清楚，你的龙族武技，其实只是掩饰，你的真正身份，应该是一尊剑修，正好，我也是一尊剑修，不知道你敢不敢来跟我打一个赌？”
<br/>　　“打赌？”
<br/>　　楚风眠反问道。
<br/>　　“赌什么？”
<br/>　　他的剑修身份，可以瞒过大部分人，但是却瞒不过真正的强者。
<br/>　　尤其是楚风眠在北邙圣域中，已经是不止一次展露过他的剑术，这风无心知晓，倒是也没什么。
<br/>　　不过令楚风眠有些好奇的是，这风无心，居然是要跟他打赌。
<br/>　　“就堵你手中的，祖龙至尊剑。”
<br/>　　风无心看着楚风眠，自信满满的开口道。
<br/>　　“你我比拼剑术，我若是赢了，你手中的祖龙至尊剑，便是归我所有！”
<br/>　　“可以，但是如果你输了呢？”
<br/>　　听到风无心的话，楚风眠脸色倒是没有任何变化。
<br/>　　赌斗，剑术？
<br/>　　赌斗别的，也许楚风眠还会考虑一二。
<br/>　　但是这赌斗剑术，楚风眠可从来不会怕。
<br/>　　他不需要任何考虑，便是就接了下来，但是说是要打赌，这风无心，也必须要有着赌注才行。
<br/>　　“我？”
<br/>　　风无心听到楚风眠的话，微微一愣，似乎是没有想到这个问题。
<br/>　　他在北域百绝榜上，虽然名列榜首，但是风家，却并非是一个大家族，他也没有太多的财富。
<br/>　　想要赌楚风眠手中的祖龙至尊剑，他也必须要拿出相应价值的赌注来才行。
