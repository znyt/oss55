<br/>    太虚。
<br/>    
<br/>        无数朵虚空花忽然开放，从虚空花之中，传出太虚帝主的声音，“杀了渡劫者。”
<br/>    
<br/>        整个太虚，都开始行动，他们要杀了渡劫者，渡劫者的实力，已经超越了常理，陈明用九太子的死告诉世人，他有着灭杀天骄的实力。
<br/>    
<br/>        而他，同样想要窃取天道的权柄。

<br/>        月宫。
<br/>    
<br/>        一道道月华落下，露出其中的一个个地仙，“我等，必定救回月宫太子，灭杀渡劫者。”
<br/>    
<br/>        太阴金桂之上传出一个声音，“去吧。”
<br/>    
<br/>        无数地仙飞空，朝着神山赶去。
<br/>    
<br/>        无冕者议会至高议会。
<br/>    
<br/>        一位位无眠者出现在空旷的大殿之中，在正上首的位置上，并没有出现至高无冕。
<br/>    
<br/>        众多无眠者对视一眼，零号笑道：“看来，至高无冕并没有兴趣参与。这件事，该如何去做，由我们决定。”
<br/>    
<br/>        有一位无眠者道：“那我们，是不是应该做我们喜欢做的事情？”
<br/>    
<br/>        还有无眠者道：“比方说，拿下几个星天。”
<br/>    
<br/>        零号道：“即便是这一次，渡劫者没有死，那么在这至高议会上，我们总会有机会，见到他的。”
<br/>    
<br/>        众多无眠者齐齐大笑，“入口之处，已经被燕山把守了，入口进去太多地仙并没有用，太拥挤会导致根本无法战斗，无法展开拳脚，四大仙国，短时间之内，根本拿不下来。”
<br/>    
<br/>        “武夫已经出关了，加上四位首席，足以灭杀这个渡劫者了。”
<br/>    
<br/>        零号笑道：“就算他们不够，我还留了一手，这一手，从天道院而来。”
<br/>    
<br/>        众多无眠者听到零号的话，都是大喜，“天道院，看来，这个渡劫者必死无疑了，诸位，四大仙国防守空虚，我们，该做事了。”
<br/>    
<br/>        身为无眠者，自然是知道天道院的存在的。
<br/>    
<br/>        “这是一场盛宴，一场充满着鲜血和死亡的盛宴。这个世界，该重新洗牌了。”
<br/>    
<br/>        零号当即举杯道：“这一杯，敬长生！”
<br/>    
<br/>        其他的无眠者也是继续举杯道：“这一杯，敬死
<br/>    
<br/>        亡！”
<br/>    
<br/>        众多无眠者齐齐起身，齐声道：“这最后一杯，敬自己！”
<br/>    
<br/>        无冕者议会，从不按照规则办事！
<br/>    
<br/>        另外一场战争，来临了！
<br/>    
<br/>        而这一场战争，更为血腥，残暴。
<br/>    
<br/>        你们争你们的天道正统，我们争我们的天下！
<br/>    
<br/>        三头龙星宿鬼门星议事大殿。
<br/>    
<br/>        四位执刀人此刻都已经汇聚一堂，步留痕道：“所有的仙路，都没有开启过，所以，三头龙星宿之外没有人知道燕山的来历。”
<br/>    
<br/>        吕清玄道：“清扫一切叛乱，这个时候，不能出任何差错，一旦让三头龙星宿关于燕山的消息暴露出去，后果，不堪设想。”
<br/>    
<br/>        段首笑道：“直接动手不就完了。”
<br/>    
<br/>        杨珲哈哈大笑道：“动手，彻底封锁三头龙星宿，不可进，也不可出，一切想要给四大仙国透露消息的，都得死！”
<br/>    
<br/>        鬼门星之上，各路大军齐齐出动，整个三头龙星宿的修行者，都能够感觉到目前有着一场镇压。
<br/>    
<br/>        无面者手下的执刀人，已经倾巢出动，要镇压一切。
<br/>    
<br/>        要么臣服，要么死。
<br/>    
<br/>        黑压压的无面者大军开始戴上面具，冲向一颗颗叛乱的星辰，血洗所有的叛乱。
<br/>    
<br/>        神山入口处。
<br/>    
<br/>        月宫太子看向身前那高大的城池，这并不是他们的手笔，月宫太子似乎是明白了什么。“我轻敌了，对方从一开始，就将这一次无字劫，当做了一场战争。”
<br/>    
<br/>        月宫太子朝着上面扫了一眼，“可是，只是这些仙兽而已，真的能够挡住我吗？”
<br/>    
<br/>        月宫太子飞身而起，待得飞到城楼之上，看见了一个穿着白衣的家伙，正在朝着他微笑，“燕山，二宗主凌仙，恭候多时！”
<br/>    
<br/>        月宫太子的双眼露出一抹精芒，“你是那个修行了道经的修行者。”
<br/>    
<br/>        凌仙道：“猜对了。”
<br/>    
<br/>        月宫太子道：“这并不难猜，你的身上，有着我厌恶的气息，那是禁忌功法的气息。”
<br/>    
<br/>        月宫太子道：“何必要拦着我，四
<br/>    
<br/>        大仙国的愤怒，你承受不起。”
<br/>    
<br/>        凌仙笑道：“渡劫的那人是燕山山主，而我，是燕山的二宗主，你说，我们是什么关系？”
<br/>    
<br/>        月宫太子了然，“你们是师徒。”
<br/>    
<br/>        “看来，唯有一战了。”
<br/>    
<br/>        师徒，在修行者是最为神圣的关系，唯一有一丢丢机会能够和这种关系一争的，大概是同门。
<br/>    
<br/>        但是同门，一般都会很尊敬爱戴师父啊。
<br/>    
<br/>        修行界，没有什么能够比师徒更为紧密。
<br/>    
<br/>        一声师父，一生师父。
<br/>    
<br/>        城墙之上，月宫太子看了一眼，有四大仙国的地仙从神山的入口处冲进来，但是，他们毕竟是一个一个的冲进来，而对方，有着十万仙兽镇守。
<br/>    
<br/>        进来的最多几百人，只会被这些仙兽瞬间淹没，若是不攻破这座城，短时间之内，根本无法增兵！
<br/>    
<br/>        仙兽在城池之上，听从着燕山领主的命令，口中吐出一道道烈火寒冰，将冲进来的仙人打得粉碎。
<br/>    
<br/>        凌仙看向身前的月宫太子，笑道：“你是月宫太子对吧，听外面的人说，你们是这世间最为强大的天骄？”
<br/>    
<br/>        月宫太子摇摇头，“这不是一个说法，这是事实。”
<br/>    
<br/>        凌仙摆摆手，然后道：“我不信，师父说过，我虽然不怎么样，天赋奇差无比，但是打你这种，还是勉强能够打赢的。”
<br/>    
<br/>        月宫太子怒道：“大言不惭，你以为，你修行了禁忌功法，便可以横行无忌吗？”
<br/>    
<br/>        凌仙认真地点头道：“对，我就是这么想的，如果不是这种功法和术法太过强大，你们何必要杀我？”
<br/>    
<br/>        月宫太子眉间露出一抹寒霜，整个肉身如同是冰晶琉璃般璀璨，“场面话都说的差不多了，现在，该看看到底是谁说的有道理了。”
<br/>    
<br/>        凌仙起身，方圆数百里的灵气暴动，朝着凌仙汇聚而来，他的境界突破了，能够掌控的范围自然更大，若不是神山之上，在一颗普通的星辰之上，他可以瞬间抽干一颗星辰的力量！
<br/>    
<br/>        此地方圆数百里的灵气汇聚而来，其他地方的灵气自然是补充了过来，整个神山之上的灵气，都是暴动！
