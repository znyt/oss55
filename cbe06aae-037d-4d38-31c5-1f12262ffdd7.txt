　　萧逸的手上，蓦然出现了一个手套，顿时火焰环绕，威势惊人。
<br/>　　“灵器。”黑白无常，天残地缺四人一惊。
<br/>　　萧逸脸色变得冰冷和认真，“四位都是半步洞玄武者吧，易某自然要认真对待。”
<br/>　　萧逸曾经感受过半步洞玄的气息强度，自然看出了这四人的修为。
<br/>　　“果真是灵器。”天残地缺嗜血的目光中露出了贪婪之色。
<br/>　　“我就说嘛，区区一个少年，就算天赋再高，也不可能有那等实力，横扫各大山贼团伙，原来靠的是灵器。”
<br/>　　“小小一个先天境，可不配拥有灵器，还是交予我等吧。”
<br/>　　黑白无常冷声笑了起来。
<br/>　　“有本事，尽管来取。”萧逸怡然不惧。
<br/>　　“哼，小子，还敢嘴硬，待会儿怎么死的都不知道。”黑白无常冷哼一声。
<br/>　　话语刚落，萧逸脚下忽然出现了一股诡异的气息。
<br/>　　“鬼藤？”萧逸眉头一皱，立刻反应过来，猛地跳起来。
<br/>　　在他脚下，一大窜的藤蔓诡异出现，朝他双脚捆去。
<br/>　　藤蔓黑色，带有尖刺，正是鬼藤。
<br/>　　同一时间，一道阴森的冷箭破空而来。
<br/>　　萧逸刚跳跃起来，躲过鬼藤，人在半空中，根本来不及躲避，只能快速凝聚出一道火刃打出。
<br/>　　火刃与冷箭对撞，轰的一声，火星四射，火刃消散，冷箭也被消去大部分威力，被强行被打偏了一些。
<br/>　　嘶，打偏了冷箭在他肩膀边擦过，擦出一条血痕。
<br/>　　伤口并不严重，但，冷箭上似乎带有莫名的力量，竟让他头脑一阵激荡，微微失神了一霎那。
<br/>　　“糟糕，是荡魂箭。”萧逸反应过来后暗道不妙。
<br/>　　果然，在他失神的一霎那，一条森然的铁链凭空而现，并将他捆绑了起来。
<br/>　　“寒冰铁链？”萧逸微微吃惊，并试图挣扎开铁链。
<br/>　　可是，铁链坚固无比，而且带有一股冰冷的力量，让他浑身冰冻，力量大减，无论如何也挣不开。
<br/>　　“小子，死吧。”地缺冷笑一声，独脚跳跃而起，另一只瘸掉的腿上，不知何时起凝聚出一把锋芒弯镰，直攻萧逸而去。
<br/>　　弯镰上黑风闪烁，阴寒无比，显然非普通东西。
<br/>　　“黑风血镰。”萧逸瞳孔一缩，看着地缺攻来的步伐，暗暗皱眉。
<br/>　　“小子，你被寒冰铁链捆绑住，动弹不得，躲避不了，我看你如何接我这一击，你的人头，我便收下了。”
<br/>　　地缺冷笑一声，脚上的黑风血镰直朝萧逸的脑袋划去。
<br/>　　萧逸怡然不惧，冷冷地看着攻来的地缺，暗道，“原来如此。”
<br/>　　如今的他，在外历练了接近三个月，见识强了许多，一眼便认出了天残地缺和黑白无常的武魂，也看出了他们的手段。
<br/>　　黑无常，半步洞玄修为，黄色巅峰武魂，鬼藤，可无声无息地在地面任何一处出现，并诡异地将人捆绑起来。
<br/>　　一旦被捆绑住，鬼藤上的尖刺便会自动攻击，将人体扎出无数血洞，让人凄惨地流血而死。
<br/>　　白无常，半步洞玄修为，黄色巅峰武魂，荡魂箭，箭无虚发，瞬间即至。
<br/>　　箭上有特殊功效，可自动根据武者的武魂气息而追击，可震荡武者的灵魂，让武者短暂失神。
<br/>　　天残，黄色巅峰武魂，寒冰铁链。
<br/>　　寒冰铁链，坚固异常，武者被捆住，将被铁链上的寒冷气息冻僵，实力大减。
<br/>　　地缺，黄色巅峰武魂，黑风血镰。
<br/>　　一旦被黑风血镰打中，哪怕只是刮出一丁点儿伤口，也会流血不止，极为麻烦。
<br/>　　天残地缺，黑白无常，本身就是两个组合。
<br/>　　天残困人，地缺杀人；黑无常困人，白无常杀人。
<br/>　　一困一杀，巧妙而默契十足。
<br/>　　以自己这般被捆绑住，无法躲避，一旦被黑风血镰打中，必死无疑。
<br/>　　这时，萧逸脑海中的想法一闪而过，手中烈焰手套火焰暴涨。
<br/>　　炽热的火焰抵消掉寒冰气息，让自己身体恢复了正常。
<br/>　　而后，体内真气澎湃而出，配合肉体力量，猛地一挣，轻松将那寒冰铁链挣破。
<br/>　　这时，地缺的瘸脚已经划来。
<br/>　　萧逸反应极快，脑袋一低，身体一歪，堪堪躲过了黑风血镰的劈杀。
<br/>　　而后脚步一跃，快速后退。
<br/>　　“嗯？竟然挣破了寒冰铁链，躲开了我的攻击。”地缺微微一惊。
<br/>　　以寒冰铁链的坚固，一般的先天九重绝对无法睁开。
<br/>　　可萧逸乃是修罗战体第一层圆满，肉体之强，媲美先天九重妖兽的肉身，配合真气，自然挣脱了。
<br/>　　黑白无常不屑地笑了一声，“小子，不是挺狂的吗？这才刚开始战斗，已经狼狈如斯，你还有何胜算。”
<br/>　　事实上，两个组合，极少一起出手，一般都是分开出任务。
<br/>　　但这次，联手而至，竟一个照面就让萧逸落入下风，更险些丧命。
<br/>　　天残地缺，黑白无常，四大杀手果真名不虚传。
<br/>　　若是换了其他武者，洞玄境之下，恐怕无人能逃脱他们的围杀。
<br/>　　萧逸淡淡一笑，反唇相讥道，“作为武者，你们有些本事，但作为杀手，你们的手段不过如此。”
<br/>　　自己刚才不过是被四人诡异的武魂打了个措手不及，如今已经看出他们的手段，要应付起来已经不难了。
<br/>　　起码，自己有足够的信心对付他们。
<br/>　　“还敢出言放肆，既然你自己找不痛快，那也不必与你多言了，我们一起上，早些杀了他回去交任务。”
<br/>　　“啧啧，这次还多得了一个灵器，奖励必然不少。”
<br/>　　黑白无常和天残地缺冷笑起来。
<br/>　　黑无常和天残联手朝萧逸包围而去，一人凝聚出鬼藤，一人放出寒冰铁链。
<br/>　　白无常站在原地，一手荡魂箭射出。
<br/>　　地缺也是站在原地，黑风血镰伺机而动。
<br/>　　黑无常和天残困人，白无常和地缺准备杀人，四人再次巧妙地配合在一起。
<br/>　　萧逸怡然不惧，身上忽然被一层火焰笼罩，身体诡异地消失在原地。
<br/>　　他已经使用了升龙。
<br/>　　鬼藤和寒冰铁链都扑了个空。
<br/>　　黑无常和天残顿时一惊，“不好，那小子的速度怎么忽然这么快了。”
<br/>　　现场，唯独只有荡魂箭在四处飞舞，找寻着目标，忽然快速飞出，而飞出的方向，竟是白无常的方向。
<br/>　　“荡魂箭追击的方向，就是那小子的方向，不好，白无常，他在你那里。”黑无常惊呼一声。
<br/>　　果然，嗖的一声，萧逸凭空出现在白无常身边。
<br/>　　萧逸一掌打出，冷声道，“用箭之人，擅远攻，不擅近战和防守，便拿你先下手了。”
<br/>　　“好快的速度。”白无常瞳孔一缩，完成跟不上萧逸的速度。
<br/>　　在他眼中，萧逸忽然成了一只快如疾风的火焰猎豹。
<br/>　　那带着炙热高温火焰的手掌，一掌轰向白无常的心脏。
<br/>　　“噗。”白无常猛地吐出一口鲜血，直接被击飞，胸膛上被烧得漆黑一片。
<br/>　　萧逸的动作很快，一连窜的攻击都在瞬间完成。
<br/>　　这时，一旁的地缺才反应过来，独脚横扫而来。
<br/>　　“滚。”萧逸冷喝一声。八十道噬火百刃同时打出，将地缺瞬间击退。
<br/>　　如今的他，单靠真气就能释放出三十道噬火百刃。
<br/>　　加上灵器的增幅，直接达到恐怖的八十道。
<br/>　　地缺被击退，黑无常和天残立刻反应过来，准备回身救助白无常，但，他们的速度可比不上萧逸。
<br/>　　萧逸身体再次消失在原地，而后再次出现在被击飞的白无常身边。
<br/>　　“噬火百刃。”
<br/>　　又是八十道恐怖火刃打出。
<br/>　　白无常本就被打伤，加之本身速度就不如萧逸，根本躲避不了。
<br/>　　轰...轰...轰.....
<br/>　　连绵不断的爆炸声响起。
<br/>　　火焰，彻底吞没了他；爆炸声中，出现一道道闷哼声，是他的一口口鲜血喷出。
<br/>　　当爆炸声消失，他的身体从半空中无力跌落，火焰仍旧在燃烧，可他，已经生机全无。
<br/>　　堂堂白无常，暗影楼有名的杀手之一，半步洞玄修为武者，短短几个照面，竟被萧逸毙于掌下。
