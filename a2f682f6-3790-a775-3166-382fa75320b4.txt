r/>　　<br/>　　这些事情发生后，陈默真的是非常的不舒服的，要是他的实力强一些，怎么会利用其他的名头，自然而然的拒绝就是了，要是威胁，反手就能灭了。<br/>　　<br/>　　所以对于提升自己实力的东西，现在还是比较渴望的。这次有这么好的机会，不用白不用。他不相信特管局这么大的一个部门，会没有龙血腾，虽然比较稀有，但是并不是没有不是。<br/>　　<br/>　　而且他也不是胡乱开药，韦俊才还真的需要疏通胸口位置的经脉，因为这里被压制的时间过久，经脉有些敝塞了！<br/>　　<br/>　　“龙血藤一定要活株，这样才能利用它的活性成份。”陈默又特意交代了一下。<br/>　　<br/>　　龙血藤，顾名思义就是一种藤本类植物，只要有一小节就能在乾坤珠内种活。所以只要能弄过来一株龙血藤，要就截留下一点活体种植，其他的配置丹药就成。<br/>　　<br/>　　“好的，我都记录下来了。”李普河说道。<br/>　　<br/>　　陈默点点头，有转头对常文斌说道：“将病人的上衣退去，我要针灸一下。他的伤必须要有针灸配合。”<br/>　　<br/>　　“好！”常文斌没有多余的话，直接按照陈默的指示做。<br/>　　<br/>　　然后陈默就将自己准备的针拿出来，这是他花高价买来的合金针。因为针对并且不同，针灸用针也是不同的，所以一大包针可是花了陈默几十万。<br/>　　<br/>　　这次，针对韦俊才的伤情，他准备的针是空心针，而且足有十几厘米长的空心针。主要是用来引导他胸口异种能量的。<br/>　　<br/>　　这种能量他还没有见过，也是想着研究一下，所以当场用针灸试试，看看能不能将异种能量引导出来。<br/>　　<br/>　　不过在施针前，由于对于异种能量的担心，还有自己使用真元，不希望别的人感受到什么不同，就先利用针灸，在他的脑袋上来了一下，让韦俊才昏迷过去。<br/>　　<br/>　　好在常文斌和李普河看到就是惊讶了一下，倒是没有其他反应。因为他们看到陈默用针灸就让韦俊才昏迷过去，倒是有些佩服他的针灸医术的。<br/>　　<br/>　　而且他们也不认为陈默会对韦俊才有什么危险的行为，因为两个人没有什么厉害关系，而且其中又有袁若珊在协调，所以两个人倒是挺配合，就这么看着陈默治病，并没有插手和插言的行为。<br/>　　<br/>　　在韦俊才准备好以后，陈默就快速的在他胸口施用了六根针灸，将异种能量所影响的经脉全部闭合住，然后就通过丝丝真元开始引导试探异种能量。<br/>　　<br/>　　等陈默的真元进入后，他才发现一种能力有种灼热的力量，可以将武者修炼出来的内力吞噬并壮大自身，虽然这种吞噬转换率比较低，但是却让武者丝毫没有办法。<br/>　　<br/>　　至于说普通的医疗手段，那就更不用说了，对这种能量根本没有任何作用。<br/>　　<br/>　　陈默的真元一接触，异种能量竟然有些像欣喜迎接一样，渐渐融入到了他的真元中，然后输入到韦俊才的真元逐渐壮大，等陈默收回的时候，发现自己的真元粗~壮了一些，处于练气三层中阶的修炼等级，竟然凭借这次吸收后，突破了练气三层中阶！<br/>　　<br/>　　好家伙，自己的真元还可以通话异种能量，倒是一个好消息。而且，这个异种能量是什么，倒是要好好了解一下。<br/>　　<br/>　　不过陈默不准备一次就将这些异种能量全部同化吸走，要是这样，岂不是自找麻烦么？<br/>　　<br/>　　反正这个能量会一直在，所以准备花上个几次针灸，将这些能量搬运走。而且既能治病，还能壮大自己的行为，当然要好好去做了。<br/>　　<br/>　　陈默很是开心的施完针，最后将韦俊才脑袋上的针一拔，他也就缓缓清醒过来。<br/>　　<br/>　　“感觉怎么样？”陈默看着韦俊才询问道。<br/>　　<br/>　　韦俊才听到陈默的话，就感觉了一下自己的胸口，却发现轻松了好多。<br/>　　<br/>　　“身体好像没有刚才那么沉重了，而且呼吸什么的都畅快了好多！”韦俊才这个时候，才真的佩服陈默，没有想到一次针灸，将自己的病情变轻了好多。<br/>　　<br/>　　“这就好，其实头次施针，只是缓解了一下。等后面必须要中药配合才行。”<br/>　　<br/>　　“好了，今天的治疗就这样了。你们尽快将中药配置好，尤其是龙血藤要尽快拿过来，我在配合施针，可能在行针三次，他的伤就能好了。”<br/>　　<br/>　　陈默对着常文斌和李普河说道，也是有催促的意思。有了龙血藤，不仅是韦俊才能够更快的回复，自己也能配置出凝气丹来。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
