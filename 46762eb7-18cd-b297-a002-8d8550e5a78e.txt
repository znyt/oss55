　　　　女娲贵为圣人，将帝俊与太一带走后，每个人心中都憋着一股气，如今女娲走后，人族只好将这股气发在剩下这些人身上。
<br/>　　剩下的妖族，有着几大武皇的帮助，瞬间便被人族屠杀一空，鲜血染红了大地，尸骸堆积如山，每个人心中都充满坚定、凶戾，接着，他们将杀意投向巫族。
<br/>　　巫族背弃人族，屠杀人族，取他们的魂魄炼制戮妖幡，这简直让人不能容忍，此刻的人族，早已不是当初的人族，当初那个还需要通过与巫族联盟来保护的人族。此时的人族，已经可以独当一面，不要再畏惧任何敌人。管他是妖族，还是巫族，只要对人族不利，人族都不会放过。
<br/>　　长虹贯破虚空，纵然是大罗金仙境的大巫，也根本没有丝毫的抵抗之力，身躯迸裂暴炸半空，化作漫天血雨，纷纷散落。
<br/>　　玄天目光一转，径直落在了以刑天为首的数万巫族大军之上，无尽的肃杀之意，铺天盖地一般瞬息笼罩而至。
<br/>　　“武祖玄天，这……这是个误会，我等听说妖族前来攻打人族，只是来看看人族是否需要帮助！”
<br/>　　曾经需要联盟的存在，如今已经成为仰望不可及的顶尖大神通者。刑天心中憋屈之余，却也不得不开口，毕竟，他的身后，还有着数万巫族儿郎，由不得他不服软。
<br/>　　可惜，巫妖二族此番对人族的屠杀，已经令玄天愤怒至极，闻得刑天此言，当即便是一声冷笑：
<br/>　　“误会？什么误会？我可没看到这其中有什么误会？巫族背弃盟友，该杀！”
<br/>　　一声该杀，顿时，虚空震动，空间变得动荡起来，玄天身后，三个巨大的黑洞缓缓出现，风云倒卷，日月垂落，煌煌魔威。
<br/>　　“轰隆隆！”
<br/>　　黑洞轻颤，仿佛能够感受到主人那心中的无限杀意，它释放出无与伦比的吞噬之力，垂落下来，对着下方的数万巫族儿郎吞噬而去。眨眼之间，便吞没了数万巫族大军。
<br/>　　“你——”
<br/>　　刑天不由得为之勃然大怒，大吼道：“你当真要与我们巫族开战吗？！”
<br/>　　闻言，玄天好似听到了天地间最好笑的笑话，他口中不由得为之一阵大笑：
<br/>　　“哈哈！刑天，你不会是在跟我开玩笑吧，难不成你忘了，是你们巫族先背弃了两族盟约，向我人族开启杀戮的。”
<br/>　　接着，他又冷笑道：“是你们教会了我人族，人族不该如此善良，如此天真，盟友又如何，朋友又如何？一样可以杀了，所以，你可以去死了！”
<br/>　　话音刚落，虚空震动，一道闪烁着黑白双色神光的转轮从玄天脑后飞出，带着浓浓的生死之意，径直穿过时间与空间的限制，转瞬，就飞到刑天的近前。
<br/>　　“尔敢！”
<br/>　　生死一瞬，突然听见一声暴虐大喝，赫见天地虚空为之一颤，一道暴躁火光，贯穿长空，呼啸而来。
<br/>　　“轰！”
<br/>　　火光与生死轮，瞬息相碰，掀起风云惊变，远处虚空一道高大的身影破空而来，正是火之祖巫，祝融！
<br/>　　“吾有何不敢？祝融，若是你们十二祖巫聚齐，摆出都天神煞大阵，吾还有几分忌惮，就你一人来，就不怕吾杀了你，彻底废了你们巫族的十二都天神煞大阵么？”
<br/>　　玄天冷然开口，随着他的话音落下，他蓦然出手，屈指间，就有数道通天剑气破空而出，化为斩仙之力，斩向祝融。
<br/>　　玄天身躯一震，目光之中，斗射而出的神光，无边威压，携带着动天之势，直冲九霄，震撼大地，瞬息之间，一股庞然伟力，自他身上倾泻而出，无与伦比的庞大力量，逆转阴阳之变，分化五行之功！
<br/>　　一声轻吟，风云惊变，赫见玄天脑后神光大涨，三道神轮浮现，岁月、命运、生死的气机显现，道则高深，玄之又玄。
<br/>　　“想杀我，也要看看你有没有那个本事！”
<br/>　　祝融轰然开口，乍见天地之间，一片赤红火光自虚空之中显现，眨眼之间，天地震荡，无穷无尽的赤红火海，汇聚在他手中，化作一柄红色神杖，庞然大势，震荡乾坤。
<br/>　　“哼，自大的东西，你活该找死！”
<br/>　　玄天单手化天，乳白色的岁月之轮在胸前划过，带着无尽沧桑之感，岁月长河流动，呼啸着，飞向祝融，势不可挡！
<br/>　　“烈焰，火神怒！”
<br/>　　祝融怡然不惧，抬手之间，转动手中红色神杖，召唤出无尽神火，天地虚空，风云一滞，无边神火凝聚成一道万丈之高的烈焰神将，操纵着无边神火，汇聚成一道惊天能量，对着玄天攻击而去。
<br/>　　然而，岁月本无息，带走无尽生命，他又如何能够抵挡？
<br/>　　岁月之轮穿过无边的火海，无声无息，划破天际，瞬息之间，便将无数烈焰化为灰烬，散落下来，烈焰神将直接被破开数半。岁月转轮没入祝融身体之内，带走他的无尽生命，使他变得一阵虚弱。
<br/>　　玄天不停，生死之轮一样划破天际，穿过生死，贯穿岁月，来到祝融身边。
<br/>　　祝融脸上闪过惊骇之色，一种不自信的表情出现在他脸上，他看到那轮生死之轮……
<br/>　　一股强烈的死亡的气息笼罩住了他，这绝对不是幻像，而是真正的死亡。从来没有什么时候比这一刻让他感觉到死亡距离自己是那么的近。
<br/>　　死亡的气息钻入他的身体之中，在疯狂的破坏他的身体组织，破坏他的五脏六腑，破坏他身体内的一切生机。而且最让他惊骇的是，那种死亡气息竟然还在疯狂增长，它就仿佛是活的一般，在吸收他体内的生机在增长。
<br/>　　他的生机在被疯狂吞噬，他的生机每被吞噬一息，那锁定他的死亡气息就会更加浓郁。
<br/>　　祝融更是在这死亡气息中找到一丝生机，可是，那一丝生机并没有在他的掌控之中，只有他的全身生机被吞噬完，那一丝生机才会出现。
<br/>　　“啊？这怎么可能？你怎会如此强大？”
<br/>　　祝融满脸不可思议，他在大声怒吼，想要挣脱这一切，可惜，他失败了。祝融挥动火神杖，卷起一片炙热的空间锁住玄天，空间之中，火光漫天，有数道赤金色的光芒撕裂虚空，轰向玄天。
<br/>　　“蠢货，吾之手段，可不是你能够探寻的。哼！”
<br/>　　玄天满脸不屑，挥手之间，拉出一片光幕，笼罩其身，光辉运转，如同一道瀑布，那几道光芒一道他身边，便被光幕挡住，反击出去。
<br/>　　紧接着，玄天一拳挥出，光芒四射，震动空间，那神杖锁定的空间，瞬间便被玄天轰碎。
<br/>　　玄天不停，想要趁机一举击杀祝融祖巫，他弹指之间，弹出数道通天剑气，剑光如虹，斩破虚空，如同要化仙而去一般，带着无尽锋锐之气，飞快斩向虚弱之中的祝融。
<br/>　　“尔敢！”
<br/>　　就在此时，又是一声大暴怒喝，轰碎虚空，自不远处传来。
